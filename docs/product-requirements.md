## 一、产品定位

| 项目 | 内容 |
|:-----|:------|
| **产品名称** | 金鉴真人·八字命理分析 |
| **目标用户** | 10岁~70岁，不限阶层，不限职业，所有对命运好奇的人 |
| **核心价值** | 确定性规则引擎驱动的严肃命理分析，不装神弄鬼，用数据说话 |

## 二、核心功能需求

### 2.1 用户输入

| 字段 | 必填 | 说明 |
|:-----|:----:|:-----|
| 姓名 | ✅ | 用户输入的真实姓名 |
| 性别 | ✅ | 男/女 |
| 出生日期 | ✅ | 年月日 |
| 日历类型 | ✅ | 阳历(公历) / 农历 |
| 出生时辰 | ✅ | 子丑寅卯辰巳午未申酉戌亥（下拉选择） |
| 出生地 | ❌ | 可选。未输入默认为北京时间/东八区 |

### 2.2 输出报告

#### 报告头部
- 姓名、出生年月日时、性别、出生地
- 八字四柱完整信息（年柱/月柱/日柱/时柱）

#### 报告正文（21§全部展示）
标准模板定义的各模块详细解读，每模块基于理论知识逐条分析。
**整份报告 ≥ 1500字**（靠分析颗粒度自然达到，不凑字）。

#### 报告操作
- PDF下载（一鍵导出为PDF文件）
- 打印支持

### 2.3 用户体验

- 美观、专业、可信的视觉设计
- 移动端优先（暗金配色）
- 流畅的交互体验
- 加载状态清晰反馈

### 2.4 用户系统（MVP简易版）

- 注册/登录（用户名+密码，或手机号）
- 报告历史记录（查看过往分析）
- 后续扩展：会员/付费/订阅

## 四、MVP交付标准

| 维度 | 标准 | 验收方式 |
|:-----|:-----|:---------|
| 输入功能 | 姓名/性别/阳历农历/时辰/出生地 全部可用 | 手动测试 |

| PDF下载 | 一键导出，排版清晰 | 手动测试 |

| 视觉体验 | 暗金配色·移动优先·专业可信 | 主观评估 |
