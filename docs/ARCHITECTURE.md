# 金鉴真人·八字命理分析系统 架构设计

## 部署架构
```
用户浏览器 → Nginx(43.162.92.9:80/443)
                ├── / → Next.js 前端 (:3000)
                └── /api/ → FastAPI 后端 (:8000)

服务端：43.162.92.9
数据库：PostgreSQL (:5432)
缓存：Redis (:6379) [将来]
```

## 技术栈
- **前端**: Next.js 14 (React) + TailwindCSS + TypeScript
- **后端**: Python FastAPI + SQLAlchemy + Pydantic
- **数据库**: PostgreSQL
- **PDF生成**: WeasyPrint / ReportLab
- **部署**: Nginx反代 + systemd

## 数据模型 (含将来扩展)
```
users          → 用户表 (会员+权限)
reports        → 报告表 (基础版/详细版)
orders         → 订单表 (将来支付)
memberships   → 会员表 (将来)
birth_info    → 生辰信息表
```
