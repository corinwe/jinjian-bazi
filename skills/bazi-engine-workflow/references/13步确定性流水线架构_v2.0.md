> **设计原则：** 所有规则写成代码，逐字检查、满足条件就加分、不满足就不加。LLM只做翻译，不做推理。

## 整体架构（三层）

```
┌──────────────────────────────────────────────────┐
│                   前端层                          │
│  用户输入（公历/农历 + 性别）→ 报告展示          │
│  纯渲染，不参与任何计算                          │
└───────────────────┬──────────────────────────────┘
                    ▼
┌──────────────────────────────────────────────────┐
│               规则引擎层（核心）                    │
│  13步流水线，全部代码写死，零LLM                  │
│  输入：天干地支×4 + 性别 + 出生年                │
│  输出：结构化JSON（约120个字段）                  │
│  路径：/root/bazi-platform/engine/                │
└───────────────────┬──────────────────────────────┘
                    ▼
┌──────────────────────────────────────────────────┐
│               数据层                              │
│  神煞表 / 纳音表 / 藏干表 / 调候用神表           │
│  案例校准库 / 用户档案库                         │
└──────────────────────────────────────────────────┘
```

**关键原则：** 规则引擎是上帝，它算出来的就是真理。前端只负责画，不负责想。
**LLM角色：** 只负责翻译结构化JSON成自然语言，不参与任何推理计算。

## 13步流水线（执行顺序严格·不可调换）

### 第1步：排盘（bazi-engine.py → constants.py BaZi对象）

| 项目 | 内容 |
|:----|:------|
| **输入** | 公历年月日时 + 性别 |
| **动作** | 年柱立春分界 → 月柱节气分界+五虎遁 → 日柱日干支算法 → 时柱五鼠遁 |
| **输出** | BaZi(year=Pillar('庚','申'), month=Pillar('癸','未'), day=Pillar('辛','亥'), hour=Pillar('辛','卯'), gender='男') |
| **工具** | `/root/weiwuji-knowledge-base/07-国学哲学/八字命格/scripts/bazi-engine.py`（ephem天文库） |
| **状态** | ✅ 已验证，立春前bug已定位但有官网验证兜底 |

### 第2步：十神判定（shi_shen.py）

| 项目 | 内容 |
|:----|:------|
| **输入** | 日主 + 四柱天干 + 四柱藏干 |
| **规则** | 生我者印（正印异/偏印同）→ 我生者食伤（食神异/伤官同）→ 克我者官杀（正官异/七杀同）→ 我克者财（正财异/偏财同）→ 同我者比劫（比肩同/劫财异） |
| **输出** | 年干十神 / 月干十神 / 日干 / 时干十神 + 每个藏干的十神 |
| **状态** | ✅ 122行，跑通 |

### 第3步：身强弱评分（shen_qiang_ruo.py）⭐ 最核心

| 项目 | 内容 |
|:----|:------|
| **输入** | 日主 + 月令 + 四柱天干 + 四柱藏干 |
| **评分规则** | **只有加分，不加不减！** |
| A. 月令本气印 | ✅ +40分（仅此！中/余气=0，其他任何位置印=0） |
| B. 月令比劫 | ✅ 全计分（本/中/余都算） |
| C. 天干比劫 | ✅ 全计分（年12/月12/日**不计**/时12） |
| D. 年/日/时支藏比劫 | ✅ 按比例（年支4×%/日支12×%/时支12×%） |
| E. 燥土规则 | 未/戌+天干丙/丁引化→当火看（不计分）；未/戌+壬/癸→当土看（计分） |
| F. 从弱 | 全无印比→恒定50分（反为强） |
| G. 从强 | 满盘印比→恒定100分时→恒定20分 |
| **评分基础分** | 年干8 / 年支4 / 月干12 / 月令40 / 日支12 / 时干12 / 时支12 |
| **藏干比例** | 本气100% / 中气60% / 余气30% |
| **输出** | {score: 73.6, label: "身强", details: {每项分数来源}} |
| **状态** | ✅ 296行，5人全部正确（家主73.6/主母7.2/子源12.0/父亲66.4/凤7.2） |

### 第4步：财星评分（cai_xing.py）

| 项目 | 内容 |
|:----|:------|
| **输入** | 日主 + 四柱藏干 |
| **铁律** | 只有**日主所克五行**才是财星！癸水日主→财=火（丙丁巳午），己土不是财！庚金日主→财=木（甲乙寅卯） |
| **计算** | 年支4×% + 月令40×% + 日支12×% + 时干12 + 时支12×% |
| **输出** | {total: 24.0, details: {nian_zhi: 0, yue_ling: 0, ...}} |
| **状态** | ✅ 126行，跑通 |

### 第5步：格局判定（ge_ju.py）

| 项目 | 内容 |
|:----|:------|
| **输入** | 月令本气十神 + 月干十神 + 全局十神 |
| **规则** | 月令本气定主格局 → 复合格局（官杀混杂/伤官生财/杀印相生/食神生财）→ 特殊格局（从弱/从强/化气） |
| **输出** | {main: "偏印格", extra: ["官杀混杂"], special: null} |
| **状态** | ✅ 218行，跑通 |

### 第6步：喜用神与调候（ge_ju.py后半段）

| 项目 | 内容 |
|:----|:------|
| **输入** | 身强弱等级 + 格局 + 出生月份 |
| **规则** | 身强→喜克泄耗（官杀>食伤>财）；身弱→喜生扶（印>比劫）；中和→补缺；调候（夏需水/冬需火） |
| **输出** | {xi: ["水","金"], ji: ["土","火"], tiao_hou: "水"} |
| **状态** | ✅ 已集成在ge_ju.py中 |

### 第7步：五行能量分析（energy.py）

| 项目 | 内容 |
|:----|:------|
| **输入** | 四柱天干+地支+藏干 |
| **计算** | 天干=1分，地支本气=1分，中气=0.6分，余气=0.3分 |
| **用途** | 健康诊断（超过3个=该五行有病）+ 行业选择（补最弱/用最强） |
| **输出** | {wu_xing: {木:1.0, 火:3.0, ...}, strongest: "土", weakest: "金"} |
| **状态** | ✅ 152行，跑通 |

### 第8步：地支关系-刑冲合害（xing_chong_he_hua.py）

| 项目 | 内容 |
|:----|:------|
| **输入** | 四柱地支 |
| **关系** | 六冲/三刑(寅巳申/丑未戌/子卯/自刑)/六害/六合/三合/半三合/拱合 |
| **能量倍数** | 天干引化+15倍/~7倍（无引化）；冲-70分；刑丑未戌-70分；害-60分；自刑10倍/5倍 |
| **用途** | 断能量基础——冲走喜用=坏事，冲走忌神=好事；三刑=藏干打架能量翻倍 |
| **状态** | ✅ 239行，跑通 |
