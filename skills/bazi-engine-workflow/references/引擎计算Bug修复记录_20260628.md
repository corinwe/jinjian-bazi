# 🚨 引擎计算Bug修复记录（2026-06-28）

## Bug 1：大运评分 — set去重导致同五行干支只计1次分

### 位置
`/root/bazi-platform/engine/da_yun.py` → `compute_da_yun_scores()`

### 问题代码
```python
dy_wx = {gan_wx, zhi_wx}  # ❌ 使用set去重！
score = 5.0
for wx in dy_wx:
    if wx in ji_shen:
        score -= 2.0  # 己丑=土+土 → {土} → 只减1次 → 虚高
```

set `{gan_wx, zhi_wx}` 去重后，若天干地支五行相同（如己丑=土+土），只计算一次加减分。

### 修复
```python
# 天干独立计分
if gan_wx in ji_shen:
    score -= 2.0 if gan_wx == ji_shen[0] else 1.5

# 地支独立计分  
if zhi_wx in ji_shen:
    score -= 2.0 if zhi_wx == ji_shen[0] else 1.5
```

### 影响
| 大运 | 修复前 | 修复后 | 正确 |
|:----|:-----:|:-----:|:----:|
| **丁亥**(纯喜用) | — | **8.5分** 🏆 | ✅ |
| **戊子**(一喜一忌) | **2分** ❌ | **5.0分** | ✅ |
| **己丑**(纯忌神) | — | **1.0分** ❌ | ✅ |

### 排查信号
用户报"大运评分不对" → 第一步检查 `dy_wx = {gan_wx, zhi_wx}` 是否用set

## Bug 2：财富评级 — 财旺阈值错误

### 位置
`/root/bazi-platform/engine/wealth_v2.py` → `_get_base_status()`

### 问题代码
```python
# ❌ 阈值30和20都是错的！正确标准应为40分
if shen_label == "身强" or shen_score >= 55:
    return "身强财旺" if cai_total >= 30 else "身强财弱"  # 30❌
else:
    return "身弱财旺" if cai_total >= 20 else "身弱财弱"  # 20❌
```

府君：身强64分 + 财星31.2分 → 31.2≥30 → 错标为"身强财旺"

### 修复
```python
# ✅ 阈值修正为40（财旺标准=≥40分）
return "身强财旺" if cai_total >= 40 else "身强财弱"
return "身弱财旺" if cai_total >= 40 else "身弱财弱"
```

### 影响
府君：身强64分+财31.2分 → 31.2<40 → 正确标为"身强财弱" ✅
财富评级对照表不再乱标 ✅/❌

### 排查信号
用户报"财富判断不对/红X绿勾乱" → 第一步检查阈值常数（30→40, 20→40）
