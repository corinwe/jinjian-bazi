# 九龙道长官网交叉验证流程

## 网站
https://www.zydx.top/paipan.php

## 🚨 核心方法：POST提交（无需登录）

### 输出内容
- **四柱八字** — 从tgline/dzline表格提取最后4个非空span
- **地支藏干** — small标签中的十神缩写（劫伤印/枭杀才/伤财/才）
- **大运序列** — data-year属性表示约几岁开始该运
- **称骨命** — 重量+评语

### 技术原理
```bash
curl -s -L "https://www.zydx.top/paipan.php" \
  --data "act=ok&name=家主&DateType=0&year=1980&month=8&date=6&hour=5&minute=30&sex=1"
```

POST参数说明：
- `act=ok` — 提交排盘请求
- `name` — 姓名
- `DateType=0` — 公历排盘（DateType=1=农历排盘）
- `year/month/date/hour/minute` — 出生时间
- `sex` — 性别（1男/0女）
- 无需cookie，无需登录

### 四柱提取关键
网站HTML结构：
```
<tr id="tgline">
  <td>天干</td>
  <td>流日(空)</td> <td>流月(空)</td> <td>流年</td> <td>大运</td>
  <td>年柱天干</td> <td>月柱天干</td> <td>日柱天干</td> <td>时柱天干</td>
</tr>
同名（地支行dzline同理）

提取方法：取所有非空<span class="big">的最后4个 = 四柱天干/地支
```
⚠️ **旧版脚本bug**：用固定索引[4]-[7]提取，但流日/流月为空时索引偏移。应使用`非空值最后4个`提取法。

### 起运年龄确认
官网返回的 `data-year` 属性即为大运起始年龄：
```
data-year='0'>甲申 — 甲申运从0岁开始（即约1岁起运）
data-year='10'>乙酉 — 乙酉运从10岁开始
data-year='30'>丁亥 — 丁亥运从30岁开始（对应2011~2020）
```

## 方法B：手动填表（备选）
1. 填写姓名
2. 选择「公历排盘」
3. 填写出生年月日时
4. 选择性别
5. 点击提交按钮

## 验证项
| 项目 | 验证方式 |
|:----|:---------|
| ✅ 四柱八字 | 年/月/日/时柱完全一致 |
| ✅ 地支藏干 | 年支/月支/日支/时支的十神一致 |
| ✅ 大运序列 | 大运天干地支顺序一致 |
| ✅ 起运年龄 | data-year值 = 大运起始年龄 |

## 注意事项
- 网站响应约1-3秒
- 返回值是静态HTML（非JS渲染），Python脚本可直接解析
- 如网站结构变更，需更新正则匹配规则
- 网站未做反爬限制，直接curl即可
