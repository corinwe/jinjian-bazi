## 常见问题排查

1. **页面空白** → JS语法错误导致整个script不执行
   - 检查是否有多余/缺少的 `}` `)` `;`
   - 检查`const`变量是否被重复声明
   - 检查模板字符串中是否引用了未定义的变量

2. **点击无反应** → `doAnalyze is not defined`
   - 脚本未加载 → 检查script标签闭合
   - `const API` 定义位置错误（放在函数外）

3. **API调用失败** → 网络错误
   - 检查 `API` 变量是否为 `'/api/v1'`
   - 检查服务器是否在运行
   - 检查CORS设置

4. **报告部分缺失** → `Cannot set properties of null (setting 'innerHTML')`
   - `${id}` 在HTML中不存在
   - 所有 `document.getElementById('r-xxx')` 对应的元素必须在HTML中有定义
   - 使用 `setHTML` 辅助函数（先检查元素存在再设innerHTML）
