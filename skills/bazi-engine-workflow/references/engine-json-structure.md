### 大运（🚨 致命错误高发区·0617教训）

```json
{
  "大运": {
    "起运": "9岁6个月9天",
    "起运年龄": 9.55,
    "序列": [
      {"大运": "甲申", "起始年龄": 9, "终止年龄": 19, "起始年份": 1990, "终止年份": 1999},
      {"大运": "乙酉", "起始年龄": 19, "终止年龄": 29, "起始年份": 2000, "终止年份": 2009}
    ],
    "当前大运": {"大运": "丁亥", "起始年龄": 39, "起始年份": 2019}
  }
}
```

🚨 **0617致命教训**：家主(1980-08-06卯时)引擎起运年龄=0.38，报告误写9岁。根因：子agent未从JSON提取起运年龄。

✅ **起运年龄提取黄金代码**：
```python
import json
with open('/tmp/bazi_batch/boss.json') as f:
    data = json.load(f)
qiyun_age = data['大运']['起运年龄']  # 0.38（唯一真值！）
qiyun_str = data['大运']['起运']      # "0岁4个月17天"
```

⚠️ **已知Bug**：大运序列中的 `起始年份`/`终止年份` 范围有重叠错误。
✅ **正确做法**：用 `起运年龄` 作为唯一真值，按连续10年手工修正。
⚠️ 起运年龄以浮点数格式存储（如0.38），取整到约1岁合理（用户确认过）。
