# 家族补运完整工作流（三步走）

> **编制时间：** 2026-07-01
> **来源：** 金鉴真人·家主/主母/子源/立 家族补运实战（多轮迭代校准）
> **核心教训：** 之前跳过前两步直接补库→方向经验证后发现能量不够→返工成本极高

## 核心场景

家族多人同时补运，涉及不同八字、不同喜忌、不同墓库方位。
**绝对不能直接跳到最后一步补库**——必须先调各人能量到40-60分区间。

## 三步走总纲

```
第0步：日主能量调整（先让人站稳）
  → 调日主能量到40-60分均衡区间
  → 场景：家主辛金64→50，主母庚金从弱不动
  
第1步：补财星+食伤到40分（造财先造工具）
  → 调财星能量+食伤能量到≥40分
  → 用能量牌/路神鹿/方位旺财/太极点催财
  → 场景：食伤44分→45分补到位，财星0分不够先戴43分能量牌
  
第2步：建财才墓库（钱造出来了得地方装）
  → 每人按自己日主五行找正确的财才墓库方位
  → 用保险柜/银行开户法建人工库
  → 前提：财星≥40分（无前提补库可以直接建，但开了也没钱装）
```

## 详细执行清单

### 第0步：日主能量检查

| 日主能量 | 状态 | 操作 |
|:--------:|:----:|:-----|
| < 40分 (身弱) | 不稳，刻不出财 | 补印比生扶到40分 |
| 40-60分 (均衡) | ✅ 最佳求财状态 | 不动 |
| > 60分 (身强过旺) | 比劫克财 | 加强克泄耗（红火+黑水+绿木各30%配比） |

**实战案例：家主辛金64分→50分方案**
```
辛金身强，忌印比，喜火(官)木(财)水(食伤)
64分 > 60分 需降

日主能量≈(印+比劫+微根)的结构：
  月令金(比劫) + 时支金(比劫) + ...

加强克泄耗的配比：
  红色火(30%) + 黑色水(30%) + 绿色木(30%) = 三种克泄耗各30%
  → 总量90%，剩下的10%留白
  → 不用纯红（火）或纯黑（水）或纯绿（木）单色——单色过猛易反噬
```

### 第1步：补财星+食伤到40分

**五大补能工具（可叠加使用）：**

| 工具 | 效果 | 价格 | 说明 |
|:----|:----:|:----:|:------|
| **能量牌** | +10~40分 | 20-30元/块 | 最精准，24h佩戴最有效。刻字可补特定五行 |
| **路神鹿** | +最多50分禄位 | 几十元 | 平和安全，不会补过头。只能用家中的禄位方向 |
| **方位旺财** | 短期催财 | 免费 | 择吉日往财方向走 |
| **太极点催财** | 月周期催财 | 免费 | 按当月月令克色穿衣 |
| **风水能量物品** | 变量 | 不等 | 风水轮/水晶等对应喜用五行的物品 |

**实操顺序：**
1. 先查八字原始财星/食伤能量分
2. 缺口分=40-当前分
3. 选能量牌补最大缺口（精确到颜色/五行）
4. 能用一块能量牌解决的问题，不要用3块

**实战案例：主母财星0→40分**
```
主母庚金从弱，财=木
日主能量0分（极弱从弱格）
财星能量0分（原局无财星透出）

方案：戴43分绿色木能量牌
  → 木补财+木耗金（从弱要耗身）
  → 一个人不能同时戴两个能量牌（能量干扰）
  → 食伤44分已经≥40分，不需额外补
```

### 第2步：建财才墓库

**补库（新建人工库）：**
- ✅ 无严格前提，可直接操作
- ✅ 补财才墓库：加强存钱蓄财能力
- ✅ 补比劫墓库：加强合伙/团队赚钱
- ⚠️ 补比劫墓库前提：日主≥40分（否则被合作方克制）
- ⚠️ 财星<40分时补库=开了也没钱装——建议先做第1步

**开库（冲开现有库门）：**
- ❌ 严格前提：该库全局能量≥40分
- ❌ 开财才墓库前提：财星≥40分（<40分叫开墓=破财）
- ✅ 条件满足后用§8.5生肖刑冲破害法

**方位核验口诀：**
```
辰东南、戌西北、丑东北、未西南
（每写一个方位前默念一遍，防方向写错）
```

## 快速检查清单

```yaml
□ 所有成员的日主能量已调至40-60分
□ 所有成员财星能量已≥40分
□ 家族成员的财才墓库方位正确（按日主五行→财才五行→墓库推导）
□ 补库和开库的区分清晰
□ 各成员之间互相不受冲克（全家合参）
□ 保险柜/开户的能量物品已配齐
```

## 核心警示

1. **补库不能跳过第0步和第1步** — 日主不稳或财星不够时补库没有意义
2. **财才墓库推导** — 日主→我克者→财才五行→该五行墓库，不能跳过中间层
3. **全家合参** — 一个方位的补能物品同时影响多人，不能只看一个人
4. **半年调整一次** — 能量牌/路神鹿都需要随大运流年调整
