## 一、合化能量倍数完整表（精确量化）

### 1.1 倍数总表

| 合化类型 | 合成+天干引化 | 合而不化 |
|:--------|:------------:|:--------:|
| **六合**（地支六合） | **10倍** | **5倍** |
| **三合**（申子辰等） | **15倍** | **7~8倍** |
| **半三合**（申子等） | **10倍** | **5倍** |
| **三会**（亥子丑等） | **20倍** | **10倍** |
| **三刑·丑未戌** | **15倍**（有土引化） | **7~8倍**（无土引化） |
| **三刑·寅巳申** | 各字能量加强（天干透出则加强对应字） | 各字能量均加强 |
| **自刑**（辰辰等） | **10倍**（天干有引化） | **5倍** |
| **岁运并临** | 能量叠加加强 | — |

### 1.2 空亡规则（能量修正）
- 空亡 = 能量减半
- **坏能量空亡 = 好事**（七杀空亡→灾减半；灾煞空亡→祸减半）
- **好能量空亡 = 坏事**（财星空亡→赚少一半；文昌空亡→学业减半）
- 计算：先算合化倍数，再÷2（原10倍→空亡后5倍；原5倍→2.5倍）

### 1.3 合化隔位规则
- **本命局**合化：需要连在一起（相邻或三合完整）
- **大运流年**合化：不需要连在一起！天上的星星照到任何位置
- 这是关键区分：流年天干＝该年值班星，可影响八字任何一柱

## 二、查一生灾祸——方法论框架

### 2.1 六条检测路线

| 路线 | 系统 | 检测对象 | 代表灾祸 | 解法 |
|:----|:----|:---------|:---------|:----|
| ① | 神煞 | **元辰** | 无妄之灾（别人做坏事牵连你） | 天乙/天德/月德（须同柱） |
| ② | 神煞 | **灾煞** | 血光横死 | 天乙/天德/月德（须同柱） |
| ③ | 神煞 | **天罗地网** | 牢狱之灾 | 能量不加强则无大灾 |
| ④ | 十神 | **印被冲坏/合化** | 保护神失效→疾病/凶灾 | 补印或加固印星 |
| ⑤ | 十神 | **七杀能量加强** | 疾病/凶灾/死亡 | 印来化解 |
| ⑥ | 十神 | **伤官/食伤能量加强** | 口舌官司/自伤 | 印来制或降能量 |

### 2.2 元辰（无妄之灾）完整查法

**分两种情况：**

**阳男阴女**（日元为阳的男性 / 日元为阴的女性）：

| 生年地支 | 见地支 | 为元辰 |
|:--------:|:------:|:------:|
| 子→未 | 丑→申 | 寅→酉 |
| 卯→戌 | 辰→亥 | 巳→子 |
| 午→丑 | 未→寅 | 申→卯 |
| 酉→辰 | 戌→巳 | 亥→午 |

**阴男阳女**（日元为阴的男性 / 日元为阳的女性）：

| 生年地支 | 见地支 | 为元辰 |
|:--------:|:------:|:------:|
| 子→巳 | 丑→午 | 寅→未 |
| 卯→申 | 辰→酉 | 巳→戌 |
| 午→亥 | 未→子 | 申→丑 |
| 酉→寅 | 戌→卯 | 亥→辰 |

**核心规则：**
- 12年遇到一次元辰年
- **不是每个元辰年都有灾！** 必须天干生地支（能量加强）才有灾
- 元辰分五种（不同天干）：火狗/木狗/水狗/土狗/金狗等
- 天干生地支（如丙戌→火生土）→ 能量加强 → 灾大
- 天干克地支（如甲戌→木克土）→ 能量被克 → 灾小
- 天乙/天德/月德在同柱 → 可解开

### 2.3 灾煞查法（速记口诀）
```
申子辰 → 灾煞在午（三合中神子→冲子为午）
寅午戌 → 灾煞在子（三合中神午→冲午为子）
巳酉丑 → 灾煞在卯（三合中神酉→冲酉为卯）
亥卯未 → 灾煞在酉（三合中神卯→冲卯为酉）
```

**注意：** 本命局无灾煞的人，第一次遇到时反应更强烈（「没见过这种能量→来了就是暴击」）。

### 2.4 天罗地网查法（年柱纳音法）

| 年柱纳音 | 天罗 | 地网 | 备注 |
|:--------:|:----:|:----:|:----:|
| **火命** | 见**戌亥**为天罗 | — | 仅火命有天罗 |
| **水/土命** | — | 见**辰巳**为地网 | 仅水/土命有地网 |
| **金/木命** | ❌ 无 | ❌ 无 | 金木纳音没有该神煞 |

### 2.5 十神系统检测

**印被冲坏的判断：**
- 一个印 → 容易化1个七杀
- 费点力也能化2个七杀
- **化不了3个七杀**（工作量太大）
- 印被流年冲走 → 七杀直克日元 → 灾

**七杀加强的判断：**
- 身强时七杀为用（身杀两停）
- 身弱时七杀为忌（需印来化）
- 七杀被合化出5倍以上 → 开始显现疾病
- 15倍+老年人 → 可能死亡

**伤官加强的判断：**
- 食神超过**5倍**→ 变伤官性质
- 伤官超过**5倍**→ 自己受伤
- 伤官稍弱 → 官司纷争
- 伤官见官 → 口舌官司

### 2.6 年龄段与灾祸严重程度

```yaml
20岁以下：能量弱 → 灾祸反应小
20-60岁：  能量最强 → 人生主要运程期
60岁以上： 能量下降 → 同样灾祸严重程度翻倍
```

**关键洞察：** 同样的丑未戌三刑+七杀加强：
- 34岁年轻人 → 官司劳欲之灾
- 65岁老年人 → 生死级（可能死亡）

### 2.7 天和地主必有灾
- **天和地主（凶）**：天干五合（表面一团和谐）+ 地支有冲害（内部藏凶）
  → 表面一套和和睦，背地有人捅刀子
- **天冲地和（吉）**：天干有冲（表面压力大）+ 地支六合（内部团结）
  → 众人帮忙克服困难，最终成功

### 2.8 岁运并临（大运=流年）
- 能量加强
- 如果同时**天克地冲**（天干相克+地支相冲）→ 最凶，可能要人性命
- 大运和流年同时与八字某柱天克地冲 → 该柱对应宫位出大事

## 三、实用补充

### 3.1 偏财旺者易惹官司
- 超过40岁 + 偏财旺 → 10个有6个有牢狱之灾或官司历史
- 因偏财旺的人多做生意/开公司 → 纠纷多

### 3.2 合化后十神需手动排
- 当前排盘软件无法自动排出合化后的十神
- 需手工计算：合化后的五行→变成对应十神
- 合出用神=好事（合财发大财，合印得证书）
- 合出忌神=坏事（合七杀得病）

### 3.3 金水两望
- 金水两旺 → 出帅哥美女（金→五官轮廓，水→皮肤白）
- 金寒水冷（无火）→ 必须走火运才能发（金寒水冷入火箱则发）
