## 第一部分：龙脉五行判定法（Part2）

### 1.1 形而尺寸（灵摆问脉法）

通过灵摆或占卜工具与龙脉沟通，确定其五行属性。

**约定规则（与灵摆商量）：**
- 金 → 顺时针1圈
- 水 → 顺时针2圈
- 木 → 顺时针3圈
- 火 → 顺时针4圈
- 土 → 顺时针5圈
- 顶级龙脉（无行劫补/五色土）→ 逆时针不停旋转

**替代方案（书面问法）：**
- 将金/水/木/火/土分别写在纸上
- 灵摆逐项确认

### 1.2 龙脉五行与八字的匹配逻辑

以火脉为例：
- 壬癸水日元 + 身强 + 火为财 → 火脉 = 财脉
- 丙丁火日元 + 身弱 → 火脉补火 → 提升日元
- 戊己土日元 + 身弱 → 火生土(印) → 补印

### 1.3 顶级龙脉的特征

- 挖出五色土（金木水火土五行俱全）
- 极其稀有，九龙道长只见过一个
- 可用于延寿（何鸿燊案例·一千万种生机）
- 深圳仙湖公园（邓小平种树处）附近曾找到

### 1.4 龙脉类型与找法

- 小区附近公园中也可找到龙脉支脉
- 龙脉在小区旁=区域整体旺
- 深圳同学在仙湖公园找到龙脉
- 东北同学在那边也找到龙脉
