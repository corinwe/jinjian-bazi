## 总纲：化解三原则 + 四大基石

### 四大基石（天时·地理·人和·神助）

```yaml
天时 = 八字命理（先天注定，不可改变）
  一个人的出厂设置，决定人生的基本走势

地理 = 阳宅风水（可调整优化）
  人住在房子里如同铁钉放在磁铁旁边——住久了受房子磁场影响
  风水调理就是调整居住环境的磁场，借地力补天时

人和 = 社交圈层（受高能量人影响）
  靠近高能量、正能量的人，自己的运势也会被带动

神助 = 神佛庇佑（宗教信仰的力量）
  有信仰的人，心理有寄托，遇事有依靠

四者关系：天时不可改，但可通过地理+人和+神助来优化
          风水调整是最实用、最直接的"借地力"手段
```

### 风水二八定律

```yaml
定律：80%的房子风水不好，只有20%是好的
原因：大部分房子在建造时没有考虑风水因素
结论：不能只靠运气撞到好房子，要主动学习风水知识趋吉避凶
      买了有问题的房子不可怕，关键是要知道怎么化解
```

### 风水两大派别

```yaml
形煞派：
  看外环境——山川河流、道路桥梁、建筑形态
  如：路冲煞、天斩煞、反弓煞等
  核心：外部环境对房屋的影响

理气派：
  看内环境——八卦九宫、五行生克、方位布局
  如：九宫缺角、门主灶厕、火烧天门等
  核心：内部空间对居住者的能量影响

两者关系：形煞派断吉凶，理气派定化解。实战中两者并用不冲突
```

### 化解三原则

```yaml
原则一：先看见了再化解
  化解的前提是能准确看出灾祸何时来、来多大
  看不清就化解 = 瞎治

原则二：能轻则轻，能用善不用法
  顺序：主动应灾（最轻）→ 五行补运 → 皈依行善 → 阵法法术（最重，弟子班内容）

原则三：解灾不消业
  化解只能把大事化小、小事化了
  不能完全消除因果，积德行善才是根本
```

## 第三章：太岁化解

### 犯太岁七大类型

| 类型 | 能量 | 触发条件 | 化法 | 说明 |
|:----|:----:|:---------|:-----|:-----|
| **值太岁（临太岁）** | ⭐⭐⭐⭐⭐ | 八字地支与流年地支相同 | 年初拜太岁+全年低调 | 本命年，最大 |
| **冲太岁** | ⭐⭐⭐⭐ | 八字地支与流年地支相冲 | 拜太岁+远离是非+重大决策暂缓 | 冲击大，易变动 |
| **刑太岁** | ⭐⭐⭐ | 八字地支与流年地支相刑 | 注意健康+避免官司 | 纠缠不清，子卯刑最轻 |
| **害太岁** | ⭐⭐ | 八字地支与流年地支相害 | 谨言慎行+防小人 | 暗中有害 |
| **破太岁** | ⭐ | 八字地支与流年地支相破 | 谨慎投资+防破财 | 轻微不利 |
| **三刑全** | ⭐⭐⭐⭐⭐ | 丑未戌/寅巳申三刑齐全 | 拜太岁+放生+行善+全年低调 | 最凶，需综合化解 |
| **天冲太岁** | ⭐⭐⭐ | 八字天干与流年天干相冲（丁癸冲/丙壬冲等） | 拜太岁+注意天干对应脏器疾病 | 天干冲，非地支 |

> **犯太岁看八字四个地支（年/月/日/时），非仅属相。** 只属相犯太岁是最表层的判断。
> **能量大小排序统一：** 值太岁 > 冲太岁 > 害太岁 > 刑太岁 > 破太岁（子卯刑是刑中最轻仅吵架）

### 二、卧室风水要诀

```yaml
1. 房间大小：卧室十几平方最佳，太大(40-50平)→消耗能量
   化解：架子床/罗汉床(有围幔)聚气，或用门帘一分为二

2. 横梁压顶：头顶有横梁→头晕/精神恍惚/不利健康夫妻
   化解：吊顶包平，或移动床位

3. 门不对床：门正对床→气流吹散磁场→人得病
   化解：移动床的位置，或挂门帘

4. 床头靠墙：背后悬空=无靠山→无贵人扶持
   原则：床头必须贴实墙

5. 床头不靠窗：窗户温度比墙低→寒气入头→中风/脑溢血

6. 镜子不对床：镜子阴气重+夜间吓人→惊吓伤肾(恐伤肾)
   中医对照：怒伤肝、恐伤肾、悲伤肺、喜伤心、思伤脾

7. 卧室少绿植：晚上植物释放CO₂吸收O₂→抢人氧气

8. 神龛不放卧室：夫妻行房对着神龛不尊敬

9. 颜色五行调理：
   木→绿/青/长方形；火→红/紫/尖角形；土→黄/棕/正方形
   金→白/圆形；水→黑/蓝/波浪形
   根据八字喜用五行选择卧室颜色和形状
   ⚠️ 与第一章五行补运颜色表一致，无矛盾

10. 未婚催桃花：卧室用粉红/橙色/米黄等鲜艳颜色
```

## 第十一章：八字-阳宅交叉分析法（人宅相合）

> 新增于2026-06-20，实战案例来源于家主（辛金身强）户型分析。

### 11.1 核心方法论

八字阳宅交叉分析 = 将家中每个成员的八字五行喜忌，与所住房间的方位五行进行匹配，判断是否"人宅相合"。

**核心理念：** 同样的房间方位，对不同八字的人吉凶不同。不能只看房子本身的风水，必须结合住的人的八字。

### 11.2 分析流程（六步法）

```
Step 1: 确定房屋朝向
  找到指北针(N)方向，确定上北下南左西右东
  无指北针时，通过入户门朝向反推（南门=坐北朝南最标准）

Step 2: 标定各房间的八卦方位
  东南(巽木) / 东北(艮土) / 西北(乾金) / 西南(坤土)
  正东(震木) / 正南(离火) / 正西(兑金) / 正北(坎水)
  中宫(中央土) — 通常是客厅

Step 3: 加载各人八字数据
  日主五行 + 身强弱分数 + 喜用神/忌神
  身强喜克泄耗（财官食伤）；身弱喜生扶（印比）
  从弱格喜克泄耗（顺从旺势）；中和看大运走势

Step 4: 交叉匹配——个人与房间
  核心公式：八字喜用五行 = 房间方位五行 → 大吉
  详细判断标准见 §11.3

Step 5: 全家五行流通分析
  各房间之间的五行相生关系
  看气运路径：入户 → 客厅 → 各房间是否一路相生
  重点看"子旺父"（子房生父房）、"夫妻相生"等关系

Step 6: 特殊格局判断 + 个性化建议
  见 §11.4（特殊格局）和 §11.5（个性化建议）
```

### 11.3 匹配判断标准

| 八字状态 | 喜用方向 | 匹配房间方位 | 效果 | 注解 |
|:--------:|:--------:|:------------:|:----:|:----:|
| **身强** | 克泄耗 | 方位五行=喜用神 | 🟢 大吉 | 住对位，能量互补 |
| **身弱** | 生扶 | 方位五行=喜用神 | 🟢 大吉 | 得地气补身 |
| **从弱** | 顺旺势(克泄耗) | 方位五行=喜用神 | 🟢 吉 | 顺从格局，不可补身 |
| **中和** | 看大运 | 方位五行=大运用神 | 🟢 吉 | 动态调整 |
| 任何状态 | — | 方位五行=忌神 | 🔴 不吉 | 需化解或调整 |

**实战案例：**

| 人物 | 八字 | 身强弱 | 房间方位 | 判断 | 理由 |
|:----:|:----:|:------:|:---------:|:----:|:------|
| **家主** | 辛金身强 | 喜木(财)火(官) | 东南巽木 | 🟢 **大吉** | 木为财星耗身，财位；巽主科技文昌 |
| **主母** | 庚金从弱 | 喜火木(克泄) | 东南巽木 | 🟢 **吉** | 从弱喜木耗身，木生火顺从格局 |
| **少爷** | 丙火中和 | 喜水平衡 | 东北艮土 | 🟢 **吉** | 艮少男本宫+土泄火秀气，利沉稳 |

### 11.4 特殊格局的八字语境判断

#### 火烧天门（西北乾金 + 厨房火）

🚨 **核心铁律（2026-06-21 老板纠正）：火烧天门是风水峦头之煞，独立于八字喜忌存在。乾宫=男主人位，这个宫位属性是固定不变的。不能因居住者八字喜火就觉得有利。九龙道长自述33岁脑梗案例已证明此煞气的独立性。**

| 八字状态 | 火烧天门判断 | 应对策略 |
|:--------:|:------------:|:---------|
| **任何八字** | 🔴 **均为煞，必须化解** | 土通关（陶土罐+银行土放灶台下）为标准化解法，不会因八字喜忌而改变 |
| **身强喜火者** | ❌ 不可认为"火为用神就有利" | 宫位煞和八字喜忌是两套独立体系，不能相互抵消。仍需土通关化解 |
| **身弱忌火者** | 🔴 煞气更重，更要强调化解 | 土通关必须做，且加强厨房门常关等管理措施 |

**标准化解方法（不分八字，通用）：**

① **灶台下方**放陶土罐+银行土（火生土→土生金通关）
② **西北角**不可放五行属金的物品（白水晶/铜器/五帝钱等）— 补金会加强乾宫金气，但火烧天门的本质问题是火克金，补金不解决问题反而可能增加五行纠缠
③ **厨房门常关**，灶台避开西北角位

**① 源头层——灶台下方放陶土罐+银行土（九龙道长秘法·转化法）**
- 原理：火生土→土生金，把克关系翻转为生关系
- 操作：去五大国有银行门口花池取土→晒干碾碎→装入陶瓷罐（陶瓷属土）→放灶台正下方
- 银行土为"官土/旺财土"，既有化解作用又能旺财
- 文昌土（清华北大门口土）、平安土（五色土/五岳土）也可替代
- 🚨 **无论主人八字喜忌如何，此层必须做**（火烧天门与八字独立）

**② 加强层——西北角摆放物品**
- ⚠️ **注意：** 此层已从补金调整为主——补金并不能化解火烧天门的本质问题（火克金的宫位煞），且可能会增加五行纠缠
- 如需补西北角能量，优先用中性色彩的陶瓷/黄水晶（土属），不用白水晶（金属）
- 白水晶如有需要，应**优先考虑家中喜金的成员**，放在该成员自己的小太极房间内（如儿子卧室文昌位），而不是放在西北大太极

**③ 管理层——厨房门常关、灶台避西北角、浅黄地砖**
- 原理：减少火气直冲乾位
- 适合所有八字状态

#### 缺角判断的八字语境

传统按九宫缺角直接断事。**八字修正法：**
- 如果一个人八字中某宫位的五行本来就是忌神 → 该方位缺角反而无害
- 如果某宫位是喜用神 → 缺角则需要化解
- 重点看：缺角方位对应的家庭成员，其八字是否忌该方位五行

#### 缺角量化标准（九龙道长原始规则）

```yaml
缺角按边长比算（不是面积比）：
  缺角宽度 ÷ 该边总长 < 33.3% → 不算缺角
  缺角宽度 ÷ 该边总长 ≥ 33.3% → 算缺角
  九龙道长原话：「整堵墙的三分之一」「缺角没缺三分之一，不算缺角」
```

### 🆕 11.8 全家合参原则（2026-06-21新增·实战修正）

> **来源：** 实战案例——老板家（一家三口）的西北角白水晶调整。原方案只从方位五行出发（西北乾金→补金），未考虑老板辛金身强忌金，导致补金反而伤主。修正后将白水晶移至少爷卧室文昌位（子源丙火偏强喜金），实现物尽其用。
> **核心教训：** 家庭风水和单人风水完全不同！必须同时考虑所有家庭成员的八字喜忌。

#### 11.8.1 核心方法论

```
家庭风水调整 ≠ 单方位五行补益
家庭风水调整 = 逐方位检查 × 逐人喜忌匹配 × 找到全家最优解
```

**三层过滤法：**

```yaml
第一层：方位五行 → 该方位本身需要什么？
  如：西北乾宫属金 → 火烧天门需土泄火
  如：大门朝西兑金 → 金气入宅需火泄金
  如：西南坤宫缺角 → 传统上补土

第二层：该方位对应的家庭成员八字喜忌（该房间主要居住者）
  如：西北乾宫=男主人 → 男主人辛金身强忌金 → 不可补金 ❌
  如：东北艮宫=少爷 → 子源喜土金 → 补土补金均可 ✅
  如：东南巽宫=主卧/女主人 → 主母从弱喜火→红紫装饰 ✅

第三层：对其他家庭成员的影响（交叉检查）
  一个调整可能同时影响多人：
  ✅ 对A有利且不伤其他人 → 实施
  ✅ 对A有利但对B不利 → 找替代方案/移位到B的小太极
  ❌ 对A不利 → 无论方位五行多正确都不实施
```

#### 11.8.2 实战优先级规则

```yaml
规则①：男主人/女主人本位优先
  乾宫（西北）= 男主人位 → 优先按男主人喜忌
  坤宫（西南）= 女主人位 → 优先按女主人喜忌

规则②：谁住谁优先
  房间主要由谁住 → 按该人的八字喜忌调整该房间
  公共区域（客厅/餐厅）→ 找三人喜用的交集

规则③：物随人走，不是物随风水位
  一个风水物品不是固定在一个位置的
  此方位对A不好但该物品对B好 → 移到B的房间（小太极）
  🔑 实战案例：白水晶从西北（伤老板）→ 移到少爷卧室西南文昌位（利子源）

规则④：全家喜用交集法（公共区域）
  客厅/餐厅等公共区域 → 找所有家庭成员喜用五行的交集
  三人喜用交集 = 共同安全色/物品
  无交集时 → 选不伤任何人的中性方案
```

#### 11.8.3 全家合参配速查表

当调整某个方位/物品时，按以下表格逐人格查：

```yaml
          方向/物品     → 对A(喜用/忌)  → 对B(喜用/忌)  → 对C(喜用/忌) → 结论
          西北补白水晶  → 老板忌金❌     → 主母忌金❌    → 子源喜金✅   → ❌不能放原位
          移至少爷卧室  → 老板不受伤✅   → 主母无影响✅  → 子源喜金✅   → ✅最优解
          门内红垫      → 老板喜火✅     → 主母喜火✅    → 子源不忌✅   → ✅完美
          西南补土(泰山石) → 老板忌土❌  → 主母忌土❌    → 子源喜土✅   → ❌不能补
          西南绿植(木克土) → 老板喜木✅  → 主母喜木✅    → 子源忌木⚠️  → ⚠️少量可
```

#### 11.8.4 常见误区

```yaml
❌ 误区一：纯风水思维——"西北缺金就补金"
  正确：先查住的人八字。如果男主人忌金，补金等于害他

## 第十二章：选房风水实战体系（江西琉落谷五行大派）

> 新增于2026-06-21，核心方法论来源于九龙道长·风水选房课程。以坐山为日元是江西琉落谷五行大派的独门秘法，与第十一章人宅相合交叉分析法互为补充：第十一章讲住进去以后的分析，本章讲住进去之前的选房。

### 12.1 核心方法论——以坐山为日元

**核心公式：** 以房屋的坐山五行 = 该宅的"日元"，看其他宫位的五行生克来定吉凶。

```yaml
坐山 = 房屋的靠山方向（背后方向）

五行生克关系转化（以坐山为日主）：
  坐山所克的方位   → 财位（吉）    — 我克者为财
  克坐山的方位     → 关鬼位（凶）  — 克我者为官杀（鬼）
  生坐山的方位     → 食神位（吉）  — 生我者为印
  同坐山的方位     → 比劫位（中性）— 同我者为比劫
  坐山所生的方位   → 印位（吉）    — 我生者为食伤
```

**与传统八字区别：**
- 八字以日干为"我"，此处以坐山为"我"
- 八字看一生运势，此处看房屋能量场
- 八字不可改，坐山可以在选房时主动选择

> 关键理解：选定坐山后，各方位吉凶就固定了。选房就是选择一个让财、食神、印位都是高大建筑的方位组合。

### 12.2 四大核心原则（缺一不可）

选房时必须四条同时满足，少一条都不行：

```yaml
原则一：有环抱
  小区前面有静水或路呈环抱状弯向小区
  原理：藏风聚气——水/路把气聚集在小区域内不外泄
  反弓水/反弓路 → 气散，凶

原则二：坐山旺
  坐山方向有高大建筑或山体做靠山
  原理：坐山旺则整个房子的气场稳定
  坐山无靠 → 人住着不安定，运势起伏大

原则三：没煞气（关鬼位不能有高大建筑）
  克坐山的方位（关鬼位）不能有高大建筑/山体/塔楼
  原理：关鬼位高大→克坐山的力量强→家运受损
  如果关鬼位有高大建筑且无法改变→此房不能买

原则四：有用神（财印食位有高大建筑为吉）
  财位（坐山所克）有高大建筑 → 旺财
  食神位（生坐山）有高大建筑 → 旺贵气
  印位（坐山所生）有高大建筑 → 旺子孙/文昌
  至少满足其中两个，最好三个全有
```

### 12.3 选房完整流程

#### 第一步：大环境——小区选择

```yaml
1. 小区形状方正，不缺角
   缺角小区 → 气不聚，住户难兴旺

2. 四大原则全部满足：
   ✅ 前面有环抱（水/路）
   ✅ 坐山方向有靠（高大建筑/山）
   ✅ 关鬼位无高大建筑
   ✅ 财/食神/印位有高大建筑

3. 避开以下不利地形：
   ❌ 路冲（道路直冲小区大门）
   ❌ 立交桥环绕（气流混乱）
   ❌ 坟场/医院/殡仪馆附近（阴气重）
   ❌ 垃圾站/污水处理厂（污秽气）
```

#### 第二步：单元楼选择

```yaml
1. 买中间不买两边：
   中间户型三面有邻居→四面围护→气场稳定
   边户三面悬空→气散→不聚财
   两边的户型只有一面有保护，另外三面受外界影响

2. 背后有靠：
   单元楼后面有更高的楼/山做靠山
   被靠的前排楼比后排楼好

3. 前面平阔：
   楼前开阔，无高大建筑遮挡
   视野好，纳气足

4. 不临街：
   临街噪音大、气流混乱
   晚上车声影响睡眠→长期耗气

5. 不买四角（辰戌丑未）：
   小区四个角落的楼栋气场最弱
   辰(东南角) 戌(西北角) 丑(东北角) 未(西南角)
   角楼受两面风吹→气散
```

#### 第三步：楼层选择

```yaml
1. 不买天地层：
   天地层 = 1楼（地）和顶楼（天）
   1楼：潮湿、阴气重、采光差
   顶楼：日晒、风雨大、气场不稳

2. 楼层五行生门向：
   门向 = 入户门朝向的五行
   楼层五行要生旺门向五行

   楼层五行计算：
   尾数1/6=水 2/7=火 3/8=木 4/9=金 5/0=土

   例：门向南（火）→ 选楼层尾数3/8（木）木生火 ✅
        或尾数2/7（火）火旺火 ✅
        忌尾数1/6（水）水克火 ❌

3. 楼层五行与居住者八字匹配（可选加分项）：
   楼层五行生旺日主喜用神 → 更佳
```

#### 第四步：内部格局

```yaml
详见本技能附录中的卧室风水、厨房风水、卫生间风水、九宫缺角等内容。
重点检查：
  ✅ 卧室门不对床
  ✅ 无横梁压顶
  ✅ 火烧天门（需八字语境判断）
  ✅ 门向与生肖不冲（入户门朝向不与居住者生肖相冲）
  ✅ 中宫问题（见§12.6）
```

### 12.4 楼层五行

```yaml
核心：楼层要看尾数，不看整栋楼号

楼层五行对照表：
  尾数 1 / 6 → 水 🔵  (1楼、6楼、11楼、16楼…)
  尾数 2 / 7 → 火 🔴  (2楼、7楼、12楼、17楼…)
  尾数 3 / 8 → 木 🟢  (3楼、8楼、13楼、18楼…)
  尾数 4 / 9 → 金 ⚪  (4楼、9楼、14楼、19楼…)
  尾数 5 / 0 → 土 🟤  (5楼、10楼、15楼、20楼…)

【选楼层总原则】
楼层五行 生 门向五行 → 大吉（门向得楼层生扶）
楼层五行 同 门向五行 → 次吉（能量相同，互相助旺）
门向五行 生 楼层五行 → 平（耗门向能量）
门向五行 克 楼层五行 → 凶（门向被楼层反克）

【门向五行对照】
  门朝南(离火) → 选木(尾3/8)或火(尾2/7)层
  门朝北(坎水) → 选金(尾4/9)或水(尾1/6)层
  门朝东(震木) → 选水(尾1/6)或木(尾3/8)层
  门朝西(兑金) → 选土(尾5/0)或金(尾4/9)层
  门朝东南(巽木) → 选水(尾1/6)或木(尾3/8)层
  门朝西南(坤土) → 选火(尾2/7)或土(尾5/0)层
  门朝东北(艮土) → 选火(尾2/7)或土(尾5/0)层
  门朝西北(乾金) → 选土(尾5/0)或金(尾4/9)层
```

### 12.5 左青龙右白虎

```yaml
青龙位 = 食神位（生坐山的方位）——吉
白虎位 = 关鬼位（克坐山的方位）——凶

【重要原则】
青龙位（食神位）越高越好：
  左边（青龙方）有高大建筑 → 得贵人扶助
  象征事业、声望、贵气

白虎位（关鬼位）越低越好：
  右边（白虎方）应平坦或低矮 → 压制煞气
  白虎位有高大建筑 → 关鬼位能量强 → 家宅不安

【量化的吉凶标准】
  青龙高于白虎 → 🟢 大吉，人丁兴旺
  白虎高于青龙 → 🔴 大凶，易有意外/是非
  青龙白虎等高 → 🟡 平，需要靠其他方位弥补

【特殊情况】
  如果坐山朝南（坐北朝南）：
    左青龙 = 东（震木）= 食神位
    右白虎 = 西（兑金）= 关鬼位
    东边建筑高于西边 → ✅ 大吉

  如果坐山朝北（坐南朝北）：
    左青龙 = 西（兑金）= 食神位
    右白虎 = 东（震木）= 关鬼位
    西边建筑高于东边 → ✅ 大吉
```

### 12.6 中宫问题

```yaml
中宫 = 整套房子的正中心
中宫五行属土（中央土）

【中宫空亡（天井）】
  定义：中宫位置是空旷的露天天井（无顶）
  效应：气从天井散走→无法聚气→不好
  判断：别墅/自建房才有天井问题
  化解：天井加顶盖，或中宫上方封住

【中宫种树】
  效应：木克土→破坏中宫气场
  中宫五行属土，种树=木克土
  影响：一家人脾胃运化不好、消化不良
  化解：树移走，或中宫不做花园

【中宫好格局】
  中宫为客厅/堂屋 → ✅ 最佳
  中宫五行土，客厅聚集家人→土气场旺盛
  中宫通明（有采光）→ 气流通畅

【中宫忌讳】
  ❌ 中宫做卫生间 → 土被水湿→污秽冲散家运
  ❌ 中宫做厨房 → 火生土强但油烟污秽
  ❌ 中宫做楼梯 → 人上下不歇→气不稳
  ❌ 中宫做卧室 → 气过旺不适宜长时间休息
```

### 12.7 五谷进宅法（进宅仪式）

```yaml
适用场景：新房装修完后择吉日进宅
核心目的：将不干净的孤魂野鬼请走，净化气场
进完宅后方可安神（供奉神明/祖先）

【准备物品】
  五谷：稻谷(大米)、小麦(面粉)、小米、玉米、豆类
      或直接用超市买来的五谷杂粮混合
  红布：一块（约1米×1米）
  铜钱/硬币：八枚（五帝钱或普通硬币均可）
  香炉/香：三根

【仪式步骤】

第一步：择吉日
  选择与房主八字无冲的吉日良辰
  一般选在早晨（阳气上升时段）
  避开与房主生肖相冲的日子
  可选择天德/月德/天乙贵人日

第二步：进宅前净宅
  进宅前三天，在房内各角落撒盐和糯米
  用意：盐=净化的能量，糯米=吸附阴气
  三天后扫干净

第三步：五谷进宅（正式仪式）
  ① 提前准备好五谷，用红布包好
  ② 择吉日良辰入门
  ③ 入门时口诵：
     \"吉日良辰进新房，五谷丰登保安康
      一切孤魂皆退散，家宅平安福满堂\"
  ④ 将红布包放于中宫位置（客厅正中央）
  ⑤ 点燃三根香，拜四方（东、南、西、北）
  ⑥ 口诵：\"此宅今日迎新主，五谷进宅驱秽土
            各方神灵来护佑，阖家平安万事福\"
  ⑦ 香燃尽后收起红布包，放于家中财位

第四步：安神
  进宅仪式完成后，方可请神/祖先牌位入宅
  安神必须在进宅之后，不可提前
  安神需择吉日，不可随意

第五步：暖宅
  进宅后第一晚，全家人在新房过夜
  开灯一整晚（增添阳气）
  煮一锅汤圆或甜粥（象征团圆甜蜜）

【注意事项】
  ⚠️ 进宅前不可建神龛/供神像
  ⚠️ 仪式严肃，不要嬉笑打闹
  ⚠️ 进宅当天不能吵架
  ⚠️ 如有孕妇，可择日回避仪式气场
```

### 印位/食神位制约关鬼化解法

```yaml
核心场景：当房屋的某个重要功能区域（如主卧门）不得不开在关鬼位时
          通过印位或食神位的能量来制约关鬼位的煞气

【基本原理】
  印位（坐山所生）= 食伤位 → 泄关鬼位之气
  食神位（生坐山）= 印位 → 生扶坐山抵抗关鬼
  利用五行通关：让煞气先经过吉位再进入关鬼位

【局势分析法】

局势一：主卧门开在关鬼位
  判断：关鬼位（克坐山的方位）开门 → 能量直冲卧室
  化解：
    A方案——在印位或食神位设屏障：
      在食神位（生坐山方位）放置屏风/柜子/高大的绿植
      原理：食神位能量生坐山 → 增强坐山抵抗力
      如坐山东(木)→食神位在北(水)→北面放大缸水或黑色屏风

    B方案——五行通关法：
      在关鬼位到主卧之间设置五行通关物品
      例：坐山水→关鬼位在北(土克水)
         → 在关鬼位和卧室之间放金⚪(白色)物品
         → 土生金→金生水→克变生

局势二：门开在财位——加强催财
  判断：门在财位（坐山所克）→ 天然旺财局
  加强：在食神位放催财物
    原理：食神位=生坐山→坐山旺→克财位的力量更强→更旺财
    操作：如坐山木→财位土(东北)→食神位水(北)
          → 北面放鱼缸/黑曜石催财

局势三：厨房在关鬼位
  判断：厨房（火）在关鬼位 → 火助关鬼位克坐山
  化解：
    在印位放水器 → 水克火减弱关鬼位火力
    在食神位放金属物品 → 增强生坐山的力量

【物品选择速查表】

| 坐山五行 | 食神位(生坐山) | 印位(坐山所生) | 食神位物品 | 印位物品 |
|:--------:|:--------------:|:--------------:|:----------:|:--------:|
| 木🟢 | 水🔵 | 火🔴 | 鱼缸/黑曜石/黑色 | 红灯笼/中国结/灯光 |
| 火🔴 | 木🟢 | 土🟤 | 绿植/文竹/富贵竹 | 陶瓷罐/黄水晶/泰山石 |
| 土🟤 | 火🔴 | 金⚪ | 红色灯/中国结 | 白水晶/铜钱/五帝钱 |
| 金⚪ | 土🟤 | 水🔵 | 陶瓷/黄水晶 | 鱼缸/黑曜石/流水摆件 |
| 水🔵 | 金⚪ | 木🟢 | 白水晶/五帝钱 | 绿植/富贵竹/木质品 |

【核心口诀】

### 16.3 刑冲合害在起名中的应用（核心方法论）

#### 六合化解冲法（名字选字的核心技术）

```yaml
核心规则：八字有冲 → 用合来化解
  子午冲 → 选丑(牛)或未(羊)化解
    - 子→丑(子丑合)
    - 午→未(午未合)
    → 合住了就不冲了

  辰戌冲 → 选酉(鸡)或卯(兔)或申(猴)或午(马)化解
    - 辰→酉(辰酉合)
    - 戌→卯(卯戌合)
    - 辰→申(申辰半合)
    - 戌→午(午戌半合)
    → 贪合忘冲

  寅申冲 → 选亥(猪)或巳(蛇)化解
    - 寅→亥(寅亥合)
    - 申→巳(巳申合)
    → 贪合忘冲

  巳亥冲 → 选申(猴)或寅(虎)化解
    - 巳→申(巳申合)
    - 亥→寅(寅亥合)
    → 贪合忘冲

  丑未冲 → 选子(鼠)或午(马)化解
    - 丑→子(子丑合)
    - 未→午(午未合)
    → 贪合忘冲
```

#### 三合化解法

```yaml
原理：利用三合局的强大能量覆盖掉冲的效应

  子午冲 → 用申(猴) → 申子辰三合
  卯酉冲 → 用亥(猪) → 亥卯未三合
  寅申冲 → 用午(马) → 寅午戌三合
  巳亥冲 → 用酉(鸡) → 巳酉丑三合
```

规则⑤ — 名字阴阳平衡

# 第二十章：合化能量倍数体系（2026-07-04新增·完整版）

> **来源：** 九龙道长新课《八字合化能量倍数体系 + 查一生灾祸方法》（2026-07-04）

## 20.1 六合（地支六合）

| 条件 | 能量倍数 | 说明 |
|:----|:--------:|:-----|
| 天干有能量引化 + 合成 | **10倍** | 天干透出合化后的五行，或被月令所生助 |
| 天干无能量引化（合而不化） | **5倍** | 地支六合关系成立，但天干无引化能量 |

**六合列表：** 子丑合土、寅亥合木、卯戌合火、辰酉合金、巳申合水、午未合土/火

## 20.2 三合局

| 条件 | 能量倍数 | 说明 |
|:----|:--------:|:-----|
| 天干有能量引化 + 合成 | **15倍** | 三合局完整且天干引化 |
| 天干无能量引化（合而不化） | **7~8倍** | 三合局成立但天干不透 |

**三合列表：** 申子辰合水、亥卯未合木、寅午戌合火、巳酉丑合金

## 20.3 半三合

| 条件 | 能量倍数 | 说明 |
|:----|:--------:|:-----|
| 天干有引化 | **10倍** | 如申子半合水，天干透水 |
| 天干无引化 | **5倍** | 半三合但天干不透 |

**半三合组合：** 申子、子辰、亥卯、卯未、寅午、午戌、巳酉、酉丑、申辰（拱合）等

## 20.4 三会局

| 条件 | 能量倍数 | 说明 |
|:----|:--------:|:-----|
| 天干有能量引化 + 合成 | **20倍** | 三者会齐且天干引化，最强 |
| 天干无能量引化 | **10倍** | 三者会齐但天干不透 |

**三会列表：** 亥子丑会水、寅卯辰会木、巳午未会火、申酉戌会金、辰戌丑未会土

## 20.5 三刑

### 丑未戌三刑

| 条件 | 能量倍数 | 说明 |
|:----|:--------:|:-----|
| 天干有土引化 | **15倍** | 三个土的地支藏干全部打架打没，只余本气 |
| 天干无土引化 | **7~8倍** | 三刑成立但天干无引化 |

**原理：** 丑未戌三刑 = 三个土互相打压，地支藏干全部打没了，剩下的本气土能量是加强的。

### 寅巳申三刑

**特殊规则：** 寅巳申三刑的三个字能量全部加强（不是单个字）。
- 天干透出哪个字，就加强哪个字的能量
- 如流月天干透甲木 → 寅的能量加强；透丙火 → 巳的能量加强；透庚金 → 申的能量加强

### 子卯刑
- 子卯刑属于较轻的相刑，主要表现为争吵

## 20.6 自刑

| 地支自刑 | 条件 | 能量倍数 |
|:---------|:----|:--------:|
| 辰辰自身 | 天干有土引化 | 10倍 |
| 酉酉自身 | 天干有金引化 | 10倍 |
| 亥亥自身 | 天干有水引化 | 10倍 |
| 午午自身 | 天干有火引化 | 10倍 |
| 同上 | 天干无引化 | **5倍** |

## 20.7 空亡（能量减半规则）

- 空亡 = 能量减半
- **坏的能量空亡 = 好事**（如七杀空亡、灾煞空亡）
- **好的能量空亡 = 坏事**（如财星空亡、文昌空亡）
- 七杀空亡 → 七杀能量减半 → 灾祸减少
- 财星空亡 → 本应赚100万，只赚50万
- 能量倍数计算方式：**先算合化倍数，再除以2**

## 20.8 合化倍数完整等级汇总表

| 合化类型 | 合成+天干引化 | 合而不化 |
|:--------|:------------:|:--------:|
| 六合（地支） | **10倍** | **5倍** |
| 三合 | **15倍** | **7~8倍** |
| 半三合 | **10倍** | **5倍** |
| 三会 | **20倍** | **10倍** |
| 三刑（丑未戌） | **15倍**（有土引化） | **7~8倍**（无土引化） |
| 三刑（寅巳申） | 各字能量加强（有天干引化） | 各字能量加强（无引化也有） |
| 自刑（辰辰等） | **10倍**（有天干引化） | **5倍**（无天干引化） |
| 岁运并临 | 能量加强（倍数为叠加） | — |
| 三合+六合同步 | 取最大倍数计算，不叠加 | — |

## 20.9 合化的\"隔位\"问题

- **本命局**的合化：需要连在一起（相邻或三合局完整）才能算合化
- **大运流年**的合化：不需要连在一起！天上的星星可以照到任何位置
- 流年天干 = 该年的值班星，可以照到八字的任何位置

## 20.10 能量计算方法（与40分法则的关系）

- 得令一次 = 40分
- 天干透出一次 = 12分
- 通根/微根叠加计算
- 大运流年也要加进去
- **合化倍数 = 在原有分数基础上乘以倍数**
- 比如财星本来12分，被三合化且天干引化 → 12×15 = 180分（极旺！）

## 20.11 合化优先级与注意事项

| 合化优先级 | 三合 > 六合 > 刑冲害破（有合化时，先论合化） |
|:----------|:--------------------------------------------|
| 合化后十神 | 合化后十神变成了另外一个五行，需手工计算。目前排盘软件无法自动排出合化后的十神 |
| 合化喜忌 | 合化出用神=好事（合财发财/合印得文凭）；合化出忌神=坏事（合七杀得病/合劫财破财） |

# 第二十二章：生死灾祸六步诊断法（2026-07-04新增）

> **来源：** 九龙道长面授班后直播《生死节的看法》（2026-07-04）

## 22.1 六种致命灾祸判断法

### ① 大运流年天克地冲·岁运并临（⭐⭐⭐⭐⭐）

**判断条件（三者缺一不可）：**
1. 大运 = 流年（岁运并临）
2. 大运+流年共同天克地冲八字的某一柱
3. 被冲的是本命局的该柱

**口诀：** 「不死自己死家人」

| 被冲宫位 | 应事六亲 | 身体部位 |
|:--------:|:--------:|:---------|
| 年柱 | 父母/祖上/家族 | 头部/颈椎/喉咙 |
| 月柱 | 兄弟姐妹/平辈 | 胸部/腹部/脾胃 |
| 日柱 | 自己/配偶 | 小腹/生殖/夫妻宫（**最凶**） |
| 时柱 | 子女 | 腿部/膝盖以下/脚 |

**实战要点：** 岁运并临但不冲八字本命局 → 没事；岁运并临+天克地冲 → 极为罕见（约200人中1个）

### ② 地支三刑+天干双七杀（⭐⭐⭐⭐⭐）

| 组合 | 致命程度 | 说明 |
|:----|:--------:|:------|
| 地支丑戌未三刑 单独 | ⭐⭐⭐ | 有灾但不必然死 |
| 天干出现两个七杀 | ⭐⭐⭐ | 有灾但不必然死 |
| **地支三刑 + 天干双七杀** | ⭐⭐⭐⭐⭐ | **精确组合，极为致命** |

**实战案例：** 87岁老人，地支丑未戌三刑+天干两七杀（合化15倍）→ 这一年必死

### ③ 七杀过多无制（⭐⭐⭐⭐）

- **15倍七杀** → 不管身强身弱都受不了，必须精确化解
- **10倍七杀** → 身强者用食神治/身弱者用印收，都有困难

### ④ 枭神夺食（自杀/偏瘫⭐⭐⭐⭐）

- **前提：** 本命局必须有食神才怕枭神夺食
- **无食神者：** 枭神当正印看，不用担心
- **年龄分水岭：** 年轻（36岁以下）→ **自杀自残**；中年以上（36岁+）→ **偏瘫中风**

**实战案例：** 2020年预测、2021年应验的15倍枭神夺食→自杀案例

### ⑤ 血刃/灾煞能量过强（⭐⭐⭐⭐）

**血刃查法（以日元查）：**

| 日元 | 血刃地支 | 日元 | 血刃地支 |
|:----:|:--------:|:----:|:--------:|
| 甲木 | 卯 | 乙木 | 辰 |
| 丙火 | 午 | 丁火 | 未 |
| 戊土 | 午 | 己土 | 未 |
| 庚金 | 酉 | 辛金 | 戌 |
| 壬水 | 子 | 癸水 | 丑 |

**灾煞五行灾害对应**（见§21.2.4）

### ⑥ 宫位受伤（⭐⭐⭐⭐）

**定义：** 八字中某个字全局无生扶、被重重克制 → 能量降到极低

**判断标准：** 该字无同五行帮它、无生它的五行、周围全是克/耗它的五行

**后果：** 代表该宫位的**六亲必然有病或早逝**；对应**身体部位必然有病**；遇大运流年冲击→灾难爆发

**实战案例：** 女命夫妻宫（火）被一圈水克，全局无木生火 → 丈夫河里淹死

## 22.3 生死灾祸六步排查流程

```
Step 1: 查四柱天克地冲岁运并临
  → 逐柱检查：一生中是否有大运+流年相同且天克地冲该柱？

Step 2: 查地支三刑+天干双七杀
  → 本命局是否有丑戌未/寅巳申三刑的地支？
  → 大运流年是否能同时触发三刑+天干双七杀？

Step 3: 查七杀过多无制
  → 大运流年的三合/三会/六合是否合化出5倍/10倍/15倍七杀？

Step 4: 查枭神夺食
  → 本命局是否有食神？（没有就不用查）
  → 大运流年是否有强枭神来夺？

Step 5: 查血刃/灾煞能量加强
  → 以日元查血刃，以年支三合局查灾煞
  → 是否被三合/三会/六合加强到5倍以上？

Step 6: 查宫位受伤
  → 每个字的生扶力量是否充足？
  → 有没有哪个字全局无生、被重重克制？
```

# 第二十三章：婚姻配婚化解补充（2026-07-04新增）

> **说明：** 以下为婚姻分析体系的化解方法补充

## 23.1 子午冲夫妻宫→生化解法

```yaml
适用场景：女命时支冲日支夫妻宫（子午冲）
核心解法：生个孩子就破了

原理：生孩子的过程改变了时柱的能量结构
     → 子午冲被解开 → 婚姻矛盾化解

操作要点：
  - 仅适用于女命时支冲日支的子午冲
  - 其他类型的夫妻宫刑冲不适合此解法
```

## 23.2 缺水火旺女命→补水解法

```yaml
适用场景：丙火女命八字缺水→火太旺脾气爆→婚姻不好

核心解法：
  ① 找水旺的老公（八字水旺者，水火既济）
  ② 用能量阵/大地阵补水

能量调整方案：
  - 佩戴黑色/蓝色饰品（水属性）
  - 家中北方布置鱼缸或流水摆件
  - 卧室用黑色/蓝色为主色调
  - 选择水旺的日子相亲/约会

原理：火旺无水则焚，水火既济则婚姻和谐
      补水可以抑制过旺的火气
      同时增强老公缘分（水为官杀）
```

## 23.3 夫妻宫自合=甜蜜关系

```yaml
夫妻宫天干地支自合 → 夫妻甜蜜，无需额外化解

常见自合日柱：
  丁亥日柱：丁壬合化木（丁火合亥中壬水）=夫妻甜蜜
  戊子日柱：戊癸合化火 = 同上
  辛巳日柱：丙辛合化水 = 同上
  壬午日柱：丁壬合化木 = 同上
  甲午日柱：甲己合化土 = 同上

注意：若自合化出忌神（七杀/伤官），则为相爱相杀型
```
