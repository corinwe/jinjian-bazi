## 架构总览

```
用户输入(姓名+出生日期+性别)
  → paipan.py 排盘（五虎遁+五鼠遁+日干支算法）

  → 前端渲染 / PDF下载
```

## 排盘模块 (paipan.py)

**位置**: `projects/bazi-platform/engine/paipan.py`

**功能**: 从公历日期+时辰 → 四柱八字

**核心算法**:
- 年柱: 以立春(2月4日)为分界，立春前算上一年
- 月柱: 五虎遁（年干定月干起始）
- 日柱: 日干支基数公式 (year-1900)*5 + (y-1)//4 + day_of_year
- 时柱: 五鼠遁（日干定时干起始）

**返回格式**:
```python
{
    "name": "姓名",
    "gender": "男/女",
    "birth_date": "1980年5月15日",
    "shi_chen": "午时(11:00-12:59)",
    "bazi": "庚申 壬午 戊寅 戊午",
    "year_pillar": {"gan": "庚", "zhi": "申"},
    "month_pillar": {"gan": "壬", "zhi": "午"},
    "day_pillar": {"gan": "戊", "zhi": "寅"},
    "hour_pillar": {"gan": "戊", "zhi": "午"},
}
```

**五虎遁口诀**: 甲己之年丙作首, 乙庚之岁戊为头, 丙辛必定寻庚起, 丁壬壬位顺行流, 戊癸何方发, 甲寅之上好追求。

**五鼠遁口诀**: 甲己还加甲, 乙庚丙作初, 丙辛从戊起, 丁壬庚子居, 戊癸何方发, 壬子是真途。
