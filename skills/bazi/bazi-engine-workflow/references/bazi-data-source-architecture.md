# 八字数据源架构（2026-07-16 老板指定）

## 核心原则

```
引擎JSON → bazi-data-source.py验证+锁定 → /tmp/{姓名}_datasource.json → 所有模块从此取数
```

**不靠记忆，全靠代码。同一份数据源，所有§共用。**

## 数据源包含的字段

| 类别 | 字段 | 来源 |
|:-----|:-----|:------|
| 8字段 | 年干/年支/月干/月支/日干/日支/时干/时支 | 引擎四柱派生 |
| 藏干 | 每个地支的藏干列表（含比例） | 引擎 |
| 藏干十神 | 每个藏干成分的十神名称 | bazi-data-source.py计算 |
| 纳音 | 四柱纳音 | 引擎 |
| 空亡 | 日柱对应旬空 | bazi-data-source.py计算 |
| 神煞 | 天乙/文昌/桃花/驿马/华盖/孤辰/寡宿/灾煞 | bazi-data-source.py计算 |
| 起运年龄 | 文字+数值两种格式 | 引擎 |
| 十大运 | 扩展至10步(到100岁) | bazi-data-source.py扩展 |
| 身强弱 | 总分+等级 | 引擎 |
| 十神 | 年干/月干/日干/时干 | 引擎 |

## 使用方式

```python
import json
DS = json.load(open('/tmp/{姓名}_datasource.json'))
# 后续每个模块从DS取数，不再重新计算
```

## 大运顺逆规则（⚠️ 常见Bug）

- **年干阴阳+性别**决定顺逆（不是日干）
- 年干阳+男→顺排；年干阴+女→顺排
- 年干阴+男→逆排；年干阳+女→逆排
