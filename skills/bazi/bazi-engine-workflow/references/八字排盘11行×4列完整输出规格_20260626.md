## 第一大步：11行×4列 + 4附加项

任何八字进来，先产出以下数据（只查表/只计算，不做任何判断分析）：

### 核心网格（11行×4列）

| 行号 | 行名 | 年柱 | 月柱 | 日柱 | 时柱 |
|:---:|:-----|:----|:----|:----|:----|
| 1 | **十神** | 劫财 | 食神/伤官 | 元男(日主) | 比肩 |
| 2 | **天干** | 庚 | 癸 | 辛 | 辛 |
| 3 | **地支** | 申 | 未 | 亥 | 卯 |
| 4 | **藏干** | 庚壬戊(带十神) | 己丁乙(带十神) | 壬甲(带十神) | 乙(带十神) |
| 5 | **纳音** | 石榴木 | 杨柳木 | 钗环金 | 松柏木 |
| 6 | **空亡** | 子丑 | 申酉 | 寅卯 | 午未 |
| 7 | **神煞** | 羊刃劫煞 | 红鸾丧门等 | 文昌亡神等 | 将星元辰等 |

### 附加项（4项）

| # | 项目 | 内容示例 |
|:-:|:----|:---------|

### 每列藏干详细格式

每柱地支的藏干必须标注十神和百分比：

```json
{
  "di_zhi": "申",
  "cang_gan": [
    {"gan": "庚", "shi_shen": "比肩", "percent": 100},
    {"gan": "壬", "shi_shen": "食神", "percent": 60},
    {"gan": "戊", "shi_shen": "正印", "percent": 30}
  ]
}
```

### 十神判定规则（精确）

| 关系 | 阴阳同 | 阴阳异 |
|:----|:------:|:------:|
| 生我者(印) | 偏印 | 正印 |
| 我生者(食伤) | 伤官 | 食神 |
| 克我者(官杀) | 七杀 | 正官 |
| 我克者(财) | 偏财 | 正财 |
| 同我者(比劫) | 比肩 | 劫财 |

**特别注意：**
- 日柱天干 = "元男"(男) / "元女"(女)，不是比肩/劫财
- 其他柱天干与日主相同 = 比肩(同阴阳) 或 劫财(异阴阳)

### 空亡计算规则（每柱独立）

每柱根据自身干支确定所属旬，独立计算空亡：

| 旬 | 干支范围 | 空亡地支 |
|:---|:---------|:--------:|
| 甲子旬 | 甲子~癸酉 | 戌亥 |
| 甲戌旬 | 甲戌~癸未 | 申酉 |
| 甲申旬 | 甲申~癸巳 | 午未 |
| 甲午旬 | 甲午~癸卯 | 辰巳 |
| 甲辰旬 | 甲辰~癸丑 | 寅卯 |
| 甲寅旬 | 甲寅~癸亥 | 子丑 |

例：庚申→甲寅旬→子丑空 | 癸未→甲戌旬→申酉空
    辛亥→甲辰旬→寅卯空 | 辛卯→甲申旬→午未空

**禁止**：所有柱子共用日柱空亡（之前犯过的错误）

### 神煞查法规则

神煞以年支和日支分别查询，结果合并去重。

**神煞规则表**（19个，配置在 shen_sha_rules.json）：
天乙贵人 / 文昌贵人 / 驿马 / 桃花 / 华盖 / 羊刃 / 劫煞 /
灾煞 / 孤辰 / 寡宿 / 亡神 / 吊客 / 金舆 / 将星 / 元辰 /
披麻 / 红鸾 / 丧门 / 太极贵人

**特殊神煞**：
- 女孤鸾煞：日柱地支为亥（特定日柱）
- 天罗地网：辰巳/戌亥
