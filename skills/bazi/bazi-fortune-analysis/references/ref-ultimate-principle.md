## 总纲：一字记之曰「能」——能量为本

> 一切断事回到能量。能量倍数×喜忌方向=断事结果。没有第二条路。

### 【核心公式】

```
断事结果 = 能量倍数 × 喜忌方向

能量倍数 = 合/会/冲/刑/害 赋予该十神的能量（5倍~20倍）
喜忌方向 = 该能量对命主是喜用（+）还是忌凶（-）

输出：
  能量强(≥10倍) + 喜用 = ✅ 好事发生（结婚/发财/升职/生子）
  能量强(≥10倍) + 忌凶 = ❌ 坏事发生（灾祸/疾病/破财）
  能量弱(<5倍) + 任何方向 = 轻微或不发生
```

## 附则：总纲与细则的关系

> **总纲是方向盘，细则是工具箱。没有总纲细则散，没有细则总纲空。**

**总纲负责**：告诉你怎么算（能量倍数×喜忌方向）→ 这是方向
**细则负责**：告诉你去哪找、用什么算（日主为金→正财甲木→三合15倍→喜用则吉）→ 这是工具

**工作时的配合方式**：
```
总纲问：这个事项要算什么能量？
细则答：结婚→男命看正财/女命看正官，没有则看替代星

总纲问：这个能量有多少倍？
细则答：三合15倍/六合10倍/冲10倍/空亡减半...

总纲问：这个能量是喜用还是忌凶？
细则答：看身强弱和用神→判断喜忌

总纲下结论：15倍×喜用=✅结婚 | 10倍×忌凶=❌灾祸
```

**简单说**：细则告诉总纲"算什么是哪个数"，总纲告诉细则"这个数×方向=什么结果"。两者缺一不可。

1. **优先级最高**：总纲高于一切具体规则。如果某个具体规则和总纲冲突，以总纲为准
2. **每个分析前念一遍口诀**：确保回到能量根本，不被散装规则带偏
3. **每个断语写完后过一遍总纲验证**：这个断语是否符合"能量倍数×喜忌方向"？
4. **所有107项知识点只是总纲的注脚**：它们是总纲在不同场景下的具体投影，不是独立规则
