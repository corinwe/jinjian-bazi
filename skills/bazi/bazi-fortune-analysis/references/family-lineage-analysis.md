## 一、核心理念：家族命理"基因"

### 1.1 什么叫家族命理基因

家族成员之间并非独立个体——他们的八字可能共享特定的**重复结构**：

- **共享地支**：同一地支（如丑、辰）在多位家族成员的八字中出现
- **共享五行倾向**：某五行在所有家族成员中均扮演核心角色（旺或弱）
- **共享格局倾向**：家族成员呈现相似的格局特征（如都带印星）

这些共享结构，就是**家族命理基因**。

### 1.2 基因的演化规律

| 代际 | 特征 | 典型表现 |
|:----:|:----|:--------|
| **第一代（祖辈）** | 基因"基础版" | 某五行旺但有制，格局常规 |
| **第二代（父母辈）** | 基因"变形版" | 同一五行或弱化或变异，格局波动 |
| **第三代（子女辈）** | 基因"集大成版" | 同一五行极致化，出现罕见格局 |

**关键观察**：一个家族的命理基因不会"消失"，只会在代际间重新组合和演化。第三代出现罕见格局，往往不是偶然——它是前两代能量累积到临界点后的**命理显化**。

## 二、分析流程

### Phase 1：个人八字独立分析（必须前置）

对每位家族成员先完成完整的个人分析（按标准7维度流程）。在个人分析中特别标注：

- 每个地支的位置和力量
- 五行分布的极值（最旺/最弱）
- 格局类型及罕见度
- 用神忌神

### Phase 2：识别家族共享"基因"

#### 2.1 地支扫描

将四柱八个地支全部列出，找出跨成员重复出现的地支：

```
令尊：丑、酉、申、午
令慈子时：辰、申、子、子
令慈善丑时：辰、申、子、丑
外甥：戌、未、丑、辰
```

**重复分析**：
- 丑土：令尊（年支）→ 令慈丑时案（时支）→ 外甥（日支） → **三代共有的"土锚"**
- 辰土：令慈（年支）→ 外甥（时支） → 第二代→第三代传递
- 申金：令尊（日支）→ 令慈（月支） → 夫妻共享的金基因
- 子水：令慈（日支+时支双见）→ 不传第三代

#### 2.2 五行极值追踪

| 五行 | 令尊 | 令慈 | 外甥 | 演化趋势 |
|:---:|:----:|:----:|:----:|:---------|
| **土** | 6（极旺） | 2-3（弱） | **62.5%（极旺+四库）** | ↗︎ 先抑后扬，第三代集大成 |
| **水** | 2（中） | **6（极旺）** | 9.4%（弱） | ↘︎ 第二代膨胀，第三代消退 |
| **金** | 3（旺） | 1-2（弱） | 9.4%（被埋） | ↘︎ 逐代减弱 |
| **火** | 1（中） | 1（极弱） | 9.4%（助土） | → 一直偏弱 |

#### 2.3 格局传承分析

| 命主 | 格局 | 与家族的关联 |
|:----:|:----|:-----------|
| 令尊 | 身旺·伤官配印+食神生财 | 土旺为根基，丑土为基因起点 |
| 令慈子时 | 财多身弱（无根） | 土崩水泛，基因在逆境中变形 |
| 令慈善丑时 | 财多身弱（有根版） | 丑土暗根，基因未断绝 |
| 外甥 | **假从印格·四库齐全** | 土基因极致化！辰戌丑未四库全 |

**核心发现**：外甥的"四库齐全"不是偶然——他的命局将家族三代分散在不同地支的土能量（令尊的丑、令慈的辰和丑）全部集齐为辰戌丑未四库。这是**家族土基因的终极表达**。

### Phase 3：家族能量关系图

#### 3.1 合冲关系矩阵

列出所有家族成员的干支之间的合、冲、刑、害关系：

| 组合 | 关系 | 含义 |
|:----|:----|:-----|
| 令尊年柱己丑 × 令慈年柱壬辰 | **天地双合**（己合壬+丑辰破） | 夫妻缘分牢固但有微裂 |
| 令尊日支申 × 令慈月支申 | **同气**（双申金） | 夫妻在思维/表达上有共鸣 |
| 令尊时支午 × 令慈日支子 | **子午冲** | 观念碰撞，但冲中带变 |
| 令慈年支辰 × 外甥时支辰 | **同支** | 祖孙共享"辰"基因 |
| 令慈年支辰 × 外甥年支戌 | **辰戌冲** | 祖孙能量有对冲 |
| 令尊年支丑 × 外甥日支丑 | **同支** | 爷孙共享"丑"基因 |

#### 3.2 能量流向图

```
          ┌──────────┐
          │  令尊    │
          │ 土6极旺  │ ← 家族土能量的"源头"
          │ 用神金水 │ ← 需要泄土
          └────┬─────┘
               │ 午火生土（自身循环）
               │ 丑土传后代
               │
     ┌─────────┼──────────────┐
     │          │               │

┌────────┐  ┌────────┐  ┌──────────────┐

│ 水6极旺│  │ 水6+丑3│  │ 土62.5%+四库 │
│ 需火土 │  │ 需火土  │  │ 需土火顺势  │
│承：辰  │  │承：辰丑 │  │承：辰戌丑未 │
└────────┘  └────────┘  └──────────────┘
```

**流向分析**：
1. 令尊的**丑土**为家族土基因起点
2. 传至令慈时，丑土沉入时支（丑时案）或完全消失（子时案）——土基因被水淹没
3. 传至外甥时，丑土回到日支，加上令慈的辰、令尊未出现的未和戌——**四库齐聚**
4. 土能量在代际间经历了"旺→衰→极旺"的U型曲线

### 3.2 家族传承启示

> 同一个家族，不同代际的五行分布、格局形态可能截然不同。
>
> 但**关键的地支和五行往往会以不同的形式在代际间反复出现**——这就是家族命理基因。

| 代际 | 格局 | 与上一代的关联 |
