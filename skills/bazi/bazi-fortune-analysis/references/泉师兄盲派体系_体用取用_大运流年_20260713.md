## 一、体用旺弱 ≠ 身强身弱（核心区分）

| 概念 | 定义 | 作用 |
|:----|:-----|:-----|
| **身强身弱** | 日主的承载能力（有根无根、得不得生扶） | 决定能否扛住财官 |
| **体用旺弱** | 体（日主+印+比劫+食神）的力量能否撬动用（财官+伤官） | 决定能否得到财官 |

> 泉师兄原话："这跟身强身弱没有半毛钱关系。身强身弱经营的是日主的承载能力，而承载能力并不能决定一个人的财富和地位。难道身弱的人一定是穷人吗？身旺的人一定是有钱吗？"

**实战案例**：
- 体弱但财官旺 → 为财官所累（满汉全席半碗饭）
- 体旺但财官弱 → 平平无奇（能力在落幕行业）

### ⚠️ 体用定义的精确拆分（2026-07-16 实战校准）

**旧定义（已废弃）**：体=日主+印+比劫，用=财+官+食伤
→ 把食神（体之工具）误归入「用」，导致体用失衡误判

**修正定义（当前标准）**：
```
体 = 日主 + 印 + 比劫 + 食神
     （我拥有的资源和工具——包括思维才华）
用 = 财 + 官 + 伤官
     （我追求的目标——地位、财富、权力）
```

**逻辑原理**：
| 十神 | 归属 | 理由 |
|:-----|:-----|:------|
| **食神** | 体 ✅ | 食神是日主自带的才华/福气/思维工具——是「我有什么」，不是「我要什么」 |
| **伤官** | 用 ✅ | 伤官是叛逆/斗争/改革——是「我要去征服/改变什么」，是追求目标 |

**记忆口诀**：
```
食神在体伤官用，思维才华是工具
追求目标看财官，加上伤官算全篇
食伤不能混一团，食神为体伤官用
```

**实战验证②——子源（丙火中和偏强·双巳伏吟）：** (2026-07-16 新增)

```
子源八字：辛卯 癸巳 丙戌 癸巳
日主：丙火 · 身强弱55.6分（中和偏强）

旧定义（体=日主+印+比劫，用=财+官+食伤）：
  体 = 比劫55.6分 + 印4分 = 59.6分
  用 = 财30.8分 + 官24分 + 食神43.2分 = 98分
  → 体弱用强 → 「能力撑不起野心」
  → 取用策略：扶体（补印比）
  → 结论❌：子源被诊断为「被压垮型」

修正定义（体=日主+印+比劫+食神，用=财+官+伤官）：
  体 = 比劫55.6分 + 印4分 + 食神43.2分 = 102.8分
  用 = 财30.8分 + 官24分 = 54.8分
  → 体强用弱 → 「大马拉小车——能力远超平台」
  → 取用策略：扶用（补财官）
  → 结论✅：子源是「怀才不遇型」，大运补财官才爆发

差异：
| | 旧定义 | 修正定义 |
|:-|:-------|:---------|
| 体 | 59.6（弱）| 102.8（强）|
| 用 | 98（旺）| 54.8（弱）|
| 诊断 | 被压垮型 ❌ | 怀才不遇型 ✅ |
| 策略 | 补印比 → 误 | 补财官 → 对 |
| 体用比 | 0.6:1 | 1.88:1 |

关键：食神（43.2分）移入体后，体由59.6→102.8，
用由98→54.8，体用关系完全翻转。
揭示了子源前半生「怀才不遇」的本质原因。
```

## 二、体用三种情况完整分类（取用神框架）

### 类型A：用克体（伤害）→ 去病

```yaml
本质：官杀攻身、财来压身 → "一辈子为财官所累"
表现：
  ├─ 官杀克禄 → 忙忙碌碌、没福气、收入降低
  ├─ 财坏印 → 学而无用、名声受损、事业下坡
  └─ 官杀克比劫 → 打架斗殴、两院之灾

取用策略：去病——压制或化掉克体的东西
  先理清楚八字最大的问题在哪儿，然后压制或化掉它

正面转变（特殊）：
  ├─ 印多了财坏印 → 可转变为「制印得权」
  └─ 比劫多了官杀克体 → 可转变为「官制群比，因祸得福」
```

### 类型B：体得用（得到）→ 平衡扶弱

```yaml
本质：用印化官杀、食伤制官杀、比劫克财 → 得到财官
核心：双方力量的平衡

三种情况：
  ├─ 体强用旺 → 体用平衡，大富大贵 ✅
  ├─ 体强用弱 → 体强能担，但财官机会少 → 补用
  └─ 体弱用旺 → 为财官所累（满汉全席半碗饭）→ 扶体

取用策略：
  尽量把弱的扶起来，而不去伤害本来就很强的东西
  不能"切胃"（不能削弱已经很弱的东西）
```

### 类型C：体与体打架（左右互搏）→ 保食伤

```yaml
本质：我自己的体之间互相打仗 → "脑子糊涂了"
经典组合：
  ├─ ⭐ 枭神夺食（最经典）
  └─ 印把食伤合住了（不是夺食，但也是大问题）

取用策略：优先保住食伤而非印！
  "食伤代表着我们的快乐、我们的技术，同时也代表着我们的思维能力"
  "而印是护身之物。我们可以这样想：
   在人生路上，我们可以不受他人扶持，也可以不被保护。
   但我们一定不能丢掉自己的立身之本，丢掉为数不多的快乐。"

消声夺食时：一定要取财来制印得全，不能用印来搞食伤
```

## 三、体受伤=灾的完整十神对应表

| 被制的体 | 应灾方向 | 具体表现 |
|:---------|:---------|:---------|
| **印被制死** | 头脑/保护 | 唯一印被坏→牢狱之灾；唯一思维被制死→脑瘫/植物人 |
| **食伤被制死** | 感官/生殖 | 不育（男）；感官系统出问题；财源被毁→感情不顺 |
| **比劫被制死** | 身体/手脚 | 手脚出问题 |
| **禄被破坏** | 身体+福气 | 第一身体受伤，第二没福气了→**一辈子受穷** |
| **阳刃被穿** | 手术/受伤 | 一刀之罪，受伤或手术；阳刃要治不治不能成才，治过头又是有灾 |

## 四、阳刃与食伤的处理铁律

### 阳刃要治但不可穿
- 最理想状态：**官来治**（不管治阳刃还是阳刃合官都可以）
- ❌ 不可穿！（穿阳刃一刀之罪）
- **食伤是万万不可以受治的**——治了食伤都没有好结果
- 除非食伤多到不行了不得不治，但那也是极少情况

## 五、阴阳制衡规则

| 阴阳属性 | 涵盖 |
|:---------|:-----|
| **阳** | 木火、戌土 |
| **阴** | 金水、辰丑土 |

**核心规则**：
当命局全是阳，只有一个或两个阴（且距离很远）；或全是阴，只有一个或两个阳（且距离很远）：
→ 这个孤立无援的阴/阳就是**命局制衡点**，**不能被冲、不能被穿**

## 六、化官杀生印三种情况 + 桥梁规则

### 三种情况

| 情况 | 官印比例 | 取用策略 |
|:-----|:---------|:---------|
| ①官印相当 | 官杀力量≈印力量 | 来官/来印都可取用 → 官来单位变大、印来权力变大 |
| ②官旺印弱 | 官特别旺、印特别弱 | 化不掉→取印为用、或取制官杀之心（财）为用；**财官皆为忌** |
| ③印多官弱 | 印太多、官太弱 | 取官杀为用、取财为用（让官杀变强，源源不断） |

### 桥梁规则
化官生印要看印能否生日主/禄：
- 能 → 力量被日主获取 ✅
- 不能 → 需要**桥梁（禄）**来承接这份力量，取禄为用
- 如果实在取不到能生到我身上的，也没关系——只要印能化掉官杀，即使我无法得到，也代表问题能解决

## 七、禄的完整规则

| 规则 | 内容 |
|:-----|:------|
| **禄见财官** | 禄最好不要见财官，见了就是**劳碌命** |
| **禄生食伤** | 禄一旦生了食伤，食伤必须做功（生财或制官），否则"白给没福气"，满脑子胡思乱想 |
| **有禄取印** | 印路相随=贵气；原局已有印路相随时，以食伤为忌 |
| **官杀克禄** | 忙忙碌碌，收入降低，没有福气 |

## 八、大运流年规则（泉师兄补充）

### 大运提前进气
大运十年干支各管五年，但需看是否**提前进气**——如果提前进气，下一个天干地支提前接管。

### 大运 vs 流年定位
| | 大运 | 流年 |
|:--|:-----|:-----|
| 性质 | 运气=大盘（锦上添花/雪上加霜） | 具体事件=个股 |
| 触发事件 | 不直接触发 | 触发具体事件 |
| 力量 | 人力难以抗衡 | 需区分来源 |

### 流年力量判定
- 流年字在原局中**有** → 力量等同于原局（如原局巳在月上，流年巳=月令力量）
- 流年字在原局中**没有** → 力量大于原局每一个字

### 大运真假辨别
**假大运**：大运某个字从未在原局中出现过 → 警惕！
- 表现：看似希望、实际失望
- 例：来印生你但印是假的=骗局；项目是真的但你抓不住（没能力/没资金）

**辨别三法**：
1. **寻根基**：天干找地支、地支找库
2. **鸠占鹊巢**：原局已有一个（如丙火），又来一个（如丁火）→ 丁火未必是真的
3. **生也是关系**：如午火来了，原局有寅木能生午→也算有关系但微妙（不算假，但比不上原局有的）

**特殊情况**：原局是上身财（辛金生亥水），大运来了个申金
→ 申金就是假的！原局已经告诉你是辛金生亥水了，现在换个方式（申金生亥水）但你不是干这个的

### 大运流年作用优先级
```
第一步：优先寻找三合/三会/六合/五合等合化组合（"熟人打招呼"）
第二步：无合化时，优先选择有深度关系的
第三步：最后才该克就克、该冲就冲
```

### 看大运流年的步骤
1. 先明确**原局的做工路线**
2. 思考这部大运/这个流年的到来**优先作用于哪条路线**
3. 判断是否**打破了原局的某个平衡**
