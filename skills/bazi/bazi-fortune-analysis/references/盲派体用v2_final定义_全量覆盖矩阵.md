## 一、一句话定义

**体=日主+印+比劫+食伤(全部)，用=财+官(唯二)**

- 食神+伤官 **全算体**——思维、才华、技术、创造力、抗争力都是日主自带的工具
- 用只有 **四个字**：正财、偏财、正官、七杀
- 除此之外的所有十神（食神/伤官/正印/偏印/比肩/劫财/日主本人）**全算体**

## 四、区分逻辑：为什么食伤是体不是用

| 十神 | 归属 | 核心理由 |
|:-----|:-----|:---------|
| **食神** | 体 ✅ | 食神是日主自带的才华/福气/思维——是「我有什么」，不是「我要什么」 |
| **伤官** | 体 ✅ | 伤官同样是日主泄秀产物——技术、创作、抗争都是日主的工具 |
| **财星** | 用 ✅ | 财富是人生的追求目标——「我要有钱」 |
| **官星** | 用 ✅ | 地位权力是人生的追求目标——「我要有地位」 |

**泉师兄视角**：食伤代表我们的快乐、我们的技术、我们的思维能力。它是我们撬动财官的工具，不是财官本身。就好比厨师的刀工（食伤）是工具，赚钱（财用）是目标——刀工再好，不做菜也赚不了钱。

## 五、体用 vs 身强弱的关系（核心区分）

> **⚠️ 泉师兄原话**：「这跟身强身弱没有半毛钱关系。」

| 概念 | 关注什么 | 判断什么 |
|:-----|:---------|:---------|
| **身强身弱** | 日主承载能力（有根无根、得不得生扶） | 扛不扛得住财官 |
| **体用旺弱** | 体（全部工具）能否撬动用（财官） | 能不能得到财官 |

**体用三种情况：**
- **A型（用克体）**：官杀攻身、财坏印 — 需去病
- **B型（体得用）**：体去追求得到用 — 需平衡
- **C型（体体打架）**：枭神夺食等 — 需保食伤

## 六、体用量化方法

```
体分 = 所有位置的 印 + 比劫 + 食伤（食神+伤官） 之总和
用分 = 所有位置的 财 + 官（正官+七杀） 之总和

位置基础分：年干8分 年支4分 月干12分 月令40分 日支12分 时干12分 时支12分
藏干百分比：本气100% 中气60% 余气30%
天干不分藏干，直接按位置分
（不扣分，不加不减，与身强弱同逻辑）

体用比例判定：
  >1.2:1 → 体强用弱 → 大马拉小车 → 扶用（补财官大运）
  0.8~1.2:1 → 体用平衡 → 能力与平台匹配 → 顺其自然
  <0.8:1 → 体弱用旺 → 为财官所累 → 扶体（补食伤/印比大运）
```

## 七、体受伤检查表

| 被制的体 | 应灾方向 | 具体表现 |
|:---------|:---------|:---------|
| **印被制死** | 头脑/保护 | 唯一印被坏→牢狱之灾；唯一思维被制死→脑瘫/植物人 |
| **食神被制死** | 感官/生殖/快乐 | 不育（男）；财源被毁；快乐被夺→抑郁 |
| **伤官被制死** | 技术/创作/表达 | 技术水平丧失、创作停滞→思维受阻 |
| **比劫被制死** | 身体/手脚 | 手脚出问题；朋友同辈受伤 |
| **禄被破坏** | 身体+福气 | 第一身体受伤，第二没福气了→**一辈子受穷** |
| **阳刃被穿** | 手术/受伤 | 一刀之罪，受伤或手术 |
