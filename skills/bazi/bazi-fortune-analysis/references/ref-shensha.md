## 1. 吉神类

```
天德贵人 / 月德贵人（最好吉神之一）：
  ├─ 查法：以月令查其他干支（排盘软件自动列出）
  ├─ 作用：得天佑神助，遇难呈祥、化险为夷
  ├─ 断事：八字带此神煞 → 一生无大灾祸、官刑不至
  │   ├─ 在配偶宫 → 配偶心地善良
  │   └─ 宫位为喜用 → 该年龄段平安顺遂
  └─ 百无禁忌，逢凶化吉

天乙贵人（最难、贵人相助）：
  ├─ 查法口诀：甲戊兼牛羊，乙己鼠猴乡，丙丁猪鸡位，壬癸蛇兔藏，庚辛逢马虎
  ├─ 查法表：
  │   年干/日元 | 天乙贵人（地支） | 对应生肖
  │   甲/戊     | 丑、未           | 牛、羊
  │   乙/己     | 子、申           | 鼠、猴
  │   丙/丁     | 亥、酉           | 猪、鸡
  │   壬/癸     | 巳、卯           | 蛇、兔
  │   庚/辛     | 午、寅           | 马、虎
  ├─ ⚠️ 额外：龙（辰）和狗（戌）是所有人的贵人，口诀中虽无但包含贵人能量
  ├─ 作用：①解灾解难（天乙一到百恶俱散）②有贵人相助 ③做事容易成功
  ├─ 断事：
  │   ├─ 在年柱 → 小学初中时易取得第一名成绩
  │   ├─ 流年遇天乙 → 该年易得贵人助
  │   └─ 可判断贵人属相，与贵人搞好关系
  └─ 可解灾煞、劫煞等凶神

文昌贵人（主学习）：
  ├─ 查法：以日元查（排盘软件自动列出）
  ├─ 作用：气质雅秀、举止温文、好学上进、有助于学习
  ├─ 断事：
  │   ├─ 在年柱 → 少年学习好
  │   ├─ 在月令 → 能量最大，学历多较高
  │   ├─ 在日柱 → 中年开始学习
  │   ├─ 在时柱 → 晚年学习
  │   ├─ 在配偶宫 → 配偶学历较高
  │   └─ 流年遇文昌 → 当年易学新技能

太极贵人（主玄学）：
  ├─ 查法：以年干和日元查（排盘软件自动列出）
  ├─ 作用：代表会遇到懂周易的人，聪明好学、好奇心重、喜欢神秘事物
  ├─ 断事：
  │   ├─ 在年柱 → 少年好奇心重
  │   ├─ 在日柱 → 中年学习周易
  │   ├─ 在时柱 → 晚年学周易并可能成为命理师
  │   └─ 天乙+太极同柱 → 利于学周易

天医（主医术）：
  ├─ 查法：排盘软件自动列出
  ├─ 作用：喜欢周易和中医
  └─ 断事：在年支 → 家中亲戚有人懂中医或周易
```

## 2. 中性神煞

```
华盖（半吉半凶）：
  ├─ 查法：以年支和日支查，寅午戌见戌、申子辰见辰、亥卯未见未、巳酉丑见丑
  ├─ 作用：艺术细胞、性格孤独、爱整洁、与佛道有缘
  ├─ 断事：
  │   ├─ 丑多 → 佛教徒倾向
  │   ├─ 未多 → 道教徒倾向
  │   ├─ 辰戌多 → 散仙
  │   └─ 说话技巧：建议说"有宗教缘分"而非直接判定教派

桃花（子卯午酉）：
  ├─ 查法：以年支和日支查，子卯午酉为桃花
  ├─ 作用：招人喜欢；过多则风流花心
  ├─ 断事：
  │   ├─ 在年柱 → 小时候招人喜欢
  │   ├─ 在中年宫位 → 可能有婚外情
  │   ├─ 合入夫妻宫 → 配偶有魅力
  │   └─ 可做法事招桃花或斩桃花
  └─ 桃花过多为桃花劫/桃花刑 → 带来负面影响

驿马（寅申巳亥）：
  ├─ 查法：以年支和日支查，寅申巳亥为驿马
  ├─ 作用：代表交通工具，奔波、出门、变动
  ├─ 断事：
  │   ├─ 流年遇驿马 → 可能买车、频繁出差、搬家
  │   ├─ 驿马逢冲 → 车祸信号
  │   ├─ 月日时柱有驿马 → 一生奔波
  │   └─ 预测车辆销售：驿马流年（2019亥、2022寅）为高峰
```

## 3. 凶神类

```
劫煞（凶神）：
  ├─ 查法：以年支和日支查，申子辰见巳、亥卯未见申、寅午戌见亥、巳酉丑见寅
  ├─ 作用：是非破财、病刑伤、东西被偷被抢
  ├─ 断事：
  │   ├─ 八字带劫煞 → 人勇敢不怕死
  │   ├─ 身强+劫煞 → 可从事武职（军警等）
  │   ├─ 两个以上劫煞 → 有暴力倾向、易犯罪
  │   └─ 家中易被盗，需管好子女
  └─ 化解：天乙/天德/月德贵人可解

灾煞（最凶神煞之一）：
  ├─ 查法：以年支查，申子辰见午、亥卯未见酉、寅午戌见子、巳酉丑见卯
  ├─ 作用：厄运缠身，主疾病、官司、牢狱、破财、血光、横死
  ├─ 五行属性断事：
  │   ├─ 灾煞在土 → 注意踏空陷落、高空坠物
  │   ├─ 灾煞在火 → 注意火灾
  │   ├─ 灾煞在水 → 注意水灾
  │   ├─ 灾煞在木 → 注意风灾、雷灾
  │   └─ 灾煞在金 → 注意利刃所伤、车祸
  └─ 化解：吉星化解+行善积德+初一十五放生+皈依佛道

血刃（血光之灾）：
  ├─ 查法：以日元查，甲见卯、乙见辰、丙见午、丁见未、戊见午、己见未、庚见酉、辛见戌、壬见子、癸见丑
  ├─ 作用：血光之灾，意外开刀住院，轻者皮肉伤重者生死意外
  ├─ 断事：
  │   ├─ 女命带血刃 → 易难产
  │   ├─ 阳刃+血刃+驿马同柱 → 易因车祸流血
  │   ├─ 日元身强+血刃 → 易意外流血
  │   ├─ 日元身弱+血刃 → 因病开刀见血光
  │   └─ 身强逢血刃 → 易出工伤/外伤
  └─ 需注意尖锐物品，流年加重时要提前防范
```

## 4. 其他神煞速查

```
丧门/披麻/吊客 → 凶，走到该年参加白事（邻居/亲戚有人去世）
国印 → 搬家之象（换宿舍/搬办公室/搬家）
红鸾 → 暗恋
亡神 → 梦到去世亲人托梦
天厨 → 吃货，喜欢吃不喜做，整天找好吃的下馆子
学堂 → 站在讲台上讲课/授课的天分

```

## 5. 神煞断事总原则

```
1. 神煞为辅，十神为主——不可只凭神煞下结论
2. 吉神所在宫位为喜用 → 福力加倍
3. 凶神所在宫位为忌凶 → 灾祸显现
4. 吉神可解凶神：天乙/天德/月德可化解灾煞/劫煞/血刃
5. 年柱无神煞是好事情（不代表祖上无根基）
6. 空亡的神煞：好神煞空亡福力减，坏神煞空亡凶灾解
7. 流年神煞比原局神煞应事更快、更明显
```

## 6. 断事表达原则

```
⚠️ 核心原则：分析对了≠表达对了

用包容性语言代替绝对性断语：

  错误表达                          正确表达
  ├─ "你跟老公吵架"              → "你今年婚姻不顺"
  ├─ "你是道家弟子"              → "你跟宗教有缘分"
  ├─ "你老婆学历高"              → "你配偶家人中有学历高的"
  ├─ "你今年离婚"                → "你今年的伴侣关系需要注意"

为什么要用包容性语言？
  ├─ 夫妻宫受冲：可能是吵架/分居/离婚/打架 → "婚姻不顺"全涵盖
  ├─ 华盖：可能是佛教/道教/散仙/基督 → "有宗教缘分"100%对
  ├─ 用包容性语言 → 分析准了表达也对 → 客户服气
  └─ 用绝对性语言 → 分析对了但细节错 → 客户说"不对" → 信任崩塌
```
