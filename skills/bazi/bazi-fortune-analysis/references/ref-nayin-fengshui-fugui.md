### 1.2 纳音断事决策树

```
第一步：看年柱纳音（代表命主的根基、祖业传承）
  ├─ 年柱纳音为金 → 祖上有积累、根基扎实

  ├─ 年柱纳音为水 → 祖上漂泊、流动性大

  └─ 年柱纳音为土 → 祖上靠土地/实业起家

第二步：看日柱纳音（代表命主自身的气质）
  ├─ 海中金 → 深藏不露、大器晚成

  ├─ 城头土 → 有权谋、适合管理
  ├─ 白蜡金 → 艺术气质、易受伤

第三步：纳音组合判断
  ├─ 年柱纳音生日柱纳音 → 祖业旺、命主享福

  └─ 日柱纳音生日柱纳音 → 自身气运好

第四步：纳音与正五行的关系

  ├─ 纳音为忌神的五行 → 命主有该五行的弱点
  ├─ 纳音正五行相同 → 表里如一
  └─ 纳音正五行不同 → 表里不一、深沉
```

## 2. 八字风水/居家布局断法

### 2.1 从八字看风水的基本原则

```
八字中的干支可以解读为命主出生地/居住地的环境特征。
这是金鉴真人"过三关"的经典技巧——只看八字就能说对居住环境。

天干看远处、大环境 → 出生地周围的大地貌
地支看近处、小环境 → 出生地周边的具体景物

方向参考：
  年干方向 → 远方
  年支方向 → 近处
  月柱 → 前面的环境
  日柱 → 居住地本身
  时柱 → 后面的环境
```

### 2.3 家中物品摆放直断

```
八字中缺失某个五行 → 家中宜摆放该五行的物品来补
八字中某个五行过旺 → 家中宜减少该五行的物品

例：八字缺水 → 家中宜放鱼缸、流水摆件、黑色装饰
例：八字火过旺 → 家中减少红色、电器的使用

具体对应：
  ├─ 木缺 → 放绿植、挂画有树林
  ├─ 火缺 → 多开灯、放红蜡烛
  ├─ 土缺 → 放陶瓷、石头摆件
  ├─ 金缺 → 放金属饰品、铜钱
  ├─ 水缺 → 放鱼缸、黑色装饰

每个宫位代表家中不同位置：
  ├─ 年柱 → 家中大门方向/客厅
  ├─ 月柱 → 厨房/餐厅
  ├─ 日柱 → 主卧/起居室
  └─ 时柱 → 阳台/儿童房

宫位受冲 → 该区域易有破损/装修变动
宫位合用神 → 该区域放喜用五行的物品 → 旺运
```

## 3. 富贵贫贱判定细化标准

### 3.1 三层次判定法

```
第一层：看格（格局清浊）
  ├─ 清格 → 格局纯粹（单一某个十神主导）
  │   正官格无七杀混杂 = 清，有七杀混杂 = 浊
  │   正财格无偏财混杂 = 清，有偏财混杂 = 浊
  │   清格 → 成就高、人生顺利
  ├─ 浊格 → 格局混乱（多个十神交锋）
  │   官杀混杂、财才混杂、伤官+正官
  │   浊格 → 人生多波折、大起大落
  └─ 变格 → 从格/化气格
      从旺/从弱 → 大成就方向单一，但易走极端

第二层：看局（用神得力程度）
  用神分为四等：
  ├─ 上等用神：天干透出 + 地支有真根 + 无刑冲克破
  │   → 格局高，富贵可期
  ├─ 中等用神：天干透出 + 地支有劫财根/余气根
  │   → 格局中上，努力可成
  ├─ 下等用神：天干透出但虚浮无根
  │   → 格局一般，有大运补根才发
  └─ 无格用神：用神不现/被克/被合化
      → 格局低微，一生平凡

第三层：看运（好运占比）
  一生大运（8步大运 × 10年 = 80年）：
  ├─ 喜用大运占 60%+ → 命好运好 → 上等
  ├─ 喜用大运占 40-60% → 命好运气一般 → 中等
  ├─ 喜用大运占 20-40% → 命一般运气一般 → 中下
  └─ 喜用大运占 20%- → 命差运差 → 下等
```

### 3.2 10级命格划分（金鉴真人细化版）

```
1级 帝王之格（9.5-10/10）：特殊格局+用神得力+好运占80%+
     例：杀印相生+调候得力+三步好运以上

2级 将相之格（8.5-9.4/10）：清格局+用神上等+好运占70%+
     例：官印相生+财官有根+大运一路顺

3级 封疆之格（8.0-8.4/10）：清格局+用神中上+好运占60%+
     例：格局清纯+财官得力+中年大运好

4级 厅局之格（7.5-7.9/10）：中清格局+用神得力+好运占50%+
     例：财官为喜+有根+运程平稳

5级 处县之格（7.0-7.4/10）：普通格局+用神有气+好运占40%
     例：正财正官格+用神有根

6级 科股之格（6.0-6.9/10）：普通格局+用神中等+好运占30%
     例：正格但用神非上等

7级 温饱之格（5.0-5.9/10）：格局偏浊+用神弱+好运少
     例：官杀混杂+身弱+运不助

8级 辛苦之格（4.0-4.9/10）：浊格局+用神无力+好运极少
     例：满盘忌神+偶尔好运

9级 飘零之格（3.0-3.9/10）：格局极差+终身无好大运
     例：从格破格+终身运背

10级 困厄之格（<3.0/10）：三刑全+无调候+终身运背
     例：满盘忌神+三刑+终身凶运
```

### 3.3 定命格的6+2综合评分法

```
六大维度（每维度0-10分）：

1. 五行平衡（权重20%）
   → 五行齐全且均衡 = 10分；严重偏枯 = 0分

2. 燥湿平衡（权重10%）
   → 不燥不湿 = 10分；纯燥/纯湿 = 0分

3. 阴阳平衡（权重10%）
   → 干支阴阳各半 = 10分；纯阴/纯阳 = 0分

4. 用神质量（权重25%）
   → 上等用神 = 10分；中等 = 5分；下等/无 = 0分

5. 好运占比（权重20%）
   → 80%+ = 10分；60% = 7分；40% = 4分；20% = 1分

6. 格局清浊（权重15%）
   → 清格 = 10分；中清 = 6分；浊格 = 2分

两个加分项：
  7. 调候用神到位（+0-1分）
  8. 有特殊贵人（天乙/天德/月德得力）+（+0-1分）

命格总分 = (六大维度加权和/10 × 9) + 调候加分 + 特殊加分

示例：
  家主魏启令 V5评分：
  ① 五行平衡 4/10 × 20% = 0.8
  ② 燥湿平衡 3/10 × 10% = 0.3
  ③ 阴阳平衡 5/10 × 10% = 0.5
  ④ 用神质量 5/10 × 25% = 1.25
  ⑤ 好运占比 6/10 × 20% = 1.2
  ⑥ 格局清浊 6/10 × 15% = 0.9
  小计 = 4.95 → /10 × 9 = 4.455
  调候加分：辛金调候喜壬水润局，年支申中壬水余气微弱 = +0.3
  特殊加分：无 = +0
  命格总分 = 4.455 + 0.3 = 4.755 → 约4.8/10
  → 对应7级"温饱之格"→ 但大运戊子/己丑为最佳用神运
     → 实际走向可达5级"处县之格"
```
