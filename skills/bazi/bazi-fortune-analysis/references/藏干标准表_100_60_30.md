## 重点记忆（两次致命踩坑）

### 巳 — 庚是余气30%（不是余气60%）

```yaml
顺序逻辑：丙(火) → 生 → 戊(土) → 生 → 庚(金)
  火生土 → 戊是中气60%（被火生，位置第二）
  土生金 → 庚是余气30%（被土生，但火克金→最弱）
→ 所以庚是余气30%，不是中气60%
```

### 戌 — 辛是中气60%（不是余气30%）

```yaml
顺序逻辑：戊(土) → 生 → 辛(金) → 被克 → 丁(火)
  土生金 → 辛是中气60%（被土生，位置第二）
  火炼金 → 丁是余气30%（火在戌中受土控制，最弱）
→ 所以辛是中气60%，丁才是余气30%
```

## 推导逻辑（10秒心算）

```
从本气出发：
  本气 → 生 → 中气（被本气生，占位第二）
  本气 → 克 → 余气（被本气克，占位最末）

巳：丙(火)→生土(中)→土生金(余)    → 庚是余气
戌：戊(土)→生金(中)→金被火炼(余)  → 辛是中气，丁是余气
```
