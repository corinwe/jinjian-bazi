# 2026-06-14 三例实战校准（金鉴真人）

> 来源：1999男/1975女/静 三份报告的反馈校准 + 老板家族五人报告修正

## 校准一：食神运≠事业上升期

**踩坑：** 1999男（己卯戊辰甲寅甲子）丙寅运（18-28岁），我判"食伤生财事业上升"，实际"没啥事业"。

**根因：** 食神运的特性被忽略——食神主安逸、享受、吃喝玩乐、不愿拼搏。与偏财格的"想赚大钱"形成矛盾。

**固化规则：**
```
断事业必查当前大运十神类型：
  ✅ 伤官运/七杀运 = 拼搏、奋斗、事业上升期
  ✅ 正官运 = 稳定、守成
  ⚠️ 食神运 = 安逸、享受、事业无作为是正常的
  ⚠️ 偏印运 = 思考多行动少

口诀：伤官七杀拼事业，食神偏印等时机
```

## 校准二：正印格≠学历高

**踩坑：** 1975女（乙卯丙戌辛卯壬辰），正印格+官印相生，我判"本科"，实际"没啥学历"。

**根因：** 只看到正印格就判学历高，没检查文昌（辛→子全局无子❌）和财坏印（双卯木耗身）。

**固化规则：**
```
正印格→学历高的充分条件：
  ① 文昌到位（查日主文昌）
  ② 无财坏印（八字财星不旺或不为忌）
  ③ 印星有根不被合化
  ④ 身不太弱（能坐得住学习）

缺以上任一条件，正印格也可能学历低。
初判"本科"前先过这四关。
```

## 校准三：学习力≠学历·四维评分

**踩坑：** 家主（偏印格+食神透）和子源（身强+双正官）都被判学业"一般"，实际两人学习力超强、学有余力。

**根因：** 用印星文昌判一切，忽略了"学得快"和"读得久"是两回事。

**固化规则：** 详见§7.8A.6 学习力双轨模型。

```
学习力四维：偏印格局力(35%)+食伤聪明力(25%)+身强精力(20%)+正官自律力(20%)
学历三要素：印星+文昌+大运配合
```

## 校准四：财富兑现≠格局兑现

**1999男案例：** 偏财格+身强财旺格局好，但丙寅食神运拖后腿→财富要推迟到40岁后甲子运才兑现。

**固化规则：**
```
格局好 ≠ 当前发财
格局兑现 = 格局 × 大运匹配 × 时机

偏财格+食神运 = 想赚钱但不急动（晚发）
偏财格+七杀运 = 敢拼敢干，快速兑现
偏财格+比肩运 = 合伙发财
```
