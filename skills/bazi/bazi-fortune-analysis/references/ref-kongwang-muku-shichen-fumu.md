## 1. 空亡详解

### 1.1 空亡判定

```
空亡以日柱为主，兼看年柱断事：
  甲子旬 → 戌亥空    甲寅旬 → 子丑空
  甲辰旬 → 寅卯空    甲午旬 → 辰巳空
  甲申旬 → 午未空    甲戌旬 → 申酉空
```

### 1.2 各柱空亡断事

```
年柱空亡：祖上无靠，与母早别离，少年辛苦
月柱空亡：父母兄弟无靠，青年学业事业多阻
日柱空亡：夫妻关系冷淡，婚姻不顺
时柱空亡：子女缘薄，晚年孤寂
```

### 1.3 纳音空亡规则

```
金空则鸣 → 金落空亡 → 吉，命主有名气
火空则发 → 火落空亡 → 吉，上进心强事业有成
水空则清 → 水落空亡 → 吉，命运飘荡
木空则折 → 木落空亡 → ❌凶，四肢易受伤
土空则崩 → 土落空亡 → ❌凶，易患慢性病
```

### 1.4 空亡填实规则

```
空亡不代表字消失，只是力量减半：
  ├─ 空亡地支继续参与刑冲破害
  ├─ 喜用空亡 → 力量减少，不吉
  ├─ 忌凶空亡 → 力量减少，好事
  ├─ 大运流年填实 → 力量恢复
  └─ 墓空亡 → 坏事不显；库空亡 → 好事减轻
```

## 2. 墓库详解

### 2.1 库 vs 墓的区别

```
同一墓库，旺为库（仓库），弱为墓（坟墓）：

库 = 该五行旺相 → 能量加倍 → 财库/官库 → 好
墓 = 该五行衰弱 → 能量归零 → 财墓/官墓 → 坏
```

### 2.2 各日主墓库对应表

| 日主 | 比劫库 | 财才库 | 官杀库 | 印枭库 | 食伤库 |
|:---:|:------:|:------:|:------:|:------:|:------:|
| 甲乙木 | 未 | 戌 | 丑 | 辰 | 戌 |
| 丙丁火 | 戌 | 丑 | 辰 | 未 | 戌 |
| 戊己土 | 戌 | 辰 | 未 | 戌 | 丑 |
| 庚辛金 | 丑 | 未 | 戌 | 戌 | 辰 |
| 壬癸水 | 辰 | 戌 | 戌 | 丑 | 未 |

### 2.3 墓库开合规则

```
1. 八字自带墓库的人，大运流年刑冲破害时开库
2. 年支月支带库 → 别人发财/升官
3. 日支时支带库 → 自己发财/升官
4. 墓库刑冲破害一次 → 打开，再刑冲破害 → 关闭
5. 八字不带墓库 → 大运流年遇库也有好事
6. 八字不带墓库 → 大运流年遇墓有灾祸
```

## 3. 月令被克制计算规则

```
月令是生助日元的，但被年干/年支/月干/日支所克时：
  被克制一次 → 还算得令
  被克制两次 → 能量变小，算通本气强根
  被克制三次 → 月令受伤，算通中气弱根
  被克制四次 → 月令只剩微根

同理适用于年支、日支、时支被周围干支克制的情况。
```

## 5. 父母寿命断法

```
父母寿命是八字分析中最难判断的部分之一，以下为金鉴真人校准体系：

第一步：看父母宫的健康信号
  年干=父宫，年支=母宫（按宫位六亲对应表）

  ├─ 父母宫受三刑 → 父母有重病缠身
  ├─ 父母宫受六冲 → 父母易有突发性疾病
  ├─ 父母宫带七杀无制 → 父母健康差
  └─ 父母宫带血刃/灾煞 → 父母有血光/意外可能

第二步：看父母的用神、忌神
  ├─ 父亲的八字比母亲的八字更关键
  ├─ 父母宫的五行是喜用 → 父母身体相对健康
  └─ 父母宫的五行是忌神 → 父母健康容易出问题

第三步：看大运流年对父母宫的冲击
  父宫被冲/害/刑 → 父有灾
  母宫被冲/害/刑 → 母有灾

  应灾年份判断：
  ├─ 父母宫的年柱/年支在流年出现伏吟 → 重点关注
  ├─ 流年天克地冲父母宫位 → 🚨 大灾信号
  ├─ 三刑+七杀同时冲击父母宫 → 🚨 非常危险
  └─ 大运流年父母宫天干地支同时伏吟 → 🚨 最严重的信号

第四步：限运法辅助判断
  子女的八字可以辅助判断父母的寿命趋势：
  ├─ 子女八字中父母宫强旺 → 父母长寿
  ├─ 子女八字中父母宫弱衰 → 父母寿命一般
  └─ 子女八字中父母宫被合化 → 需结合大运看应期
```

## 6. 天干落地检查·补充规则（金鉴真人核心）

```
天干落地检查的完整范式（用于所有八字分析）：

检查每个天干在地支是否有本气根：
  ├─ 甲→寅（本气）、卯（中气）、辰（余气乙）
  ├─ 乙→卯（本气）、寅（中气）、辰（中气乙）、未（余气乙）
  ├─ 丙→巳（本气）、午（中气）
  ├─ 丁→午（本气）、巳（中气）、未（余气丁）
  ├─ 戊→巳（本气）、午（本气）、辰（余气戊）、戌（本气）
  ├─ 己→未（本气）、午（中气）、辰（余气己）、戌（余气己）
  ├─ 庚→申（本气）、酉（中气）
  ├─ 辛→酉（本气）、申（中气庚）、戌（余气辛）、丑（余气辛）
  ├─ 壬→亥（本气）、子（中气）、申（中气壬）
  └─ 癸→子（本气）、亥（中气壬）、丑（余气癸）、辰（余气癸）

天干落地判定：
  ├─ 有本气根 → 力量100%，真实有效
  ├─ 有中气根 → 力量60%，半落地
  ├─ 有余气根 → 力量30%，微弱
  └─ 无根 → 虚浮天干，力量打3折

天干落地规则 → 直接影响用神判断：
  喜用神天干有根 → 真正可用
  喜用神天干虚浮 → 需要大运补根才发
  忌神天干有根 → 真正有害
  忌神天干虚浮 → 危害有限
```
