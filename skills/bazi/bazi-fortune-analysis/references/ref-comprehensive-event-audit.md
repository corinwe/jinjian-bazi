## 审计方法论总结

### 近N年审计 vs 全生命周期审计的区别

| 对比项 | 近N年审计 | 全生命周期(回顾性) | 
|:------|:---------|:------------------|
| **时间范围** | 指定时间段（如近10年） | 从出生到去世全部 |
| **数据精度** | 可精确到年、大运、流年机制 | 约数/范围（"约1970s"） |
| **事件类型** | 破财/病痛/买房/存款为主 | 全部+子女/成家/立业/去世 |
| **输出格式** | 表格式渐进展开 | 完整的档案式章节输出 |
| **难度** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
