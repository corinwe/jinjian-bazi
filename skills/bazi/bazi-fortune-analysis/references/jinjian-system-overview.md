# 金鉴真人八字核心方法论速查

> 完整理论文献见知识库：`07-国学哲学/八字命格/课程资料/`（理论体系全量版）

## 7大独特理论

| # | 理论 | 一句话核心 |
|:-:|:----|:-----------|
| 1 | **燥湿平衡** | 未戌=燥土脆金，辰丑=湿土生金。与五行平衡并列判断 |
| 2 | **子为阴水** | 与"子为阳水"的传统不同，金鉴体系认为子是阴水 |
| 3 | **一位为真** | 喜用神一位为真、二位为争、三位为病，非越多越好 |
| 4 | **十神吉恶平比例** | 吉(正官正印食神)、恶(七杀伤官劫财偏印)、平(正偏财比肩)。统计比例定性格 |
| 5 | **7种强弱状态** | 从强→偏强→身强→中和→身弱→偏弱→从弱。新增"偏强""偏弱" |
| 6 | **五行体用** | 体=本质属性，用=表现形式。体用结合判断真实影响力 |
| 7 | **八字方向** | 上南下北左东右西（与地图相反） |

## 分析流程强制步骤

1. **天干落地检查**（最核心）→ 每个天干查地支本气根。无根=虚浮
2. **燥湿平衡** → 未戌燥土脆金vs辰丑湿土生金
3. **十神吉恶平比例** → 吉神/恶神/平神各多少，比例定性格
4. **一位为真** → 用神一个就够，多了反而是"争"
5. **7种强弱定位** → 从强→偏强→身强→中和→身弱→偏弱→从弱
6. **特殊格局校验** → 炎上/润下/稼穑/从革必须有成局的地支
7. **合冲刑害+大运流年** → 动态推断

## 核心口诀

五行平衡是根本，燥湿阴阳两相参。
十神分作吉恶平，一位为真多则病。
身强食伤能生财，身弱印比来帮扶。
合冲刑害看动态，流年大运定吉凶。
