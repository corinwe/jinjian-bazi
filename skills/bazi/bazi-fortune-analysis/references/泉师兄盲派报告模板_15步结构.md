## 第3步 · 体用分析（泉师兄核心）

### 3.1 体用 vs 身强弱——核心区分
泉师兄原文：「这跟身强身弱没有半毛钱关系。身强身弱经营的是日主的承载能力，而承载能力并不能决定一个人的财富和地位。」

### 3.2 体用明细（2026-07-16勘误·段建业原典）

> **⚠️ 错误来源**：旧版写「用=财官食伤」，源自 `体用判定_用包含食伤.md` 方向全反。2026-07-16老板校准后已删除该错误文件。

**体（我自己及使用的工具）：** 日主 + 印 + 比劫 + **食神**（才华/技能/思维工具）
**用（人生追求的目标）：** 财 + 官 + **伤官**（成就/表达/作品）

```yaml
食神→体（内部工具）: 日主所生阴阳相同→内在才华·思维模式·手艺
伤官→用（外部目标）: 日主所生阴阳不同→进取·表达·追求·作品
```

### 3.2A 体用量化方法（子源案例提炼·2026-07-16新增）

**体用分数计算规则（与身强弱共用位置基础分体系）：**
```yaml
位置基础分：年干8分 年支4分 月干12分 月令40分 日支12分 时干12分 时支12分
藏干百分比：本气100% 中气60% 余气30%
天干不分藏干，直接按位置分

体分 = 所有位置的印 + 比劫 + 食神之总和
用分 = 所有位置的财 + 官 + 伤官之总和
（不扣分，不加不减，与身强弱同逻辑）
```

**体用比例判定标准：**
| 体用比 | 类型 | 实战含义 | 取用策略 |
|:-------|:-----|:---------|:---------|
| 0.8~1.2:1 | 体用平衡 ✅ | 能力与平台匹配 | 顺其自然 |
| >1.2:1 | 体强用弱 ⚠️ | 「大马拉小车」 | 扶用（补财官大运）|
| <0.8:1 | 体弱用旺 ⚠️ | 为财官所累 | 扶体（补印比大运）|

**口诀：** 体用平衡看比例，零点八到一点二；体强用弱扶财官，体弱用旺扶印比

### 3.3 体用三种情况归类（泉师兄分类法）
```
Type A: 用克体（伤害）→ 需「去病」
Type B: 体得用（得到）→ 需「平衡」扶弱方
Type C: 体与体打架（互搏）→ 需「保食伤」
```
