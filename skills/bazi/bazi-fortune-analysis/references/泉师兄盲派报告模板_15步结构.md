## 第3步 · 体用分析（泉师兄核心）

### 3.1 体用 vs 身强弱——核心区分
泉师兄原文：「这跟身强身弱没有半毛钱关系。身强身弱经营的是日主的承载能力，而承载能力并不能决定一个人的财富和地位。」

### 3.2 体用明细（2026-07-16勘误v2·段建业原典+魏启令v2实战校准）

> **⚠️ 2026-07-16 v2修正**：v1（本参考文件此前版本）误将伤官归入「用」中。此错误于魏启令v2报告实战中发现并纠正。正确盲派定义：体含食伤（食神+伤官全部算体），用只财官（正财偏财正官七杀只有这四个）。伤官和食神一样是日主才华的表达方式，是撬动财官的工具，不是追求的目标。

**体（我自己及使用的工具）：** 日主 + 印 + 比劫 + **食伤（食神+伤官）**
**用（人生追求的目标）：** 财 + 官（正财偏财正官七杀，只有这四个）

```yaml
体（所有工具性十神）：
  • 日主 — 自身
  • 印（正印/偏印）— 滋养/庇护/学习能力
  • 比劫（比肩/劫财）— 帮手/朋友/竞争
  • 食伤（食神/伤官）— 才华/技术/智慧/创造力

用（所有目标性十神）：
  • 财（正财/偏财）— 物质财富的追求
  • 官（正官/七杀）— 名声/地位/权力的追求

区分原则：体=撬动财官的工具，用=被撬动的财官本身
食神归体和伤官归体的原因相同——它们都是日主所生，都是工具，不是目标
```

### 3.2A 体用量化方法（子源案例提炼·2026-07-16新增）

**体用分数计算规则（与身强弱共用位置基础分体系）：**
```yaml

藏干百分比：本气100% 中气60% 余气30%
```

#### 5.4.1 基本定义

| 类型 | 位置 | 形式 | 盲派内涵 |
|:----|:-----|:-----|:---------|
| **家里做功** | 日支+时支 | 独立做事、自己当老板 | 能量内聚，自我驱动，内心世界 |
| **外面做功** | 年支+月干（含月令） | 打工、给别人做事、合作 | 能量外显，社会角色，外在表现 |

> 口诀：家里能求不求外，回归本心是真功

#### 5.4.2 家里核心配置分析（日柱+时柱）

看日支与时支的组合性质，决定家里的做功效率：

| 组合 | 效率 | 机制 | 实战表现 |
|:----|:----:|:-----|:---------|
| **自刑**（如午午自刑） | 🟢 中高 | 能量内聚不散逸，自我驱动强 | 自己卷自己，做功集中 |
| **六合/三合** | 🟢 高 | 内部组合好，合力做功 | 家庭/自身助力强 |
| **食伤在家**（时干食伤） | 🟡 中高 | 体之精华为自己所用 | 才华/智慧服务于自身目标 |
| **相冲**（如子午冲） | 🔴 低 | 能量内耗，内心矛盾 | 内部消耗大，做功分散 |
| **空亡/入墓** | ⚫ 极低 | 内部资源不可用 | 自身力量难以发挥 |

#### 5.4.4 家里家外比例判定

```
判定依据：
  家里能量 = 日支十神得分 + 时支十神得分（含藏干）
  家外能量 = 年支十神得分 + 月令十神得分（含藏干）+ 天干透出力量

比例类型及断语：
  ✅ 家里>家外（如60:40） → 自我驱动为主，内心世界主导
  ✅ 家里<家外（如40:60） → 社会角色为主，外在环境主导
  ✅ 家里≈家外（如50:50） → 内外均衡，表里如一

⚠️ 修正说明（实战易犯错误）：
  自刑（如午午自刑）能量过于集中 → 容易高估家里比例
  双官透干+外部财局 → 家外力量不容忽视，易被低估
  需综合天干透出和地支组合来判定，不能单看能量分
  最终归宿始终看家里做功（时柱=晚年归宿）
```

#### 5.4.5 家里家外交互关系

```
家里（自刑/食伤）→ 内聚能量 → 自我驱动
                 ↓
家外（透干/外局）→ 外显身份 → 社会角色
                 ↓
回家（时柱归宿）→ 晚年归宿 → 体之精华回馈自身
                 ↓
闭环：外面争得的身份地位，最终回到自己的人生中
```

**断语样板——家里家外统一型：**
```
此命内外一体：家里的热（午午自刑）要往外冒到社会（丁透），
社会上的身份地位又会收回到家里（时柱食伤为终点）。
不是那种「外面风光家里狼藉」的命，
而是「外面是什么，家里就是什么」的命。
内外一致、表里如一的格局。
```

### 9A.3 盲派库墓区分法则

```
核心法则：
  库：被冲开 → 财/官释放 → 得财/得官（好）
  墓：未被冲 → 财/官关闭 → 弱/存而不发（中性偏凶）

判定步骤：
  1. 原局有财库/官库吗？（在年/月/日/时）
  2. 若有，原局有冲开吗？（辰戌冲？丑未冲？）
  3. 无冲开 → 库=墓（关闭状态）
  4. 需大运流年来冲开 → 财/官释放

特别规则：
  ✅ 辰戌冲 → 水库火库同时开，能量最大（10倍土）
  ✅ 丑未冲 → 金库木库同时开
  ✅ 三合/三会也能激活财库（如亥卯未→激活未中乙木）
  ❌ 辰与辰/戌与戌：同字不冲，不开库
  ❌ 有合化优先 → 合化后不开库（贪合忘冲）
```

### 9A.4 财库激活的两种方式对比

| 方式 | 机制 | 能量 | 性质 |
|:-----|:-----|:----:|:-----|
| **三合/三会激活** | 引动库中藏干（如亥卯未激活未中乙木） | 80%激活 | 渐进来财，市场机会 |
| **冲开（丑未/辰戌）** | 直接冲开库门 | 100%释放 | 集中得财，库存释放 |

```
两种方式的本质区别：
  三合激活 = 气场生发（用气场激活）→ 温和渐进来财
  冲开库门 = 暴力破门（用刑冲破害）→ 集中爆发得财
  
  实战组合：
    【三合+冲开】= 双重激活 → 大富
    如：辛亥运三合（激活）+ 癸丑运丑未冲（破门）
    → 先渐进激活，再集中释放
```

### 9A.5 财库在体用框架下的定位

```
体（月令为印/比时）：
  月令 = 印（体），但印中藏财（用）
  例：未土=正印（体），未中乙木=正财（用）
  → 印中藏财 → 体（庇护/学习）中藏有用（财富）

体用透视：
  印星作为「体」→ 功能受限（如燥土不生金/不助水）
  印星作为「财库」→ 藏着用（财星）在内部
  → 大运冲开时：体无用有 → 用（财）释放 → 财富爆发
  → 体（印）功能受限但用（财库）潜力巨大
```
