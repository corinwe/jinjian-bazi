## 一、单柱分析框架（适用于年/月/日/时四柱）

每个柱独立分析时，必须回答以下6个问题：

### 1.1 六问清单

```
□ 问①：天干是什么十神？喜神还是忌神？
□ 问②：地支是什么十神？藏干有哪些？喜神还是忌神？
□ 问③：天干×地支构成什么组合？（四象限定位）
□ 问④：该柱代表什么人生阶段/领域？（柱位含义）
□ 问⑤：该柱的维度影响是什么？（事业/财富/健康/婚姻/子女）
□ 问⑥：空亡是否存在？对冲刑害关系如何？
```

### 1.2 四象限对抗分析法（核心）

天干vs地支的喜忌组合，决定了该柱的根本性质：

| 天干\地支 | 地支为喜神 | 地支为忌神 |
|:---------:|:----------:|:----------:|
| **天干为喜神** | 🏆 **最佳配置**<br>上下一致，内外皆吉<br>例：主母时柱壬午（食神用+正官喜） | ⚠️ **表面风光，根基有争**<br>有权位但内心不踏实<br>例：子源时柱癸巳（正官喜+比肩忌） |
| **天干为忌神** | 🔄 **表面竞争，根基是财**<br>问题在外，福气在内<br>例：老板时柱辛卯（比肩忌+偏财喜） | ❌ **最辛苦配置**<br>内外皆敌，需外力化解<br>例：立时柱庚午（七杀忌+伤官忌） |

**判断口诀**：
```
喜上喜下是格局，一生顺遂少波折
喜上忌下外风光，里子辛苦自己知
忌上喜下藏真金，表面纷争实有福
忌上忌下最艰难，印星化杀是救星
```

### 1.3 四柱各位置的核心含义

| 柱位 | 代表阶段 | 代表领域 | 核心判断方向 |
|:----|:---------|:---------|:------------|
| **年柱** | 0~16岁·少年 | 祖上、父母、出身、遗传 | 家庭背景、少年运势、先天体质 |
| **月柱** | 17~32岁·青年 | 父母、兄弟、学业、职场起步 | 青年运势、职场起点、学习能力 |
| **日支** | 33~48岁·中年 | 配偶、婚姻、中年事业 | 婚姻质量、配偶特征、中年运势 |
| **时柱** | 49岁+·晚年 | 子女、晚年、事业终点 | 最终成就、子女状况、退休状态 |

### 1.4 时柱五维分析（时柱专属深度）

每个时柱分析必须覆盖以下5个维度：

| 维度 | 核心问题 | 判断依据 |
|:-----|:---------|:---------|
| **① 事业终点** | 这个人最终靠什么立足？ | 时柱十神定方向<br>财=资本化、官=权力、食伤=才华、杀=技术权威 |
| **② 第二事业** | 下班后/退休后做什么？ | 时干十神定倾向<br>比劫=合作、食伤=输出、印=学习 |
| **③ 创作产出/IP** | 会留下什么无形资产？ | 食伤→技术/内容/IP<br>财→股权/资产<br>官→制度/规则 |
| **④ 晚年状态** | 晚年是安逸还是操劳？ | 时柱喜忌定基调<br>喜=安逸、忌=操劳<br>空亡填实=晚年爆发 |
| **⑤ 财富最终格局** | 财富量级落在哪里？ | 时柱财星/食伤=有财源<br>空亡=延迟爆发<br>比劫夺财=需防分财 |

## 二、柱间关系分析框架

### 2.1 柱间递进逻辑

```
年柱（出身）→ 月柱（打基础）→ 日支（成家立业）→ 时柱（结果）
```

四柱是一个人的人生剧本：
- 年柱写好了「出身条件」
- 月柱决定了「努力方向」
- 日支决定了「合作伙伴」
- 时柱给出了「最终答案」

### 2.2 柱间生克制化

| 关系 | 含义 | 判断方向 |
|:-----|:-----|:---------|
| **年柱生时柱** | 祖上荫庇，晚年享祖上余荫 | 年干生时干 |
| **时柱克年柱** | 晚辈超越长辈，但可能有冲突 | 时干克年干 |
| **月柱同日柱** | 婚姻对象像父母/兄弟姐妹 | 月干同日干 |
| **时柱冲月柱** | 晚年与青年时期的选择完全相反 | 地支相冲 |
| **年时伏吟** | 晚年回到原点，或子女重复自己的路 | 年柱时柱干支相同 |

### 2.3 官星双透（头尾呼应）

当 **月柱天干 = 时柱天干** 时，构成「官星双透，头尾呼应」：
- 月柱透官→青年入体制
- 时柱透官→晚年也在体制
- 一生都在一套体系中
- 例：子源月柱癸巳，时柱癸巳

### 2.4 财局被空亡（延迟爆发）

当年/月/日形成三合财局，但时支为空亡时：
- 40岁前：空亡引而不发，「有钱但差一步」
- 空亡被大运填实后：财富指数级增长
- 例：老板亥卯未三合木局（财局），时支卯为空亡→辛卯运填实

## 三、应用示例

### 3.1 年柱分析示例（以老板为例）

年柱庚申：天干庚=劫财忌神，地支申=劫财忌神
- 四象限定位：忌+忌 = 少年辛苦配置
- 结论：祖上竞争环境，早年资源需争夺
- 少年运：甲申运一喜一忌，有财星但被劫

### 3.2 月柱分析示例（以子源为例）

月柱癸巳：天干癸=正官喜神，地支巳=比肩忌神
- 四象限定位：喜+忌 = 表面风光内部有争
- 青年阶段有权力但根基不稳
- 官星双透+月柱时柱同为癸巳→头尾呼应

### 3.3 日支分析示例（以主母为例）

日支午火：丁火正官喜神+己土正印
- 从弱格中火为第一喜用→配偶是最大贵人
- 午火为夫星→婚姻是命运转折点

### 3.4 时柱分析完整示例（以老板为例）

时柱辛卯：天干辛=比肩忌神，地支卯=偏财喜神
- **四象限定位**：忌+喜 = 表面竞争，根基是财
- **①事业终点**：偏财在时柱→资本巨子/技术资本化
- **②第二事业**：比肩在时干→与朋友合作，但防分财
- **③创作产出**：食伤生财→专利/IP/技术体系
- **④晚年状态**：辛卯运填实空亡→愈老愈富
- **⑤财富格局**：空亡延迟→前40年差一步，后40年爆发

## 五、五鼠遁反推最佳时柱方法论

### 5.1 核心原理

> 时柱的**天干不是自由选择**——由**日干通过五鼠遁**决定。
> 反推最佳时辰的正确方法：给定日干 → 五鼠遁列出全部12个时柱 → 逐一四象限评估 → 挑出最优。

**五鼠遁口诀**：
```
甲己还加甲     乙庚丙作初     丙辛从戊起
丁壬庚子居     戊癸何方发     壬子是真途
```

### 5.2 12倍速查表（所有日干×所有时柱）

| 日干 | 子 | 丑 | 寅 | 卯 | 辰 | 巳 | 午 | 未 | 申 | 酉 | 戌 | 亥 |
|:----|:--|:--|:--|:--|:--|:--|:--|:--|:--|:--|:--|:--|
| **甲/己** | 甲 | 乙 | 丙 | 丁 | 戊 | 己 | 庚 | 辛 | 壬 | 癸 | 甲 | 乙 |
| **乙/庚** | 丙 | 丁 | 戊 | 己 | 庚 | 辛 | 壬 | 癸 | 甲 | 乙 | 丙 | 丁 |
| **丙/辛** | 戊 | 己 | 庚 | 辛 | 壬 | 癸 | 甲 | 乙 | 丙 | 丁 | 戊 | 己 |
| **丁/壬** | 庚 | 辛 | 壬 | 癸 | 甲 | 乙 | 丙 | 丁 | 戊 | 己 | 庚 | 辛 |
| **戊/癸** | 壬 | 癸 | 甲 | 乙 | 丙 | 丁 | 戊 | 己 | 庚 | 辛 | 壬 | 癸 |

> 例：丙日主→时干只能是戊己庚辛壬癸甲乙丙丁戊己→**不可能有丙午时**。
> 例：甲日主→时干只能是甲乙丙丁戊己庚辛壬癸甲乙→**确实有甲子时**。

### 5.3 六步评估流程

```
Step 1: 查日干 → 确定五鼠遁口诀
Step 2: 按速查表列出全部12个时柱（天干+地支）
Step 3: 用引擎查喜忌 → 标注各天干×地支的喜忌
Step 4: 四象限定位每个时柱（喜+喜 / 喜+忌 / 忌+喜 / 忌+忌）
Step 5: 剔除与时支/月支/日支存在刑冲破害的选项（见下方⚠️）
Step 6: 选出最优（双喜 > 忌上喜下/喜上忌下 > 双忌）

⚠️ 必查约束：
□ 时支冲月支（如亥巳冲、子午冲）
□ 时支冲日支（如巳亥冲日支）
□ 时支刑/害日支或月支
□ 日时伏吟 → 确认是否可接受
□ 是否破坏原局三合/六合局
```

### 5.4 判定优先级

| 优先级 | 组合 | 说明 |
|:------|:-----|:-----|
| 🥇 最优 | **天干喜+地支喜** 且 无刑冲破害 | 最佳配置，内外皆吉 |
| 🥈 次优 | **天干忌+地支喜** 或 **天干喜+地支忌** | 一好一坏，看哪个更重要 |
| 🥉 可用 | 双喜但有伏吟/半冲 | 吉但需评估副作用 |
| ❌ 最差 | **天干忌+地支忌** | 双忌，需大运填实或化解 |

### 5.5 四大典型模式

| 类型 | 命例 | 当前 | 最佳 | 原理 |
|:----|:-----|:-----|:-----|:------|
| **当前已很好** | 庚日从弱（主母） | 壬午双喜 ✅ | 丁亥双喜更优 ✅ | 从弱格火水齐来，亥未半合木局 |
| **忌+喜可接受** | 辛日身强（老板） | 辛卯忌+喜 | **当前即最优** ✅ | 三合财局>单柱双喜，原局最佳 |
| **喜+忌需注意** | 丙日身强（子源） | 癸巳喜+忌 | 癸巳最佳/壬辰并列 | 双官透头尾呼应 vs 辰戌冲权变 |
| **提升最大** | 甲日身弱（立） | 庚午双忌 ❌ | 甲子双喜 ✅ | 七杀伤官→比肩正印，逆转人生 |

### 5.6 常见误区

| 误区 | 正确理解 |
|:-----|:---------|
| "丙日主配丙午时最好" | 丙日主**不可能有丙时**（丙辛从戊起→午时甲午） |
| "甲木喜水→甲子时肯定有" | 甲日主确实有甲子时 ✅ （甲己还加甲→子时甲子） |
| "地支午是喜神→午时都好" | 天干也要看！甲日庚午=七杀忌+伤官忌=双忌 ❌ |
| "最佳时柱=当前时柱" | 最佳时柱是理论最优解，是分析工具，不改变原局 |
| "不考虑原局三合直接换时辰" | 换时辰必须检查是否破坏原局的成型格局 |

### 5.7 ⚡ 2026-07-06 交叉验证修正点（两个AI答案对比）

**修正①：午火对辛金日主=七杀，非正官**
- 辛金日主：午中丁火→**七杀**（非正官），丙火才是正官
- 错误案例：甲午时对老板 = 财生杀（非财生官）
- 正官/七杀判断口诀：丙=正官，丁=七杀（辛日主）

**修正②：三合财局 > 单柱双喜**
- 老板原局亥卯未三合财局（全局价值极高）
- 甲午时虽时柱自身双喜，但破坏了财局 → 整体不如原局
- **评估全局价值时，三合/六合局的力量优先级高于单柱的四象限**

**修正③：五鼠遁必须先查速查表/口诀双重验证**
- 甲日干五鼠遁为"甲己还加甲"→ 子时=**甲子时**（不存在丙子时）
- 每次分析前必须：加载bazi-wushidun-verify → 查表确认

**修正④：全局冲刑合害 > 时柱自身喜忌**
- 任何冲刑都必须标注并评估严重性，不能默认"冲了就是好"
- 子源辰戌冲日支 = 有人看好（开库有权变），有人看衰（伤夫妻宫）
- **流派差异要标注，但不能默认某一边**

**修正⑤：三合财局是最高优先级的命局结构**
- 老板三合财局+空亡 = 延迟爆发的大器晚成格局
- 这个结构比单柱的双喜配置更珍贵
- 换时柱前必须先问：**原局最核心的结构是什么？能不能破？**

## 七、🚨 发布前校验清单（强制执行）

**每次做「最佳时柱分析」后，结论发布前，必须逐项跑完以下校验。**

跑法：`python3 /root/bazi-platform/scripts/pillar-verify.py` 或手动逐项检查。

### □ 关卡① 五鼠遁校验

```
步骤：对照12倍速查表（§5.2），确认每个候选时柱的天干真的存在
检查：候选列表外是否还有更好的时柱被遗漏？
反例：甲日主不存在丙子时 → 对方AI犯了这个错
```

### □ 关卡② 藏干十神校验

```
步骤：对每个候选时柱的地支，查藏干 → 确定十神
重点标注所有「容易混淆」的：
  □ 辛日主午火=七杀（非正官）
  □ 庚日主午火=正官（正确）
  □ 丙日主巳火=比肩（非劫财）
  □ 甲日主午火=伤官（非食神）
  □ 其他所有地支的十神都确认正确
```

### □ 关卡③ 结构优先级校验

```
对最佳候选问：
  □ 它是否破坏了原局更重要的结构？
  □ 三合/六合局的优先级 > 单柱双喜配置
  □ 官星双透（头尾呼应） > 单柱一喜一忌
  
换时柱前的4个必须问：
  ① 原局最核心的结构是什么？
  ② 这个结构能破吗？
  ③ 破了之后有什么替代？
  ④ 替换后的整体得失是否净为正？
```

### □ 关卡④ 全局冲刑校验

```
对每个候选时柱问：
  □ 时支冲月支？ → 致命（父母宫/青年运）
  □ 时支冲日支？ → 致命（夫妻宫）
  □ 时支刑/害月支或日支？ → 需要标注
  □ 时支是否导致日时伏吟？ → 需要标注
  
判断标准：
  冲日支/月支 = 不可取 ❌
  刑/害/破 = ⚠️ 标注严重性
  无冲刑 = ✅ 
```

### □ 关卡⑤ 最优性校验

```
对最终结论问：
  □ 最佳时柱真的是所有候选中最优的吗？
  □ 有没有双喜无冲刑的候选被忽略了？
  □ 如果另一个AI来做分析，它会同意吗？
  □ 结论是否标注了「流派差异」（如辰戌冲的好坏）？
```
