## 1. 五重人格分析法总纲

> **来源：** 综合原始文档01/02/12号（十神性格）、19号（木身高）、25号（性格分析流程）+实战校准
> **来源标注：** bazi-foundation-analysis §23 + bazi-fortune-analysis §9.5

五重人格分析法是金鉴真人体系的核心性格分析框架，从五个层次逐层深入，全面还原一个人的性格底色。

```yaml
五重人格分析流程：

第一重：日主五行底色
  → 看日主天干的五行属性（木/火/土/金/水）
  → 定性格基调（正直/热情/稳重/刚毅/智慧）

第二重：十神人格体系
  → 看天干十神（外表性格）+ 地支十神（真实性格）
  → 看格局（人生追求方向）+ 月干（社会角色）
  → 定十神组合与互动

第三重：刑冲合害对性格的影响
  → 看地支之间的刑冲合害关系
  → 定性格中的矛盾/冲突/调和/转化

第四重：神煞天赋特征
  → 看神煞体系（桃花/华盖/驿马/天乙贵人等）
  → 定特殊天赋和性格特征

第五重：成长与人生建议
  → 综合前四重 + 身强弱 + 喜忌
  → 定性格优缺点总结和人生建议方向

口诀：
  一重日主看底色，二重十神看人格
  三重刑冲看矛盾，四重神煞看天赋
  五重综合出建议，五层走完性格现
```

### 1.1 性格分析的两个层面

> **来源：** bazi-foundation-analysis §23.3

```yaml
⚠️ 区分两个层面：
  天干十神 = 给人的第一印象（外表气质、社交面具）
  地支十神 = 深入了解后的真实性格（内在本质）

  断语示例：
    「月干七杀在年 → 外人觉得严肃不好惹，实则日坐正印温暖细腻」
    「天干劫财 → 见面熟、善于交际；地支七杀 → 实则防备心强、杀伐决断」
```

### 1.2 性格与格局的关系

> **来源：** bazi-fortune-analysis §6

```yaml
主格（月令透干）= 人生追求方向（如偏财格一生想赚钱）
辅格一：日支本气十神 = 真实脾气性格
辅格二：月干十神 = 外在表现/社会角色
辅格三：日支暗合/合化 = 隐藏性格

⚠️「脾气」和「性格」是两回事：
  - 格局（主格）= 人生追求（如偏财格追求赚钱）
  - 日支（辅格）= 真实脾气（如日坐七杀脾气爆）
  - 月干 = 外在表现/社会角色

例：偏财格·自坐七杀·月干食神
  → 人生追求：赚钱（偏财格）
  → 真实脾气：暴脾气（日坐七杀）
  → 外在表现：随和好相处（月干食神）
```

### 3.2 十神深度详解

> **来源：** bazi-foundation-analysis §9.5.1 十神本质论

#### 正官（克我者，阴阳不同）
```yaml
本质：秩序、规范、权威、社会认可
├─ 天干透正官 → 做事讲规矩、有底线、重视名誉
├─ 正官为用 → 做人正气，社会地位稳
└─ 正官为忌 → 被规矩束缚、做事迂腐
细分（公众号原文）：
  木正官（戊己日生）→品行端正，能适应复杂社会环境
  火正官（庚辛日生）→性急躁，有威力，敢管敢做
  土正官（壬癸日生）→敏锐判断力，经济方面有才能
  水正官（丙丁日生）→谦虚圆滑，富于理性，仁慈善良有同情心
```

#### 七杀（克我者，阴阳相同）
```yaml
本质：压力、征服、魄力、危机处理
├─ 天干透七杀 → 做事果断、敢于打破常规
├─ 七杀为用有制 → 大魄力大成就（杀印相生/食神制杀）
├─ 七杀无制 → 性格暴躁、易招是非、健康受损
├─ 七杀旺 → 果敢决断、竞争意识强、掌控欲强、性格刚烈
├─ 七杀旺 → 越挫越勇（抗压能力强）
└─ 七杀旺 → 易身心俱疲，长期高压导致失眠/焦虑/内分泌失调
```

#### 正印（生我者，阴阳不同）
```yaml
本质：滋养、学习、庇护、传承
├─ 天干透正印 → 有慈悲心、爱学习、有长辈缘
├─ 身弱正印为用 → 学业好、得贵人提携
├─ 身强正印过旺为忌 → 依赖性强、缺乏主见
├─ 身强遇印为忌 → 保守、不争、不开窍
└─ 用神大运 ≠ 舒服大运：七杀/伤官为用神→过程极苦
```

#### 偏印/枭神（生我者，阴阳相同）
```yaml
本质：特殊技能、冷门知识、神秘学、逆向思维
├─ 天干透偏印 → 有独特见解、适合研究/玄学
├─ ⚠️ 偏印 ≠ 枭神：
│   命局中无食神时叫偏印（中性偏凶）
│   有食神时叫枭神（凶神，枭神夺食）
├─ 偏印为用 → 在冷门领域有建树
├─ 偏印为忌 → 性格孤僻；枭神夺食有健康/心理问题
├─ 月令偏印必有一技之长 → 在某技能领域特别擅长
└─ 偏印与食神的宫位对立 → 对应宫位六亲关系不和
```

#### 正财（我克者，阴阳不同）
```yaml
本质：稳定收入、家庭观念、务实主义
├─ 天干透正财 → 靠工资收入、会过日子
├─ 正财一位为用 → 婚姻稳定、收入稳
└─ 正财多位 → 财多身弱易成财奴
```

#### 偏财（我克者，阴阳相同）
```yaml
本质：大财、投资、人脉、慷慨、异性缘
├─ 天干透偏财 → 善投资、交游广阔
├─ 偏财为用 → 财路广、有经商头脑
└─ 偏财为忌 → 乱花钱、投资必亏
```

#### 比肩（同我者，阴阳相同）
```yaml
本质：自我、自尊、竞争、独立性
├─ 天干透比肩 → 有主见、好竞争、不喜受人管
├─ 比肩为用 → 得同辈助力、能合伙
└─ 比肩过旺为忌 → 固执己见、兄弟反目、易犯小人
```

#### 劫财（同我者，阴阳不同）
```yaml
本质：朋友、同辈、合伙、争夺
├─ 天干透劫财 → 善社交、讲义气、但易破财
├─ 身弱劫财为用 → 得朋友帮，共渡难关
├─ 身强劫财为忌 → 被朋友陷害、争夺家产、男命克妻

劫财性格区分（天干vs地支）：
  劫财在天干 → 见面熟、善于交际、情商高（社交型）
  劫财在地支 → 心眼多、善于谋划（谋略型）
  没劫财 = 项羽 → 单打独斗，不善于合作
  有劫财 = 刘邦 → 善于交际、善用人
```

#### 食神（我生者，阴阳相同）
```yaml
本质：才华、福气、享受、创造力
├─ 天干透食神 → 有才艺、懂生活、性格温和
├─ 食神生财 → 靠才华/技术赚钱
├─ 食神过多为忌 → 沉迷享乐、不务正业
├─ 食神旺 → 心宽体胖、不发愁
└─ 食神在哪个宫位 → 该年龄段/身体部位发胖
```

#### 伤官（我生者，阴阳不同）
```yaml
本质：才华锋芒、叛逆、创新、不服管
├─ 天干透伤官 → 聪明有才、爱挑战权威
├─ 伤官配印 → 才华得以正向发挥，是贵格
├─ 伤官见官 → 与领导冲突、事业坎坷、女命克夫

伤官能量四级递进（实战核心）：
  伤官能量小 → 吵架
  能量大 → 打架、口舌纷争
  能量再次变大 → 官司（到法院起诉）
  能量最大 → 自己受伤（动手术）

伤官泄秀（身强）≠ 伤官配印（身弱）：
  伤官配印 = 身弱 + 伤官 + 印 → 主贵（官场/名望）
  伤官泄秀 = 身强 + 伤官 → 主艺术（才华发挥）
```

## 4. 第三重：刑冲合害对性格的影响

> **来源：** bazi-foundation-analysis §3 字间互动规则库

### 4.1 合化对性格的影响

```yaml
合化本质：加强合化出来那个五行的能量
合出喜用 → 性格中的优点被放大
合出忌凶 → 性格中的缺点被放大

天干五合：
  甲己合土 → 原本甲木性格（正直固执）→ 增加戊土（敦厚稳重）
  乙庚合金 → 原本乙木性格（柔韧适应）→ 增加庚金（刚强果断）
  丙辛合水 → 原本丙火性格（热情急躁）→ 增加壬水（智慧变通）
  丁壬合木 → 原本丁火性格（细腻文雅）→ 增加甲木（正直有担当）
  戊癸合火 → 原本戊土性格（敦厚稳重）→ 增加丙火（热情开朗）

合化条件与性格：
  合而成化 → 性格转换明显
  合而不化 → 性格矛盾，内心拉扯
  争合/妒合 → 多角关系，性格复杂
```

### 4.2 六冲对性格的影响

```yaml
冲走忌凶（不好的能量）→ 性格中的缺点被释放，人变好
冲走喜用（需要的能量）→ 性格中的优点被削弱，人变差

具体冲刑：
  子午冲 → 水火交战 → 性格急躁、冲动
  卯酉冲 → 金木交战 → 刚强与柔韧的矛盾，易有决断困难
  寅申冲 → 金木交战 → 开拓与保守的冲突
  巳亥冲 → 水火交战 → 热情与冷静的矛盾
  辰戌冲 → 土自战 → 内心纠结、固执
  丑未冲 → 土自战 → 敦厚与内敛的拉扯

宫位冲的性格断事：
  年月冲 → 背祖离家 → 性格独立、叛逆
  月日冲 → 性格矛盾多、易有情绪波动
  日时冲 → 克子女 → 性格急躁
```

### 4.3 三刑对性格的影响

```yaml
三刑的本质：
  三刑不是「扣分」，也不是简单的伤害
  三刑 = 地支藏干「打架」→ 余气全部打没 → 只剩本气 → 本气能量翻倍

丑未戌三刑（最重·15倍土）：
  → 比劫能量极强 → 性格固执、强势、不听劝
  → 男命：比劫克偏财=父缘薄；比劫克正财=妻矛盾

寅巳申三刑（12倍）：
  → 木火金交战 → 性格起伏大、易怒易静
  → 每个被加强的字，喜=吉表现，忌=凶表现

子卯刑（5倍）：
  → 子水生卯木 → 木的能量加强
  → 性格容易敏感、容易吵架
  → 能量最小，一般只代表小摩擦

自刑（辰辰/午午/酉酉/亥亥）：
  → 该字的能量被加强
  → 该字是喜用→性格中的优点明显
  → 该字是忌凶→性格固执、钻牛角尖

口诀：
  三刑不是扣分是打架，余气打没本气加倍
  被加强的字看喜忌，喜则吉表现忌则凶
  丑未戌出倔强牛，寅巳申出暴脾气
  子卯小吵小摩擦，自刑偏执难回头
```

### 4.4 六害对性格的影响

```yaml
六害 = 穿 = 拿利刃刺穿身体
  寅巳害 → 急躁 + 火爆 = 暴脾气
  卯辰害 → 柔韧 + 敦厚 = 表面温和内心固执
  酉戌害 → 精致 + 燥热 = 追求完美但内心焦虑
  申亥害 → 刚强 + 智慧 = 表面坚强内心多虑
  丑午害 → 敦厚 + 热情 = 表面老实体内有火
  子未害 → 智慧 + 燥热 = 聪明但容易受伤
```

### 9.4 群阴独阳/群阳独阴（泉师兄盲派实战·2026-07-13新增）

> **来源**：泉师兄抖音视频「看八字的思路」逐字精读
> **核心概念**：比纯阳/纯阴更常见的极端情况——八个字中只有一个阳（群阴独阳）或只有一个阴（群阳独阴）

#### 9.4.1 定义与判定

```yaml
群阴独阳 = 八个字中只有一个是阳干或阳支，其余七个都是阴
群阳独阴 = 八个字中只有一个是阴干或阴支，其余七个都是阳

判定标准：
  ┌─ 群阴独阳：7个阴字 + 1个阳字
  │   → 「孤阳不生」——独阳难以独立
  │   → 独阳被群阴包围，能量被压制
  │
  └─ 群阳独阴：7个阳字 + 1个阴字
      → 「孤阴不长」——独阴难以发展
      → 独阴被群阳包围，能量被消耗

❗ 全阴全阳不算极端（大家一样=平衡）
❗ 群阴独阳/群阳独阴才是真正的阴阳失衡
```

#### 9.4.2 心理状态判断

```yaml
群阳独阴 → 狂躁、激烈
  └─ 阳多阴少 → 能量外泄、难以收敛
  └─ 群阳之中一点阴 → 那点阴被压制得出不来
  └─ 表现为：暴躁易怒、难以平静、表面刚强内心脆弱

群阴独阳 → 阴郁、深沉
  └─ 阴多阳少 → 能量内收、难以释放
  └─ 群阴之中一点阳 → 那点阳拼命挣扎
  └─ 表现为：内心压抑、情绪低沉、表面平静内心翻涌
  
特别——水火交战加重精神问题：
  ├─ 群阴独阳 + 水火交战 → 🚨 精神层面问题突出
  ├─ 丙火在年干（独阳）→ 替命主抗灾的亲人
  ├─ 巳火/午火在家（日支/时支）→ 自己名下受灾
  └─ 应事顺序：年上亲人出问题 → 日支自己出问题
```

```yaml

  "独阴独阳就是我人生中所追求的东西"

独阳（阳少反而珍贵）：
  → 这个人一生所追求的就是**阳属性**的东西
  → 阳 = 火 = 光明、事业、名声、地位、热情
  → 事业上：追求和离卦（火）相关的事业
  → 感情上：独阳可以作为感情星/桃花星

  → 应事：大运/流年来火的时候，感情应事

独阴（阴少反而珍贵）：
  → 这个人一生所追求的就是**阴属性**的东西
  → 阴 = 水 = 智慧、财富、隐秘、艺术、灵性
  → 事业上：追求和水/坤卦相关的事业
  → 感情上：独阴可以作为感情星/桃花星
  → 应事：大运/流年来水的时候，感情应事
```

#### 9.4.4 独阳的十神对应与六亲应灾

```yaml
独阳所在的位置决定其具体意义：

独阳在年干（丙火为例）：
  → 丙火就是替我抗灾的亲人
  → "丙火在年上所带的六亲会出问题"
  → 丙火为偏财（对辛金日主）→ 父亲早亡/父缘薄
  → 丙火为七杀（对癸水日主）→ 祖父有灾/早逝

独阳在家（日支/时支的巳火为例）：
  → "巳火在家里代表的是我自己"
  → "下一个受灾的就是我自己了"
  → 应事晚于年柱，但应灾程度类似

六亲应灾规则：
  独阳在年/月 → 先应六亲（父亲/祖父/长辈）
  独阳在日/时 → 后应自己（中年/晚年）
  应事时间：大运流年来引化独阳时

案例（泉师兄原文）：
  满盘全是金水阴的东西，唯独有两个火是阳的
  一个在年上（丙），一个在时上（巳）
  → 年上的丙火所带的六亲出问题→父亲早亡
  → 时上的巳火代表自己→自己在高速车祸
  → 命主性格：本身比较阴沉
```

#### 9.4.5 实战使用流程

```yaml
拿到八字后的阴阳检查流程：

Step 1 — 数阴阳个数
  阳干（甲/丙/戊/庚/壬）+ 阳支（子/寅/辰/午/申/戌）
  阴干（乙/丁/己/辛/癸）+ 阴支（丑/卯/巳/未/酉/亥）
  → 统计总阳数 vs 总阴数

Step 2 — 检查是否群阴独阳/群阳独阴
  □ 7阴1阳 = 群阴独阳（阳为追求）
  □ 7阳1阴 = 群阳独阴（阴为追求）
  □ 非此二类 → 转到常规阴阳失衡

Step 3 — 独阳/独阴的定性与定位
  独阳/独阴是什么字？在哪个宫位？
  □ 年柱 → 六亲应先灾
  □ 月柱 → 父母/兄弟
  □ 日柱 → 自己
  □ 时柱 → 子女/晚年

Step 4 — 独阳/独阴的十神属性
  这个字对日主是什么十神？
  □ 财 → 财富追求
  □ 官 → 地位追求
  □ 印 → 知识追求
  □ 食伤 → 才华追求
  □ 比劫 → 自我追求

Step 5 — 大运流年应期
  什么大运/流年会引化这个独阳/独阴？
  → 该大运/流年就是这个领域的人生转折点

口诀：数阴阳，看比例；七比一，极端起
      独阳追求阳的事，独阴追求阴的事
      独阳在年六亲灾，独阳在家自己害
      大运流年引化时，人生转折就在此
```

## 10. 燥湿平衡对性格的影响

> **来源：** bazi-foundation-analysis §11.2 + 综合实战校准

### 10.1 燥湿判断方法

```yaml
八字燥湿判断：
  燥土（未/戌）多 → 燥性体质 → 性格急躁、缺乏耐心
  湿土（辰/丑）多 → 湿性体质 → 性格温和、但较拖延
  火旺 → 燥热 → 热情急躁
  水旺 → 湿润 → 智慧但多虑

影响性格的燥湿因素：
  燥湿不平衡 → 炎症体质 → 影响情绪稳定
  火太旺 → 燥热体质 → 阴虚火旺 → 性格激进
  水太旺 → 寒性体质 → 性格内敛、容易伤感

燥湿平衡的性格表现：
  燥多湿少 → 热情开朗但急躁
  湿多燥少 → 温和内敛但拖延
  燥湿平衡 → 情绪稳定、心理成熟
```

### 10.2 燥湿与情绪的关系

```yaml
八字燥热的性格表现：
  ├─ 脾气来得快去得快（🔥 点火就着）
  ├─ 做事有冲劲但缺乏持久性
  ├─ 喜欢热闹、人多的地方
  └─ 情绪波动大、容易被外界影响

八字湿寒的性格表现：
  ├─ 情绪稳定、不易动怒
  ├─ 做事有耐心但容易拖延
  ├─ 喜欢安静、独处的时间
  └─ 不容易被外界影响、但也缺乏热情

口诀：
  燥土旺人急躁，湿土旺人温和
  火多人疯，水多人精，水土各半最太平
```

## 11. 综合判断流程

### 11.1 拿到八字后的性格分析顺序

> **来源：** bazi-foundation-analysis §23.5 综合判断流程

```yaml
拿到八字判断性格 → 按以下5步走：

Step 1：看日主五行 → 定性格底色基调
  X木/火/土/金/水 → 对应§2的基础性格

Step 2：看十神分布 → 定人格类型
  天干十神 = 外表性格，地支十神 = 真实性格
  格局 = 人生追求，日支 = 真实脾气，月干 = 社会面具

Step 3：看刑冲合害 → 定性格中的矛盾/调和
  合化 = 性格修饰
  冲刑 = 性格矛盾
  三刑 = 性格中的激烈面

Step 4：看神煞 → 定特殊天赋和性格特征
  桃花/华盖/驿马/天乙贵人等 → 对应§5特征

Step 5：看阴阳燥湿 → 定整体气质和情绪倾向
  阴阳比例 → 刚柔倾向
  燥湿平衡 → 情绪稳定性
```
