# 金鉴真人·子女实战校准案例表

> 2026-06-15 更新
> 来源：家族5人八字分析 + 父母合参反推

## 案例1：父亲（己丑癸酉癸亥戊午）

| 项目 | 内容 |
|:-----|:------|
| **八字** | 己丑 癸酉 癸亥 戊午 |
| **日主** | 癸水（中和偏强·阴干） |
| **子女星** | 年干己土七杀=子，时干戊土正官=女 |
| **十二长生** | 戊土(女)在午=帝旺 → 旺中五子；己土(子)在午=帝旺 → 旺中五子 |
| **实际子女** | **4人**（2女2子）：1978长女→1980次子→1981次女→1983次子 |
| **应期验证** | 1978戊午→时柱伏吟·正官双透→长女 ✅ |
| | 1980庚申→印星帮身→次子 ✅ |
| | 1981辛酉→偏印夺食但大运帮身→次女 ⚠️ |
| | 1983癸亥→亥午暗合引动子女宫→次子 ⚠️ |

**启示：** 十二长生基数法断5个但实有4个。基数法是"命理缘分"，实际受时代政策制约（1970-1980年代计划生育）。

## 案例2：家主（庚申癸未辛亥辛卯）

| 项目 | 内容 |
|:-----|:------|
| **八字** | 庚申 癸未 辛亥 辛卯 |
| **日主** | 辛金（身弱·阴干） |
| **子女星** | 丁火正官=子，丙火七杀=女（传统规则） |
| **实际子女** | **1子**：子源(2011辛卯) |
| **应期验证** | 2011辛卯→丙辛合水引动七杀+亥卯未生火→子源 ✅ |
| **幼子验证** | 2017丁酉→双丁火正官信号但实际无→身弱能量不足 |

**启示：** 子女星引动+有能量=应期。单有信号不够，身弱能量不足则信号不兑现。

## 案例3：主母（丁卯丁未庚午壬午）

| 项目 | 内容 |
|:-----|:------|
| **八字** | 丁卯 丁未 庚午 壬午 |
| **日主** | 庚金（身弱） |
| **子女星** | 时干壬水食神=子（女命） |
| **实际子女** | **1子**：子源(2011辛卯) |
| **应期验证** | 2011辛卯→流年与时柱午未合→引动+丙辛合→生子上注意：身弱8分，通常难生育，但2011年亥卯未木局生火→火生土→土生金→连环相生补身 |

**启示：** 极端身弱（8分）仍然能生育，需看大运流年是否有通关相生。

## 案例4：父母合参（双癸水日主）

| 父亲八字 | 母亲八字 |
|:---------|:---------|
| 己丑 癸酉 癸亥 戊午 | 壬辰 戊申 癸卯 癸丑 |
| 癸水·中和偏强 | 癸水·身强 |
| **共4子女** | |

**父母同频规律：**
- 1978戊午：父→戊土伏吟(女星) + 母→戊土透干(正官) = 共振
- 1980庚申：父→庚申帮身 + 母→庚申印星 = 双金印帮身
- 1981辛酉：父→辛酉偏印 + 母→辛酉偏印 = 双偏印同频
- 1983癸亥：父→癸亥比劫 + 母→癸亥比劫 = 双癸水伏吟
