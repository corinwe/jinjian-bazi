## 完整判断流程（engine 实现步骤）

### 核心原理

```
子女出生年份 = 大运流年引动子女星的应期
子女出生时父亲的年龄 = 子女出生年份 - 父亲出生年份
```

### 应期信号（按强度排序）

```yaml
⭐⭐⭐⭐⭐ 子女星在流年天干透出（最强）
  例：原局戊土正官藏辰中 → 流年戊午年透干 → 得女

⭐⭐⭐⭐ 子女宫（时支）被伏吟/合/冲
  时支被伏吟 → 强烈子女信号
  时支被六合/半合 → 引动子女宫
  时支被冲 → 变动中得子

⭐⭐⭐ 流年合化出子女星
  天干五合/地支六合化出子女星五行

⭐⭐ 大运子女星透干+流年配合
  大运透子女星→十年趋势
  流年落子女宫→当年应事

⭐ 身旺有能量生育的年份
  身弱遇印比帮身→有能量生育
```

  （2017丁酉推为女儿但实际无 → 推理需校准）

### 父母合参子女规律

> 来源：父亲(己丑癸酉癸亥戊午) + 母亲(壬辰戊申癸卯癸丑) → 4个子女的实战反推

#### 规律①：父母同频
父母日主相同→命理同频，子女应期在双方命盘上都有信号。
非单方面看父或母，而是合参。

#### 规律②：帮身是生育前提
身旺才有能量生育。印星/比劫帮身的年份最容易有子女。
1980庚申/1981辛酉→金印帮双癸水身→连年生两胎。

#### 规律③：子女星引动 + 有能量 = 应期
子女星到位（透干/伏吟/冲合）+ 身旺有能量 = 两者缺一不可。
