## ⚠️ 引擎实现铁律 — 身强弱参数必须被分析逻辑使用（2026-07-05审计教训）

**来源：** 2026-07-05 health_v2.py全量审计发现

```yaml
【Bug发现】health_v2.py传入shen_label参数，但仅存在输出元数据（dict）中，
始终未被任何分析逻辑使用。导致WEAK_SYSTEMS表中的身强弱高危特征
（如「身强水旺无火调候→心血管高危」）虽然有数据但从不触发。

【修复】将shen_label传入4个核心函数：
  check_wuxing_energy_imbalance → 身强水旺无火调候/身弱火旺无水救→心血管预警
  check_dry_wet_balance        → 身强/身弱下的燥湿表现不同
  assess_severity              → 身弱→严重度+1（耐受力差）
  generate_disease_summary    → 结论中体现身强弱影响

【铁律】任何传入analyze_health_full的shen_label参数：
  □ 必须在至少3个分析逻辑函数中作为条件使用
  □ 不能仅存在return dict的元数据中
  □ 四大薄弱系统(WEAK_SYSTEMS)的高危特征必须有代码检查逻辑
  □ 引用链必须验证：comprehensive_v2.py → health_v2.analyze_health_full
```

### 手指看疾病法
- 大拇指→脾胃 | 食指→肝胆 | 中指→心脏 | 无名指→肺部 | 小指→肾脏
- 哪根手指弯曲不直→对应器官有问题
- 双手一起看

### 面相痣疤断身体（全息映射法）
- 眉心/额头→头 | 左眉毛→左胳膊/肩 | 右眉毛→右胳膊/肩
- 鼻子→躯干/身体 | 左法令纹→左腿 | 右法令纹→右腿
- 面部痣/疤/伤口/青春痘→对应身体部位也有
- 动态变化：疾病好了，面部斑点也会消失

### 耳朵断疾病法
- 耳朵倒过来像婴儿在母体中的形态
- 耳垂→头部；往上依次→颈/胸/腹/腰/腿
- 耳垂痣/疤/横纹/折痕→心脑血管疾病（脑梗风险）
- 左耳→左半身，右耳→右半身
- 只算自然形成（痣/疤/纹路），耳洞不算

## 🆕 §核心方法论：八字疾病诊断体系（2026-07-10 全量审计补全）

> **来源**：九龙先生八字课程·第23-24课 + bazi-foundation-analysis 综合整理

### 1. 五行过旺过弱 → 对应脏腑（基础框架）

| 五行 | 对应脏腑 | 过旺症状 | 过弱症状 |
|:-----|:--------|:---------|:---------|
| **木** | 肝、胆、筋骨、眼睛 | 肝火旺、易怒、头晕 | 肝血虚、视力差、筋骨弱 |
| **火** | 心、小肠、血液、舌头 | 心火旺、失眠、口疮 | 心血虚、怕冷、循环差 |
| **土** | 脾、胃、肌肉、口腔 | 消化不良、腹胀、肥胖 | 脾胃虚、消瘦、食欲差 |
| **金** | 肺、大肠、皮肤、鼻子 | 肺热、便秘、皮肤病 | 肺气虚、易感冒、过敏 |
| **水** | 肾、膀胱、骨骼、耳朵 | 肾火旺、腰疼、浮肿 | 肾虚、怕冷、精力差 |

### 2. 五行过三原则（核心定量方法）

> 引擎能量评分中，某五行能量 **≥3倍基础值** = 过三，对应器官易发病

| 五行 | 能量≥3 | 对应疾病 |
|:-----|:-------|:---------|
| **木过三** | 木≥3 | 肝胆疾病、筋骨疼痛、头部不适 |
| **火过三** | 火≥3 | 心脏问题、血液循环、眼疾 |
| **土过三** | 土≥3 | 脾胃病、消化不良、肌肉酸痛 |
| **金过三** | 金≥3 | 肺病、皮肤过敏、便秘 |
| **水过三** | 水≥3 | 肾病、水肿、腰膝酸软 |

### 3. 七杀断病法（九龙特色）

七杀（偏官）在特定位置对应特定疾病：
- 年柱七杀 → 头面部疾病（祖上遗传）
- 月柱七杀 → 胸肺部疾病
- 日柱七杀 → 腹部/脾胃
- 时柱七杀 → 下肢/生殖系统

**加重条件**：
- 七杀 + 刑冲 → 病情加重
- 七杀 + 空亡 → 凶事化解（病情减轻）
- 七杀无制 → 慢性病

### 4. 偏印断病法

偏印在特定位置：
- 年柱偏印 → 头部神经问题
- 月柱偏印 → 呼吸道/精神压力
- 日柱偏印 → 消化系统
- 时柱偏印 → 生殖/泌尿系统

**偏印为忌** → 精神类疾病倾向（抑郁、焦虑、失眠）

### 5. 燥湿平衡与疾病

| 状态 | 疾病倾向 |
|:-----|:---------|
| **命局太燥**（未戌多、水少） | 皮肤干燥、便秘、心火旺、易怒 |
| **命局太湿**（辰丑多、火少） | 湿气重、关节疼、浮肿、乏カ |
| **燥土被火引化** | 火旺克金→肺病、皮肤过敏 |
| **湿土被水浸泡** | 水旺克火→心脏病、循环差 |

### 6. 五行受克 → 对应脏腑

某五行被克 = 该五行对应的脏腑易患病：
- 木被金克 → 肝胆/筋骨受伤
- 火被水克 → 心脏/血液问题
- 土被木克 → 脾胃/肌肉疾病
- 金被火克 → 肺/皮肤过敏
- 水被土克 → 肾/泌尿系统问题

### 7. 体质类型（引擎输出 → 可读判断）

| 引擎输出 | 实际含义 | 健康建议 |
|:---------|:---------|:---------|
| 火过旺 | 心火旺体质 | 多补水、少辛辣、注意睡眠 |
| 水过旺 | 寒湿体质 | 多补火、少冷饮、注意保暖 |
| 木过旺 | 肝火旺体质 | 多补金、少熬夜、注意情绪 |
| 金过旺 | 肺燥体质 | 多补水、少辛辣、注意呼吸 |
| 土过旺 | 湿热体质 | 多补木、少油腻、注意消化 |
| 五行中和 | 平衡体质 | 维持现状即可 |
- 春木→脾胃差 | 夏火→脾气爆/肺差 | 秋金→勇猛/肝胆差 | 冬水→冷静/心脏差

## §XX 五行健康完整判断（易道知识应用萃取）

### 五行对应身体完整表
| 五行 | 核心功能 | 外在表现 | 疾病倾向 |
|:----|:---------|:---------|:---------|
| 木 | 肝主疏泄、藏血 | 目、筋、指甲 | 肝胆不适、头痛脱发 |
| 火 | 心主血脉、神明 | 舌、脉、面色 | 血压异常、心血机能弱 |
| 土 | 脾主运化、统血 | 口、唇、肌肉 | 消化不良、胃胀腹泻 |
| 金 | 肺主气、调节水液 | 鼻、皮肤、毛发 | 呼吸不畅、肺炎、便秘 |
| 水 | 肾主精、生长发育 | 耳、二便、骨骼、脑 | 肾虚、泌尿炎症、体寒 |

### 健康判断三大原则
1. **五行缺失**（某五行个数为0）：该五行对应的身体部位先天储备不足
2. **五行过旺**（某五行≥3个）：该五行能量过亢，对应器官易发炎、拥堵
3. **五行受克**（某五行弱且有强五行克它）：慢性损伤或功能衰退

### 意外伤害判断三大逻辑
1. **印星护身被破坏**：印星被财坏或刑冲→保护层被击穿→意外几率大
2. **日干之根被冲死**：日干在地支的根被冲克→根基动摇→交通事故/跌倒
3. **五行严重失衡**：金木/水火硬性对战→岁运引发→对应部位伤害

### 五行克伤害参考表
- 火克金：电击、火灾、烫伤
- 水克火：溺水
- 木克土：棍棒伤、高空坠落
- 金克木：钝器伤、车祸
- 土克水：跌打损伤、摔伤
