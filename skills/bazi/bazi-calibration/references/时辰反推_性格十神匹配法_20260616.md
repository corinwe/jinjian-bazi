## 适用场景

用户给了多个候选出生日期/时辰，但不确认哪个是对的，并且知道命主的性格特征。

## 方法论（五步匹配法）

### 第1步：日主五行匹配

根据性格描述，判断命主更接近哪种日主：

| 五行 | 性格特征 | 匹配关键词 |
|:----|:---------|:----------|
| **庚金（阳金）** | 刚强、果断、有主见、刀剑之金 | 「刚强有主见」「硬气」「不服输」 |
| **辛金（阴金）** | 精致、细腻、内秀、珠宝之金 | 「精致细腻」「敏感」「讲究」 |
| **甲木（阳木）** | 正直、高大、领导力 | 「正直」「有担当」 |
| **乙木（阴木）** | 柔韧、适应力强、细腻 | 「柔韧」「灵活」 |
| **丙火（阳火）** | 热情、光明、大方 | 「热情」「外向」 |
| **丁火（阴火）** | 细腻、文雅、温暖 | 「文雅」「内敛」 |
| **戊土（阳土）** | 敦厚、稳重、包容 | 「稳重」「厚道」 |
| **己土（阴土）** | 内敛、包容、善于蓄积 | 「内敛」「包容」 |
| **壬水（阳水）** | 智慧、流动、浩瀚 | 「智慧」「大度」 |
| **癸水（阴水）** | 含蓄、应变、渗透 | 「含蓄」「机灵」 |

⚠️ 如果命主兼具两种特征（如「刚强有主见但也精致细腻」），则以主要特征定日主，次要特征看时柱十神。

### 第2步：官杀扛力验证

根据「能抗压/扛得住压力」的描述，用**官杀扛力精确规律**验证各候选：

```
扛力判定（基于身强分数）：
  ≥60分 → 能担大多数官杀组合
  50~59分 → 能担双正官，扛单七杀有压力
  40~49分 → 仅能担单正官
  <40分 → 任何官杀皆为压力（除非有格局豁免）
```

如果命主明显「能抗压」：
- ✅ 一票否决身弱<20分+双官杀/七杀的组合（扛不住）
- ✅ 从弱50分（五行皆喜用）→ 能抗压
- ✅ 日主有真根（禄位）→ 能抗压

如果命主「不能抗压、容易焦虑」：
- ✅ 一票否决身强无根+官杀重的组合（理论上能但实际不能）
- ✅ 优先身弱+印比帮身的组合

### 第3步：时柱十神性格匹配（核心步骤）

这是最关键的步骤。用命主的「脾气/说话方式/情绪」等特征匹配时柱十神：

| 时柱十神 | 性格特征 | 匹配关键词 |
|:---------|:---------|:----------|
| **劫财** | 脾气大、说话直、讲义气、重朋友、情绪外露 | 「脾气大」「说话直」「讲义气」「容易激动」 |
| **七杀** | 霸气、果断、不受欺负、暴躁 | 「脾气大」「霸气」「不受欺负」 |
| **伤官** | 说话直、才华外露、不服管、情绪化 | 「说话直」「有才华」「不服管」「情绪化」 |
| **食神** | 温和、聪明、有才华、佛系、好相处 | 「温和」「聪明」「好相处」「佛系」 |
| **正官** | 守规矩、有分寸、温和、负责任 | 「守规矩」「负责任」「温和」 |
| **正印** | 温和、有耐心、稳重 | 「温和」「有耐心」「稳重」 |
| **偏印** | 独特、孤僻、研究型 | 「独特」「孤僻」「爱钻研」 |
| **正财** | 务实、稳重、节俭 | 「务实」「稳重」「节俭」 |
| **偏财** | 慷慨、大方、人脉广 | 「慷慨」「大方」「人脉广」 |
| **比肩** | 独立、有主见、好竞争 | 「独立」「有主见」 |

**匹配度评分公式：**

```
匹配度 = Σ(权重 × 匹配值)

权重分配（按描述强度）：
  "能抗压" → 抗压权重2分
  "脾气大"/"说话直" → 性格权重3分（核心！）
  "容易情绪激动" → 情绪权重1分
  "精致细腻" → 细腻权重0.5分
  "刚强有主见" → 日主权重2分

匹配值：
  完全匹配 = 1.0
  部分匹配 = 0.5
  不匹配 = 0.0

总分 = 抗压 + 性格×3 + 情绪 + 细腻 + 日主
```

### 第4步：时柱与日柱的互动检查

时柱的性格必须与日柱的五行特征一致：

| 日主 | 时柱劫财 | 时柱食神 | 时柱七杀 |
|:----|:--------|:--------|:--------|
| 庚金（刚强） | 刚+直=合适 ✅ | 刚+柔=中和 | 刚+霸=合适 ✅ |
| 辛金（精致） | 精+直=硬朗化 | 精+柔=更精致 ✅ | 精+霸=压力大 |
| 甲木（正直） | 直+直=过于刚 | 直+柔=中和 | 直+霸=冲突 |

如果时柱与日主性格相悖（如辛金精致+七杀暴躁），需检查是否有其他因素化解（如合局/印星）。

### 第5步：大运序列与已知事件交叉验证

对筛选出的最优候选，看大运序列是否与命主已知人生事件吻合：

| 大运阶段 | 事件验证 |
|:---------|:---------|
| 第一步大运 | 童年/学业：是否符合命主早年经历？ |
| 第二步大运 | 青年/初入社会：是否有对应的转折？ |
| 当前大运 | 现状：与命主当前状态一致？ |
| 最佳大运 | 命主最顺的时期是否对应？ |

## 实战案例：主母时辰反推

### 输入
```
三个候选日期：①6.15庚申日 ②6.25庚午日 ③闰6.6辛巳日
每个日期跑巳时/午时/未时 → 9种可能
命主性格描述：「能抗压、脾气大、说话直、容易情绪激动、刚强有主见、精致细腻」
```

### Step 1 — 日主匹配

「刚强有主见」→ **庚金（阳金）** ⭐⭐⭐⭐
「精致细腻」→ 时柱食神/伤官/合水带来的细腻面，不改变日主

→ 淘汰③辛巳日（辛金为主，主调是精致非刚强）

### Step 2 — 官杀扛力验证

候选①6.15：庚金坐申禄根+从弱50分→能抗压 ✅
候选②6.25：庚金无根+从弱50分→靠食神智慧化解，非硬扛 ⚠️

→ 候选①略优

### Step 3 — 时柱十神匹配

「脾气大+说话直+容易情绪激动」= 时柱应有**劫财**或**七杀**

各候选时柱对比：
```
6.15巳时 → 辛巳（劫财）→ 脾气大+说话直 ✅ | 巳申合水→细腻 ⭐
6.15午时 → 壬午（食神）→ 温和、不暴躁 ❌
6.15未时 → 癸未（伤官）→ 说话直但非暴躁 ⚠️
6.25巳时 → 辛巳（劫财）→ 脾气大+说话直 ✅ 但无金根→能抗压存疑
6.25午时 → 壬午（食神）→ 温和 ❌
6.25未时 → 癸未（伤官）→ 才华型 ⚠️
闰6.6各时 → 辛金为主→主调不匹配
```

**匹配度评分：**
```
①6.15巳时（辛巳劫财）= 有根能扛2 + 劫财3 + 情绪1 + 巳申合水0.5 + 庚金刚强2 = 8.5分
①6.15未时（癸未伤官）= 有根能扛2 + 伤官1 + 情绪1 + 伤官细腻0.5 + 庚金刚强2 = 6.5分
②6.25巳时（辛巳劫财）= 无根0 + 劫财3 + 情绪1 + 0 + 庚金刚强2 = 6.0分
①6.15午时（壬午食神）= 有根能扛2 + 食神0 + 0 + 食神细腻0.5 + 庚金刚强2 = 4.5分
```

### Step 4 — 时柱与日柱互动

6.15庚申日巳时（辛巳）：
- 庚金刚强+辛金劫财=刚强+直来直去，匹配 ✅
- 巳申合水→合出食伤智慧，精致细腻的来源 ✅
- 辛金劫财坐巳火正官→脾气大但有分寸（劫财+正官=讲义气但守规矩）✅

### Step 5 — 大运验证

6.15庚申日大运：戊申(1996)→己酉(2006)→庚戌(2016)
起运年龄与已知事件吻合度需结合命主实证确认。

### 最终结论

**6.15庚申日巳时（辛巳劫财）** 以8.5分胜出，远超第二名6.5分。
