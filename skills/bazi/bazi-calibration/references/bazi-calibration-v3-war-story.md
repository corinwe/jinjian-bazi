# 八字校准 v3.0 实战踩坑记录

> **背景：** 2026-06-04 全库审计发现家族八字全部错误，经老板纠正后重新校准为 v3.0 最终版
> **关键教训：** 日柱基准错误 → 全盘皆错；月柱索引偏移1位 → 全盘皆错

---

## 一、日柱基准选择（致命错误）

### ❌ 错误假设

```
已知 2020-01-01 日柱 = ???

❌ 最开始错误假设：2020-01-01 = 丙午日 (gan=2, zhi=6)
  导致所有日柱计算结果偏位
  1980-08-06 算成甲寅日（实际是辛亥日）
  全盘偏差，无法通过任何验证
```

### ✅ 正确基准

```
✅ 2020-01-01 = 癸卯日 (gan=9, zhi=3)
   来源：日历网 (calendar.8s8s.net)
   
   delta = (target_date - 2020-01-01).days
   gan_idx = (9 + delta) % 10
   zhi_idx = (3 + delta) % 12
```

### ⚠️ 基准选择的补救措施

```
如果对某个基准有疑虑：
1. 用多个独立来源验证同一个日期
2. cantian.ai、calendar.8s8s.net 是两个可靠来源
3. 选一个日期互验：1980-08-06 = 辛亥日 (cantian.ai确认)
   → 用这个反推基准日是否正确
```

## 二、月柱索引偏移（最隐蔽的 Bug）

### ❌ 错误代码

```python
ZHI = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥']
# ZHI.index('未') = 7
zhi_idx = 7  # ❌ 直接用了7
gan_idx = (start['庚'] + 7) % 10  # = (4+7)%10 = 1 → 乙
# 月柱 = 乙未 ❌（应该是癸未）
```

### ✅ 正确代码

```python
# 五虎遁月支索引：寅=0，因为正月是寅月
# ZHI中寅=2，所以必须减2
zhi_idx = (ZHI.index('未') - 2) % 12  # = (7-2)%12 = 5  ✅
gan_idx = (start['庚'] + 5) % 10  # = (4+5)%10 = 9 → 癸
# 月柱 = 癸未 ✅
```

### 根本原因

```
ZHI 列表从 子=0 开始，但月柱计算从 寅=0（正月）开始
差异 = 2 位偏移
```

## 三、Python 负数取模行为

### ⚠️ Python 的 % 对负数的结果与数学期望不同

```python
# delta = (1980-08-06 - 2020-01-01).days = -14392

# Python 中：
-14392 % 10  # = 8（不是 -2！）
-14392 % 12  # = 8

# 在索引计算中直接用：
gan_idx = (9 + (-14392)) % 10  # = (9 + 8) % 10 = 7 → 辛 ✅
# 这是正确的！Python 的 % 自动转为正数
```

### 关键认知

```
⚠️ Python 的 % keeps the sign of the DIVISOR (always positive for positive divisors)
所以计算 delta 后直接：
  gan_idx = (base_gan + delta) % 10  # 不管 delta 正负都正确
  zhi_idx = (base_zhi + delta) % 12  # 不管 delta 正负都正确
不需要额外转换！
```

## 四、大运方向校验

### 口诀

```
阳男阴女顺走，阴男阳女逆走

家主庚申年（阳年），男 → 阳男 → 顺排？
  庚申是阳还是阴？庚=阳，阳年男=阳男 → 顺排 ✅
```

### ⚠️ 实际错误

```
知识库中所有家主文件用的是阳男逆排（❌）
阳年男性应该顺排 ✅
```

## 六、验证清单（排盘后逐一过）

```yaml
□ 年干正确？  GAN[(year-4)%10]
□ 年支正确？  ZHI[(year-4)%12]
□ 月支正确？（节气检查，非农历月）
  立春→寅、惊蛰→卯、清明→辰、立夏→巳
  芒种→午、小暑→未、立秋→申、白露→酉
  寒露→戌、立冬→亥、大雪→子、小寒→丑
□ 月干正确？（五虎遁 + zhi_idx -2 偏移）
  甲己→丙、乙庚→戊、丙辛→庚、丁壬→壬、戊癸→甲
□ 日柱正确？（用至少一个在线工具互验）
  可靠源：zydx.top、cantian.ai
□ 时柱正确？（五鼠遁 + 时辰索引）
□ 大运方向正确？
  阳男阴女顺，阴男阳女逆
□ 起运节气正确？
  顺排找出生后下一个节气
  逆排找出生前上一个节气
  ⚠️ 一个月两个节气！不要跳过一个！
```
