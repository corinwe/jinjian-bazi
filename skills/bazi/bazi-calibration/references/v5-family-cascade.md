# V5家族连环分析工作手册

> **编写背景**：2026年5月30日，主母成、儿子元、父亲、母亲、外甥五位家族成员并行分析实战总结。
> **基于**：bazi-master-agent 标准分析流程 + 金鉴真人修正规则（V5体系）
> **用途**：多命主家族场景的标准化执行指南

---

## 一、适用范围

| 场景 | 适用 | 说明 |
|:----|:---:|:------|
| 2-5位家族成员同时分析 | ✅ | 核心场景 |
| 6人以上家族 | ✅ | 分批执行，每批最多5人 |
| 单人八字分析 | ❌ | 直接用标准分析流程即可 |
| 跨代际对比（2代/3代） | ✅ 特别适合 | 可加「家族传承分析」模块 |

---

## 二、执行前准备

### 2.1 信息收集清单

```
对每个命主需要收集：
  □ 完整八字（年柱+月柱+日柱+时柱）
  □ 日主/性别
  □ 当前大运 + 已过大运 + 未来大运（至少提供当前和下一大运）
  □ 已知信息（职业/学业/婚姻状况/重大事件）
  □ 与其它家族成员的关系（夫妻/父子/母女等）
```

### 2.2 排盘确认

对每个命主做以下排盘确认后再开始分析：

| 检查项 | 说明 |
|:------|:------|
| 年柱确认 | 按立春分年界，非春节 |
| 月柱确认 | 按24节气划分，非公历/农历月初 |
| 日柱确认 | 用万年历/排盘软件校准 |
| 时柱确认 | 确认时辰的准确性（子时尤其注意早晚子时） |

## 三、执行顺序与流程

### 3.1 标准化执行序列

```
第一步：命主排序
  按辈分从高到低（祖辈→父辈→子辈）或用户指定顺序
  ↓
第二步：逐个分析（每命主独立运行标准分析流程）
  命主A：16步全流程 → 保存为文件A
  命主B：16步全流程 → 保存为文件B
  ...（以此类推）
  ↓
第三步：交叉验证
  跨命合冲关系检查 → 补充到各报告中
  ↓
第四步：可选——输出家族能量总图
  跨代际五行极值追踪
  家族命理基因传承总结
```

### 3.2 标准分析流程 → 报告格式映射

| 决策树步骤 | 报告章节 | 必须/可选 |
|:----------|:---------|:---------:|

| 第2步：身强弱 | 得分明细 + ⚠️假旺真弱排查 | **必须（含验证框）** |

## 五、特殊日主处理速查

### 5.1 庚金在未月（经典脆金）

| 步骤 | 处理 |
|:----|:------|
| 得令 | 0分（未月燥土脆金，不得月令生） |
| 假旺排查 | 必做——天干透三金表象强但无酉真根 |
| 关键救应 | 辰/丑湿土为命局最重要救应 |
| 用神 | 辰丑湿土 > 申酉金 |

### 5.2 辛金无酉时（真根在酉不在申）

| 步骤 | 处理 |
|:----|:------|
| 申金 | 劫财根，+10~15分（非真根） |
| 酉金 | 真根（禄位），+30分（如不在局则帮扶有限） |
| 常见陷阱 | 误判申为辛金强根 |

### 5.3 壬/癸水在未月

| 步骤 | 处理 |
|:----|:------|
| 得令 | 0分（水囚于未月） |
| 子/亥为根 | 子为劫财根（+10分），亥为真根（+30分） |
| 子未害 | 子水根受损，需特别标注 |
| 用神 | 申酉金印星 > 亥子水比劫 |

### 5.4 丙火在巳月（建禄格）

| 步骤 | 处理 |
|:----|:------|
| 得令 | +20分（火旺于巳月） |
| 真根 | 巳为丙火真根（禄位）+30分 |
| 从格排查 | 天干有财官破格则不从 |
| 用神 | 极强→泄克耗（土>水>金） |

## 七、坑洞警示（来自实战）

### ⚠️ 坑洞1：大运起运计算冲突

**现象**：你手动计算的起运/大运与用户提供的大运表不一致。
**处理**：**以用户提供的大运表为准**。即使在备注中标注差异，也不要更改用户数据。

**根本原因**：
- 不同排盘软件算法不同
- 节气交割时刻不同（GPS真太阳时 vs 标准北京时间 vs 地方时）
- 用户从其他命理师处获得的大运排法

### ⚠️ 坑洞2：假旺真弱漏判

**容易遗漏的场景**：
- 天干透出多个比劫（表象身强）但无真根
- 月令为印但燥土脆金（名义生实则不生子金）
- 三合/三会局看似助身但被刑冲破害

**应对**：强制排查不可跳过，即使看起来状态简单。

### ⚠️ 坑洞3：家族排序混乱

**表现**：多命主分析时，切换命主导致分析逻辑混乱
**应对**：严格按照「辈分从高到低」或用户指定顺序，完成一个再开始下一个

### ⚠️ 坑洞4：财富量级脱离实际

**表现**：给学生/初入职场者预测过高收入
**应对**：必须按命主当前社会身份锚定——学生说学业不说收入，老人说安康不说发财

### ⚠️ 坑洞5：十神缺失遗漏

**表现**：漏掉了某个十神缺失的整体影响
**应对**：第16步不可省略——"缺正印"往往就是命主人生困境的解读书眼

## 八、文件输出规范

### 8.1 命名规则

```
八字深析_称谓_V5.md
例：八字深析_主母成_V5.md
```

### 8.2 文件结构——命主表头统一格式

每份文件包含（按顺序）：

**头部（报告最上方）必须包含以下完整信息：**

```markdown
> **八字**：XXXX XX XX XX
> **日主**：XX（阴阳） | **性别**：男/女
> **格局**：[正八格名称 + 特殊组合]
> **身强弱**：[七种状态之一]（XX分/100分制）
> **命格评分**：**X.X/10（说明）**  ← 必填！10分制综合评分
> **用神**：... | **忌神**：...
> **当前大运**：干支（XX-XX岁，年份范围）
```

| 评分区间 | 命格描述 |
|:--------|:---------|
| 8.0-10 | 上等命格 |
| 7.0-7.9 | 中上等命格 |
| 6.0-6.9 | 中等偏上 |
| 5.0-5.9 | 中等命格 |
| 4.0-4.9 | 中等偏下 |
| 0-3.9 | 普通命格 |

**主体：第1-16步完整分析**

**结尾必须包含：**
- 🎯 综合评分总表（含命格总分行）
- 金鉴真人跋（文言/半文言风格总结）
- 初判待验声明

### 8.3 文件保存位置

- 统一存放在 `/root/` 目录
- 多人分析时，所有文件在同一目录即可
