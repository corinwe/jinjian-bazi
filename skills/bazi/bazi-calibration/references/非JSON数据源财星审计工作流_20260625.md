> **创建时间**：2026-06-25
> **触发案例**：仔仔/涛涛/杨昌玉/魏齐/魏弋甯 5人财星审计
> **核心问题**：5人均不在`家族八字核心数据源_v1.json`中，需手动审计旧报告

---

## 触发条件

人物不在JSON数据源中，但有旧版报告（v1.0 pre-ALLNEW）需要做财星校准。

## 标准工作流

### 第1步 — 确认人物是否在JSON数据源中

```bash
cat /root/weiwuji-knowledge-base/07-国学哲学/八字命格/02-人物档案/家族八字核心数据源_v1.json
# 搜姓名：json.load → 遍历各['姓名']，模糊匹配
```

- **在JSON中** → 用bazi-pipeline.sh跑引擎，读engine.json数据
- **不在JSON中** → 只能手动审计旧报告（第2步起）

### 第2步 — 读取旧报告头部获取基本信息

从报告§1读取：
```
姓名 | 八字 | 日主 | 性别 | 身强弱 | 财星分数 | 格局
```

### 第3步 — 手动计算财星分数（月令40分规则）

**规则**（九龙道长原始规则）：
```
月令本气财 = 40分 × 100%
月令中气财 = 40分 × 60% = 24分
月令余气财 = 40分 × 30% = 12分
天干财 = 12分（透干直接计）
日支本气财 = 12分 × 100%
日支中气财 = 12分 × 60%
日支余气财 = 12分 × 30%
年支财 = 8分(年支)×藏干百分比
时支财 = 12分(时支)×藏干百分比
```

**关键注意**：
- 月令不是财星（本气不是财）→ 月令不财计分
- 月令本气是财星才适用「在月看父母/平台」规则；余气财星不算引化点
- 财星五行由日主决定：我克者为财（阳我克阳=偏财，阳我克阴=正财，阴我克阴=偏财，阴我克阳=正财）

### 第4步 — 对比报告与手动计算的财星分

| 偏差 | 行动 |
|----|------|
| < 2分 | 做标记但不一定修改（可能是四舍五入差异） |
| 2~10分 | 修改§1和§8的财星分数，调整财富等级描述 |
| > 10分 | 大修：§1/§8/财富五级表/身弱不胜财分析/版本记录全面更新 |

### 第5步 — 修正§1总览表

修改财星分数行：
```
旧：财星分数 | 8分（极低）
新：财星分数 | **24分**（月令戌丁火偏财12分+日支巳丙火正财12分=24分）
```

### 第6步 — 修正§8财星评分/财富等级表

- 更新§8.1财星评分行
- 更新财富五级对照表（巨富/大富/中富/小富/小康/温饱逐条检查）
- 更新身弱不胜财/身强财弱等分析段落
- 补充§8.x如财富大运匹配表（如果缺此节且行数不足1500）

### 第7步 — 报告行数验证

确保报告 ≥ 1500行。如不足，补充§8.x子节：
- §8.5 财富大运匹配总表（大运/年龄/财星条件/财富潜力/核心建议）
- §8.6 最佳发财年份一览表
- §8.7 财库与财源分析
- §8.8 财富积累策略补充

### 第8步 — 版本记录更新

添加v1.1版本记录：
```
| v1.1 | 2026-06-25 | 财星修正：8分→24分（月令戌丁火12+日支巳丙火12=24） |
```

### 第9步 — §1与§8一致性强制验证

```python
# 必须对比§1和§8的财星分数，不一致以§8.1为准修正§1
grep -n '财星分数' report.md
# §1行和§8行应显示相同数值
```

### 第10步 — 推库

```bash
git add -A
git commit -m "人名 v1.1财星校准：旧值→新值"
git push
```

## 特殊场景处理

### 场景A：人物只有推算八字，无精确公历日期

如仔仔（己亥甲戌癸巳癸丑）和涛涛（壬辰庚戌丙寅庚寅）：
- 无精确公历日期 → 无法通过bazi-pipeline.sh跑引擎
- 无法做官网验证（日柱确认）
- **只能手动审计旧报告**
- 财星分数仍可手动计算（基于八字本身，不依赖出生日期）

### 场景B：人物有精确公历日期但年柱跨年

如杨昌玉（2003-01-08出生，农历壬午年腊月初六）：
- 引擎输出：癸未年（因用公历2003）
- 官网验证：壬午年（因农历还在壬午年）
- **以官网验证为准**
- 在pipeline报告中标注⚠️跨年差异

### 场景C：报告行数不足1500行

补充方法（按优先级）：
1. 添加§8.x财富分析子节（大运匹配表/发财年份/财库分析等）
2. 补充§11学业文昌分析
3. 补充§12事业大运详解
4. 补充§20.3人生阶段寄语
