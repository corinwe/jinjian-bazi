# 金鉴真人八步法速查

## 八步法

```
Step 1：排盘·先找六恶神（七杀伤官枭印劫财灾煞血刃）
Step 2：判身强弱（得令/得势/得地）
Step 3：格局判定 + 身财关系 + 财星入墓/入库
Step 4：大运全盘（起运年龄 + 每十年一步）
Step 5-6：逐流年关键年份 × 倍数计算 × 事件判定
Step 7：墓库分析（如有辰戌丑未）
Step 8：断语总结 + 完整总结表
```

## 话术五则

1. **绝对化表述**：不说"可能"，说"断他"、"必有"
2. **量化倍数**：用精确倍数描述严重程度
3. **互动验证**："这年心情怎么样？您自己反馈一下"
4. **故事+案例**：穿插典故或名人案例
5. **给解法**：先断灾，后给预防方案
