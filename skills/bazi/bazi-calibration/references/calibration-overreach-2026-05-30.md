## 四个错误案例的根因分析

### 错误1：家主2002壬午年 — "伤官见官→冲突离职" ❌

**我的分析链条：**
- 丙戌大运（七杀+印）
- 壬午年：壬水伤官透干 → 丙火七杀在大运天干 → 伤官见官信号
- 22岁初入社会 → 推断：与上级冲突离职

**为什么错了：**
- 伤官壬水与七杀丙火只是"天干见天干"的远合关系，不是紧邻冲克
- 壬水伤官虚浮无根（壬在命局地支无根）→ 力量极弱
- 22岁刚大学毕业（正常应届生身份），正常轨迹是"求职/入职"
- 信号强度不足1/10级 → 完全不应作特殊事件描述

**正确基调：** 22岁大学毕业求职/入职的普通一年，最多说"工作上有些自己的想法，但整体平稳"

---

### 错误2：家主2009己丑年 — "双土印星→跳槽大晋升" ❌

**我的分析链条：**
- 丙戌大运末，即将换丁亥大运
- 己丑年：双土印星 → 用神年 → 推断大晋升/跳槽

**为什么错了：**
- 己丑确实是用神（土生金），但印星本质是"稳定/滋养/保守"，不是"剧变"
- 丙戌大运末 → 换运前一年，能量衰减期（大运末年6折）
- 印是地支缓进，不是天干突进
- 把"印到"等同于"剧烈晋升"是错误的——印到意味着稳步提升

**正确基调：** 工作中得到上级或长辈认可的一年，有稳步的提升，但不是剧烈跳槽

### 错误3：主母2015乙未年 — "月柱伏吟→二胎/家庭质变" ❌

**我的分析链条：**
- 庚戌大运（最压抑十年）中
- 乙未年：月柱丁未被伏吟 → 推断家庭重大事件

**为什么错了：**
- 伏吟在月柱 + 庚戌忌神大运 → 应理解为"重复枯燥的日常"，不是"重大事件"
- 伏吟的断事必须结合大运吉凶：
  - 伏吟+喜用大运 = 好事重复（如升职/加薪）
  - 伏吟+忌神大运 = 压抑/重复/停滞
- 没有考虑庚戌大运对主母的压制作用

**正确基调：** 家庭生活/工作陷入重复和压抑感，没有大的变化，但内心疲惫

---

### 错误4：主母2022壬寅年 — "水木双用神→职业大突破" ❌

**我的分析链条：**
- 辛亥大运（一生唯一用水神的大运）中
- 壬寅年：水木双用神 → 推断职业大突破收入飞跃

**为什么错了：**
- 用神到 ≠ 大突破。用神到应理解为"困境缓解"
- 壬寅年对金日主来说，是用神水木到 → "从被埋没到出土"
- 寅亥合（岁运合）加强了能量，但更多是"破局感"而非"飞跃感"
- 用神力度评级：壬水食神在天干虚浮无根（壬在命局无本气根）
  寅木偏财在地支有根（寅中甲木本气）→ 偏财有根，但食神虚浮
  → 整体信号约4/10级，可做"改善/破局"描述，不可做"大飞跃"

**正确基调：** 从压抑的环境中走出来，找到了适合自己的节奏，收入有改善，但非大飞跃

## 信号强度分级系统（金鉴真人·过推防治）

```
1/10级 — 天干常见互动（单一合/冲）
  → 几乎不影响正常轨迹，不作事件描述
  → 例：天干见一个合，但不紧邻

2-3/10级 — 地支半合/六合/轻微冲害
  → 有变化但不剧烈
  → 可用"有轻微波动""略有好转"等温和表述

4-5/10级 — 三合局引动/用神真根到/天干五合（隔位）
  → 可触发中等事件描述
  → 可用"有较大的变化/机遇""需要关注"

6-10/10级 — 岁运并临/三刑+七杀/枭神夺食紧邻/用神真根+三合
  → 可触发重大事件描述
  → ⚠️ 只有在6/10级以上才说重大事件
```
