# 34人精确评分数据格式说明

> **编制人**：金鉴真人
> **编制时间**：2026年06月22日
> **来源**：本会话中34位外部客户评分从估算升级为精确评分的全量计算
> **数据文件**：`/root/all_34_scoring_merged.json`（67条记录）

## 数据格式

```json
{
  "id": "20",
  "name": "王子",
  "version": "主推",
  "time": "未",
  "wealth": 8.3,
  "career": 9.0,
  "marriage": 7.0,
  "education": 7.0,
  "health": 6.7,
  "total": 8.03
}
```

## 字段说明

| 字段 | 说明 | 公式 |
|:----|:----|:----|
| id | 客户编号（19~52） | |
| name | 姓名 | |
| version | 主推/最差 | |
| time | 时辰（不含「时」字） | |
| wealth | 财富综合分 | 原局财分×0.7 + 大运财分×0.3 |
| career | 事业综合分 | 原局事分×0.7 + 大运事分×0.3 |
| marriage | 婚姻综合分 | 原局婚分×0.7 + 大运婚分×0.3 |
| education | 学业综合分 | 原局学分×0.7 + 大运学分×0.3 |
| health | 健康综合分 | 原局康分×0.7 + 大运康分×0.3 |
| total | 综合总分 | wealth×35% + career×30% + marriage×15% + education×10% + health×10% |
