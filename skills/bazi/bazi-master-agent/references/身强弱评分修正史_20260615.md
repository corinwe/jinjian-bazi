# 身强弱评分修正史（2026-06-16 终版）

## 🔴 原始错误 → 🟢 最终正确规则

### 错误①：加减相减（已修正✅）
**旧版**：印比加分 + 食伤财官杀减分 → 相减得总分
**修正**：只算加分（印+比劫），不加不减
**原文依据**：09号文档第105行「只算加分就行了，不用算减分。总共100分，你算他加了多少分，不就知道他日元多少分了？」
**老板原话**：「你不得分的项不用管它就行了，你只算得分的项那就是它总分啊」

### 错误②：己土对辛金判成正印（已修正✅）
**旧版**：己土生辛金 = 正印
**修正**：己土(阴)+辛金(阴) = 阴阳相同 = 偏印
**口诀**：生我者为印，比阴阳定正偏；阴阳相同是偏印，不同是正印

### 错误③：燥土未月令计印分（已修正✅·老板确认保留）
**旧版**：未月己土偏印=计40分
**修正**：未是燥土(含丁火)，燥土不生金 → 不计分
**规则**：辰丑=湿土生金 ✅ | 未戌=燥土不生金 ❌
**老板确认**：「好像是说燥土不生金 这个先保留吧」（2026-06-16）

### 错误④：月令比劫不计分（已修正✅）
**旧版**：月令比劫不计分（传统派规则）
**修正**：月令比劫同样计分（素材09行89原文：「这个算，因为这是比劫」）
**关键辨析**：九龙道长「比劫不入格局」仅指格局判定（正八格），**不是**身强弱评分（素材25行93）

### 错误⑤：印在所有位置都计分（已修正✅·2026-06-16）
**旧版**：「印星在所有位置都算（天干+地支藏干本/中/余气均计）」
**修正**：印**只**在月令本气算分，月令中/余气不计，其他任何位置不计
**原文依据**：
  素材20行1038：「印除了在月令这里本气算分，印在其他的任何宫位都不算分」
  「月令中气的不算分...月令余气算分不算？不算」
  素材09行89：「印在月令算，但是印在其他的工位是不算的」

## 🟢 最终评分法（2026-06-16 老板确认）

```
✅ 月令本气印 = +40分    （"印在月令本气算分"）
❌ 月令中气/余气印 = 0分 （"月令中气的不算分"）
❌ 其他位置印（年/日/时/天干）= 0分 （"印在其他任何宫位都不算分"）
✅ 月令本气比劫 = +40分
✅ 月令中/余气比劫 = +该位置分数（"比劫算"——所有位置都算）
✅ 其他位置比劫 = +该位置分数
✅ 大运流年印 = +分数（大运流年不分位置）
❌ 燥土（未/戌）对庚辛金印不计分（金鉴·老板确认保留）
❌ 食伤/财/官杀等不扣分
```

核心口诀：**印只看月令本气，比劫满盘都算；燥土对金不算，大运印分另计。**
