# 身强弱评分修正史（2026-06-16 终版·九龙道长原始规则）

## 📜 最终规则总表（2026-06-16老板确认）

### 加分规则（九龙道长原始·7处原文验证）

```
✅ 月令本气印 = +40分    （素材20行1038）
❌ 月令中气/余气印 = 0分 （同）
❌ 其他位置印（年/日/时/天干）= 0分 （"印在其他任何宫位都不算分"）
✅ 月令本气比劫 = +40分
✅ 月令中/余气比劫 = +该位置分数（"比劫算"——所有位置都算）
✅ 其他位置比劫 = +该位置分数
✅ 大运流年印 = +分数（大运流年不分位置）
```

### 燥土规则（条件版·素材05+09）

```
被火引化（天干丙/丁/卯戌合火/寅午戌合）→ 当火看 → ❌ 对金不计分
被水灭火（天干壬/癸）→ 当纯粹土看 → ✅ 对金计分
无火无水引化（天干戊/己/庚/辛等）→ 当土看 → ✅ 对金计分
```

### 身强弱分段（素材15行161+17行125+25行389）

```
<40分   → 身弱   （固定喜印比帮身，忌克泄耗）
40-60分 → 中庸  （素材25行389：最佳状态，无固定喜忌，随大运流年变）
>60分   → 身强   （固定喜克泄耗，忌印比帮身）
从弱0分→恒定50分（所有五行皆喜用）
从强>100分→恒定20分（喜印比）
```

### 核心口诀

```
印只看月令本气，比劫满盘都算；
燥土条件看水火，从弱50分恒定；
40-60中庸不分段，无固喜忌随运变；
帮身之时喜克耗，克耗之时喜印比。
```

## 🔴 修正历史

### 错误①：加减相减（已修正）
**旧**：印比加分 + 食伤财官杀减分
**新**：只算加分（印+比劫），不加不减
**依据**：09号文档第105行

### 错误②：己土对辛金判成正印（已修正）
**旧**：己土生辛金 = 正印
**新**：己土(阴)+辛金(阴)=阴阳相同=偏印

### 错误③：月令比劫不计分（已修正）
**旧**：月令比劫不计分（误读"比劫不入格"为身强弱规则）
**新**：月令比劫同样计分（素材09行89）

### 错误④：印在所有位置都计分（已修正）
**旧**：印星在所有位置都算
**新**：印只在月令本气算分
**依据**：素材20行1038

### 错误⑤：中和三档细分（已修正）
**旧**：40-49偏弱/50正中和/51-60偏强
**新**：40-60不分段，统称中庸，无固定喜忌随大运变
**依据**：素材25行389+369
