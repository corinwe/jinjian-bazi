### 矛盾2：「本气根」定义在不同文件中指代不同概念

**涉及文件：**
- `SKILL.md` 第447-454行（4.1节）
- `references/tiangan-luodi-checks.md` 第17-21行
- `references/ref-kongwang-muku-shichen-fumu.md` 第209-228行（第6节）

**SKILL.md 4.1 定义「本气根（禄位/帝旺）」：**
- 本气根 = 禄位或帝旺
- 戊→巳(禄)、午(帝旺)
- 己→午(禄)、未(帝旺)
- 得分：+30分/个

**tiangan-luodi-checks.md 定义「本气根（地支藏干）」（第17-21行）：**
- 戊→辰、戌、丑、未 (10成力量)
- 己→辰、戌、丑、未 (10成力量)
- 巳/午 → 戊是余气/中气，不是本气

**ref-kongwang 第6节 定义「本气根（地支藏干）」（第211-227行）：**
- 戊→巳（本气??）、午（本气??）、辰（余气戊）、戌（本气）
- 己→未（本气）、午（中气）

**矛盾分析：**
- 在藏干体系中，巳中本气是丙不是戊，说"巳是戊的本气根"是错误的
- tiangan-luodi-checks说戊的藏干本气根是辰戌丑未（全部土库），而ref-kongwang说戌和巳午都是戊的本气根（巳午实际不是）
- 三份文件对戊己土"本气根"的定义互不一致

| 文件 | 戊的本气根 | 己的本气根 | 定义标准 |
|:----|:-----------|:-----------|:--------|
| SKILL.md 4.1 | 巳、午 | 午、未 | 禄位/帝旺 |
| tiangan-luodi-checks | 辰、戌、丑、未 | 辰、戌、丑、未 | 藏干本气 |
| ref-kongwang 第6节 | 巳、午、辰(余气)、戌 | 未、午(中气) | 混合标准 |

**建议统一方案：**
1. SKILL.md 4.1 应将「本气根（禄位/帝旺）」改为「禄旺根」或「强根（禄位/帝旺）」，以区别于藏干体系的「本气根」
2. 统一tiangan-luodi-checks和ref-kongwang中戊己土的藏干表
3. 戊土的藏干标准根应为：辰(本气戊)、戌(本气戊)、丑(本气己)、未(本气己)；巳/午仅为禄旺位但非藏干本气

### 矛盾3：流年「冲破原局合化」与「贪合忘冲」的潜在冲突

**涉及文件：**
- `references/ref-liunian.md` 第96-106行（6.1贪合忘冲） vs 第339-352行（6.2流年太岁优先级最高）
- `SKILL.md` 第1207-1211行（12.4要素三）

**贪合忘冲规则（ref-liunian.md 6.1）：**
> 流年某字既有「冲」又有「合」→ 优先合化，不冲了

**流年大哥大规则（ref-liunian.md 6.2）：**
> 流年太岁憋了60年才值班一次，能量最大，可以冲破原局合化

**潜在冲突场景：**
流年某字有两个关系同时存在：
- 它与原局A字有「合」关系
- 它与原局B字有「冲」关系
- 同时该字还试图冲破原局的另一个三合局

在这种情况下，贪合忘冲说"合优先不冲了"，但流年大哥大说"流年最大可冲破原局合化"——两个规则给出相反的导向。

**建议统一方案：** 明确两个规则的适用范围：
- 贪合忘冲：适用于**同一个字**既有「冲」又有「合」的关系时，取合弃冲
- 流年冲破原局合化：适用于流年字与原局合化中的某个字发生**刑冲破害**时，破坏原局合化
- 两者不冲突：贪合忘冲管的是冲合取舍（不冲了），流年最大管的是破合（破掉原有的合）
- 建议在ref-liunian.md中增加一个综合示例来解释两者的协同关系

### 矛盾4：刑冲破害优先级表的能量分数与伤害分值矛盾

**涉及文件：**
- `SKILL.md` 第3.9节（能量倍数速查表）
- `SKILL.md` 第12.2节（犯太岁能量扣分）
- `references/ref-liunian.md` 第0A.3节（能量伤害量化表）

**问题：**
- SKILL.md 3.9：三刑(丑未戌)=15倍(土) — 这是能量加强（正面），与伤害扣分不在同一维度

- 15倍（能量倍数表）→ 用于分析"土的能量被加强了多少"
- -70分（伤害量化表）→ 用于分析"命主受到多大的伤害"

**建议统一方案：** 在SKILL.md 3.9表和ref-liunian.md 0A.3表中分别标注用途，例如：
- 能量倍数表（3.9）标注为「五行能量变化倍数」— 仅用于分析五行力量变化
- 伤害量化表（0A.3）标注为「灾祸伤害分值」— 仅用于断事应灾等级

### 矛盾6：日柱纳音生日柱纳音（疑似笔误）

**涉及文件：**
- `references/ref-nayin-fengshui-fugui.md` 第79行

**原文：**
```
└─ 日柱纳音生日柱纳音 → 自身气运好
```

"日柱纳音生日柱纳音" = 自己生自己，没有意义。应该是笔误。

**推测正确版本：**
```
└─ 时柱纳音生日柱纳音 → 自身气运好
```
或
```
└─ 日柱纳音生年柱纳音 → 自身气运好
```

**建议统一方案：** 修正为"时柱纳音生日柱纳音"或"日柱纳音生年柱纳音"。

### 矛盾7：四吉神中的「印」是否包含偏印（枭）

四吉神（印、食、官、财）→ 和谐、有序、坦然

### 矛盾8：身强偏财格喜比劫 vs 四恶神中劫财为恶

**涉及文件：**
- `SKILL.md` 第665行（6.1节偏财格）
- `SKILL.md` 第607行（5.3节四恶神）

**SKILL.md 6.1 偏财格：**

喜用：食伤（食伤生财）、比劫（能担财）
```

```

- 偏财格本身就不怕比劫（偏财为众人之财），比劫反而能帮身担财
- 但四恶神的分类是基于十神本质属性（劫财的本质是争夺、消耗）
- 同一个十神在不同格局中有不同表现是正常的

**建议：** 可加注明确说明"四恶神是本质分类，不意味着任何时候都是忌神；偏财格局中比劫可为喜用"，消除困惑。

### 矛盾10：四柱空亡 vs 纳音空亡的能量表述不一致

**涉及文件：**
- `references/ref-kongwang-muku-shichen-fumu.md` 第44-49行（1.4节）
- `references/ref-nayin-fengshui-fugui.md` 第99-102行

**ref-kongwang 1.4：**
```
空亡不代表字消失，只是力量减半
```

**ref-nayin 1.3：**
```
纳音空亡的能量约等于正五行空亡的70%
```

**问题：** 正五行空亡「力量减半」= 50%，纳音空亡「约等于正五行空亡的70%」= 70% × 50% = 35%。但金空则鸣的效果反而大于正五行空亡——这里"效果大于"和"能量约70%"是不同概念，容易混淆。

**建议统一方案：** 明确区分「能量留存比例」和「吉凶效果强度」两个维度，在ref-nayin中加注说明。
