

## 二、选房前准备（实用必读）

```yaml
1. 预算：各地房价差异巨大（深圳2-30万/平 vs 三四线1万/平）
   地段（学区/市中心/繁华）直接影响房价
   贵的房子通常有更好的环境、配套和物业

2. 生活配套（🏥 第一优先！）：
   医院 - 急救时间就是生命！父亲头孢配酒急救案例（3分钟路程救命）
   超市/菜市场、理发店、药店、饭店
   学校（学区房溢价明显）

3. 现房 vs 期房（⚠️ 2021年起大量烂尾）：
   期房=开发商资金链断裂→血汗钱打水漂
   作者亲身经历：2013年买安置房，十年后仍未盖好
   ✅ 推荐现房（价格贵一点但无风险）

4. 物业质量：
   直接影响居住体验——卫生/绿化/安全
   物业差→垃圾堆积、臭气熏天→影响升值和居住舒适度

5. 停车位配比：
   老小区下午4点就没车位
   选房时确认车位配比
   电动车/充电桩也要考虑
```

## 三、小区风水四大核心原则（缺一不可）

> 完整理论见 `bazi-remission-methods` §12.2

```yaml
【原则一：有环抱】✅
  小区前面有静水（湖）或路呈环抱状
  环抱→藏风聚气
  反弓水/反弓路→❌ 大凶

【原则二：坐山旺】✅
  坐山方向（大门朝向的对面）有高大建筑或山体做靠山
  无靠山→❌ 天斩煞/坐山无力——风水楼盘第一大忌

【原则三：没煞气（关鬼位）】✅
  克坐山的方位（关鬼位）不能有高大建筑
  关鬼位有高大建筑→压力大/公检法找事/严重者有牢狱之灾
  例：子山午向（坐山=水）→关鬼位=离火宫（火克水）
      朝向=大门朝南=离火宫=关鬼位→进的全是关鬼气→白送也别住

【原则四：有用神】✅
  财位（坐山所克）有高大建筑→旺财
  印位（坐山所生）有高大建筑→平安/出高材生/学习好
  食神位（生坐山）有高大建筑→福气/子孙兴旺/财运好
  比劫位（同坐山）❌ 不喜高大→太旺容易破财
```

## 四、选址十要诀

### 4.1 小区要方正（忌异形）
```yaml
圆形/三角形/异形小区 → 四大皆空（坐山空/朝向空/两边空）
  案例：圆形小区盖好7年入住率不到40%
  ✅ 方正小区气场稳定
```

### 4.2 不买小区四个角（辰戌丑未）
```yaml
辰(东南角) 戌(西北角) 丑(东北角) 未(西南角)
四角楼最容易破损→掉漆/裂缝/墙皮脱落
原理：辰戌丑未代表土，土打架（辰戌冲/丑未冲/辰丑破/未戌破）
三年一个辰戌丑未年→三年破损一次
```

### 4.3 不临街 / 不买最前排
```yaml
临街三大问题：
  ① 噪音（汽车声晚上不停）
  ② 灰尘/尾气污染
  ③ 易失窃（案例：入住四个月被盗）
风水格局：临街房=军队的兵和卒在前面当炮灰
         将帅在后→普通住宅业主是"将"，不该在炮灰位
```

### 4.4 中间为贵
```yaml
小区中间位置的房子比周围贵10%-20%
原因：藏风聚气最好
✅ 中间最抢手
```

### 4.5 子山午向（坐北朝南）最好
```yaml
北半球最佳朝向
原理：
  冬天阳光斜射进屋→温暖杀菌
  夏天阳光直射→进屋少→不热
  大门在离火宫（南）→进火气
  坐山为水→火克水=我克者为财→进财气
```

### 4.6 楼间距要够（采光通风）
```yaml
间距太近→低楼层（1-3楼）采光差
阴暗潮湿→滋生细菌→人得病
```

### 4.7 远离外部阴煞
```yaml
500米内不要有：
  寺庙（阴气重，孤魂野鬼聚集）
  坟地/火葬场（阴阳相隔）
  监狱（怨气重）
  垃圾站/污水处理厂（污秽气）
  高架桥/立交桥（气流混乱）
```

### 4.8 不买海景房
```yaml
海风大（日常3级起步，常达6-7级）
房子比普通住宅早破损20年以上
  正常30年维修→海景房5年就开始出问题
当地人不买→专门卖给外地人
  想感受→租民宿（一天几十到百元）
```

### 4.9 不买法拍房自住
```yaml
法拍房：原房主因财务纠纷被银行强制回收
原房主住这个房子都破产/坐牢了→磁场有坏信息
✅ 可买来出租（便宜，3-5折起拍）
❌ 不可搬进去住
```

### 4.10 不买比邻居高的房子
```yaml
别人盖3层→你也盖3层
比邻居高→别人嫉妒→背后议论→败家
枪打出头鸟，和光同尘
```

## 五、选单元楼

```yaml
1. ✅ 买中间不买两边
   边户三面悬空→冬天温度低几度（同样供暖）
   风水=坐椅子只有一边扶手→做事少人帮助
   中间有左右护→做事有朋友相帮

2. ✅ 背后有靠山
   楼后要有更高的楼或山做靠
   背后落空→无贵人扶持→一生难遇明主
   案例：许昌市政府搬到皇后坟为靠山

3. ✅ 楼前要平阔
   一眼开阔→事业好
   前面有水池/喷泉/假山流水→更佳

4. ✅ 楼间距够（见4.6）
```

## 六、选楼层

### 6.1 不买天地层
```yaml
❌ 一楼（地）：
  采光差（高楼遮挡）/高空抛物/脏
  低层建筑（6-8层）买一楼问题不大
  高层建筑（二三十层）不要买一楼和顶层

❌ 顶层（天）：
  太热/风大（无法藏风聚气）
  案例：开发商给自己留顶层17楼被批评
  老人上楼不便/电梯停电/防水问题
```

### 6.2 楼层五行生门向（核心）
```yaml
楼层五行按尾数定：1/6=水 2/7=火 3/8=木 4/9=金 5/0=土

门朝南(离火) → 选木(尾3/8)或火(尾2/7)层
门朝北(坎水) → 选金(尾4/9)或水(尾1/6)层
门朝东(震木) → 选水(尾1/6)或木(尾3/8)层
门朝西(兑金) → 选土(尾5/0)或金(尾4/9)层
门朝东南(巽木) → 选水(尾1/6)或木(尾3/8)层
门朝西南(坤土) → 选火(尾2/7)或土(尾5/0)层
门朝东北(艮土) → 选火(尾2/7)或土(尾5/0)层
门朝西北(乾金) → 选土(尾5/0)或金(尾4/9)层

楼层生门向→做事顺利 ✅
楼层克门向→做事阻碍、壮志难酬 ❌
```

### 6.3 门向与生肖不冲
```yaml
命主（和家人）生肖不能和门向相冲
  属鸡(酉)→不买门朝东(卯)→卯酉冲
  属兔(卯)→不买门朝西(酉)→卯酉冲
  属鼠(子)→不买门朝南(午)→子午冲
  属马(午)→不买门朝北(子)→子午冲
家人属相也要考虑进去
```

## 七、户型内部风水

### 7.1 户型方正
```yaml
✅ 方正户型 → 气场稳定
❌ 多角 → 种流房（家人易得增生/肿瘤/癌症）
   案例：全身每个穴位长疙瘩→多角房导致
   化解：朱砂红线埋地板缝（朱砂粉浸泡24h→晾干→切割埋入）
❌ L型/缺角房 → 需补角
```

### 7.2 中宫不可损坏
```yaml
中宫=整套房子正中心，五行属土
中宫出问题的房子→改都没法改
❌ 中宫空亡（天井）→ 气散
❌ 中宫种树 → 木克土+困字
❌ 中宫做卫生间 → 污秽冲散家运
❌ 中宫做厨房/楼梯 → 气不稳
✅ 中宫做客厅/堂屋 → 最佳
```

### 7.3 卧室风水（重中之重）
```yaml
① 房间大小：十几平方最佳，太大(40-50平)消耗能量
   化解：架子床/罗汉床聚气

② 横梁压顶→头晕/精神恍惚
   化解：吊顶包平或移动床位

③ 门不对床→气流吹散磁场→得病

④ 床头靠墙→背后有靠→有贵人扶持

⑤ 床头不靠窗→寒气入头→中风/脑溢血

⑥ 镜子不对床→阴气+夜间吓人→恐伤肾

⑦ 卧室少绿植→晚上植物抢人氧气

⑧ 神龛不放卧室

⑨ 主卧门开在关鬼位→印位/食神位设屏障制约
   主卧门开在财位→食神位放催财物→旺财局
```

### 7.4 鬼门线
```yaml
鬼门线 = 坤宫(西南) + 艮宫(东北)的连线
卧室在鬼门线上→灵异事件/邪病/睡眠不好/做噩梦
⚠️ 卧室要避开鬼门线
  化解：五月真营图（五岳真形图）镇宅辟邪
```

### 7.5 外部形煞
```yaml
检查房屋外部：
  ❌ 路冲（路直冲房子）
  ❌ 反弓煞（道路/水反弓）
  ❌ 天斩煞（两楼间缝道）
  ❌ 剪刀煞（剪刀形物体对着房子）
  ❌ 尖角煞（尖角对窗/门）
```

## 八、改补房子风水（重点）

> 买了有问题的房子不可怕，关键是要知道怎么化解！

### 8.1 火烧天门化解（第1优先级）
```yaml
判断：厨房在西北角（乾宫）= 火烧天门
性质：独立于八字喜忌存在！任何人住都得化解！
🚨 九龙道长本人33岁脑梗案例已证明

标准化解三步法：
① 源头层（必做）：
   灶台下放陶瓷罐+银行土
   （火生土→土生金通关）
   
② 加强层（可选）：
   西北角放黄色陶瓷/黄水晶（土属性）
   ❌ 不放白水晶（金属性）

③ 管理层：
   - 厨房门常关
   - 灶台避开西北方向
   - 地砖用浅黄色（土属性）
```

### 8.2 缺角化解
```yaml
量化标准：缺角宽度÷该边总长≥1/3才算缺角
八字修正：缺角方位如果为喜用→必补
          如果是忌神→不补反好

补角物品速查：
  乾宫(西北)缺 → 老虎/五帝钱/铜金器
  坤宫(西南)缺 → 泥塑/陶瓷/女娲像
  巽宫(东南)缺 → 文竹/富贵竹/木雕青龙
  艮宫(东北)缺 → 假山/泰山石
  坎宫(正北)缺 → 流水摆件/鱼缸
  离宫(正南)缺 → 火炉/红色中国结
  震宫(正东)缺 → 梅花/兰花
  兑宫(正西)缺 → 宝剑/金蟾/龙龟

特殊缺角组合：
  西北+西南都缺 = 离婚房
  西南+东北都缺 = 难怀孕/流产
```

### 8.3 尖角煞 → 血光之灾
```yaml
来源：室内尖形摆件/尖角家具/仙人掌等
效应：血光之灾（案例：两只公鸡摆件→一天内4人同时血光）
化解：移除尖角物品→用圆角替代
```

### 8.4 种流房（多角房）化解
```yaml
来源：非方正户型多出一个角
效应：增生/肿瘤/癌症
化解：红线浸泡朱砂粉24h→晾干→地板切缝埋入→水泥美缝
```

### 8.5 人工靠山法
```yaml
适用：高层/背后无靠
方法：三块砖摆品字形→上面砖用朱砂写"大算蒜计"→放座山位置
```

### 8.6 门对门化解
```yaml
门气散冲：门对门→气流混乱→不聚财
化解：挂门帘/三才五帝尺/随手关门

对门煞（邻居对门）：
  ❌ 八卦镜对外→不利邻里和谐
  ✅ 五帝钱/山海镇挂自家门后
  ✅ 天官赐福/中国结
```

### 8.7 别墅风水注意事项
```yaml
【南北方差异】
  北方：坐北朝南必须、客厅挑空、墙体厚、窗户小、房顶平、层高高
  南方：朝向随便、窗户大（落地窗）、房顶尖、层高低（~2.3米）、墙面薄

【院子风水】
  ① 前庭=进财口→保持整洁少放杂物→断舍离
  ② 中间不种树→木克土=困字
  ③ 种树宜：桃木/枣树/石榴/葫芦（驱邪或多子多福）
     忌：夹竹桃/含羞草等有毒植物、带刺植物
  ④ 大门前不要有树（枯树最凶）
  ⑤ 前低后高（不要前高后低→哭哭啼啼）
  ⑥ 保持阳光充足→防阴气湿重→石材防滑
  ⑦ 不养鸡鸭猪羊→异味→呼吸道疾病

【建筑注意事项】
  ① 官鬼位（克坐山的方位）保持安静
     不能有洗衣机/大门等会动的东西
     食伤位/财位可以动→放冰箱/洗衣机→越动越旺
   
  ② 大门不直冲房门→建影背墙（照壁）
     高3.1米，宽度大于大门
   
  ③ 一层一太极→各层独立看缺角
     三楼缺角只影响住三楼的人
   
  ④ 上下要协调
     一楼卧室×二楼厕所→❌漏水/异味
   
  ⑤ 正在住的房子装修→必须搬出去
     房子磁场和主人融为一体→动土=动命运
     案例：整栋楼西北角挖坑→所有男主人出问题
   
  ⑥ 房子后边空旷→人工靠山石
     至少房子宽度的1/4
```

### 8.8 进宅仪式（五谷进宅法）
```yaml
① 择吉日（与房主八字无冲）
② 进宅前三天→各角落撒盐+糯米→三天后扫净
③ 进宅当天：
   - 准备五谷用红布包好
   - 入门口诵"吉日良辰进新房"
   - 红布包放中宫（客厅中央）
   - 点三炷香拜四方
   - 口诵"此宅今日迎新主"
④ 安神（进宅后才能请神/祖先牌位）
⑤ 暖宅（第一晚全家人过夜，开灯整晚）
```

## 九、全家合参原则（核心）

```yaml
【三层过滤法】
```
