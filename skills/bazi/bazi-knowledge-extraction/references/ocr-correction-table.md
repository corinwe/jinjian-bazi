# 命理PDF·OCR常见错别字修正表

> 来源：易道知识应用7份PDF + 进阶6份PDF OCR实战萃取
> 适用场景：处理八卦命理类扫描件PDF时的OCR后处理

## 爻位类（最高频错误）

| OCR误识 | 正确 | 说明 |
|:--------|:-----|:------|
| 芝位 | 爻位 | 最常见，OCR把"爻"识别为"芝" |
| 区位 | 爻位 | OCR把"爻"识别为"区" |
| 功位 | 爻位 | OCR把"爻"识别为"功" |
| 攻位 | 爻位 | OCR把"爻"识别为"攻" |
| 芝（单独） | 爻 | 如"1芝"→"1爻"、"三六芝"→"三六爻" |
| 莹（单独） | 爻 | 如"3莹"→"3爻" |
| 有（单独）| 爻 | 如"2有"→"2爻"（罕见但出现过） |
| 艾 | 爻 | OCR把"爻"识别为"Ai/艾" |
| 受 | 爻 | OCR把"爻"识别为"受" |

## 命理术语类

| OCR误识 | 正确 | 说明 |
|:--------|:-----|:------|
| 印组 | 印星/印绶 | "印组护身"→"印星护身" |
| 天王 | 天干 | "天王与身体"→"天干与身体" |
| 俯伤 | 食伤 | 常见于十神分析段落 |
| 胡、某章 | 六、某章 | 目录中数字OCR错误 |
| 玉十甲子 | 五十甲子 | 六十甲子相关 |

## 执行原则

1. **先批量替换，后逐页校验** — 先按上表全量替换，再随机抽3-5页确认
2. **上下文判断** — "区位"在指"爻位"时替换，指"区域"时保留
3. **页眉装饰忽略** — 扫描件页眉的"(的 吻道元承"等OCR噪声直接删除
4. **卦名保留原文** — "乾、兑、离、震、巽、坎、艮、坤"OCR通常正确
5. **小六壬掌诀名保留原文** — "大安、留连、速喜、赤口、小吉、空亡"不需修正
6. **数字+爻位合一处理** — 正则替换：`(\d)\s*芝/莹/区` → `\1爻`

## 验证方法

```
修正前：芝位基础知识篇→1芝-基础员工→三六芝相应
修正后：爻位基础知识篇→1爻-基础员工→三六爻相应
搜索确认：grep -c "芝位|功位|攻位|印组" 修正后文件 → 应为0
```
