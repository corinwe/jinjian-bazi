# 八字报告自动配图生成

## 触发机制
完整报告生成后 → gen_report_images.py 自动读取DS → 调用火山引擎豆包API → 出JPG到人物档案目录。

## 后端配置
- Provider: 火山引擎豆包 (volcengine)
- Endpoint: ep-20260718232041-dslrc
- Model: doubao-seedream-5-0-pro (TextToImage)
- VLM备用: doubao-seed-2-0-mini-260215 (仅推理不出图)
- API: https://ark.cn-beijing.volces.com/api/v3/images/generations
- Size: 1920x1920 (最小368万像素)
- Timeout: 180秒 (模型生成较慢)
- ARK_API_KEY 存于 ~/.bashrc 持久化

## 八字象法全图 v3 — 有机整体意境

### 核心理念（老板校准）
❌ v1错误：只画日主的五行（如"辛金"画个金属）
✅ v3正确：八个天干地支各自五行元素有机交融，形成山水画般的整体意境

### 画面必须包含
1. **八字符号全部**：年柱+月柱+日柱+时柱的天干地支各配其五行色彩和元素
2. **五行生克关系视觉化**：金生水=山泉、木生火=燃薪、火生土=灰烬、土生金=矿脉、水生木=溪边树
3. **五行克制关系**：金克木=刀砍树、木克土=根系破土、土克水=山拦水、水克火=水灭火、火克金=熔炼
4. **有机整体**：不是八个标签拼贴，而是一幅完整的自然山水/天地画卷，每个元素与其他元素有互动
5. **日主居中**：日主放大居中，其他七字符环绕，体用比通过山水比例暗示

### 日主五行配色方案

| 日主 | 主色 | 辅色 | 画面元素 | 关键词 |
|:----|:----|:-----|:---------|:-------|
| 庚/辛(金) | 白/金 | 灰/银 | 金属/刀剑/珠宝 | 白金色调，金属质感 |
| 甲/乙(木) | 绿/青 | 棕 | 树木/竹林 | 青绿山水，森木 |
| 壬/癸(水) | 蓝/黑 | 银灰 | 波浪/江海 | 水墨氤氲，流水 |
| 丙/丁(火) | 红/紫 | 橙 | 火焰/太阳/光芒 | 朱红暖色，光焰 |
| 戊/己(土) | 黄/棕 | 橙 | 大地/山峦 | 赭石色调，山峦 |

### prompt模板（v3完整版本）

**通用框架：**
```
一幅中国传统水墨山水画，八字{八字}。{日主}日主居中，八字符散落。
畫面以{主色调}为主，融合{多色调}。
画面关系：{生克关系链1}；{生克关系链2}；{生克关系链3}。
体用比{体}:{用}通过山水比例暗示。
传统水墨风格，意境深远，竖版公众号配图。
```

**实例（魏启令·身强辛金）：**
```
一幅中国传统水墨山水画，八字庚申癸未辛亥辛卯。辛金日主居中，八字符散落。
整體白金/青绿/墨蓝/赭黄色调。画面关系：金属山石间涌出清泉(金生水)，
溪流两岸草木繁茂(水生木)，树木长于大地(木克土)，大地蕴藏金属(土生金)。
体用比64:36通过山水比例暗示。传统水墨风格，竖版公众号配图。
```

**实例（主母成·从弱庚金）：**
```
一幅中国传统水墨山水画，八字丁卯丁未庚午壬午。庚金日主居中。
以红紫/白金/墨蓝/青绿/赭黄色调交融。关系：烈日(午)照耀大地(未)(火生土)，
大地蕴藏金属(土生金)，金属锻造需烈火(火克金)，金生流水(金生水)，流水润草木(水生木)。
体用比50:50。传统水墨风格。
```

**实例（子源·丙火巳月）：**
```
一幅中国传统水墨山水画，八字辛卯癸巳丙戌癸巳。丙火日主居中。
以红紫/墨蓝/白金/青绿色调交织。关系：双巳火(熔炉/烈日)烘烤大地戌土(火生土)，
双癸水(溪流/雨露)调和烈焰(水克火)，卯木树木生在土地上(木克土)。
画面中水火交融形成蒸汽云雾，丙火日主如一轮明日照亮全图。
体用比55:45。传统水墨风格。
```

### prompt构建步骤
1. 从DS读取四柱八字和日主
2. 统计五行出现次数（天干+地支各算一次）
3. 按主次排序确定主色调
4. 构建生克关系链（有哪几种五行就列哪几种关系）
5. 日主居中放大，体用比通过山水比例暗示

## 注意（学习教训）
- ❌ v1的错误：prompt只写"辛金日主八字元素"，出图只画了单一日主
- ✅ v3的正确做法：逐个描述每个天干地支的五行元素，并明示生克制化关系
- timeout必须设180秒以上，豆包seedream-5-0生成单张图约30-60秒
- 文件命名统一用 _八字象法全图.jpg（不写日期避免版本冲突）

## 公众号配图规范
- 竖版（1920x1920近似1:1，公众号正文最佳）
- 水墨风格，意境深远
- 八个字的五行必须全部可辨
- 生克关系通过画面自然呈现（线条/融合区/过渡色）
