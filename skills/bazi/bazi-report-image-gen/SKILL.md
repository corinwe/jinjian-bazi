# 八字报告自动配图生成

## 触发机制
完整报告生成后 → gen_report_images.py 自动读取DS → 调用火山引擎豆包API → 出JPG到人物档案目录。

## 后端配置
- Provider: 火山引擎豆包 (volcengine)
- Endpoint: ep-20260718232041-dslrc
- Model: doubao-seedream-5-0-pro
- API: https://ark.cn-beijing.volces.com/api/v3/images/generations
- Size: 1920x1920
- ARK_API_KEY 存于 ~/.bashrc

## 日主五行配色方案

| 日主 | 主色 | 辅色 | 画面元素 | 关键词 |
|:----|:----|:-----|:---------|:-------|
| 庚/辛(金) | 白/金 | 灰/银 | 金属/刀剑/珠宝 | 白金色调，金属质感 |
| 甲/乙(木) | 绿/青 | 棕 | 树木/竹林 | 青绿山水，森木 |
| 壬/癸(水) | 蓝/黑 | 银灰 | 波浪/江海 | 水墨氤氲，流水 |
| 丙/丁(火) | 红/紫 | 橙 | 火焰/太阳/光芒 | 朱红暖色，光焰 |
| 戊/己(土) | 黄/棕 | 橙 | 大地/山峦 | 赭石色调，山峦 |

## prompt模板

### 八字象法图（总览配图）
```
中国传统八字命理象法图，{日主}日主{八字}，体用比{体}:{用}，
{主色调}色调，{元素}元素，
水墨风格，天干地支符号装饰，阴阳鱼图案，留白意境，
竖版公众号配图，简约大气，高级灰背景
```

### 财富路线图（§8用）
```
八字财星象法图，{财星位置}财星分布，
{主色调}色调线条构成财富路线，
抽象水墨风格，现代与传统结合，竖版配图
```

### 事业图（§10用）
```
八字事业官杀象法图，官杀{官杀位置}，
{主色调}色调，{元素}元素，
上升线条寓意晋升阶梯，竖版配图
```

## 公众号配图规范
- 竖版（9:16比例，公众号正文最佳）
- 水墨风格为主，简约为辅
- 留白不低于画面30%
- 色调按日主五行选择，禁止色混乱
- 文件命名：{姓名}_{主题}_{日期}.png
- 存放路径：{人物档案目录}/assets/
