# 八字报告自动配图生成 v5

## 触发机制
完整报告生成后 → gen_report_images.py 自动读取DS → 调用火山引擎豆包API→ 出JPG到人物档案目录。

## 后端配置
- Provider: 火山引擎豆包 (volcengine custom_provider)
- Endpoint: ep-20260718232041-dslrc
- Model: doubao-seedream-5-0-pro (TextToImage)
- API: https://ark.cn-beijing.volces.com/api/v3/images/generations
- Size: 1920x1920 (最小368万像素)
- Timeout: 180秒
- ARK_API_KEY 存于 ~/.bashrc

## ⚠️ 核心约束（五轮迭代校准）

### 1. 色彩 > 水墨
❌ 不要画成黑白水墨画（v3/v4错误）
✅ **必须色彩丰富鲜艳**，五行的代表色要鲜明可辨
提示词中用"色彩丰富鲜艳，非水墨黑白"明确约束

### 2. 八个字全画，每个字有正确的元素
八字四个天干+四个地支，每个字都要有对应的五行元素和形态：

| 天干 | 五行 | 色彩 | 元素形态 |
|:----|:----|:----|:---------|
| 庚 | 金 | 银灰/金属色 | **刀剑/铁矿石/金属矿石** |
| 辛 | 金 | 晶莹剔透白玉色 | **珠玉/珠宝/首饰** |
| 甲 | 木 | 翠绿 | 参天大树/古松 |
| 乙 | 木 | 翠绿 | 花草藤蔓 |
| 丙 | 火 | 金红/炽白 | **太阳/烈日（在天上）** |
| 丁 | 火 | 红橙 | 烛焰/灯火 |
| 壬 | 水 | 蓝 | 大江大河 |
| 癸 | 水 | 蓝/墨蓝 | **溪流小河（在地面）** |
| 戊 | 土 | 黄棕 | **高山** |
| 己 | 土 | 棕 | 田园沃土 |

### 3. 地支特殊形态（多次校正）

| 地支 | 正确形态 | 错误（不要） |
|:----|:---------|:------------|
| **卯** | **低矮藤蔓/花草/灌木**（翠绿） | ❌ 高树（那是寅/甲木） |
| **戌** | **砖窑燥土/城墙**（低平赭黄） | ❌ 高山（那是戊土） |
| 巳 | 熔炉窑火（红橙） | |
| 午 | 烈日当空（金红） | |
| 申 | 刀剑/兵器（银灰金属） | |
| 酉 | 宝珠（晶莹） | |
| 亥 | 深潭（墨蓝） | |
| 子 | 长河（蔚蓝） | |

### 4. 丙火=太阳挂天上，癸水=溪流在地面
- 子源的丙火是**高悬天空的烈日**，光芒万丈照亮全图
- 癸水是**地面蜿蜒的溪流**，水汽蒸腾与天上烈日呼应
- 不要画丙火在水里

### 5. 意境氛围感 > 元素标签
不是画"这里有水，那里有火"的元素标签，而是画**整体的氛围**：
- **子源（丙火+双癸水）**: 烈日当空+地面溪流→热气蒸腾、水汽氤氲的夏日感
- **主母成（双丁双午+壬水）**: 四火炽烈→扑面而来的炎热感，壬水调和但水汽蒸腾
- **魏启令（庚申+辛亥）**: 金属冷冽+溪水清澈+藤蔓生机→清凉中带着生机的氛围

### 6. 年柱不要写错字
- 庚申不是庚甲
- 年柱核对: 年干+年支要从DS八字字段逐字取

## Prompt工程模板

### 通用结构
```
[风格] + [八字] + [日主居中] + [逐柱元素描述] + [氛围意境] + [色彩约束] + [竖版]
```

### 逐柱描述格式
```
年柱{天干}{地支}：{天干}为{元素}({色彩})，{地支}为{形态}({色彩})
```

### 实例（魏启令·庚申癸未辛亥辛卯·彩色风格）
```
一幅生动鲜艳的中国传统风格插画。八字庚申癸未辛亥辛卯。辛金日主如一枚温润珠玉居中。
年柱庚申：庚为刀剑铁矿石金属矿石(银灰金属色)，申为兵器金属光泽。
月柱癸未：癸为清澈小溪(蓝色)蜿蜒流过田园沃土未(棕色)。
日柱辛亥：辛为珠玉珠宝首饰(晶莹剔透白玉色)散落，亥为深潭(墨蓝)静谧。
时柱辛卯：辛为小颗珠玉(白)，卯为低矮藤蔓花草灌木(翠绿)丛生非高树。
氛围：金属冷冽与溪水清澈交融，藤蔓增添生机。
色彩丰富鲜艳，非水墨黑白。竖版公众号配图。
```

### 实例（子源·辛卯癸巳丙戌癸巳·热气蒸腾意境）
```
一幅生动鲜艳的中国传统风格插画。八字辛卯癸巳丙戌癸巳，丙火日主。
一轮炽热烈日(丙火)高悬天空，光芒万丈照耀大地。
天空中有两股癸水气流化为白云/雨雾，与烈日的热气相遇形成蒸腾雾霭。
地面上：辛为珠玉(白色晶莹)，卯为低矮藤蔓花草(翠绿)，
巳为熔炉窑火(红橙)，戌为砖窑燥土城墙(赭黄)。
双癸水化为两条溪流从两侧蜿蜒流过，水汽蒸腾与天上烈日呼应。
氛围：炎热但水汽充盈，冷暖交织。
色彩丰富鲜艳，非水墨黑白。竖版公众号配图。
```

### 实例（主母成·丁卯丁未庚午壬午·炎热感）
```
一幅生动鲜艳的中国传统风格插画。八字丁卯丁未庚午壬午。庚金日主居中。
年柱丁卯：丁火如烛焰(红橙)照耀在卯木藤蔓花草(翠绿)上。
月柱丁未：丁火(红橙)与未土田园沃土(棕色)如夏日田野。
日柱庚午：庚为金属(银白)被午火烈日(金红)炙烤。
时柱壬午：壬水小河(蓝色)流过午火大地(金红)，水汽蒸腾。
双丁双午四火炽烈，夏日的炎热感扑面而来。
虽有壬水河流调和，但水汽被热气蒸腾成云雾。
色彩丰富鲜艳，非水墨黑白。竖版公众号配图。
```

## 执行流程
1. gen_full_report.py 生成报告 → 自动调用 gen_report_images.py
2. gen_report_images.py 读取DS → 调用 build_prompt()
3. build_prompt() 逐柱构建描述 → 调用火山引擎文生图API
4. 出图JPG → 入 /tmp/ → 复制到知识库人物档案目录

## 常见错误（已踩坑）

| 错误 | 后果 | 修复 |
|:----|:-----|:-----|
| prompt写"水墨风格" | 图成了黑白 | 改"色彩丰富鲜艳，非水墨黑白" |
| 卯木写"树木" | 画成高树像甲木 | 写"低矮藤蔓花草灌木(非高树)" |
| 戌土写"火山/高山" | 画成大山像戊土 | 写"砖窑燥土城墙(非高山)" |
| 丙火和癸水不区分天地 | 太阳画到水里 | 白天上，水地面 |
| 庚写"金属" | 不够具体 | 庚=刀剑铁矿石, 辛=珠玉珠宝 |
| 年柱写错字 | 庚申写成庚甲 | 从DS八字逐字取 |
| timeout低于180秒 | 请求超时 | 设180秒以上 |
| 只写日主不写其他七字 | 图只有日主 | 逐个描述四柱天干地支 |
