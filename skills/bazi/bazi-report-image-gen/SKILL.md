# 八字报告自动配图生成

## 触发规则
每次完整报告生成后，自动触发配图生成。

## 日主五行配图方案

| 日主五行 | 主色调 | 画面元素 | |
|:--------|:------|:---------|:--|
| 金 | 白/金 | 金属/珠宝 | 庚刚辛秀 |
| 木 | 绿/青 | 树木/竹 | 甲参天乙柔 |
| 水 | 蓝/黑 | 波浪/江海 | 壬浩荡癸涓 |
| 火 | 红/紫 | 火焰/日 | 丙炎上丁烛 |
| 土 | 黄/棕 | 大地/山 | 戊厚重己润 |

## prompt模板

### 八字象法图
```
中国传统八字象法图，{日主}日主{五行}属性，
体用比{体}:{用}，大运{大运}，
{主色调}色调，{元素}元素，
水墨风格，天干地支符号，留白意境，
竖版公众号配图
```

### 财富事业图
```
八字{模块}象法图，{主色调}色调，
线条抽象风格，{元素}元素，
传统现代结合，竖版配图
```
