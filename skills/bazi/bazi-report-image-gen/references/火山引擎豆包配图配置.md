# 火山引擎豆包配图配置

## API配置
- Provider: volcengine (custom_provider)
- Base URL: https://ark.cn-beijing.volces.com/api/v3
- API Key: config.yaml中存储
- 文生图模型: doubao-seedream-5-0-pro-260628
- 端点: ep-20260718232041-dslrc

## 出图API
```
POST https://ark.cn-beijing.volces.com/api/v3/images/generations
Authorization: Bearer {ARK_API_KEY}
Content-Type: application/json

{
  "model": "ep-20260718232041-dslrc",
  "prompt": "...",
  "n": 1,
  "size": "1920x1920"
}
```

## prompt演变教训
| 版本 | 问题 | 修正 |
|:----|:-----|:-----|
| v1 | 只画日主五行 | 八个字全画 |
| v2 | 水墨黑白 | 彩色鲜艳 |
| v3 | 卯木画成高树 | 卯=藤蔓花草 |
| v4 | 戌土画成高山 | 戌=砖窑燥土 |
| v5 | 丙火画在水里 | 丙火=太阳在天上 |

## prompt黄金公式
```
"一幅生动鲜艳的中国传统风格插画。八字{八字}，日主{日主}居中。
四柱逐字描述：{每个字的五行元素+地支形态}。
注意：卯木为低矮藤蔓花草灌木非高树，戌土为砖窑燥土城墙非高山。
色彩丰富鲜艳非水墨黑白。竖版公众号配图。"
```
