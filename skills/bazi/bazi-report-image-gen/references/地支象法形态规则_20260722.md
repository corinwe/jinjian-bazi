# 地支象法形态规则（2026-07-22定稿）
用于AI配图生成时，每个地支对应的正确视觉形态。

## 核心原则
每个地支有确定的视觉形态，不能混淆——如卯≠甲、戌≠戊。

## 地支形态速查表

| 地支 | 五行 | 正确形态 | 常见错误（禁止） |
|:----|:----|:---------|:----------------|
| 子 | 水 | 蜿蜒长河/清泉 | 大江大海（那是壬） |
| 丑 | 土 | 冻土/寒冰/沼泽 | 高山 |
| 寅 | 木 | 参天古松/高树 | 藤蔓（那是卯） |
| 卯 | 木 | 低矮藤蔓/花草/灌木 | 高树（那是寅/甲） |
| 辰 | 土 | 水库/湿地 | 高山 |
| 巳 | 火 | 熔炉窑火/炉灶之火 | 森林大火（那是午） |
| 午 | 火 | 烈日当空/篝火 | 烛火（那是丁） |
| 未 | 土 | 田园沃土 | 砖窑（那是戌） |
| 申 | 金 | 刀剑/兵器/金属矿石 | 珠宝（那是辛） |
| 酉 | 金 | 珠宝/首饰/宝珠 | 刀剑（那是庚） |
| 戌 | 土 | 砖窑/燥土/城墙 | 高山（那是戊） |
| 亥 | 水 | 深潭/暗流 | 小溪（那是癸） |

## 天干五行对应元素（配图用）

| 天干 | 五行 | 视觉元素 |
|:----|:----|:---------|
| 甲乙 | 木 | 甲=参天大树；乙=花草藤蔓 |
| 丙丁 | 火 | 丙=烈日/大火；丁=烛火/灯光 |
| 戊己 | 土 | 戊=高山/巨石；己=田园/沃土 |
| 庚辛 | 金 | 庚=刀剑/铁矿石；辛=珠宝/珠玉 |
| 壬癸 | 水 | 壬=大江大海；癸=溪流/雨露 |

## 丙火（太阳）在画面中的位置
**丙火日主 → 烈日高悬天空，光芒万丈**
不是在水里，不是在地面。太阳在天上。

## 双癸水的表现方式
双癸水（如月干+时干双透）→ 两条溪流从两侧汇入，或两股雨雾
水量充沛但不汹涌——不是大江大河，是连绵流水。
