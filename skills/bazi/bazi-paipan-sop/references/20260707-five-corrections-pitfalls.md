# 2026-07-07 报告五项致命错误与修定规则

> **来源：** 老板魏启令审阅v29.0精简版报告后指出的5个问题
> **根因：** 规则存在但未执行（大运上限）、规则错误（文昌/格局）、缺规则（婚姻子女深度）
> **教训：** 每次写完报告必须先过这五项自查，再推库

## ① 格局命名规则（不可再犯）

**错误案例：** 辛金日主·月令未土(偏印主气)·月干癸水(食神) → 我写了「食神配印格」
**正确：** 应以月令主气为先 → **「偏印格」**（偏印配食神）

**强制规则：**
```
Step 1 → 月令地支是什么？主气藏干是什么？
Step 2 → 主气对应的十神是什么？→ 这就是格局的主名
Step 3 → 月干是否透出月令主气或辅助十神？→ 有则加「配X」
Step 4 → 格局名称写完后反问：月令主气是这个吗？✅
```

**常见错误模式：**
| 错误 | 正确 | 原因 |
|:-----|:-----|:-----|
| 食神配印（月令未偏印） | 偏印格 | 月令主气是偏印，不是食神 |
| 伤官配印（月令未偏印） | 偏印格 | 癸水是食神不是伤官（辛生癸=阴生阴=食神） |
| 以透干为主命名 | 月令主气优先 | 透干是「配」不是「主」 |

## ② 大运只列到80岁（不可再犯）

**错误案例：** 报告列了11步大运至110岁（甲申→甲午）
**正确：** 只列前8步至约80岁（甲申→辛卯）

**规则：**
```
✅ 精简版/深度版统一：大运序列只保留前8步（至约80岁）
❌ 禁止列超过80岁的大运（壬辰·癸巳·甲午等）
❌ 禁止以「引擎算了11步」为理由全列
依据：老板2026-07-07亲令「列到八十几岁就够了」
```

## ③ 文昌仅限未成年人（不可再犯）

**错误案例：** 老板（46岁）的报告里输了完整文昌改进方案
**正确：** 年龄>25 岁 → 完全不输出文昌段

**判断逻辑：**
```
IF 当前年份 - 出生年份 ≤ 25 THEN
  → 强制包含文昌改进方案（方位/摆件/颜色/时间/饮食/关键学业年份）
ELSE
  → 完全不输出（连「无需补」的判断都不用写）
```

**注意：** 即使原局文昌需补（引擎wen_chang.has_wen_chang=false），成年人也不输出文昌段。文昌补法只针对25岁以下未成年人。

## ④ 补财库必须含充库+开库（不可遗漏）

**错误案例：** 只写了5种补库方法，没有充库策略和开库窗口
**正确：** 三步完整

```
补库（从无到有建立财库）→ 充库（在财星大运主动充实）→ 开库（刑冲年份把握窗口）

充库示例：庚寅运是木运（寅木正财），此运期间每年收入增量的30%回填专用账户
开库示例：丑未冲年份（如2037丁丑年），偏印库被冲开，有意外之财机会
```

## ⑤ 婚姻和子女必须深度展开（不可简略）

**错误案例：** 婚姻只写了配偶性格一句话，子女只写了一句话「子源已出生」
**正确：** 婚姻至少涵盖6项，子女至少涵盖6项

**婚姻检查清单：**
```
□ 夫妻宫（日支）十神+喜忌
□ 配偶星定位（男正财/偏财 女正官/七杀）
□ 配偶特征：长相（只写实际地支的分类，不列全组）、五行、性格
□ 结婚窗口（3个，精确到年份）
□ 桃花机会（配偶星出现的流年）
□ 感情走势（早/晚婚+质量）
```

**子女检查清单：**
```
□ 子女星定位（男官杀/女食伤）
□ 子女宫（时柱）喜忌
□ 子女出息程度（上等/中等/一般，依据时柱十神+大运）
□ 子女性别倾向（从子女星阴阳+时支类型）
□ 添丁年份
□ 子女性格特征
```
