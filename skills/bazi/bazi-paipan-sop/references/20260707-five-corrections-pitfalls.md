## ① 格局命名规则（本会话纠正3次才正确）

**错误史**：
- v29: 「食神配印格」❌ 月令不透不能做主格名
- v30: 「偏印格」❌ 月令己土虽为偏印但未透干
- 正确：**「杂气格」** ✅ 月令未的己/丁/乙全不透天干

**强制规则（逐字套用）：**
```
Step 1 → 月令地支的本气/中气/余气分别是什么？
Step 2 → 本气是否透出天干？
  ├─ 是 → 以本气定正八格 ✅
  └─ 否 → 中气或余气是否透出？
      ├─ 是 → 以透出者定正八格 ✅
      └─ 否 → 杂气格 ⚠️
```

**实例对照**：
| 八字 | 月令藏干 | 透干 | 正确格局 |
|:----|:---------|:-----|:---------|
| 庚申癸未辛亥辛卯 | 己偏印/丁七杀/乙偏财 | 己丁乙全不透 | **杂气格** |
| 丁卯丁未庚午壬午 | 己正印/丁正官/乙正财 | 丁透月干 | **正官格** |

**常见错误模式：**
| 错误 | 正确 | 原因 |
|:-----|:-----|:-----|
| 食神配印（未月偏印） | 杂气格 | 月令己丁乙全不透，不能定正八格 |
| 偏印格（未月偏印不透） | 杂气格 | 本气不透就不是偏印格 |
| 以月干十神命名 | 看月令透出 | 月干≠月令透出 |
| 混用食神/伤官 | 癸水对辛金=食神 | 阴生阴为食神 |

## ② 大运只列到80岁

**错误案例：** 报告列了11步大运至110岁（甲申→甲午）
**正确：** 只列前8步至约80岁（甲申→辛卯）

**规则：**
```
✅ 精简版/深度版统一：大运序列只保留前8步（至约80岁）
❌ 禁止列超过80岁的大运（壬辰·癸巳·甲午等）
依据：老板2026-07-07亲令「列到八十几岁就够了」
```

## ③ 文昌仅限≤25岁未成年人

**错误案例：** 给成年人（46岁）的报告中输出了完整文昌改进方案
**正确：** 年龄>25岁 → 完全不输出文昌段

**判断逻辑：**
```
IF 当前年份 - 出生年份 ≤ 25 THEN
  → 强制包含文昌改进方案
ELSE
  → 完全不输出（连「无需补」的判断都不写）
```

**注意：** 即使原局文昌需补（引擎wen_chang.has_wen_chang=false），成年人也不输出文昌段。
