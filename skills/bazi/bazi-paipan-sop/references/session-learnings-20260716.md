## 1. 体用定义最终版（不可再改）

```
体 = 日主 + 印 + 比劫 + 食伤（食神+伤官全部归体）
用 = 财 + 官（正财偏财正官七杀，只有这两个）
做功 = 体去追求用的方式
```

**口诀**：体是我和我的工具，印比食伤都在体；用是追求的目标，财官才是真用途。

**误区史**：
1. ❌ 2026-07-14: 体=日主+印+比劫，用=财+官+食伤（缺食伤在体）
2. ❌ 2026-07-16 初修: 体=日主+印+比劫+食神，用=财+官+伤官（错拆食伤）
3. ✅ 2026-07-16 终版: 体=日主+印+比劫+食伤，用=财+官

## 2. 引用口诀前必须校验条件

**典型案例**：「四柱皆带财，库在日时来，食伤能生财，不富没天理」
- 家主八字年柱（庚申）无财 → 四柱不满足 → 不能直接引用
- 应写：三柱有财，年柱无财，但日时有库+食伤生财路线清晰

**通用规则**：任何口诀/断语引用前，逐条核对其条件是否满足本八字。

## 3. 大运真假规则 — 藏干也算有

原局申中戊(余气30%) → 大运戊子天干戊为真（非假）
原局亥中甲(中气60%) → 大运甲申天干甲为真（非假）

**规则**：只要藏干（本气/中气/余气）存在对应的天干，该天干在大运出现即为真。

## 4. 算术错误易发点

**本期错误**：用能量12+7.2+12+24=55.2，子代理写成40.8
**教训**：任何算术结果必须逐项加总验证，不信任子代理的闭眼计算。
**防御**：穷举表+加总验证行，两项数字必须一致。

## 5. 子代理交付前检查清单

子代理经常犯的低级错误：
