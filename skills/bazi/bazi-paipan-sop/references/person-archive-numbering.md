# 人物档案目录编号规则（老板铁律）

> **来源**：2026-07-16 老板严厉纠正（我先建了00-魏启令和00-文颖，破坏了已有01-56的编号体系）

## 规则

**禁止使用00作为新人编号。** 00只用于归档类目录（`00-v8.0评分法归档`、`00-原始素材`）。

## 操作步骤

```bash
# 第1步 — 查看当前最大编号
ls -1d /root/weiwuji-knowledge-base/07-国学哲学/八字命格/02-人物档案/[0-9]*/ | \
  sed 's|.*/||' | sort | tail -1

# 第2步 — 取最大数字+1作为新编号
# 例：最高56 → 新人=57（需检查57是否已被占用）
# 如果有57-梦 → 用58

# 第3步 — 创建目录
mkdir -p "/root/weiwuji-knowledge-base/07-国学哲学/八字命格/02-人物档案/<编号>-<姓名>/"
```

## 常见错误

```
❌ 错误①：创建 00-文颖
   根因：没查当前目录，想当然用了00
   正确：ls 看最大号→57已被57-梦占用→取58

❌ 错误②：创建 00-魏启令
   根因：不知道已有01-家主-魏启令
   正确：不要把新人目录和已有目录搞重复

❌ 错误③：子代理写错父路径
   「八字命理/」≠「八字命格/」
   根因：目标路径写成了 八字命理 而不是 八字命格
   验证：ls 目标父路径，确认包含「八字命格/」
```
