# 盲派分析报告参考格式（以立为例）

> **说明**：本文件是盲派分析报告书写规范的参考案例，基于2026-07-14立的盲派分析实战验证。
> **对应SOP**：bazi-paipan-sop Phase 4.3C

## 报告结构

```
标题: {姓名}·盲派八字命理深度分析报告 v1.0
声明块:
> 分析体系：⛩️ 盲派体系（废旺衰·弃用忌·理法·象法·技法三法合一）
> 与九龙体系的关系：独立并存。本报告以盲派方法论为主，九龙体系数据仅作参考。
> 八字+日主+性别+当前年龄

内容分四大块：
┌─ 📐 盲派理法篇（做功分析）
├─ 🛠 盲派技法篇（根基/三垣/暗象/神煞）
├─ 🎨 盲派象法篇（七维扫描/画图解读）
├─ 📅 大运与流年（盲派视角）
└─ 📋 21§ 标准报告内容（保持与九龙模板一致）
```

## 四块的详细内容规范

### 📐 理法篇

```yaml
结构：
一、体用宾主
  ├─ 体=日主+印+比劫 ｜ 用=财官
  ├─ 主位（家里）=日柱+时柱 ｜ 宾位（家外）=年柱+月柱
  └─ 体用判定（体多/用多→体用匹配度）

二、做功结构
  ├─ 化用结构 / 合制结构 / 生用结构 / 合用结构 / 墓用结构
  ├─ 注明核心做功方式（最核心的1-2种）
  └─ 核心做功总结（一句话概括）

三、功神与废神
  ├─ 逐字判断（四柱每个字：参与做功=功神，不参与=废神）
  └─ 功神效率排名

四、正局与反局
  ├─ 原局正局/反局判定
  └─ 大运反局预警

五、家里家外·归属权
  └─ 每个财官的位置和归属判定
```

### 🛠 技法篇

```yaml
结构：
六、根基与份额
  ├─ 每个天干的根基（本气根/中气根/余气根/无根）
  ├─ 份额判定（多根力量叠加+刑冲破坏打折）
  └─ 家里（日时）根vs家外（年月）根分析

七、三垣分析
  ├─ 胎元（体质/父母/祖籍）→ 月干前一位+月支前三位
  ├─ 命宫（事业）→ 惊门推算法
  ├─ 身宫（财富）→ 另一套推算法
  └─ 三垣天干落财官印+坐支生助→富贵直断

八、暗象体系
  ├─ 暗合（藏干暗合）
  ├─ 暗拱（两字之间空出虚神）
  ├─ 暗带/暗邀
  └─ 实战含义

九、盲派神煞
  ├─ 文昌/桃花/驿马/孤辰寡宿/元辰/灾煞/血刃
  └─ 仅列出命局中有的
```

### 🎨 象法篇

```yaml
结构：
十、七维扫描
  ├─ 五行扫描：五行流通/缺什么
  ├─ 天干扫描：十天干基础画像
  ├─ 地支扫描：十二地支环境画像
  ├─ 十神扫描：十神关系定位
  └─ 逐个列出

十一、画图解读
  ├─ 每个字是什么物体
  ├─ 物体之间的匹配/冲突
  ├─ 整幅画说了什么
  └─ 和谐度判断
```

### 📅 大运与流年

```yaml
结构：
十二、大运精析（盲派视角）
  ├─ 每步大运的功神变化
  ├─ 大运真假验证（三法：寻根基/鸠占鹊巢/生也微妙）
  ├─ 正局/反局变化
  └─ 纯忌神运标注 🔴

十三、串宫压运（可选·当前年份）
  ├─ 大运十二神定基调
  ├─ 日柱十二神看个人状态
  └─ 信号叠加判断
```

### 📋 21§标准内容

与九龙体系报告的§结构一致（§1-§21），但每个§的内容改用盲派视角撰写：
- §1 总览 → 增加「⛩️盲派——做功结构/功神/家里家外」行
- §2 格局 → 九龙写格局名，盲派写成败（杀印相生→贵/化用得力）
- §3 身强弱 → 标为「参考数据」，注明盲派不看
- §4 喜用神 → 标为「参考数据」，注明盲派不用
- §8 财富 → 盲派角度：财库归属+做功路线
- §12 婚姻 → 鸳鸯门+阴阳定位+定份额法
- 其余§保持标准内容但融入盲派视角

## 冲突点标注规范

报告末尾必须标注九龙vs盲派的冲突点（如有）：
```
⛩️ 九龙与盲派冲突点：
├─ 身强弱4.0分（九龙）vs 不看身强弱（盲派）
├─ 用神喜忌（九龙）vs 功神废神（盲派）
└─ 格局命名（九龙·七杀格）vs 做功结构（盲派·杀印相生化用）
两者不矛盾——是同一件事的不同分析角度。
```
