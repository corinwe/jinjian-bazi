## 一、发财核心公式

```
发财 = 身强（40~60分） + 财旺（≥40分）
```

## 二、财富五级速查

| 等级 | 条件 | 量级 | 与官运关系 |
|:----:|:-----|:----:|:-----------|
| 👑 巨富 | 身强+财旺+日/时柱有库+大运配合 | 几十亿+ | 财生官，官运亨通 |
| 💰 大富 | 身强+财旺+日/时柱有库+大运刑冲 | 数亿 | 以财谋权 |
| 🥈 中富 | 身强+财旺但<40分+无库+财星刑冲 | 数千万 | 财可助官 |
| 🏠 小富 | 身弱+财弱+无库+刑冲 | 上千万 | 财不能助官 |
| 🥉 贫穷 | 身弱无财+比劫过旺+枭神夺食 | 千万以内 | 财官双弱 |

## 三、破财5信号（事业官运同步警戒）

1. **财星受冲** → 不投资不借出
2. **食伤受冲** → 财根断，备B计划
3. **身强遇比劫** → 不合伙不担保
4. **财星被合化** → 防合同陷阱
5. **遇墓（<40分）** → 等库年再出击

## 四、合作伙伴规则（身强vs身弱）

| 身强 | 身弱 |
|:----|:----|
| 以财官食为喜用→消耗能量 | 以印比为喜用→补充能量 |
| 切记不可与人合作→自己单干 | 必须与人合作→靠团队 |
| 财为喜用→找有钱老板 | 比劫为喜用→找朋友同事 |
| 官杀为喜用→与领导合作 | 印为喜用→找师长强者 |
| 食伤为喜用→与下属员工 | 天乙贵人合作法永远适用 |

## 五、财官互动六状态

| 状态 | 财富断语 | 官运断语 | 综合 |
|:-----|:---------|:---------|:-----|
| 身强+财旺+官旺 | 大富 | 大官 | 名利双收 |
| 身强+财旺+官弱 | 大富 | 中等官运（财可买官） | 富比贵显 |
| 身强+财弱+官旺 | 中富 | 大官（清官） | 贵比富显 |
| 身弱+财旺+官旺 | 富屋贫人 | 压力大/官非风险 | 需印比帮身 |
| 身弱+财弱+官弱 | 小富 | 小官/无官 | 平凡之命 |
| 从弱+财旺+官旺 | 大财~巨富 | 管官之人 | 特殊格局·爆发 |
