# 入赘/赘婿倾向判断·推理依据

## 八字符号机制

入赘的本质是 **男方身弱不胜财 → 财（妻星/妻家）反克自身**：

```
身弱(4分) + 财旺(43分)
  → 财为忌神
  → 财星代表妻子 + 妻家资源
  → 身弱扛不住财 = 接不住妻家
  → 婚姻关系中处于被支配地位
```

## 关键规则链

```
身弱财旺（素材05行415）
  → 怕老婆（九龙道长原话）
  → 财克印
    → 妻子压制母亲（婆媳关系规则）
    → 原生家庭助力弱
  → 财入夫妻宫为忌
    → 妻家势力直接压在夫妻关系上
    → 婚姻中的能量流向：妻家→夫方
  → 比劫弱
    → 无兄弟帮衬
    → 男方只能独自面对妻家压力
```

> 您不能这么说哈，您身弱，您只能说比较怕老婆"

## 使用注意

1. 这是一套**推理体系**，非九龙道长师传原文
2. 使用时需标注"根据现有规则推理"
3. 入赘是婚姻形式的一种，不等同于婚姻不幸
4. 实际是否入赘还需结合现实因素（户籍/家庭结构/经济条件等）
