# 十神性别对应速查表

## 命主性别 vs 配偶星（核心铁律）

```yaml
          男命                    女命
配偶星 → 正财/偏财（妻星）       正官/七杀（夫星）
───────────────────────────────────────────
正财   → 妻子（正妻）            父亲
偏财   → 情人/偏妻/生意财         —
正官   → 女儿                    丈夫（正夫）
七杀   → 儿子                    情人/偏夫
正印   → 母亲                    祖父/女婿
偏印   → 祖父                    母亲
食神   → 孙子                    女儿
伤官   → 祖母/孙女               儿子
比肩   → 兄弟                    姐妹
劫财   → 姐妹                    兄弟
```

## 常见致命错误（本场会话0617案例）

### 错误1：男命写夫星
```
❌ 「配偶星=甲木正官（时支寅中藏），正官为夫星」
→ 正官对男命=女儿，不是配偶！
→ 正确：男命·己土日主→偏财癸水为妻星（年干癸水透出）
```

### 错误2：男命以正官为配偶星
```
❌ 「日主丙火男命：阳日干男命，以正官为配偶星」
→ 正官对男命=女儿/子女，不是配偶！
→ 正确：丙火男命→正财辛金为妻星
```

### 口诀

```
男命妻星是正财，女命夫星是正官
正官写进男命里，配偶性别全搞反
先看性别再下笔，男女规则各分清

男命中见到夫星→停！这是女命规则
女命中见到妻星→停！这是男命规则
```
