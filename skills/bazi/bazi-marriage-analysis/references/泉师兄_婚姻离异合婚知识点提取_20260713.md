## 一、鸳鸯门口诀体系（←最高优先级补入）

泉师兄称鸳鸯门为"盲派对于八字婚姻方面的一个大总结"，包含7条口诀公式，是现有体系**完全缺失的总纲性知识**。

### 口诀①：宫坏心伤，徒伤情
**原文：** "宫坏心伤，徒伤情"
- 宫=夫妻宫（日支）=住着另一半的家
- 心=夫妻心（男财女官）=具体的人
- 宫和心受刑冲穿客过于严重→婚姻一定出问题
- 现有体系：夫妻宫独立评分（100分体系扣分）+ 配偶星独立检查，但**无宫心同时受损的叠加扣分规则**

### 口诀②：公心相配，和谐美
**原文：** "公心相配，和谐美"
- 夫妻心正好是命主喜用神
- 宫位（日支）里坐着的正好就是夫妻心（宫心同位）
- 宫心同位+为喜用=和谐美好
- 现有体系：夫妻宫十神断婚姻（定性）+ 100分体系（定量），但**无"宫心同位+为喜用→加分"规则**

### 口诀③：男为阳人女为阴，阴阳相配好夫妻；阴阳失位，婚不美
**原文：**
- 万物负阴抱阳，阴阳和谐分庭抗礼=最好状态
- 阴盛阳衰/阳盛阴衰=非好状态
- 八字体现：男生身弱+财旺被财欺 / 女生官杀克身太过
- 生活体现：地位不平等→一个高傲一个卑微→破裂
- 现有体系：有"身弱财旺怕老婆"等零散规则，但**无阴阳平衡理论框架**

### 口诀④：工多性杂，婚不顺
**原文：**
- 夫妻心多现=婚姻混乱
- 戊癸合→两个癸水（两个妻星）; 壬午自合→两个戊土（两个官星）
- 古代=纳妾；现在=婚姻混乱/感情混乱
- 现有体系：有"官杀混杂"和"财多"规则，但**无"夫妻心多现"（工多性杂）这个整合概念**

### 口诀⑤：财是命中情缘线；劫财见财婚可恼，配偶要跟别人跑
**原文：**
- 男女都看财（非男命才看）
- 女命官杀无力时→财为官的源头→财被劫=官伤→离婚
- 广义：财不只是钱，指代感情生活与精神世界
- 现有体系：有"劫财夺财=离婚"但**女命"财门关断"权重不足**

### 口诀⑥：伤官生旺柱无财，一对鸳鸯两分胎
**原文：**
- 女命伤官旺→需要财来通关（伤官生财→财生官）
- 伤官旺+无财→伤官直接克官→婚姻出问题
- 案例：辛金日主/火为官→金水一片治火→需要木财通关但没有→官星受伤
- 现有体系：✅ 有"伤官克官"规则，但**通关机制说明不足**

### 口诀⑦：男命丧官落空亡，夫妻同房阳不刚；女命无财财空旺，男女世上少欲望
**原文：**
- **空亡≠神煞空亡**，而指**太弱/完全无力**（泉师兄特别说明）
- 男命身旺比劫旺→需食伤泄秀→食伤太弱（丧官落空亡）→性功能弱
- 食伤=生殖器（与[9-02]呼应）
- 现有体系：**完全未覆盖**。无"食伤太弱=性功能弱"判断规则

## 二、定份额法（←最高优先级补入）

### 核心方法
盲派定量方法论，判断"命局中的某个天干（夫妻星）有几个人份"。

**天干寻根基（对号入座法）：**
- 天干有甲乙→找地支寅卯辰未
- 天干有丙丁→找地支巳午未戌
- 天干有庚辛→找地支申酉戌丑
- 天干有壬癸→找地支亥子辰丑
- 天干有戊己→找地支辰戌丑未

**定份额4份框架：**
1. **从哪里出来的地方** = 1份（天干的根基地支）
2. **在哪儿待着的（所在位置）** = 1份（天干坐的地支）
3. **家里不管几份算1份** = 1份（家里=日时柱，多份合并为1）
4. **谁生了我** = 1份（生这个天干的地支）

**连体规则：** 天干与脚下根基在一起（如甲木坐寅木）→算1份（不分出1+1=2）

**原理：** 我开公司赚一万块，有N个合伙人→分N份。婚姻中"有几个人分我的妻子/丈夫"。

### 应用①：体与夫妻心发生多次关系→离婚
1. 看夫妻心（男财女官/阴阳独取）
2. 看与夫妻心发生关系的"那个字"有几人份
3. 多人份→多人来合一个财/官→另一半容易犯桃花→离婚

**案例1：** 乙庚合→庚金几份？月上庚壹份+家里不管几份=1份→共2份→两人来合一个乙木财→离婚特征

**案例2：** 己亥自合→己土几份？家外未土1份+巳火1份+家里巳火1份→共3份→三人合己土财→多段婚姻

### 应用②：食伤生财有别人的份→离婚
我用食伤对老婆好，但食伤有别人的份→跑来一个人也对我老婆好→离婚

**案例：** 亥水食伤→丑中1份+自己1份+家里亥水1份=3人份；亥水又从家外丑中出来=2份→结构复杂

### 应用③：劫财见财→离婚
简单直接：劫财把财克掉。现有体系有覆盖。✅

## 三、阴阳定位夫妻星（←高优先级补入）

群阴独阳/群阳独阴取独为夫妻星。

**阴阳划分：**
- 阳：甲乙丙丁戊（天干）+ 寅卯辰巳午未戌（地支）
- 阴：己庚辛壬癸（天干）+ 申酉亥子丑（地支）

**规则：** 命局全阳围绕一个阴→取此阴为夫妻星；全阴围绕一个阳→取此阳为夫妻星
**原理：** "冷的时候要取暖，热了想吹空调——人之本性"

**替代夫妻心规则（无财官时）：**
- 无财官≠无婚姻
- 身旺比劫一片→食伤=夫妻心（需要泄秀/依靠）
- 金水寒一片→火=夫妻心（需要温暖）
- 木火燥一片→水=夫妻心（需要润泽）
- 群阴独阳/群阳独阴→取独为夫妻心

现有体系：替代规则只有"无正财→偏财为妻/无正官→七杀为夫"，**完全缺失阴阳替代法**。

## 四、体用关系与家内家外

### 体用定义
- **体** = 日主 + 禄神 + 比劫（我的身体、头脑手脚、思维）
- **用** = 财官（我这一辈子要用的东西，同时指代妻子和丈夫）

### 家内家外
- 年月 = 家外（社会/外在的东西）
- 日时 = 家内（我的东西，可随时支配取与夺）
- 优先取**家里的财官**为夫妻星（进了家门的另一半）
- 警惕**家外的财官**（还待在外面/被拦在门外/跑出去了）
- 优先取**与我体发生关系的财官**（合为先）

### 体用多次关系=多次婚姻
体和用发生几次关系=几次婚姻。

## 六、其他知识点

### 食神=生殖器 / 禄神=下半身
- 食神=男女生殖器
- 禄神=日主最亲近的体=下半身
- 现有体系：**未覆盖**

### 宫vs星的二分判断法
泉师兄全篇以"宫vs星"为框架：
- 宫=日支=家
- 星=男财女官=人
- 断婚姻重点在星不在宫（但宫也重要）
- 现有体系有宫/星概念但未形成完整的二分法判断框架

### 家外财官的三层含义
家外的财官要思考：
1. 是从家里面跑出去了？
2. 还是只是像宾客一样待在家外，家里另有其人？
3. 还是被拦在门外不让进家的？
