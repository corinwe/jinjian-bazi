# 母子/亲子合参方法论

> 编制人：金鉴真人
> 编制时间：2026年06月17日
> 来源：主母成与子源·母子合参实战

## 核心维度（6维度合参法）

### 维度1：八字命盘对照
- 四柱逐柱对比
- 日主生克关系（母日主生子日主=天然母生格局）
- 身强弱对比

### 维度2：五行能量互补
- 天干逐字互动（生克合冲）
- 地支逐字互动（伏吟最重—同根基）
- 核心发现：年支伏吟=家族根基一脉相承
- 喜用互补检视：A的喜用是否补B的不足

### 维度3：大运同步分析
- 做时间线对照表（每10年/每步大运）
- 标注关键同步节点（两人同时进入最佳大运=顶级同步）
- 分析各自关键事件时间点的交互影响

### 维度4：财富轨迹对照
- 分阶段对比两人财富走向
- 家庭总财富叠加评估
- 组合投资策略建议

### 维度5：关系特征定性
- 日主生克基础分（母生=5⭐/母克=3⭐/无关=4⭐）
- 年支伏吟加分（同年支=5⭐）
- 地支合局加分（午戌合/亥卯合等）
- 喜用互补加分（完美=5⭐）
- 总分20⭐制

### 维度6：需注意年份排查
- 两人同时走忌神大运的年份
- 一方低谷+另一方高峰的年份（经济落差影响关系）
- 流年冲动年支（伏吟被冲=家庭根基动摇）

## 口诀
```
母子合参六维度，命盘五行大运步
财富轨迹关系质，注意年份也莫忘
日主生克定基调，年支伏吟根基同
最佳大运若同步，天时已至莫错过
```
