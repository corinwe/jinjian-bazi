# 婚姻质量：原始100分体系（2026-07-05代码实现）

## 原始100分体系

九龙道长原始理论：夫妻宫(日支)评分基准100分，扣分项如下：

| 扣分项 | 扣分 | 说明 |
|:------|:----:|:-----|
| **三刑** | -100 | 归零离婚（寅巳申/丑未戌完整三刑） |
| **六冲（日支被冲）** | -70/次 | 每冲一个其他地支扣一次 |
| **二刑** | -70 | 寅巳/巳申/丑未/未戌等二刑 |
| **六害（日支被害）** | -30/次 | 每害一个其他地支扣一次 |
| **自刑** | -30 | 辰辰/午午/酉酉/亥亥 |
| **六破（日支被破）** | -20/次 | 每破一个其他地支扣一次 |
| **男命日支比肩** | -10 | |
| **男命日支劫财** | -20 | |
| **女命日支食神** | -10 | |
| **女命日支伤官** | -20 | |

### 加分项

| 加分项 | 加分 | 说明 |
|:------|:----:|:-----|
| **配偶星入本位** | +30 | 男命正财/女命正官入夫妻宫 |
| **有配偶星** | +15 | 配偶星在原局存在 |

### 等级映射

| 分数范围 | 等级 |
|:--------|:----|
| ≥80 | 优秀 |
| ≥65 | 良好 |
| ≥50 | 中等 |
| ≥30 | 偏差 |
| <30 | 差 |
| <50（原始规则） | 会离婚 |
