# 婚姻分析0617新增校验（琼+晓案例触发）

## 女命婚姻排查增补

以下检查项在0617会话中被反复遗漏，必须逐项过：

### 检查18：三重正官不透干
女命是否有正官在地支而不透天干？
- 数量是否 >= 3？
- 是 -> 官多为杀 -> 婚姻偏差/家暴风险
- 案例：琼壬戌癸卯癸巳壬戌，年戌中戊土+日巳中戊土+时戌中戊土=三重正官不透干

### 检查19：劫财透干女命
女命天干是否有劫财？
- 是 -> 性格刚硬不服输、争吵不断、情感竞争
- 对配偶星是抑制还是加剧冲突？

### 检查20：时支冲日支（夫妻宫被冲）
时支是否与日支六冲（子午/丑未/寅申/卯酉/辰戌/巳亥）？
- 是 -> 婚姻动荡结构性因素
- 案例：晓甲子壬申辛丑甲午，时支午火与日支丑土？不对，丑午是害不是冲。
  但若日支丑与时支未 -> 丑未冲 -> 夫妻宫被冲

### 检查21：夫妻宫为忌神
日支十神为喜用还是忌神？
- 忌神 -> 婚姻带来消耗，配偶是负担
- 案例：琼日支巳火正财克身（身弱32分，财为忌），婚姻是消耗源

口诀：
  三重正官不透干，官多为杀在眼前
  劫财透干女刚硬，夫妻一冲难安宁
  夫妻宫忌神婚消耗，配偶虽好也难熬
