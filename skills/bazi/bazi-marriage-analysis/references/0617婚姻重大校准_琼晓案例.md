# 2026-06-17 婚姻分析重大校准

## 案例一：琼（壬戌癸卯癸巳壬戌·身弱32分）

### 最初误判
- 婚姻排名第1(90分) — 严重错误！
- 根因：#仅看了日支巳中戊土正官就认为婚姻好

### 正确分析
1. **三重正官不透干**（年戌·日巳·时戌→戊土均在地支）
   → 内在压抑、表面上看不出来
   → 官多为杀倾向 → 家暴风险极高

2. **双壬劫财透干**（年干+时干）
   → 女命双劫财透 → 性格刚硬不堪、争吵不断
   → 感情中容易有竞争/第三者隐患

3. **夫妻宫巳火本气为丙火正财（非官星）**
   → 婚姻偏物质连接而非情感连接
   → 巳火对身弱癸水为忌神 → 婚姻是消耗源

4. **卯戌合火强化忌神**
   → 婚姻越投入、消耗越大

### 正确婚姻评分：45分(偏差)
- 之前给的90分完全错误

## 案例二：晓（甲子壬申辛丑乙未·中和43.6分）

### 最初遗漏
- 婚姻评为"中等偏上"(丑土印库包容)
- **完全遗漏了丑未冲！** 时支未直接冲日支丑

### 正确分析
1. **时支未冲日支丑** = 子女宫冲夫妻宫
2. 导致婚姻结构性动荡：价值观冲突、冷战周期、边界侵入
3. 但大运有合可化解部分冲力（子年子丑合、午年午未合）

### 修正
- 婚姻质量：中等偏上 → 中平偏上(丑未冲)
- §12.1/§12.6/§21.5全部补充丑未冲分析

## 教训总结

```
婚姻看似好，实际未必好 — 以下信号最易遗漏：

① 官星在藏不在透 → 内在压抑、表面看不出
② 官多而旺 → 官多为杀 → 暴力/压制
③ 夫妻宫本气非官星 → 婚姻重物质轻情感
④ 时支冲日支 → 先天结构性矛盾
⑤ 女命劫财透 → 性格刚硬争吵不断
```

每次写婚姻分析前，逐项过以上5条检查。
