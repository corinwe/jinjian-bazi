# 主母成·婚姻评分校准案例（2026-06-22）

> **八字：** 丁卯 丁未 庚午 壬午  
> **格局：** 从弱50分（原局0分印比·弃命从势）  
> **配偶：** 家主（庚申 癸未 辛亥 辛卯·身强64分）  
> **来源：** 老板反馈「我俩就是一家，评分差太多不合理」

---

## 📋 问题：主母婚姻评分5.65，家主8.30 → 差异2.65分

主母婚姻5.65分的构成（原有）：
- 原局5.5×0.7 + 大运6.0×0.3 = 5.65分
- 主要扣分：午午自刑、丁壬合

家主婚姻8.30分的构成（原有）：
- 原局8.5×0.7 + 大运7.8×0.3 = 8.30分
- 主要加分：正财入夫妻宫（亥藏甲木）+日支喜用+亥卯合

**问题：差异2.65分 > 1.5分阈值 → ❌ 必有一方评分逻辑错误**

---

## 🔍 根因分析

### 错误1：低估"夫星为从弱第一喜用"的权重

主母是从弱格，夫星（午火正官）是**格局第一喜用神**。

```
从弱格的本质：弃命从势，完全顺应环境的克泄耗
对于主母来说：
  - 火（官杀）= 格局最需要的"势"
  - 夫星正官 = 格局第一喜用
  - 配偶 = 格局的"势"之来源
  → 没有配偶的"势"，从弱格局就不成立
  → 婚姻评分应给最高区间（≥8分），而非中等分
```

原评分5.65 → 相当于给了"中等偏低"的婚姻评价，与"夫星为格局第一喜用"的核心命理特征严重矛盾。

### 错误2：遗漏亥午暗合（合婚信号）

主母日支午（丁火正官）与家主日支亥（壬水伤官）存在**亥午暗合**：

```
亥中藏干：壬（本）+ 甲（中）
午中藏干：丁（本）+ 己（中）

亥中壬水 ↔ 午中丁火 = 丁壬暗合
→ 夫妻宫暗合 = 灵魂层次的天然默契
→ 比明面上的六合/三合更深层
→ 明合是表面默契，暗合是内心共鸣
```

原有评分完全遗漏了这个最关键的合婚信号。

### 错误3：午午自刑被过度扣分

```
午午自刑对不同格局的意义：
  - 普通格局（忌火）：午午自刑=内耗、矛盾、自我折磨
  - 从弱格（喜火）：午午自刑=向上的动力、高标准、自我鞭策
  
  主母从弱喜火，午午自刑意味着：
  - 对双方标准都高 → 这不是婚姻质量低，而是质量高的代价
  - 自我要求高 → 反而推动双方共同成长
```

原评分把午午自刑当作大扣分项，但对于从弱喜火的格局，这不应被视为负面扣分。

### 错误4：定性描述与量化评分不匹配

```
报告描述：🌕 良好
实际评分：5.65分

定性≠量化的对应：
  优秀   →  ≥8分
  良好   →  ≥7分
  中等   →  5~6分
  一般   →  3~4分
  差     →  <3分

"良好"但给5.65分 → 致命错误
```

## 💡 经验教训总结

### 关键铁律

```
1. 夫妻评分必须交叉校验 → 差异≤1.5分
2. 夫星/妻星为格局第一喜用 → 婚姻评分≥8分
3. 夫妻宫暗合（亥午暗合等） → 必须纳入评分（+1~1.5分）
4. 刑冲扣分看格局 → 喜用刑冲=动力，忌凶刑冲=扣分
5. 定性描述必须匹配量化评分 → "良好"对应≥7分
```

### 检查清单

```
□ 是否知道此人的配偶八字？
   → 是 → 做夫妻评分一致性校验
□ 夫星/妻星是否为格局第一喜用？
   → 是 → 婚姻评分≥8分
□ 双方日支是否存在暗合关系？
   → 是 → 婚姻评分+1~1.5分
□ 刑冲是否根据格局喜忌判断？
   → 喜用刑冲≠扣分
□ 定性描述和量化评分是否匹配？
   → "良好"→≥7分  "优秀"→≥8分
```
