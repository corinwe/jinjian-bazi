#### ① 删除杜撰内容 — GONG_WX_SEX 火/金/木

**问题：** 代码中 `GONG_WX_SEX` 定义了五行全量映射：
```python
GONG_WX_SEX = {"土": "配偶肾功能偏弱（土克水）", "水": "肾功能强",
               "火": "精力旺盛", "金": "适度", "木": "和谐"}
```

九龙道长原始理论只覆盖了土和水（土克水→肾功能偏弱、水旺→肾功能强），火/金/木的性功能断语**无原始依据**。

**修正：** 删除火/金/木断语，保留土和水。

**⚠️ 2026-07-05审计发现「修正过正」陷阱：** 校准说「保留土和水,删除火/金/木」→ 代码实现中 GONG_WX_SEX 被整个删除,土/水的性功能断语也未实现。每次按校准修正代码后必须用真实案例验证「保留A+B,删除C」没有变成「删除A+B+C」。

**强制验证清单(每次校准修正后):**
```yaml
□ 校准修正范围是否比skill规定的更宽？

□ 配偶宫五行=土时是否有「肾功能偏弱」断语？
□ 配偶宫五行=水时是否有「肾功能强」断语？
```

### 深层机制

```yaml
入赘的本质：男方身弱不胜财 → 财（妻星/妻家）反克自身
  不是简单的"怕老婆"，而是：
  - 财星能量太强 → 妻家提供的资源/条件远超男方自身能力
  - 身弱接不住 → 婚姻关系中处于被支配地位
  - 印星被克 → 连原生家庭也帮不上忙（财克印）
  - 最终结果：男方"入"到女方家，以换取妻家资源的支撑

所以入赘不是道德问题，是能量匹配问题：
  身强财旺 → 男方能驾驭 → 妻家资源为己所用
  身弱财旺 → 男方被财反克 → 被妻家所控
```

### 14.2 定份额法（最高优先级·完整规则）

盲派核心定量方法，用于判断"命局中的某个天干（夫妻星）有几个人份"——即婚姻中潜在的竞争者和复杂程度。

#### 天干寻根基（对号入座法）

| 天干 | 找地支根基 |
|:----|:----------|
| 甲乙 | 寅卯辰未 |
| 丙丁 | 巳午未戌 |
| 庚辛 | 申酉戌丑 |
| 壬癸 | 亥子辰丑 |
| 戊己 | 辰戌丑未 |

#### 份额4框架（定份额核心）

每个天干的"份数"由4个来源构成：

| # | 来源 | 说明 | 算份数 |
|:-:|:----|:-----|:-----:|
| ① | **出处（从哪出来）** | 天干的根基地支 | 1份 |
| ② | **所在（在哪儿待着）** | 天干坐的地支 | 1份 |
| ③ | **家里（日时柱）** | 不管日时柱有多少份，合并算1份 | 1份（合并） |
| ④ | **生我（生这个天干的地支）** | 生这个天干的干支 | 1份 |

#### 连体规则

天干与脚下根基在一起（如甲木坐寅木）→ 算1份（不分出1+1=2）
> 原理：天干坐在自己根上=不可分割的整体

#### 应用①：体与夫妻心多次关系→离婚

**步骤：**
1. 确定夫妻心（男财女官 / 阴阳独取）
2. 看与夫妻心发生关系的"那个字"有几人份
3. 多人份→多人来合一个财/官→另一半容易犯桃花→离婚

**案例1：** 乙庚合→庚金几份？月上庚1份+家里不管几份=1份→共2份→两人来合一个乙木财→离婚特征

**案例2：** 己亥自合→己土几份？家外未土1份+巳火1份+家里巳火1份→共3份→三人合己土财→多段婚姻

#### 应用②：食伤生财有别人的份→离婚

用食伤对老婆好，但食伤有别人的份→跑进来一个人也对老婆好→离婚

**案例：** 亥水食伤→丑中1份+自己1份+家里亥水1份=3人份；亥水又从家外丑中出来=2份→结构复杂

#### 应用③：劫财见财→离婚

简单直接：劫财把财克掉。（现有体系已有覆盖 ✅）

#### 原理总结

> "我开公司赚一万块，有N个合伙人→分N份。婚姻中就是'有几个人分我的妻子/丈夫'。"

### 14.4 体用+家内家外框架（完整规则）

> ⚠️ 泉师兄定义：体=日主+禄神+比劫，用=财官（夫妻）
> 与九龙道长定义（五行本质vs表现形式）不同，两者并行，分别使用。

#### 体用定义

| 概念 | 包含 | 含义 |
|:----|:-----|:-----|
| **体** | 日主+禄神+比劫 | 我的身体、头脑手脚、思维、能力 |
| **用** | 财星+官杀 | 我一辈子要用的东西=财富+事业，同时指代妻子和丈夫 |

#### 家内家外定义

| 区域 | 柱位 | 含义 |
|:----|:-----|:-----|
| **家外** | 年柱+月柱 | 社会/外在的东西 |
| **家内** | 日柱+时柱 | 我的东西，可随时支配取与夺 |

#### 夫妻星取用原则（按优先级）

```
优先取家里的财官为夫妻星（进了家门的另一半）🥇
  ↑
优先取与体发生关系的财官（合为先）🥈
  ↑
警惕家外的财官（还待在外面/被拦在门外/跑出去了）⚠️
```

#### 家外财官的三层含义（泉师兄核心贡献）

当财官在家外时，需要深入思考：

1. **是从家里跑出去了？** → 另一半心不在家/缘分已流出
2. **像宾客一样待在家外，家里另有其人？** → 家外有情人家内有配偶
3. **被拦在门外不让进家？** → 有阻碍（比劫拦路/印星阻挡等）

#### ⭐ 体用多次关系=多次婚姻（泉师兄核心贡献）

> **体与用发生几次关系=几次婚姻**

**体用关系类型：**
| 关系类型 | 含义 | 例子 |
|:--------|:-----|:-----|
| **体合财官** | 日主/比劫与财官相合 | 甲己合、乙庚合 |
| **体生财官** | 食伤/禄神生财官 | 食伤生财、禄生财 |
| **体克财官** | 比劫克财、日主克官 | 比劫夺财、日主抗官 |
| **体被财官克** | 财克印、官克日主 | 财坏印、官杀克身 |

**判断方法：**
1. 确定体（日主+禄神+比劫）的干支
2. 确定用（财星+官杀）的干支
3. 数体中哪些字与用发生了关系（合/生/克/被克）
4. 每种关系=1次婚姻

**案例：** 一个八字中，日主甲木合己土（正财）=1次，月干劫财庚金又合乙木（偏财）=再1次→2次婚姻

**与现有体系的关系：**
- ✅ 现有体系有"二婚断法"（头婚看原局+二婚看大运）
- ❌ 但**无"体用关系次数=婚姻次数"这个定量方法**
- 定份额法 + 体用多次关系 → 可定量判断婚姻次数
