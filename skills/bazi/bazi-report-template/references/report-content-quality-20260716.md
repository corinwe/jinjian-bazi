# 报告内容质量标准（2026-07-16 老板校准）

> **根因**：2026-07-16 共5轮对话老板指出报告"是数据堆砌不是命理分析"。每轮报告都是骨架+字段值，没有分析文本。

## 🚨 铁律：报告内容是命理分析，不是数据堆砌

**禁止**这种格式（数据堆砌）：
```markdown
## 性格分析
外:正官 内:食神 社交:正财 晚年:正官
```

**必须**是这种格式（命理分析）：
```markdown
## 性格分析

外在给人的印象是正官。守规矩、责任心强、注重社会形象和他人评价。外在表现成熟稳重。
内心深处真正的人格是食神。温和友善、乐于分享、有口福有才华。为人随和易相处。
社交场上的做派是正财。务实稳健、精打细算、对财富有规划。追求稳定收入。
```

## 三要素检查清单

每个§写完检查：

| 要素 | 说明 | 通过标准 |
|:-----|:-----|:---------|
| ✅ 数据正确 | 所有数字从DS来，不凭记忆 | 数字可追溯DS['xxx'] |
| ✅ 有分析 | 对数据做了解读，不是列数据 | 有"这意味着""表现为"等分析词 |
| ✅ 有建议 | 给出了可执行的建议 | 不是"视大运调节"这种废话 |

## 各§最低分析深度

| § | 禁止 | 要求 |
|:---|:-----|:-----|
| 性格 | 只列十神名 | 每个十神至少一句性格描述 |
| 财运 | 只列财星位置 | 财星含义+财富路线+建议 |
| 婚姻 | 只说"婚姻宫为X" | 十神在婚姻宫的含义+配偶特征 |
| 大运 | 只写"✅喜用" | 该运对人生的具体影响分析 |
| 健康 | 只列五行次数 | 五行过三断病法+脏腑保养建议 |
| 事业 | 说"随大运调节" | 具体行业建议+事业曲线 |
| 人生建议 | "根据喜用神" | 至少5条可执行建议 |

## 字数基准

- 全报告 ≥ 800行（质量门禁G3标准）
- 每个§ ≥ 15行
- 关键§（格局/财富/大运） ≥ 25行
