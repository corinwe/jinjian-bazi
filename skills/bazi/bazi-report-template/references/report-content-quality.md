# 报告内容质量标准
> 2026-07-16 老板多次指出报告内容"空"后固化。
> 2026-07-17 深度修正：每个§必须有命理分析，不能只输出数据标签。

## 核心教训：报告 ≠ 数据堆砌 ≠ 十神标签

数据堆砌 vs 命理分析的根本区别：
- **数据堆砌**：陈述十神名称（如"配偶宫为伤官"）
- **命理分析**：解释该十神在对应宫位的具体含义（如"伤官在配偶宫：才华过人但也挑剔苛刻，对配偶要求高，婚姻中自己占主导地位"）

## ❌ 反面案例（本会话2026-07-17）

```markdown
## 婚姻分析（❌ 错误）
配偶宫（日支亥）为伤官。 配偶宫一般，需看大运配合。
命理珍宝·绊禄桃花：禄入主与他神合为桃花。

## 子女分析（❌ 错误）
时柱卯辛。时干比肩，时支偏财。
```

## ✅ 正确示范（2026-07-17修正版）

```markdown
## 婚姻分析（✅ 正确）
配偶宫（日支亥）：伤官。
伤官在配偶宫：才华过人但也挑剔苛刻。对配偶要求高，容易因小事不满。
配偶多半有才但性格较弱，婚姻中自己占主导地位。
绊禄桃花判定：日支亥与时支卯相合→桃花（内向合，熟人关系）。
财星明现，若大运流年穿倒财星为关财门，婚姻有变数。

## 子女分析（✅ 正确）
时柱卯辛：时干比肩，时支偏财。
时支偏财(乙)→子女经济头脑好、社交能力强。
时干比肩透出→子女独立性强者但人际关系需引导。
```

## 模糊描述禁止清单

以下描述属于"说了等于没说"，禁止使用：

```
❌ "配偶宫一般，需看大运配合"（→必须解释具体含义）
❌ "楼层尾数宜选水木火相关数字"（→直接说3、8、2、7、1、6）
❌ "时柱卯辛。时干比肩，时支偏财。"（→必须解释这个组合意味着什么）
❌ "年柱劫财→祖上。"（→必须解释劫财在年柱对祖业的影响）
```

## 命理珍宝规则运用检查

每报告必须至少包含以下命理珍宝规则中的3条：
1. 六冲七型（冲凶/冲旺/冲动/冲开/冲出/冲去/冲破）
2. 宾主关系（以命主为主，何柱为宾）
3. 虚实喜忌（虚忌神吉/实忌神凶）
4. 忌神三制（旺制/平衡制/弱制）
5. 绊禄桃花（女命禄入主与他神合）
6. 关财门（财被制服=离婚应期）
7. 体用比（盲派财富等级判定）
8. 十干喜忌（各天干特有喜忌）
