# 报告内容质量标准
> 2026-07-16 老板多次指出报告内容"空"后固化。
> 核心教训：报告不是把DS字段打印一遍，而是「DS数据+规则→分析结论」。

## 报告 ≠ 数据堆砌

```markdown
# ❌ 错误示例（2026-07-16 子源报告）
## 性格分析
外:正官 内:食神 社交:正财 晚年:正官
## 事业分析
中和随大运
## 学历分析
文昌:申

# ✅ 正确示例（2026-07-16 v4版本）
## 性格分析
子源的性格外显为正官，做事守规矩、注重社会形象——给人的第一印象是稳重可靠。
但内心深处（日支）是食神，真实个性温和友善、有才华、乐于分享。
这种外正官内食神的组合，表面正经内心活泼，属于「人前一套人后一套」不是贬义，
而是对不同场合有不同应对策略的表现。
社交场上是正财型（务实），晚年转为正官型（稳重）。
```

## 每条数据的呈现方式

| 数据 | ❌ 堆砌式 | ✅ 分析式 |

| 婚姻宫食神 | 婚姻宫:巳藏食神 | 食神在婚姻宫，配偶温和包容、有才华。但食神在宫者对感情期望较高，容易理想化。|
| 大运壬辰忌 | 壬辰 9-18 忌 | 壬辰运(9-18岁)天干壬水为七杀忌神，青少年期容易感到外界压力大、与长辈关系紧张。|

## 每§最低内容要求

- 性格(§6): 至少分析3个不同人格层面（外在/内在/社交），每层有解释
- 财富(§8): 财星表+位置解读+身强弱影响+大运窗口
- 事业(§10): 月干十神分析+身强弱影响+大运窗口+行业建议
- 健康(§14): 五行过三对照+每项有保养建议
- 大运(§17): 每步判运+一句具体影响分析（不止「喜用」/「忌」）
