## 引擎JSON降级策略

引擎JSON不直接输出以下字段，需要手工从技能规则推断：

| 缺失字段 | 推断方法 | 引用技能 |
|:---------|:---------|:---------|
| **格局名称** | 月令藏干无透干→杂气格；有透干→透干十神为格；全局最显著特征命名 | foundation fortune |
| **财星等级** | wealth skill §2.2 五级表：身强财旺→巨富，身强财弱→中富，身弱财旺→身弱财弱→小富 | wealth |
| **财星分数** | 按§5规则手工计算：各位置基础分×藏干% | wealth |
| **神煞列表** | 年干查文昌/天乙·年支查华盖/驿马·日干查文昌 | foundation |
| **做功路线** | §24 禄→库→官→财→食伤优先级，逐路检查 | wealth |
| **驱吉避凶** | 喜用决定宜做什么，忌神决定忌做什么，结合大运窗口 | fortune remission |
