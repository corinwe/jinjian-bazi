## 核心理念

```yaml
学业判断 = 学业基因 × 兑现条件

学业基因 = 年柱有印（天干透 OR 地支藏正印/偏印）
兑现条件 = 文昌 + 大运/流年补文昌 + 身强身弱 + 印根是否被合化消耗 + 十神配合

月令有印决定深度（能否高学历）
年柱有印决定基础（有没有学业基因）
12岁前决定效率（能否兑现）
```

**铁律：学习力好 ≠ 学业一定差。** 年柱有印是"有学业基因"的信号，六步排查决定"能兑现多少"。正常情况下学习力好学业不会差，除非兑现条件全面拉垮（文昌缺+大运忌+身极弱）。

## 第一步：第0层·年柱有印三档法（3秒定性）

> **2026-06-14修正：年柱藏印也算，不可只看透干。**

### 查法

```yaml
① 年柱有印（天干透印 OR 地支藏正印/偏印均可）
   → 判✅学业好（有学业基因，下限不低）

② 不符合①，但12岁前大运/流年遇文昌贵人
   → 判✅学业中等（文昌补印）

③ ①②都不符合
   → 判❌学业一般（无学业基因）
```

### 年柱藏印质量速查（⚠️ 基于年支基础分=4分计算）

| 藏印质量 | 说明 | 约当分数 |
|:--------|:-----|:--------:|
| **本气藏印**（如子源·卯藏乙木100%）| 学业基因最强 | 4分(年支4×100%) |
| **中气藏印** | 学业基因良好 | 2.4分(年支4×60%) |
| **余气藏印**（如家主·申藏戊土30%）| 有基础但偏弱 | 1.2分(年支4×30%) |
| **藏印被合化消耗**（如静·戌中戊土被卯戌合火）| 打五折 | 原分×0.5 |
| **合化出的印局** | 三合/三会/六合影印可分别计分 | 按倍数算 |

### 查法细化（逐宫检查）

```

  -    无论正印偏印，只要带印就算
  -    本气强、余气弱，根据100/60/30标准
```

## 第二步：六步精细排查（确认第0层结论）

> 第0层判断"有学业基因"后，六步排查决定"能兑现到什么程度"。

### 六步清单

```yaml
□ 第1步：印在月令本气？
  印在月令本气 → +40分，印星有力 ✅ 学历基础深
  印在月令中气/余气 → 不计分 ❌
  印不在月令 → 不计分 ❌

□ 第2步：印根是否被合化/刑冲消耗？
  印根被合化（如卯戌合火→戌中戊土变火）→ 印根虚 ❌
  印根被冲 → 印根动摇 ❌
  印根被刑/害 → 印根受损 ❌
  印根完整无伤 → ✅

□ 第3步：文昌是否存在？
  查文昌贵人（甲巳乙午丙申丁酉戊申己酉庚亥辛子壬寅癸卯）
  文昌在月令 → ✅ 最强，学历至少本科以上
  文昌在各柱 → ✅ 有用
  文昌在大运 → ✅ 10年学习运好
  文昌在流年 → ✅ 一年利学习
  文昌缺失 → ❌

□ 第4步：18岁前大运走喜用还是忌神？
  喜用运（印比帮身）→ 学习黄金期有人托底 ✅
  忌神运（偏财/比劫/食伤耗身）→ 学习黄金期被破坏 ❌

□ 第5步：印运到来时，学习窗口是否还在？
  早期教育（7-15岁）是关键窗口
  印运在12岁前到 → ✅ 最利
  印运在18岁前到 → 中
  印运在18岁后才到 → 窗口已关闭 ❌

□ 第6步：全局综合验证
  ≥4项✅ → 高学历
  2-3项✅ → 中等学历
  ≤1项✅ → 低学历
```

### 第0层与六步的关系

```yaml
第0层判✅学业好 + 六步≥4项✅ = 高学历（如子源）
第0层判✅学业好 + 六步2-3项✅ = 中等学历（如家主）
第0层判✅学业好 + 六步≤1项✅ = 低学历（有基因但兑现差）
第0层判❌一般 + 六步≤1项✅ = 低学历（如静/主母）
第0层判❌一般 + 六步≥2项✅ = 中等（文昌补救）
```

### 🚨 文昌查法双轨制（2026-06-24新增·子源文昌全错教训）

> **致命错误**：子源（丙火偏强）报告中，补文昌方案按年干辛→文昌子(正北)设计。但补文昌应按日干丙→文昌申(西南)。两套查法用途不同，混用则全盘错。

```yaml
【核心规则】两个查法各司其职，不可混淆：

① 命理文昌（判断有没有文昌星）→ 用年干
   用途：判断这个人先天有没有文昌星、文昌何时到位
   例：子源年干辛→文昌子(正北)，原局无子→文昌未到位，戊子运(48岁)到位
   报告中§1表/神煞表/文昌贵人栏 → 用年干查

② 补文昌实操（文昌证法布局）→ 用日干
   用途：风水布局、文昌塔摆放、文昌笔佩戴、考试方位、颜色搭配
   例：子源日干丙→文昌申(西南)→白水晶文昌塔(金为丙火喜用，双重效果)
   报告中§21文昌证法冲刺方案 → 用日干查

【为什么不能混用】
  年干文昌是先天命理的判断标准（是否有文昌星）
  日干文昌是后天补运的操作标准（如何催旺文昌）
  两者是不同的系统，用途不同，不矛盾
	
口诀：命理文昌看年干，补文昌实操用日干
      年干断有没有，日干定怎么补
      子源教训要牢记，北南相错全盘输
```

## 第八步：常见误区对照

```yaml
误区① "年干没印=没学业"
  → 错！年支藏印也算，本气/中气/余气均可

误区② "藏余气=没意义"
  → 错！余气藏印（1.2分，基于年支4分）也代表有学业基因

误区③ "文昌缺失=一定差"
  → 错！大运文昌可补10年，流年文昌可补1年

误区④ "身弱=一定差"
  → 错！年柱有印+身弱=学得快但不持久

误区⑤ "正印透干=高学历"
  → 错！印根被合化/早年忌神运/文昌缺失都可能拉低

误区⑥ "偏印格=学历高"
  → 偏印主分析能力，不代表学历一定高

误区⑦ "偏印运=学业好/学历高"
  → ⚠️ 致命错误！偏印运 ≠ 正印运！
  → 偏印运主偏门智慧、悟性高，但不保正统学历
  → 正印运才主文凭、学历、正统教育
  → 实战案例：主母/成(庚金)戊申偏印运(6-16岁)，成绩好但因财破印频繁辍学
    偏印有悟性≠正印保学历，偏印运中财破印=聪明但没钱上

误区⑧ "学习成绩好=学历高"
  → 错！成绩好是学习能力，学历是兑现结果
  → 财破印(经济原因辍学)是重要干扰因素
  → 同"印透≠高学历"一样，都是「学业基因好但兑现条件拉垮」的变体

误区⑨ "官杀旺=学霸"
  → 官杀自律有助于学习，但无印星加持则难到顶尖
```

### 一、身弱补印 vs 身强补文昌（核心方法论区分）

```yaml
【身弱之人】
  可补印（印为喜用神）+ 可补文昌（神煞系统不受身强弱限制）
  → 最佳策略：印+文昌双补，效果最大化
  → 注意：印星在原局超过3个，即使身弱也不能补印（印多成病）

【身强之人】
  只能补文昌（补印会使八字更失衡）
  → 纯文昌策略：勿碰任何补印行为

【中和之人】
  印和文昌均可用，以文昌为主、印为辅
```

#### 印多成病判断标准

```yaml
印星≥3个（正印+偏印合计）→ 即使身弱也不可补印
  补印后果：印多塞脑、思维僵化、过度依赖、缺乏干劲
  替代方案：只补文昌 + 补食伤（食伤泄秀可化解印多之病）

印星=2个（身弱）→ 可补印，但以文昌为主
印星=0-1个（身弱）→ 大胆补印+文昌双管齐下
```

#### 3.3 大太极→小太极定位法

```yaml
第一步：大太极定位
  站在房屋（或住宅）正中心，用罗盘确定八个方向
  找出本命文昌地支对应的方位角
  如文昌在巳（东南方）→ 房屋的东南角

第二步：小太极定位
  如果大太极方位在厕所、厨房、阳台等不利位置
  则缩小范围到个人的卧室/书房
  在卧室中重新用罗盘定位东南方

⚠️ 铁律：大太极不可用转小太极
  → 每个房间独立分九宫，独立定位
  → 不能在客厅定完文昌位后说\"这个方位在卧室的东南角\"
  → 大太极=房屋整体，小太极=单独房间
    
标准做法：
  ① 大太极定位：整个房子的文昌位（整栋/整层的东南）→ 放主文昌塔
  ② 小太极定位：孩子卧室的文昌位 → 放孩子书桌+小文昌塔
  ③ 如果是租房/单间：以大房间为单位做一次九宫定位即可
```

```yaml
❌ 文昌位不可在厕所（水污文昌）→ 需化解
❌ 文昌位不可在厨房（火烤文昌）→ 需化解

  厕所文昌位 → 放绿植（富贵竹/文竹）+ 常关厕所门
  厨房文昌位 → 放黄水晶/白水晶隔火
```

### 十一、⚠️ 文昌方案个性化定制（八字喜忌交叉匹配法） — v1.2新增

> **核心发现（2026-06-21实战校准）：** 补文昌不能只看文昌五行选色。必须将文昌五行与目标人的八字喜忌做交叉匹配，否则可能出现'文昌催旺了但喜忌平衡被破坏'的反效果。

#### 11.1 五步交叉匹配流程

```yaml
拿到一个人的八字 → 设计文昌方案前，按此流程跑一遍：

Step 1：判断身强弱
  → 身强/中和偏强 → 只补文昌（不补印）
  → 身弱 → 印+文昌双补
  → ⚠️ 印≥3个→即使身弱也不补印，只补文昌

Step 2：查文昌星（日干→地支）
  → 确定文昌五行和对应颜色
  → 确定文昌方位

Step 3：查八字喜忌
  → 找出喜用神（按优先级）和忌神
  → 这是决定「文昌塔选什么颜色」的关键

Step 4：交叉匹配（核心！）
  ┌─────────────────────────────────────────────┐
  │  文昌五行 vs 喜忌 → 有三种情形：              │
  │                                              │
  │  🟢 情形A：文昌五行=喜用神                    │
  │     → 双重效果！催文昌+补喜用，最佳匹配        │
  │     → 按文昌五行选色即可（如子源·金=喜用）     │
  │                                              │
  │  🟡 情形B：文昌五行=中性（既非喜用也非忌神）   │
  │     → 纯文昌效果，按文昌五行选色即可            │
  │     → 无额外喜忌影响                          │
  │                                              │
  │  🔴 情形C：文昌五行=忌神                      │
  │     → ⚠️ 需要谨慎处理！                       │
  │     → 方案：文昌塔颜色改用喜用神颜色（不以文昌 │
  │       五行定色），但文昌方位/文昌笔/方位走法照 │
  │       文昌五行操作。以喜用色为主，文昌形制为辅  │
  └─────────────────────────────────────────────┘

Step 5：输出完整方案
  → 包含：文昌位定位 + 文昌塔颜色（喜忌修正后） 
  → 北斗七星灯颜色（同塔色）
  → 五行音乐（按文昌五行，不按喜忌—音乐走神煞）
  → 文昌笔（通用型，不分喜忌）
  → 佩戴饰品（走喜用神五行）
  → 考试当天颜色（走喜用神/文昌色综合）
  → 考试前方向（文昌方位，不按喜忌）
```

#### 11.2 实战校准案例

**案例A：子源（丙火·文昌申金·喜水>土>金·忌火>木）— 🟢 情形A**

```yaml
分析过程：
  日干丙火 → 文昌申（金）
  文昌五行 = 金
  喜用神 = 水 > 土 > 金
  文昌金 = 第三喜用 ✅ → 🟢 情形A

方案决策：
  文昌塔颜色 → 白色/金色（金=喜用，直接按文昌五行选）
  北斗七星灯色 → 白色（同塔色）
  佩戴饰品 → 白水晶（金=喜用）
  考试当天颜色 → 白/米/黑（喜用色 + 文昌色）
  考试前方向 → 西南方（文昌申位）
  🚨 绝对禁止：红色文昌塔（火=忌神）、绿色文昌塔（木=忌神）
```

**案例B：鱼亚晴女儿（乙木·文昌午火·身弱喜水印）— 🟡 情形B**

```yaml
分析过程：
  日干乙木 → 文昌午（火）
  文昌五行 = 火
  喜用神 = 水（印星）
  文昌火既不喜也不忌 → 🟡 情形B

方案决策：
  文昌塔颜色 → 红色（直接按文昌五行选火色）
  同时补印（水）→ 蓝黑色床品、黑曜石、北方走
  补印和补文昌独立进行 ➡ 互不干扰
```

**假设案例C：假如某人身强忌神为金，文昌也属金 — 🔴 情形C**

```yaml
分析过程：
  假设：日干庚金身强，文昌亥（水），金为忌神
  文昌五行 = 水
  喜用神 = 火/木
  文昌水 ≠ 喜用→属于中性 → 🟡
  
  实际🔴情形C罕见，因为文昌地支固定且多样：
  如果文昌恰好是忌神——解决方法：
    - 文昌塔颜色：不用文昌五行色，改用喜用神颜色
    - 文昌笔/文昌符：照常（属神煞系统，不受喜忌限）
    - 文昌方位：照常走（方位本身无喜忌）
  🚨 核心原则：神煞系统（文昌笔/文昌符/文昌方位走法）
     不受喜忌影响。只有文昌塔颜色/北斗七星灯色需要
     按喜忌调整。
```

#### 11.3 核心原则总结

```yaml
【唯一需要喜忌修正的项目】
  ✅ 文昌塔颜色 → 需匹配喜忌
  ✅ 北斗七星灯色 → 需匹配喜忌（同塔色）
  ✅ 佩戴饰品五行 → 走喜用神，不走文昌五行
  ✅ 考试当天衣服颜色 → 走喜用神为主

【永远不需要喜忌修正的项目（神煞系统独立）】
  ✅ 文昌笔 → 通用，不分五行喜忌
  ✅ 文昌符 → 神煞系统，独立运作
  ✅ 文昌方位走法 → 按文昌位走，不管喜忌
  ✅ 五行音乐 → 按文昌五行播放
  ✅ 文昌位定位 → 按日干查地支定方位

【双重效益场景（文昌=喜用）】
  当文昌五行恰好是喜用神时 → 催文昌+补喜用，一举两得
  此时方案设计最顺畅：所有颜色全部对齐
  这是最理想的文昌补运场景
```

#### 11.4 重要陷阱：别把文昌当印补

```yaml
🚨 常见误区："文昌补运 = 补印" ❌
  文昌属神煞系统，补文昌不受身强身弱影响
  印星属十神系统，补印受身强身弱严格限制
  两者是不同的系统！

🚨 实践中的混淆：
  "文昌属木→孩子忌木→不能补文昌" ❌
  → 错！文昌属木指的是文昌地支的五行，但补文昌
    走的是神煞系统，不是五行补益系统
  → 文昌塔颜色才需要考虑喜忌（因为塔色本身有五行）
  → 但文昌笔/文昌符/文昌方位走法不需要考虑喜忌

🚨 正确理解：
  补文昌 = 借文昌星的神煞能量（天文系统）
  补印   = 借印星的十神能量（命理系统）
  两者平行运作，互不替代，但可以互补
```

### 9.2 学历层级定位规则（本科/研究生/博士）

决定学历层级（学士/硕士/博士）的关键因素：

```yaml
【核心原理】学历 ≤ 学位，学位由「学习窗口持续长度」决定
  本科 = 18-22岁持续学习 → 需18岁前大运不极端（或可扛）
  硕士 = 22-25岁继续深造 → 需印运在22岁前到位且为喜用
  博士 = 25-30岁长期钻研 → 需月令印强+文昌强+大运持续补印

【三层判定规则】

🏅 本科（学士）
  ✅ 条件：18岁前大运喜忌不极端 + 月令有印 + 文昌在局
  ✅ 命理：能扛住18岁前的学习压力，考入大学
  ⚠️ 限制：印运在18岁后才到→本科毕业即工作，不继续深造
  实战：家主·985本科——印运丙戌(21-30)在毕业后到→本科学历

🏅 研究生（硕士）
  ✅ 条件：本科条件全满足 + 印运在18-22岁到位 + 文昌在月/日
  ✅ 命理：大学毕业时正好印运发力→有动力继续深造
  ⚠️ 限制：研究生的关键窗口是22-25岁
  实战：待校准案例

🏅 博士
  ✅ 条件：硕士条件 + 月令印为喜用 + 文昌在月令 + 大运25-30持续补印
  ✅ 命理：长期学术钻研的命理支撑
  ⚠️ 限制：博士需要持续的学术耐力（身强+印旺+文昌）

【印运时间线决定学历层级口诀】
  印运在18岁前→学历基础好，保底本科以上
  印运在18-22岁→硕士窗口，有深造潜力
  印运在22-25岁→硕士可冲，博士随缘
  印运在25岁后→学历上限本科（印运虽好但学龄已过）
```
