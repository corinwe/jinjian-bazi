## 完整10步方案

### 准备工作（第1-5步）

```
第1步：买罗盘 — 淘宝"东定"或"万星堂"7寸（200-500元）
第2步：定位文昌位
  大太极：全屋西南方 → 33cm白水晶文昌塔
  小太极：少爷卧室西南方 → 13cm白水晶文昌塔
第3步：选文昌塔 — 白水晶13层（白色/金色）
第4步：北斗七星灯阵 — 7盏白色LED灯，排列成北斗七星
第5步：五行音乐 — 播放《商》音（金音），学习时15-30分钟
```

### 佩戴与饮食（第6-8步）

```
第6步：文昌笔 — 白色/金色，放书包文具盒随身带
第7步：佩戴白水晶饰品 — 吊坠/手链
第8步：迷迭香 — 考前1-2周每日1杯茶或1粒胶囊
```

### 考试当天（第9-10步）

```
第9步：出门前
  ① 先往西南方走99步或绕一圈 → 再折返去考场
  ② 喝迷迭香茶
  ③ 佩戴文昌笔+白水晶
第10步：入场后
  ① 文昌笔放桌面右上角
  ② 默念九字真言9遍：'临兵斗者皆阵列前行'
  ③ 深呼吸7次（吸4→停4→呼6）
  ④ 请天乙贵人（属猪/鸡者）送考
```

### 🚨 特别禁忌

- ❌ 不可用红色/绿色文昌塔（火/木为忌神）
- ❌ 不可穿红/绿衣服考试
- ❌ 考试当天不可先往东/南方走
- ✅ 推荐色：白/金/银（主色）、蓝/黑、黄/米
