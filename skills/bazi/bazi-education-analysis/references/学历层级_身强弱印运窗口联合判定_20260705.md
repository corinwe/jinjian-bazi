# 学历层级：身强弱×(十神正反面+印运窗口)联合判定（2026-07-05 v2.2）

## 🚨 铁律：每次判定学历必须先考虑正反面

**身强弱是第一优先级**，因为它决定**什么十神对学习有利/不利**（不是"能不能扛"）。

千万不要只加正面逻辑不加反面逻辑！正反面必须同时考虑。

---

## 身强弱决定的十神正反面

| 身强弱 | ✅ 正面十神（利于学习） | ❌ 反面十神（不利学习） |
|:-------|:----------------------|:----------------------|
| **身强(≥60分)** | 官杀(自律·学习好)·食伤(聪明·爱好广)·财星(有动力) | **印枭(学非所用)**·比劫(拖累) |
| **中和(40-60分)** | 平衡型，印运更佳但无印也能学 | — |
| **身弱(<40分)** | **印枭(学习好)**·比劫(同学帮助) | **官杀(压力大发挥失常)**·食伤(贪玩偏科)·财星(早恋杂念) |
| **身极弱(<15分)** | 需大运印比强力补救 | 几乎学不动 |

## 大运十神正反面分析

代码中 `_analyze_da_yun_shi_shen_effect()` 函数：
- 对每步大运，根据身强弱判断该运的十神是正面/反面/中性
- 按时间窗口（0-18/18-22/22-25/25-30）统计正反面数量
- 净得分 = 正面运数 - 反面运数
- 净得分越高→学业越有利

### 身强正反面示例

身强日主走：
- 官杀/食伤/财运 → ✅ 正面（喜用），学得好
- 印运 → ❌ 反面（忌凶），学非所用。**尤其8岁前遇印运=基础学习不好**

### 身弱正反面示例

身弱日主走：
- 印/比运 → ✅ 正面（喜用），学得进去
- 官杀运 → ❌ 反面（忌凶），压力大发挥失常
- 食伤运 → ❌ 反面（忌凶），贪玩偏科
- 财运 → ❌ 反面（忌凶），早恋杂念弃学

## 综合判定逻辑

```python
# education.py v2.2 _determine_degree()

身强(≥60分):
  0-18岁正面≥0 + 六步≥4 + 文昌 → 顶尖·硕士及以上
  0-18岁正面≥0 + 六步≥3 + 文昌 → 985·硕士(印18-22岁)/本科
  0-18岁正面≥0 + 六步≥3 → 211·本科
  0-18岁正面≥-1 + 六步≥2 → 普通本科
  六步≥2 → 普通本科·专科
  else → 大专/职校
  
  特别：8岁前遇印运(反面)→基础不好→降一档

中和(40-60分):
  + 印运18岁前 + 六步≥4 → 985·硕士及以上
  + 印运18岁前 + 六步≥3 → 985·硕士
  + 六步≥3 → 211·本科
  + 六步≥2 → 普通本科
  else → 大专/职校

身弱(<40分):
  0-18岁正面>0 + 六步≥3 + 文昌 → 211·本科
  0-18岁正面>0 + 六步≥2 → 普通本科·本科
  0-18岁正面≥0 + 六步≥2 → 普通本科·专科
  有印运补救 + 六步≥2 → 大专/职校
  else → 初中/高中

身极弱(<15分):
  0-18岁正面>0 + 六步≥3 + 文昌 → 普通本科
  else → 初中/高中
```

## 常见反面逻辑遗漏陷阱

```yaml
陷阱① 「身强什么运都能学好」
  → ❌ 忽略印运为忌凶 → 身强遇印运=学非所用
  → ✅ 身强正面运是官杀食伤财，反面运是印比

陷阱② 「身弱需要印运托底」
  → ❌ 只看正面不看反面 → 没考虑官杀食伤财对身弱的破坏
  → ✅ 身弱走官杀/食伤/财运=学不进去

陷阱③ 「有文昌就能学好」
  → ❌ 忽略文昌只是锦上添花，不能逆转身强弱决定的十神正反面
  → ✅ 文昌是辅助因素，身强弱×大运十神才是主因

陷阱④ 「印运来了就是好事」
  → ❌ 身强遇印运=反面（学非所用），只有身弱遇印运=正面
  → ✅ 同一印运对身强身弱影响相反
```
