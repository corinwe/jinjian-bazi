# 十神学业正反面判定表（2026-07-05 · 引擎代码固化）

> **来源：** 2026-07-05 审计发现 engine/education.py v2.1 自创了「身强遇正财/食伤=学得好」的错误逻辑。
> 经老板多次纠正后，回归九龙道长原始理论，建立完整正反面判定体系。

## 核心原理

**身强弱决定十神对学业的真实影响，不是只看喜忌。**
每个十神有其固有的行为属性（正财=搞钱、食伤=吃喝玩乐），喜用只说明「喜欢这个能量」，
不说明这个能量会导向学习。

## SHI_SHEN_STUDY_EFFECT 表（engine/education.py）

| 十神 | 身强影响 | 身弱影响 | 原始依据 |
|:----|:---------|:---------|:---------|
| 正印 | bad（学非所用） | good（学得好） | 印为忌凶/喜用 |
| 偏印 | bad（学非所用） | good（偏门智慧） | 同上 |
| 比肩 | bad（拖累） | good（同学帮助） | 比劫为忌/喜用 |
| 劫财 | bad（拖累） | good（同伴支持） | 同上 |

| 正财 | **mixed**（喜用但搞钱无心学习） | bad（杂念早恋） | ⚠️ 正财=钱财，不是学习 |
| 偏财 | **mixed**（喜用但社交分心） | bad（杂念多） | 同上 |
| 食神 | **mixed**（聪明但吃喝玩乐贪玩） | bad（贪玩偏科） | 素材02行129 |

## Python 实现模式（engine/education.py）

```python
SHI_SHEN_STUDY_EFFECT = {
    "正印": ("bad", "good"),    # (身强效应, 身弱效应)
    "偏印": ("bad", "good"),
    "比肩": ("bad", "good"),
    "劫财": ("bad", "good"),
    "正官": ("good", "bad"),
    "七杀": ("mixed", "bad"),
    "正财": ("mixed", "bad"),
    "偏财": ("mixed", "bad"),
    "食神": ("mixed", "bad"),
    "伤官": ("mixed", "bad"),
}

# 用法：根据身强弱选择对应列
effect_map = SHI_SHEN_STUDY_EFFECT.get(shi_shen, ("neutral", "neutral"))
if shen_score >= 60:
    effect = effect_map[0]  # 身强列
elif shen_score < 40:
    effect = effect_map[1]  # 身弱列
else:
    effect = "neutral"       # 中和=中性
```

## 反模式陷阱

1. ❌ 「引擎说喜用=学习好」→ 正财说喜用，实际是搞钱
2. ❌ 「只看印运不看十神」→ 身弱走官杀/食伤/财运全在反面
3. ❌ 「身强框架大=什么运都能学好」→ 印运反面学非所用
4. ❌ 「只看一面不看反面」→ 必须正反面同时看
