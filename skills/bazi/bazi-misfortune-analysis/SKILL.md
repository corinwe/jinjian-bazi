## 第一部分：牢狱/官非四大神煞

### 一、元辰（大耗）— 无妄之灾

**定义：** 自己没做坏事，别人做坏事牵连你（担保纠纷、下属出事等）。

**查法：** 分阳男阴女与阴男阳女两种规则。

```yaml
阳男阴女（日元为阳的男性/日元为阴的女性）：
  子→未 | 丑→申 | 寅→酉 | 卯→戌
  辰→亥 | 巳→子 | 午→丑 | 未→寅
  申→卯 | 酉→辰 | 戌→巳 | 亥→午

阴男阳女（日元为阴的男性/日元为阳的女性）：
  子→巳 | 丑→午 | 寅→未 | 卯→申
  辰→酉 | 巳→戌 | 午→亥 | 未→子
  申→丑 | 酉→寅 | 戌→卯 | 亥→辰
```

**应灾标准：** 看流年遇到的字与八字组合。干支相生加强（如火生土/土生金）→ 应灾；干支相克减弱（如木克土/土克水）→ 问题不大。

**化解：** 天乙/天德/月德同柱可解。

### 二、灾煞 — 血光横死

**定义：** 最凶神煞之一，主血光横死。能量加强时→能坐牢都是最轻的。

**查法：** 以年支查，三合局中神所冲之字。
```
申子辰→午（子午冲）
亥卯未→酉（卯酉冲）
寅午戌→子（午子冲）
巳酉丑→卯（酉卯冲）
口诀：三合局中神，所冲即为灾煞
```

**原理：** 12年一次。能量弱时无事，被合化加强（5倍+）时必有灾。

**例外：** 灾煞与天乙/天德/月德同柱→自动解灾。

**定义：** 主刑罚、牢狱、疾病之灾。

**查法：** 看年命纳音。
```
戌亥为天罗，辰巳为地网
年命纳音火 → 见戌或亥 = 天罗
年命纳音水或土 → 见辰或巳 = 地网
年命纳音金或木 → 无此神煞 ✅
```

**宫位：** 出现在哪一柱即应在哪个年龄段。


### 宫位受伤理论（2026-07-04新增）

> 来源：九龙道长2026-07-04直播《生死节的看法》

**定义：** 八字中某个字全局无生扶、被重重克制 → 该字的能量被降到极低，和空亡类似。

**判断标准：**
- 该字没有同五行来帮它
- 该字没有生它的五行来生它
- 周围全是克它/消耗它的五行
- → 该字「受伤了」

**宫位受伤的后果：**
- 代表该宫位的六亲**必然有病或早逝**
- 代表该宫位对应的**身体部位必然有病**
- 再遇大运流年冲击 → 灾难爆发

**实战案例：**
1. **死老公八字**：女命夫妻宫（火）被一圈水克，全局无木生火、无火帮火 → 丈夫河里淹死
2. **死妈妈八字**：宫位（土）全局无火生土，被重重木克 → 妈妈早逝 + 严重颈椎病/富贵包
3. **儿子斑秃案例**：伤官宫位（木）受伤 → 儿子斑秃（头发代表木），父亲人间蒸发

**检验方法：** 逐字检查八字中每个字的生扶力量
```
Step 1: 该字的同五行存在吗？帮助还是被耗？
Step 2: 生该字的五行存在吗？远还是近？
Step 3: 克该字的五行有多少？力量多重？
Step 4: 全局加权 → 该字实际能量还剩几分？
如果接近0分 = 受伤
```

#### 枭神夺食的两种情形

```yaml
情形① — 本命局有食神+大运/流年见偏印 ⚠️ 最严重
  机制：偏印把命主原本拥有的福气夺走了
  效应：严重抑郁、自杀自残想法
  原因："曾经拥有，突然失去"比"从未拥有"更痛

情形② — 本命局无食神+大运/流年见偏印 ⚠️ 较轻
  机制：偏印虽到，但没有食神可夺
  效应：轻度不开心、小郁闷
  原因："未曾拥有，何来伤心"

判断口诀：
  本命有食怕枭来，夺走福气最伤心
  本命无食枭来了，未曾拥有不太痛
```

#### 按五行不同断事

| 枭神五行 | 所夺食神五行 | 断事方向 | 身体部位 |
|:--------:|:-----------:|---------|:--------:|
| 金 | 木 | 手脚受伤、筋骨损伤、车祸 | 四肢/骨骼 |
| 木 | 土 | 血光之灾、意外流血 | 皮肤/肌肉/血管 |
| 水 | 火 | 车祸、烫伤、交通事故 | 心脏/眼睛 |
| 火 | 金 | 刀伤、金属伤、手术 | 肺/呼吸系统 |
| 土 | 水 | 溺水、水厄、肾脏病 | 肾脏/泌尿 |

#### 实战案例对照
```
枭金夺木 → 手脚受伤，容易骨折/扭伤
枭木夺土 → 意外血光，摔伤磕碰流血
枭水夺火 → 车祸/溺水，交通意外高发年
枭火夺金 → 刀伤/手术，整年不宜动刀
枭土夺水 → 肾脏/妇科问题，需防泌尿疾病
```

#### 枭神夺食的年龄分界线
- 36岁前 → 偏向外伤（车祸/骨折/打斗）
- 36岁后 → 偏向内疾（肿瘤/经络淤堵/慢性病恶化）

#### 附加判断
- 枭神为喜用但被食神反克 → 轻度夺食（注意休息）
- 枭神为忌凶+大运流年加强 → 重度夺食（必须化解）
