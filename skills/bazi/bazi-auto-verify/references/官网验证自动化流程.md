## 触发条件
每次排盘新八字或重要八字后，必须运行官网验证。

## 自动化脚本
```bash
bash /root/.hermes/profiles/jinjian-zhenren/scripts/bazi-zydx-verify.sh [年] [月] [日] [时] [分] [名字] [性别]
```

## 脚本提取内容
- 四柱八字（天干地支）
- 地支藏干（如申=劫伤印）
- 称骨命（重量+评语）
