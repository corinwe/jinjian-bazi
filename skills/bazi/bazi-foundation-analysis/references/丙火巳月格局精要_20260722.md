# 丙火巳月格局精要 —— 逐字精读笔记
来源：命理经典·四月丙火总评
精读日期：20260722

## 一、核心要义

### 格局总纲
四月丙火（巳月），丙火建禄。火势如烈日当空，炎威极盛。

### 唯一真用神：壬水
- 丙火没有壬水，就是"孤阳无辅"，难以展现光辉
- 壬水的作用：消解炎热，达成水火既济
- 壬水是阳水，丙火是阳火 → 阴阳搭配晶莹透彻，性格光明磊落

### 庚金的作用：为壬水提供源头
- 壬水到了巳宫（巳火）处于绝地，没有庚金引发水源，壬水无根易涸
- **壬庚双透 + 无戊己土** → 最高格局（湖海广阔映照太阳 → 文明之象）
  - 科举成绩优异 → 荣耀封赏、显赫官职
  - 若不应验 → 可能是出生地风水不佳

## 二、壬水藏支的层次

| 藏壬位置 | 层次 | 效果 |
|:--------|:---:|:-----|
| **申中壬水** | 上等 | 申庚得禄生壬水，科举有成 |
| **亥中壬水** | 中等 | 亥甲木泄水气，通常为秀才之才 |
| 巳中壬水（绝地） | 无效 | 火旺水绝，难以为用 |

申vs亥的差异：
- 申：庚金得禄 → 泄土气而生水 → 好
- 亥：甲木虽在亥长生，但力量弱，不能有效制土 → 不如申

## 三、癸水替代方案

| 组合 | 结果 | 说明 |
|:----|:----|:-----|
| 壬水 | 贵格 | 阳火配阳水，晶莹透澈 |
| 无壬有癸 | 富格 | 不晴不雨，难以显贵 |
| **庚癸双透** | **富而不贵** | 善于谋划、能言善辩、才能出众 |
| 壬癸皆无 | 贫贱 | 火炎土燥，僧道之命，可能夭折 |

癸水 vs 壬水的本质差异：
- 壬水（阳）配丙火（阳）→ 光明磊落、豪迈无私
- 癸水（阴）配丙火（阳）→ 阴晴不定、难以显贵

## 四、特殊格局

### 全庚金无比劫
- 有金无水 → 富而不贵
- 有火透出 → 实则贫穷

### 丙午日 + 壬水多 + 无戊土
- 阳刃煞重 → 光棍/盗贼之命
- 地支水局 + 壬水透 + 无戊 → 盗贼
- 有己土（不能克水，只能浑水）→ 下贱粗鄙
- **七杀阳刃格局：一杀制一刃→威权；多杀多刃→凶暴**

### 炎上格
- 金水皆无却有甲木 → 炎上格
- 火太旺无依 → 孤苦贫夭

## 五、六亲
- 以壬水为用神
- 财星金为妻
- 官杀水为子女

## 六、核心口诀
> **四月丙火，建禄炎威。壬为真用，庚作水源。**
> **壬庚双透，文明贵显。无壬有癸，富厚能言。**
> **壬癸皆无，贫贱堪忧。丙午壬多，煞重须防。**

## 七、对子源八字的对照判断（精读后）

子源八字：辛卯 癸巳 丙戌 癸巳

对照原文逐条验证：
1. 丙火巳月建禄 ✅ → 火旺
2. 双癸水透干 ✅ → 有调候，但癸水（阴）不如壬水（阳）
3. 辛金（正财）在年干透出 → 辛为阴金，生水之力不如庚金
4. **无壬水** ❌ → 难以大贵
5. **无庚金** ❌ → 壬水无源（即使大运来壬也无根）
6. **无戊己土** ✅ → 不克水
7. 双巳火 → 火炎土燥

综合定位：
- 有癸水调候，比壬癸皆无的人好得多
- 癸水不如壬水 → 难以达最高贵格
- 辛金正财透干 → 有财富基础
- 双癸水正官 → 有管理才能，走正道
- **整体：中上格局，富大于贵**
