# 盲派体用判定 · "用"包含食伤

> **来源**：2026-07-14 家主体用判定校准——原将"用"仅限定为财官，漏了亥水食伤
> **核心修正**：盲派体用判定中，「用」= 财 + 官 + 食伤（人生追求的目标）

## 一、体用定义

```
盲派体用体系：

体 = 我所拥有、我的根基
  ├─ 日主
  ├─ 印星（生日主者）
  └─ 比劫（同五行者）

用 = 我所追求、我的人生目标
  ├─ 财星（日主克者 → 财富）
  ├─ 官杀（克日主者 → 地位/压力）
  └─ 食伤（日主生者 → 才华/成就/作品）
```

## 二、常见错误

```
❌ 错误：只把"财官"算作用，漏了食伤
  → 体用判定中漏了亥水（藏壬水伤官）

✅ 正确：用 = 财 + 官 + 食伤
  → 亥水藏壬水伤官 → 属于"用"的范畴
  → 必须列入体用判定

影响：
  ❌ 漏食伤 → "用少而弱" → 低估命主的追求能量
  ✅ 含食伤 → "用有合力" → 三合财局中亥水是生神
```

## 三、口诀

```
体是根基用是求，日主印比是体筹
财官食伤都是用，人生追求全在这
写体用前先自问：用里有没有食伤？
食伤也是人生目标，漏了就是分析偏
```
