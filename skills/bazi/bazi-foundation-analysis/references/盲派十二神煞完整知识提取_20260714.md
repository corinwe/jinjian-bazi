#### 3. 四柱定格（四季分配）
- 年柱 = 春季（天干=前1.5月，地支=后1.5月）
- 月柱 = 夏季
- 日柱 = 秋季
- 时柱 = 冬季

#### 4. 实战判断优先级
```
① 先看大运十二神 → 定年度基调
② 再看日柱十二神 → 直接影响个人状态
③ 然后看月柱十二神 → 事业和环境
④ 最后看年柱和时柱 → 早年环境和晚年归宿
```

### 与子平的区别
> **子平无此体系**——串宫压运十二神是盲派独有技法，子平八字没有这种以流年地支起12神循环的体系。它是盲派断流年最核心的「不传之秘」，完全不依赖五行生克，只靠十二神在四柱的排列位置断吉凶。

| **与子平区别** | 查法相同。盲派在串宫中将其纳入破财星体系与丧门/大耗并用 |
| **来源** | 通用 |

### ⑮ 丧门·吊客 ⭐ 盲派特殊用法

| 项目 | 内容 |
|:----|:-----|
| **查法** | **丧门**：年支前两位（子→寅，丑→卯...）。**吊客**：年支后两位（子→戌，丑→亥...）或串宫十二神第11位 |
| **用法** | 丧门主孝服、损财、生病；吊客主哀伤、孤独、白事。盲派串宫中丧门+吊客同时出现→死人星，判断有孝服 |
| **与子平区别** | 子平中丧门和吊客各自独立存在；盲派将其纳入串宫压运体系作为固定成员，且有叠加断事规则（丧门+吊客同现=大孝） |
| **来源** | 盲派串宫压运 + 子平通用 |

## 四、盲派 vs 子平 神煞态度根本差异对比

| 维度 | 子平派 | 盲派 |
|:----|:-------|:-----|
| **神煞地位** | 辅助参考，权重低，不进入核心判读 | **判读核心之一**，与象法结合使用，直接对应具体事件 |
| **断事方式** | 以用神为主线——用神得令则吉、忌神当令则凶 | 神煞+象法触发——特定神煞在特定宫位=直接断事件 |
| **串宫压运** | 无此体系 | **本门不传之秘**，12神不看生克直接断年运 |
| **元辰/灾煞** | 一般凶煞，不作为断事主力 | 纳入四大牢狱神煞体系（元辰+灾煞+天罗地网+印星被冲），有能量倍数定量判断 |
| **血刃** | 作一般神煞参考 | 重视+合化倍数定量断事 |
| **孤辰寡宿** | 单见即可论孤寡 | **孤寡同时出现才断女命大忌** |
| **大耗** | 即元辰，单一概念 | 区分「生年大耗=元辰」和「流年大耗=串宫十二神」两个独立体系 |
| **小耗/丧门/吊客** | 有不常用 | 串宫十二神中每年必查的固定成员 |
| **神煞数量** | 大量神煞（数百个），多数不常用 | 精选常用约8-12组，结合串宫体系集中使用 |
