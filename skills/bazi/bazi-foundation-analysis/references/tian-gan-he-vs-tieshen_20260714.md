# 天干五合 vs 贴身区分规则（2026-07-14新增）

> **来源**：2026-07-14 少爷(辛卯癸巳丙戌癸巳)实战校准——"双癸水正官合丙火日主"中丙癸无天干五合，错用了"合"字
> **核心原则**：天干五合才是真正的"合"。相邻但无五合关系的天干之间是"贴身"不是"合"。

---

## 一、天干五合（真正的"合"）

| 组合 | 化气 |
|:-----|:------|
| 甲己合 | 化土 |
| 乙庚合 | 化金 |
| 丙辛合 | 化水 |
| 丁壬合 | 化木 |
| 戊癸合 | 化火 |

**只有这5组是天干五合。其他任何天干之间都不构成"合"。**

## 二、贴身（非合·正确术语）

相邻的两个天干如果没有五合关系，称为"贴身"：

| 关系 | 含义 | 例（丙火日主） |
|:-----|:-----|:--------------|
| **五合** | 天干五合，合化某五行 | 丙+辛 = **丙辛合化水** ✅ |
| **贴身** | 相邻但无五合 | 丙+癸 = **贴身非合**（癸克丙） |
| **相隔** | 不相邻 | 年丙+时癸 = 隔柱不作用 |

**核心断句**：「合」必须有天干五合。没有五合的天干之间只能用"贴身""相邻""相克""相生"来描述。

## 三、常见错误模式

```
❌ 双癸水正官合丙火日主（丙癸无合）
   ✅ 双癸水正官贴身丙火日主（官星贴身）

❌ 丙火日主被双癸水争合（丙癸合不存在）
   ✅ 丙火日主被辛金正财合（丙辛合为真五合）

❌ 任一天干之间用"合"描述但没有在五合表中
   ✅ 写"合"前先查天干五合表，确认存在再写
```

## 四、检查流程

```
写任何"X合Y"前，先问三个问题：

① X和Y在天干五合表中吗？
   查表：甲己·乙庚·丙辛·丁壬·戊癸
   在表中 → ✅ 可以写"合"
   不在表 → ❌ 不能用"合"字，改用"贴身""相邻""相克"

② 是邻合、隔合还是遥合？
   邻合（年+月/月+日/日+时）→ 直接合 ✅
   隔合（月+时/年+日） → 合而不化 ⚠️
   遥合（年+时） → 力量极弱

③ 日主合官还是合财？
   丙+辛=辛为财→日主合财（为求财）
   甲+己=己为财→日主合财
   ✅ "合"决定关系性质，"贴身"不能替代"合"
```

## 五、实战案例

### 案例1：子源（少爷）
```
八字：辛卯 癸巳 丙戌 癸巳 | 日主：丙火
丙+辛（年干） = 丙辛合化水 ✅ 真五合 → 财来合身
丙+癸（月干） = 丙癸 ❌ 无合 → 官星贴身
正确：丙辛合化水（财合身化官），双癸水贴身
```

### 案例2：家主（魏启令）
```
八字：庚申 癸未 辛亥 辛卯 | 日主：辛金
辛+庚（年干）= ❌ 无合（庚合乙）
辛+癸（月干）= ❌ 无合（癸合戊）
辛+辛（时干）= 同干不比
→ 日主辛金无合
```
