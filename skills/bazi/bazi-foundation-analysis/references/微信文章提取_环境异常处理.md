# 微信公众号文章提取 · 环境异常处理

## 背景

微信公众平台对自动化请求有环境检测机制（检查是否在微信App/浏览器环境内），当检测到异常时会弹出「环境异常」/「captcha」页面，阻止内容加载。

## 提取方案对比

| 方案 | 成功率 | 适用场景 |
|:----|:------|:---------|
| `scripts/extract_wx_article.py`（主方案） | 80% | 普通公众号文章 |
| curl + 多手机UA轮换 + 手动grep | 95% | 被captcha拦截的文章 |
| 浏览器（browser_navigate） | ⚠️ 不推荐 | 微信captcha在headless浏览器中更难解 |

## 降级方案（captcha绕过）

```
Step 1: 尝试主方案提取
  python3 /path/to/scripts/extract_wx_article.py "<url>"
  ✅ 成功 → 直接使用输出
  ❌ 输出含"环境异常" → go Step 2

Step 2: curl + 多手机UA轮换
  UA列表（按优先级）：
  a) Mozilla/5.0 (Linux; Android 13; SM-S9080) AppleWebKit/537.36 ... Mobile Safari/537.36 MicroMessenger/8.0.33
  b) Mozilla/5.0 (iPhone14,3; U; CPU iPhone OS 15_0 like Mac OS X) AppleWebKit/602.1.50 ...
  c) Mozilla/5.0 (Linux; Android 11; Pixel 5) AppleWebKit/537.36 ... Chrome/93.0.4577.62 Mobile Safari/537.36

  curl -s -L -A "<UA>" "<url>" -o /tmp/wx_article.html

Step 3: BeautifulSoup提取
  python3 -c "
  import re, html
  with open('/tmp/wx_article.html') as f: data = f.read()
  from bs4 import BeautifulSoup
  soup = BeautifulSoup(data, 'html.parser')
  # 提取标题
  og = re.search(r'<meta[^>]*property=\"og:title\"[^>]*content=\"([^\"]+)\"', data)
  title = og.group(1) if og else ''
  # 提取正文
  m = re.search(r'<div[^>]*id=\"js_content\"[^>]*>(.*?)</div>\\s*<script', data, re.DOTALL)
  if m:
      clean = re.sub(r'<[^>]+>', '\\n', m.group(1))
      clean = re.sub(r'\\n{3,}', '\\n\\n', clean).strip()
      body = html.unescape(clean).replace('&nbsp;', ' ')
  else:
      body = '\\n'.join(soup.stripped_strings)
  print(f'标题: {title}')
  print('='*60)
  print(body)
  "

  ✅ 成功 → 提取完成
  ❌ 失败 → 手动复制+保存（最后手段）
```

## 注意事项

1. **UA头是核心**：微信环境检测主要看User-Agent，移动端+MicroMessenger标记最容易通过
2. **cookie无关**：captcha不依赖cookie，多UA轮换通常够用
3. **频率限制**：同IP短时间内请求多个不同文章可能触发限流，间隔1秒+再请求即可
4. **文章尾部广告**：提取后检查末尾，常有「以道相通」等宗教/无关内容，需剔除
