# 天干隔合/遥合/争合判定规则

> **来源**：2026-07-14 主母成案例（丁卯丁未庚午壬午·丁壬隔合+争合）
> **规则来源**：九龙道长原始理论 line 4288「合而不化：天干合但中间有其他天干相隔→合而不化」
> **用途**：每次做天干合化判断前，先过本检查表

---

## 三类天干合有效性判定

### 条件A: 紧邻（直接相邻）
年干合月干 / 月干合日干 / 日干合时干
→ **正常天干合**，可成化也可能合而不化（看根+月令）
→ Step 2：检查天干是否有根+月令是否生该五行
  □ 有根+月令生 → 成化 ✅
  □ 无根或月令不生 → 合而不化

### 条件B: 隔一柱
月干合时干（隔日干） / 年干合日干（隔月干）
→ **隔合**，力量较弱
→ **合而不化**，不能按直接合论断
→ 需大运流年引动方可成合

### 条件C: 隔两柱
年干合时干（隔月干+日干）
→ **遥合**，力量最弱
→ **合而不化**
→ 基本不能直接合，需大运流年强力引动

### 条件D: 争合
一天干被多天干争合（如双丁争一壬） / 多天干争合一干
→ **力量分散**，谁也合不彻底
→ 代表多方关系/竞争局面
→ 合而不化

---

## 实战判定步骤

```
Step 1 — 判断是否紧邻？
  ├─ 紧邻 → 进入正常合化判断
  └─ 隔一柱 → 隔合 → 合而不化 ✅
      隔两柱 → 遥合 → 合而不化 ✅

Step 2 — 判断有无争合？
  ├─ 两天干→一天干 → 争合 → 力量分散
  └─ 一对一 → 无争合

Step 3 — 紧邻条件下的成化判断
  天干有根（地支有藏干）？→ 有/无
  月令生日干或合化的五行？→ 是/否
  两者都满足 → 成化
  任一不满足 → 合而不化

Step 4 — 成化后的新五行，按新十神论吉凶
  □ 合化为喜用 → 吉事
  □ 合化为忌神 → 凶事
```

## 常见错误模式

```
❌ 错误1：看到"丁壬合"就直接当合论断
  主母：丁（月干）+壬（时干）隔日干庚 → 隔合 → 合而不化
  正确：隔合，力量弱，需大运引动

❌ 错误2：看到"合"就只写合，不写隔合/争合/合而不化
  正确：必须标注属于哪种合，以及是否成化

❌ 错误3：用断语时引用错误十神
  丁壬合 → 丁是七杀（对辛金）或正官（对庚金/壬水），不是印！
  正确：逐藏干定性，不看地支属性看藏干十神
```

**口诀**：天干合化先看邻，隔位合来力不真；争合多人争一物，合而不化基础论；有根月令生才化，无根无生仅合名。
