# 十神阴阳判定铁律（2026-07-16 校准）

> **根因**：`calc_shishen()` 中同阴阳/异阴阳规则完全倒置。
> **触发案例**：子源年支乙木对丙火，应为**正印**（阴生阳，异阴阳），错判为**偏印**。
> **连锁影响**：主母成月支丁火对庚金，应为**正官**（阴克阳，异阴阳），错判为**七杀**。

## 正确规则

**同阴阳 = 偏的/凶的 | 异阴阳 = 正的/吉的**

| 十神对 | 关系 | 同阴阳 | 异阴阳 |
|:-------|:-----|:-------|:-------|
| 偏印/正印 | 生我 | **偏印** | 正印 |
| 七杀/正官 | 克我 | **七杀** | 正官 |
| 劫财/比肩 | 同五行 | **劫财** | 比肩 |
| 伤官/食神 | 我生 | **伤官** | 食神 |
| 偏财/正财 | 我克 | **偏财** | 正财 |

## 完整代码

```python
def calc_shishen(tg: str, rizhu: str) -> str:
    if tg == rizhu: return '日主'
    wx = {'甲':'木','乙':'木','丙':'火','丁':'火','戊':'土','己':'土','庚':'金','辛':'金','壬':'水','癸':'水'}
    yy = {'甲':'阳','乙':'阴','丙':'阳','丁':'阴','戊':'阳','己':'阴','庚':'阳','辛':'阴','壬':'阳','癸':'阴'}
    tw = wx[tg]; rw = wx[rizhu]; same = yy[tg] == yy[rizhu]
    if tw == rw: return '比肩' if same else '劫财'
    sheng = {'金':'土','木':'水','水':'金','火':'木','土':'火'}
    if sheng.get(rw) == tw: return '偏印' if same else '正印'  # 同阳=偏印
    wosheng = {'金':'水','木':'火','水':'木','火':'土','土':'金'}
    if wosheng.get(rw) == tw: return '食神' if not same else '伤官'  # 同阳=伤官
    kewo = {'金':'火','木':'金','水':'土','火':'水','土':'木'}
    if kewo.get(rw) == tw: return '七杀' if same else '正官'  # 同阳=七杀
    wok = {'金':'木','木':'土','水':'火','火':'金','土':'水'}
    if wok.get(rw) == tw: return '偏财' if same else '正财'  # 同阳=偏财
    return '?'
```

## 验证用例

```python
tests = [
    ('丙','乙','正印','异阴阳'),('丙','甲','偏印','同阴阳'),
    ('丙','壬','七杀','同阴阳'),('丙','癸','正官','异阴阳'),
    ('丙','辛','正财','异阴阳'),('丙','庚','偏财','同阴阳'),
    ('丙','戊','食神','异阴阳'),('丙','己','伤官','同阴阳'),
    ('庚','丁','正官','异阴阳'),('庚','乙','正财','异阴阳'),
    ('辛','丁','七杀','同阴阳'),
]
for rz,tg,exp,note in tests:
    r = calc_shishen(tg,rz)
    assert r == exp, f'{tg}对{rz}: 期望{exp}, 实际{r}'
```
