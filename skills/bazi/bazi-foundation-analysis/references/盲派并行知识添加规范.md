# 盲派并行知识添加规范

> **来源：** 2026-07-13 session：向 bazi-foundation-analysis 批量添加 §3B-§3H + §3A.8.5 盲派专项知识
> **用途：** 当后续需要向本 skill 添加与其他体系（九龙/子平）**冲突**的知识时，遵守此规范。

## 核心原则：独立并存，不修改现有体系

本 skill 可能同时承载多个派系的知识（九龙道长体系、盲派体系、其他实战校准体系）。当不同派系对同一知识点有**相反的结论**时：

```
❌ 错误做法：修改现有§3.8的九龙优先级（合>冲>刑>害>破）→改成盲派优先级
✅ 正确做法：新增§3C独立存放盲派优先级（三合>三会>六合>五合>生>冲>克>穿），标注冲突
```

## 盲派专项标记体系

| 标记 | 含义 | 使用场景 |
|:----|:-----|:---------|
| ⛩️盲派专项 | 整个章节/规则属于盲派体系 | 独立章节标题、章首说明 |
| ⛩️盲派独有 | 与九龙体系完全冲突，仅盲派有 | 具体规则（如寅辰不论克） |
| ⛩️盲派专项补充 | 在现有九龙章节下的延伸补充 | 如§3A.8.5枭神夺食归属权 |

## 冲突说明结构

每一条与九龙体系冲突的盲派规则，必须包含一个**冲突说明块**：

```markdown
> **⚠️ 冲突说明**：本节的优先级顺序与九龙体系（§3.8合>冲>刑>害>破）完全不同。
> 两者**独立并存**，不能混合使用。使用本节规则时必须标注「⛩️盲派优先级」。
```

对于有多条冲突的章节，还应附加**冲突汇总对比表**：

```markdown
| 冲突点 | 九龙体系（§3.8） | 盲派体系（本节） | 使用场景 |
|:-------|:----------------|:----------------|:---------|
| 三会 vs 三合 | 三会 > 三合 | 三合 > 三会 | 完全不同，依流派选择 |
| ... | ... | ... | ... |
```

## 使用规则口诀

```
使用九龙体系分析时，遵守 §3.8 的原有规则
使用盲派体系分析时，遵守各节的⛩️规则
❌ 不可在同一分析中混用两种优先级
✅ 遇到冲突时，以分析时所用的流派为准
```

## 来源标注规范

所有盲派专项知识都必须标注来源：
- 「泉师兄抖音视频」→ 标明具体视频标题
- 「实战验证」→ 标注实战案例来源
- 「交叉验证」→ 标注与哪个体系做了对比

## 版本号升级

添加盲派并行知识后，版本号递增小版本（如 v2.27→v2.28），description 中注明「⛩️盲派独立并存」。
