# 藏干作用优先级规则（2026-07-14新增）

> **来源**：2026-07-14 家主(庚申癸未辛亥辛卯)实战校准——申中壬水伤官应优先泄庚金劫财(同支)，而非优先去合未中丁火七杀(隔柱)
> **核心原则**：同柱藏干之间优先作用 > 不同柱藏干之间的作用
> **跨柱引动**：不同柱藏干之间的作用需要大运/流年引动才能激活，原局中不自动发生

---

## 一、规则等级

| 优先级 | 作用范围 | 条件 | 示例 |
|:------|:---------|:-----|:------|
| **🥇 第1优先** | 同柱藏干之间 | 同一地支内的藏干自动相互作用 | 申中庚金→壬水（庚生壬，同支自动） |
| **🥈 第2优先** | 邻柱藏干之间 | 相邻地支的藏干，原局有基础关系的可作用 | 年支申+月支未（申未非邻，不自动） |
| **🥉 第3优先** | 隔柱藏干之间 | 需要大运/流年引动才能激活 | 申中壬水+未中丁火（原局不自动合） |

## 二、判断流程

```
Step 1 — 确认作用双方是同柱还是不同柱
  □ 同柱（同一地支内的藏干之间）→ 🥇 原局自动作用
  □ 不同柱 → 进入Step 2

Step 2 — 确认是否是邻柱
  □ 相邻柱（年+月 / 月+日 / 日+时）→ 🥈 原局可能有基础关系
  □ 不相邻柱（年+日 / 年+时 / 月+时）→ 进入Step 3

Step 3 — 确认是否有大运/流年引动
  □ 大运或流年出现触发字 → 🥉 跨柱作用被激活
  □ 无大运/流年引动 → ❌ 原局中不自动发生
    禁止写"原局中戊癸合"（隔柱且无引动）
```

## 三、常见错误模式

```
❌ 申中壬水→跨柱合未中丁火→"伤官合七杀"
   ✅ 申中壬水优先泄庚金(同支)，不走未中丁火
   → 正确描述：申中庚金被壬水伤官泄→比劫夺财力减弱

❌ 原局戊癸合（年柱戊+月柱癸，隔月干/年支等）
   ✅ 仅邻柱的戊癸合才在原局考虑
   → 年戊+日癸 = 隔两柱 → 原局不自动合

❌ "巳亥冲"（巳在月支，亥在日支）
   ✅ 月巳+日亥是邻柱 → 原局有冲的基础关系
   → 但若双巳抗一亥（2:1）→ 冲不成立
```

## 四、口诀

```
同柱优先，隔柱需引
同支自动生克合，跨柱不动不关联
写藏干关系先问：同柱还是不同柱？
大运流年来引动，跨柱才能被激活
不引不吭不出声，原局自己不成事
```
