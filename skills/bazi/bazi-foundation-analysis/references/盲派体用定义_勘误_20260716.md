# 盲派体用定义 · 勘误（2026-07-16 老板校准·二次修正）

> **来源**：段建业盲派原典 + 老板两次口头校准
> **错误历史**：2026-07-14 创建时将体用定义写反（用包含食伤）；2026-07-16 第一次修正时误拆食伤（食神归体伤官归用），老板第二次指出：食伤全在体，用只有财官

## 一、体用定义（正确·最终版）

**口诀**：体是我和我的工具，用是我追求的目标，做工是用体去得用的方式。

```
体 = 我自己 + 我所使用的工具
  ├─ 日主（自身）
  ├─ 印星（支持系统/贵人/长辈缘）
  ├─ 比肩/劫财（帮手/同伴 — 比肩=日主本人，劫财=伙伴）
  ├─ 禄神（身体/根基）
  └─ 食伤（食神+伤官 — 才华/技能/思维/创造力）

用 = 人生追求的目标（外物）
  ├─ 财星（正财+偏财 — 财富）
  └─ 官杀（正官+七杀 — 地位/事业/权力）

做功 = 体去追求用的方式（核心方法论）
  体（印比食伤）→ 用什么工具 → 追求用（财官）
  例：食伤生财 = 用才华（体）去赚钱（用）
  例：印化官杀 = 用智慧/支持（体）去得权（用）
  例：比劫扛财 = 用团队/伙伴（体）去求财（用）
```

## 二、关键概念区分

```yaml
日主 vs 比肩：
  日主 = 我本人（在日柱）
  比肩 = 与我相同五行者（可代表我，也可代表我的分身/合作者）
  老板原话：「日主就是比肩」— 两者本质相同，都是「我」

禄 vs 比肩：
  禄 = 日主在地支的临官位（甲禄寅、乙禄卯等）
  禄就是我的身体/根基 — 也是体的一部分
  比肩在天干，禄在地支 — 都是「我」

食神 vs 伤官（都在体中）：
  食神 = 正面才华（温和、享受、福气）
  伤官 = 进取才华（表达、叛逆、创造力）
  两者都是「我的思维工具」— 都在体中
  不分拆，统一算作体的组成部分
```

## 三、常见错误（两次勘误汇总）

```
❌ 错误①（2026-07-14）：体=日主+印+比劫，用=财+官+食伤
   影响：食伤被误判为追求目标 → 体用关系判断全错
   
❌ 错误②（2026-07-16 第一次修正）：体=日主+印+比劫+食神，用=财+官+伤官
   影响：错误拆分了食伤，伤官被误归入用
   
✅ 正确（最终版）：体=日主+印+比劫+食伤，用=财+官，做功=体追求用的方式
```

## 四、体用量化计算规则

```
体总分 = 日主(不计分) + 印星得分 + 比劫得分 + 食伤得分（食神+伤官全算）
用总分 = 财星得分 + 官杀得分

⚠️ 注意：
  食伤全算进体（不分食神/伤官）
  用里没有食伤（财官才是追求目标）
```
