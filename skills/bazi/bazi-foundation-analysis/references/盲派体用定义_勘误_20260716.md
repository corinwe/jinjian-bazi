## 一、体用定义（正确）

```
盲派体用体系（段建业原典）：

体 = 我自己及使用的工具
  ├─ 日主（自身）
  ├─ 印星（支持系统/贵人）
  ├─ 禄神（根基/稳定）
  ├─ 比肩/劫财（帮手/同伴）
  └─ 食神（才华/技能/思维工具）⚠️ 以前漏了
  
用 = 人生追求的目标
  ├─ 财星（财富）
  ├─ 官杀（地位/事业/压力）
  └─ 伤官（进取/表达/作品）⚠️ 以前归类错了
```

### 为什么食神在体、伤官在用？

```yaml
食神 → 体（内部工具）：
  食神=日主所生，阴阳相同 → 代表内在才华、技能、思维模式
  是「我自己」的延伸——我的脑子、我的手艺、我的创作能力
  类比：我是木匠，我手里的刨子是我的工具（体）

伤官 → 用（外部目标）：
  伤官=日主所生，阴阳不同 → 代表进取、表达、追求、叛逆
  是「我向外追求的东西」——我的作品、我要证明的事
  类比：我用刨子做出来的家具，是我要卖出去换钱的（用）
```

### 口诀

```
体是我和我的工具：日主印比加食神
用是追求的目标：财官伤官才是用
食神伤官要分清：食在内来伤在外
写体用前先自问：食神归体伤归用？
```

## 二、常见错误（本次勘误）

```
❌ 旧错误①：体=日主+印+比劫，缺食神
   影响：食神才华不算在「自己」里 → 低估体的能力

❌ 旧错误②：用=财+官+食伤（包含食伤）
   影响：食神作为内在才华被误判为外部目标 → 体用关系判断错误

✅ 修正：体=日主+印+比劫+食神，用=财+官+伤官
```

## 四、正确的体力量化计算

```
体总分 = [日主自身(不计分)] + 印星得分 + 比劫得分 + 食神得分

注：食神归体后，食神的得分要加进体能量中。
   但伤官仍在用中，伤官的得分不能算进体。
```

**实操指南（每次算体用前执行）：**
```
□ Step 1 — 列出八字中所有食神的位置和分数
□ Step 2 — 把食神分加入「体」的总能量
□ Step 3 — 伤官分留在「用」的总能量
□ Step 4 — 核对：「我有没有把食伤（食神+伤官）混在一起算？」
```
