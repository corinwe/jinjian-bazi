## 三、常见陷阱与校验

### ⚠️ 陷阱1：庚阳生壬阳=食神（非伤官）— ⭐主母案例

**错误版本（v1.0）**：庚金日主，时干壬水 → 误判为「伤官」
**错误原因**：混淆了阴阳规则——以为「庚生壬=异阴阳」
**正确**：庚(阳金)生壬(阳水) = 阳生阳 = **同阴阳** = **食神** ✅

```
庚金日主时干壬水 → 庚阳生壬阳
→ 同阴阳 → 食神 ✅
→ 格局为「食神见正官」而非「伤官见官」
```

### ⚠️ 陷阱2：辛金生壬水=伤官（非食神）

**常见错误**：以为辛生壬=食神（因为壬/辛都是金水相生）
**正确**：辛(阴金)生壬(阳水) = 阴生阳 = **异阴阳** = **伤官**

### ⚠️ 陷阱3：日干阴阳记忆法

```
阳干: 甲丙戊庚壬（刚健主动）
阴干: 乙丁己辛癸（柔顺主静）

口诀：甲丙戊庚壬为阳，乙丁己辛癸为阴

同阴阳 = 正(食神/正官/正财/正印/比肩)
异阴阳 = 偏(伤官/七杀/偏财/偏印/劫财)
```

### ⚠️ 陷阱4：神煞四柱查法

神煞以四柱地支查，不混入十神映射中。
```
辛→子文昌：辛年以子为文昌（10天干文昌口诀：甲巳乙午丙申丁酉戊申己酉庚亥辛子壬寅癸卯）
丁→酉文昌：丁年以酉为文昌
```
文昌位的十神性质根据所属地支而不同，不影响文昌本身的功能。

## 四、引擎交叉验证方法

当手动计算十神映射时，用引擎排盘交叉验证：

```bash
# 主母案例验证
bash projects/bazi-platform/scripts/bazi-must-run-engine.sh \
  -n 成 -g 1 -y 1987 -m 7 -d 20 -h 12
```

引擎输出中 `ten_shen` 字段会显示十神映射结果。手动计算结果必须与引擎一致。

**验证重点：**
□ 天干十神：年/月/日/时干逐干校对
□ 藏干十神：本气/中气/余气逐一校对
□ 食神 vs 伤官：特别注意同阴阳规则
□ 正官 vs 七杀：同阴阳=七杀（不是正官）
□ 正印 vs 偏印：同阴阳=偏印（不是正印）
