# 五月庚金格局精要 —— 逐字精读笔记
来源：命理经典·五月庚金总评
精读日期：20260722

## 一、核心要义

### 格局总纲
五月（午月）丁火主事，庚金处于败地（沐浴）。
庚金被猛烈丁火过度煅制——需要壬水来调节。

### 核心用神：壬水（第一）+ 癸水（辅助）+ 庚辛金（生水之源）

| 层级 | 配置 | 结果 |
|:----|:-----|:-----|
| 🥇 | 壬透+癸藏+地支见金 | 富贵轻而易举 |
| 🥈 | 金透+壬藏+地支金生壬 | 被选拔任用 |
| 🥉 | 癸透+见金 | 也不错，富贵较低，他途发展 |
| ⚠️ | 仅壬或癸独现无金生 | 被旺火燥土烘干 |
| ❌ | 无水 | 奔波贫贱 |
| ❌ | 成火局+无水+无土 | 贫贱 |
| ❌ | 火局+无水+戊己土透 | 可免早逝 |

## 二、关键规则

### 水源问题
「五月的金处于沐浴，水处于绝。仅仅见到一两处壬水或癸水，没办法克制旺火，反而会被旺盛的火和干燥的土烘干。所以如果壬水透出天干，还需要癸水来辅助，地支见到庚辛金来生助，这样的水才是有源之水，获得富贵就容易了。」

→ 壬水必须要有庚辛金来生，否则是"无源之水"，会被火烘干。

### 忌神
- 戊己土透干克水 → 普通人
- 戊土藏地支 + 木克之 → 文人中的优秀者

### 火局无水
地支组成火局（巳午三合），如果没有水：
- 必须有戊己土透出 → 泄火势，免早逝
- 土也不透 → 贫贱

### 从杀格
全木火，无比劫印食伤 → 从杀格，不小的富贵。
但五月午宫有己土印星，不能轻易论从杀。

## 三、行运关键
「就算四柱中没有水，也最好行运到北方水地。等待遇到水的时候才能发展，这就是所谓的巧妙借助元武（水）。」

→ 大运走北方水地（亥子丑），可以弥补原局无水的不足。

## 四、关键口诀
> **五月庚金，败地沐浴。**
> **丁火猛烈，壬水为救。**
> **壬透癸辅，金生乃贵。**
> **无源之水，烈火烘干。**
> **元武借力，水运方发。**

## 五、对主母成八字的对照结论

八字：丁卯 丁未 庚午 壬午

### 优势
1. 五月庚金，时干壬水透出 ✅ — 第一用神到位
2. 己土（未）藏而不透 ✅ — 不直接克壬水
3. 卯中乙木藏 → 可克己土 ⚠️ 但力量弱
4. **大运37岁后辛亥、壬子、癸丑连续水运** ✅ — 原文说的"行运北方水地"

### 不足
1. **地支无庚辛金生壬水** ❌ — 壬水是无源之水
2. 无癸水辅助 ❌ — 独水难救烈火
3. 双丁+双午 → 火势猛烈，过度煅制 ❌
4. 年支卯木生火 ❌ — 助长火势

### 格局定位
中等格局。壬水透干是亮点，但无金生之。好在37岁后大运走水地，补足了原局的不足。
