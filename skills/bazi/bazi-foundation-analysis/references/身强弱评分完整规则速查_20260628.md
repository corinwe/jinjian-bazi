## 一、位置基础分

| 位置 | 基础分 | 备注 |
|:----|:-----:|:-----|
| **年干** | **8** | 天干不分藏干 |
| **年支** | **4** | 藏干×4 |
| **月干** | **12** | 天干 |
| **月令** | **40** | 藏干×40（最大单项） |
| **日干（日主）** | **0** | ⚠️ 日主=自身，不计入评分 |
| **日支** | **12** | 藏干×12 |
| **时干** | **12** | 天干 |

## 二、藏干百分比

| 藏气等级 | 比例 | 速算 |
|:--------|:----:|:----|
| **本气**（第1个） | **100%** | 位置分×1.0 |
| **中气**（第2个） | **60%** | 位置分×0.6 |
| **余气**（第3个） | **30%** | 位置分×0.3 |

| 时支 | 12 | 12.0 | 7.2 | 3.6 |

## 三、加分规则（只加不减）

> **九龙道长原文**：「只算加分就行了，不用算减分。总共100分，你算他加了多少分，不就知道他日元多少分了？」

### ✅ 印星加分

| 条件 | 分数 | 原因 |
|:-----|:----:|:-----|
| 月令**本气**印 | **+40分** | ✅ 唯一算印的位置 |
| 月令中气印 | **0分** | ❌ 不算 |
| 月令余气印 | **0分** | ❌ 不算 |
| 年干/月干/时干印 | **0分** | ❌ 非月令不算 |
| 年支/日支/时支藏干印 | **0分** | ❌ 非月令不算 |

> **口诀**：印只看月令本气，别处再透不算分

### ✅ 比劫加分

| 条件 | 分数 | 说明 |
|:-----|:----:|:------|
| **月令比劫**（本/中/余气全算） | 按藏干比例×40 | ✅ |
| **天干比劫**（年干/月干/**时干**） | 年8+月12+时12 | ✅ |
| **日干（日主）** | **0分** | ⚠️ 日主=自己，不计比劫 |
| **日支藏干比劫** | 12×比例 | ✅ |
| **年支藏干比劫** | 4×比例 | ✅ |
| **时支藏干比劫** | 12×比例 | ✅ |

> **口诀**：比劫满盘都算，日主自己不算；年干月干与时干，逐字排查才算真

## 四、燥土规则

**判断优先级**：
```
① 天干有丙/丁？ → 未/戌当火看 → 不计印分，不生金
② 无丙/丁但有壬/癸？ → 灭火变土 → 当土看 → 计印分，生金
③ 无火无水？ → 当土看 → 计印分，生金
```

| 案例 | 月令 | 天干 | 燥土效果 | 印分 |
|:----|:----|:-----|:---------|:---:|
| 老板（辛金） | 癸未 | 癸水灭火 | 当土→生金 | +40 |
| 主母（庚金） | 丁未 | 双丁引化 | 当火→不生金 | 0 |

> **口诀**：燥土判断有顺序，先看丙丁后看水；丙丁优先当火看，壬癸浇水变回土

## 五、总分 = 印分 + 比劫分

所有加分项相加，**不加不减**，无任何扣分项。

## 六、判定标准

| 总分区间 | 状态 | 喜忌 |
|:--------:|:----|:-----|
| **>60分** | **身强** | 喜克泄耗（食伤>财>官） |
| **40~60分** | **中和** | 无固定喜忌，随大运变化 |
| **<40分** | **身弱** | 喜印比帮身 |
| **≤0分** | 从弱→恒定**50分** | 所有五行皆为喜用 |
| **≥100分** | 从强→恒定**20分** | 喜印比帮身 |

### 中和段精准分层（子源55.6分案校准）

| 分段 | 实战表现 | 喜忌 |
|:----|:---------|:-----|
| **55~60分** | 接近偏强侧 | 按固定喜忌：喜克泄耗，忌生扶 |
| **45~55分** | 正中间 | 喜忌随大运灵活变化 |
| **40~49分** | 接近偏弱侧 | 按固定喜忌：喜生扶，忌克泄耗 |

### 自坐比劫永不从弱

日支为比肩或劫财→**永远不从弱**，即使被克到1分也不从。

## 七、计算示例
