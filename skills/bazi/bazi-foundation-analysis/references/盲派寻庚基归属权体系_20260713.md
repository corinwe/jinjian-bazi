# 盲派寻庚基（寻根基）体系 · 完整方法论

> **来源：** 泉师兄18篇视频文案逐字精读（2026-07-13全量提取）
> **核心定义：** 寻庚基 = 给八字中每个五行找到「老家」（根基和出处）。这是盲派所有分析（归属权/枭神夺食/做功/食伤生财等）的前置条件。

## 1. 寻庚基标准口诀

```yaml
天干找地支，地支找库
一句话概括所有：天干找地支，地支找出处

【天干找地支】
  甲/乙木 → 找寅、卯、辰、未
  丙/丁火 → 找巳、午、未、戌
  戊/己土 → 找辰、戌、丑、未（戊己土的根较特殊）
  庚/辛金 → 找申、酉、戌、丑
  壬/癸水 → 找亥、子、辰、丑

【地支找库/出处】
  寅/卯木 → 优先找未库，无未找辰库
  巳/午火 → 优先找戌库，无戌找未库
  申/酉金 → 优先找丑库，无丑找戌库
  亥/子水 → 优先找辰库，无辰找丑库
  辰/戌/丑/未 → 本身就是库，不再往外找

【无根基无出处时的判定】
  在家外（年月）→ 就是家外的（社会的/别人的）
  在家里（日时）→ 就是家里的（自己的）
```

## 2. 家外/家里归属二分法

```yaml
盲派将八字四柱一分为二：
  年月 → 家外（代表社会、他人、外部环境的东西）
  日时 → 家里（代表自己、完全属于自己的东西）

【根在年月=家外→别人的/社会的】
  例：财星的根在年支 → 这个财是社会的，不是自己的
  例：比劫的根在月令 → 这些比劫是外面的人

【根在日时=家里→自己的】
  例：财星的根在日支 → 这个财是我自己的
  例：食伤的根在时支 → 这个技术是我自己的

【多个根=混血儿（部分社会+部分自己）】
  例：天干甲木，地支有寅木在家外、辰土在家里
  → 这个甲木部分属于社会、部分属于自己
  → 吉凶祸福都与自己相关
  → 创业合作模式：自己+别人一起

【根的位置决定力量】
  根在年上 = 摇根（太远，借不上力量）
  根在月令 = 强根（力量最大）
  根在日支 = 坐根（与日主连体，最稳固）
  根在时支 = 次强根（力量中等）
```

## 3. 定份额法

```yaml
定份额 = 计算一个五行被「几个人份」占有
这是断婚姻（体用多次关系=多次婚姻）的核心方法

【四份框架】
  第1份：从哪里出来的地方占一份（根基出处）
  第2份：在哪儿待着的地方占一份（所处宫位）
  第3份：家里不管几份都算一份（家里统算）
  第4份：谁生了我一下也算一份（给我投资）

【连体规则】
  天干与同柱地支连体 → 只算一份
  如甲木在月干+寅木在月支 → 甲从寅出+待在一起 → 只算一份

【应用示例】
  - 乙庚合 → 找庚金有几份
    月干庚金一份 + 家里庚金一份 = 两份
  → 两个人来合一个财 → 婚姻中另一半容易犯桃花 → 离异特征
  
  - 己亥自合 → 找己土有几份
    家外未土一份 + 巳火一份 + 家里巳火一份 = 三份
  → 三个人合这个己土财 → 多段婚姻
```

## 4. 归属权分析2×2矩阵（劫财见财）

```yaml
【核心框架】先定财的归属（自己/别人），再定比劫的归属（自己/别人）

                    财是自己的          财是别人的
  比劫是自己的   ①心甘情愿投资/扩张   ③去碰不该碰的→官司牢狱
  比劫是别人的   ②破财/被偷被抢       ④和自己没关系

【力量对比修正】
  日主根强 + 比劫根弱 → 别人抢不动 → 不破财
  比劫根强 + 日主根弱 → 必破财

【混血儿归属】
  一个财有多个根（如卯在家外+寅在家里）→ 合伙开公司
  食伤也有同样规则：食伤是我的筷子vs别人的筷子
```

## 5. 食伤当财看的条件

```yaml
【条件①】食伤在家外（跟家里一点关系都没有）→ 可以当财看
【条件②】原局没有财官 → 食伤当财看（无奈之举）
【优先规则】有财官优先用财官，没有才用食伤当财

【制 vs 合治的区别】
  - 制食伤 → 发财但自己和六亲出问题
  - 合治食伤 → 单纯发财，不出问题
  
【食伤当财的代价】
  食伤是快乐之本、身体 → 把食伤当财看 = 损亲损身
  容易触发枭神夺食
```

## 6. 寻庚基的实战应用链

```yaml
寻庚基 → 定归属 → 组合判断 → 出断语

Step 1 — 给每个字寻庚基（天干找地支，地支找库）
Step 2 — 定每个字的家外/家里归属
Step 3 — 如果是混血儿，列出所有份额
Step 4 — 根据归属组合判断吉凶：
  
  枭神夺食判断：
    枭神家外 + 食伤家内、阴枭制阳食 → 🚨 牢狱之灾
    枭神家外 + 食伤家内 → 精神层面问题
    官生印夺食 → 事业问题（追根溯源）
  
  财星归属判断：
    财在家里 + 比劫家外 → 破财（别人伸手掏口袋）
    财在家外 + 比劫家里 → 官司牢狱（碰了不该碰的）
    
  食伤生财判断：
    财在家里 → 自己做事业，有话语权
    财在家外 → 给别人打工，借他人资源
    食伤有别人份 → 必须合作
```
