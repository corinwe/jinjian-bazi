# 大运真假判定精确规则

> **来源**：2026-07-16 家主魏启令盲派报告校准
> **触发**：子代理将甲申(甲藏亥✅)和戊子(戊藏申✅)误判为「假大运」
> **核心规则**：泉师兄大运真假三法 + 藏干口径

## 一、泉师兄大运真假三法

```yaml
三法判定大运天干地支的真假：

法一 — 寻根基（最基础）：
  大运天干 → 在原局有根（透干或藏干）→ ✅ 真
  大运地支 → 在原局有出现 → ✅ 真
  无根/无出现 → ⚠️ 假

法二 — 鸠占鹊巢（进阶）：
  大运来了一个字 → 占了原局另一个字的位置 → 真假看占的是喜是忌

法三 — 生也微妙（高阶）：
  大运的字与原局合化出新的五行 → 合化的结果决定真假
```

## 二、藏干口径精确规则（2026-07-16 校准）

> **错误历史**：子代理只查天干有无透出，未查地支藏干，导致甲申（甲藏亥√）、戊子（戊藏申√）被判假

```yaml
【藏干口径判定规则】

天干在原局的「有」= 以下任一：
  ├─ 天干有明字（甲木在原局天干出现）→ 有 ✅
  ├─ 天干无明字，但地支藏干**本气/中气/余气**有该字 → 有 ✅
  ├─ 天干无明字，地支也无该藏干 → 无 ❌
  └─ ⚠️ 特别注意：余气(10-30%)也算「有」，不能因比例低就判无

例：甲申大运的甲
  原局天干无甲 ❌
  但原局亥中藏甲(中气60%) → 有根 → ✅ 真大运 ✅

例：戊子大运的戊
  原局天干无戊 ❌  
  但原局申中藏戊(余气30%) → 有根 → ✅ 真大运 ✅

例：丙戌大运的丙
  原局天干无丙 ❌
  原局地支无丙火藏干 ❌ → ⚠️ 假大运 ⚠️

口诀：真假先找根在哪，天干没有看藏干
      藏干本中余气都算，别嫌比例低就划假
      甲藏亥里是真实，戊藏申中也有根
```

## 三、大运真假判定表模板

每次分析大运时，逐运检查并填入下表：

```yaml
| 大运 | 天干 | 原局有/无 | 依据 | 真假 | 地支 | 原局有/无 | 依据 | 真假 |
|:----|:----|:---------:|:----|:----:|:----|:--------:|:----|:----:|
| 甲申 | 甲 | ✅ 藏亥 | 亥中甲(中气) | ✅真 | 申 | ✅ 有申 | 年支 | ✅真 |

| 丙戌 | 丙 | ❌ 无丙 | 原局无丙 | ⚠️假 | 戌 | ❌ 无戌 | — | ⚠️假 |

```

## 四、大运真假的影响

```yaml
真大运 → 该大运的能量真实进入命局，产生实际影响
  ├─ 真天干 + 真地支 = 双重真实，影响最大
  └─ 真天干 + 假地支 = 天干影响但地支不实在

假大运 → 该大运的能量是虚的，影响打折扣
  ├─ 假天干 = 表面的趋势，内在没变化
  └─ 假地支 = 看起来有舞台但站不稳

特殊：藏干余气算「有」
  → 余气比例低（10-30%），虽算有但力量弱
  → 该大运的能量不会太强（天干有根但根不深）
  → 结论：算真，但力度打八折
```
