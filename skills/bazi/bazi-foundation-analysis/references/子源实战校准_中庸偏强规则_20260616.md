# 子源实战校准：中庸边界处理规则

> 案例：子源（辛卯 癸巳 丙戌 癸巳），丙火日主，55.6分
> 校准日期：2026-06-16

## 校准触发

v12.0按九龙道长「40-60中庸·无固定喜忌」规则处理，结果3个关键事件全部断错：
- 2013年搬迁 → 错断为2017 ❌
- 2025年考入更好学校 → 误判为"健康注意" ❌
- 2025-2026学业持续长进 → 完全遗漏 ❌

而v8.1/v11.1按「中和偏强·固定喜土金水忌木火」判断，3事件全部正确 ✅。

## 结论

**40-60分中庸区间内，靠近两端时应按偏侧处理：**

| 分数段 | 处理方式 | 喜忌 |
|:------|:--------|:-----|
| 55-60分 | 按偏强处理 | 固定喜克泄耗，忌印比 |
| 45-55分 | 真正的中间带 | 喜忌随大运灵活变化 |
| 40-49分 | 按偏弱处理 | 固定喜印比帮身，忌克泄耗 |

## 规则修正

九龙道长原文「40-60中庸不分段·喜忌随大运变」在靠近两端（55-60/40-49）时，实战中表现出固定的喜忌方向。应将此作为补充规则：

```yaml
中庸边界规则：
  55-60分 → 实战偏强，按身强喜忌（固定喜克泄耗）
  40-49分 → 实战偏弱，按身弱喜忌（固定喜印比）
  45-55分 → 真正的中间平衡，按九龙道长原文（喜忌随大运变）
```

| 项目 | v17.0（旧版·错） | v18.0（本版·正确） |

| 🔴 忌神 | 火(比劫)>土(食伤)>金(财) ❌ | 火(比劫)>木(印星) ✅ |

### 根因分析

**错误逻辑链：**
1. 看到丙火日主+双癸水正官→误以为身弱/需要补木印生身
2. 55.6分靠近60上限→实际偏强侧
3. 月令巳火+双巳比肩+戌火库→火势已旺→补木是火上浇油

**正确逻辑链：**
1. 丙火生于巳月当令+双巳比肩+戌火库→火势偏旺
2. 55.6分偏强侧→需要泄耗（克泄耗=水土金）
3. 水制火（官杀）、土泄火（食伤）、金耗火（财）

### 易混淆辨析

特别容易混淆的组合：
```
丙火偏强 + 天干有双癸水正官 → 容易误判为「需要木印生身」
实际上双癸水克火已经消耗了丙火，再补木印→木生火→火上浇油
正确的理解：双癸水制住了火势，不需要再补木印助火
```
