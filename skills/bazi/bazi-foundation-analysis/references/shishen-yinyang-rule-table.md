## 口诀

**同阴阳=偏的/凶的（偏印、劫财、七杀、伤官、偏财）**
**异阴阳=正的/吉的（正印、比肩、正官、食神、正财）**

## 规则表

| 类别 | 关系 | 同阴阳（阳见阳/阴见阴） | 异阴阳（阳见阴/阴见阳） |
|:-----|:-----|:-----------------------|:-----------------------|
| 生我(印) | 印星 | **偏印** | 正印 |
| 克我(官杀) | 官杀 | **七杀**(阳克阳/阴克阴) | 正官(阳克阴/阴克阳) |
| 我生(食伤) | 食伤 | 食神(阳我生阳/阴我生阴) | **伤官**(阳我生阴/阴我生阳) |
| 我克(财) | 财星 | **偏财**(阳我克阳/阴我克阴) | 正财(阳我克阴/阴我克阳) |
| 同五行 | 比劫 | 比肩(阳见阳/阴见阴) | **劫财**(阳见阴/阴见阳) |

## 验证工具

```python
# 快速验证日主X对天干Y的十神
def quick_ss(rizhu, tg):
    YY = {'甲':'阳','乙':'阴','丙':'阳','丁':'阴','戊':'阳','己':'阴','庚':'阳','辛':'阴','壬':'阳','癸':'阴'}
    WX = {'甲':'木','乙':'木','丙':'火','丁':'火','戊':'土','己':'土','庚':'金','辛':'金','壬':'水','癸':'水'}
    same = YY[tg] == YY[rizhu]
    rw, tw = WX[rizhu], WX[tg]
    if tw == rw: return '比肩' if same else '劫财'
    if {'金':'土','木':'水','水':'金','火':'木','土':'火'}[rw] == tw: return '偏印' if same else '正印'
    if {'金':'水','木':'火','水':'木','火':'土','土':'金'}[rw] == tw: return '食神' if same else '伤官'
    if {'金':'火','木':'金','水':'土','火':'水','土':'木'}[rw] == tw: return '七杀' if same else '正官'
    if {'金':'木','木':'土','水':'火','火':'金','土':'水'}[rw] == tw: return '偏财' if same else '正财'
```
