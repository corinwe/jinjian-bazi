## 六冲七型
| 类型 | 条件 | 结论 |
|:----|:----|:------|
| 冲凶 | 忌神得根冲入命中 | 主凶事 |
| 冲旺 | 太岁冲极旺之神 | 冲旺喜神吉，冲旺忌神凶 |
| 冲动 | 冲静止之字使动 | 影响命局 |
| 冲开 | ①合者被冲 ②墓库被冲 | 婚期/转折 |
| 冲出 | ①冲年/时 ②冲墓 | 外出/释放 |
| 冲去 | 冲衰神 | 离开不死亡 |
| 冲破 | 冲极衰受克者 | 主大凶死亡 |

只有冲凶和冲破主凶。

## 六合三用
1. **合留**：配偶星旺相逢冲刑，太岁合之→婚成
2. **合绊**：合绊喜神凶，合绊忌神吉。六合能解三合
3. **合凶**：某字太弱注定死→应合之年死亡

## 以禄当财
**条件**：财官食伤都无用时有禄→禄当财。财官食伤有用时不可用。
**忌**：怕劫财(分禄)、怕财(禄神怕财)、怕复吟(重抑大凶)

## 伤官去官格
以官为忌神，伤官有力去官→得官。不论日主衰旺。

## 宾主七层
1. 日主为主，他干支为宾
2. 日柱为主，他柱为宾
3. 日时为主，年月为宾
4. 贴日主者为主，远离为宾
5. 八字为主，大运流年为宾
6. 禄印比肩为主，他神为宾；食神财星为主，杀刃劫为宾
7. 用神喜神为主，忌神仇神为宾

## 穿害六组
子未、丑午、寅巳、卯辰、申亥、卯戌。
"穿倒正印归不了正" — 相克又带穿最严重。

## 从格四条件
1. 日主无根无生扶者从
2. 有根但根被坏者从(年支最远→易从)
3. 无根印也无根者从
4. 无根印有根但根被坏者从

## 干支虚实
| 类型 | 判定 |
|:----|:------|
| 虚忌神(无根忌) | 吉 |
| 实用神(有根用) | 吉 |
| 实忌神(有根忌) | 凶 |
| 虚用神(无根用) | 凶 |

例外：调候用神虚也吉，泄神虚也吉(秀气宜虚)。

## 忌神三制
1. **旺制**：弱忌神→彻底制服称贼神
2. **平衡制**：略偏旺→相制衡，化敌为友
3. **弱制**：过强不从→周旋

**贼神**：客贼神(忌神被制如警察所中的贼)→大富贵命。主贼神(喜神被制)→凶恶残疾命。
