# 微信公众号文章提取技巧（反爬绕过方案）

> **背景：** `web_extract` 工具对微信公众平台文章（mp.weixin.qq.com）通常失效，因为微信有强反爬机制（CAPTCHA滑块验证）。
> **已验证方案：** curl + 移动端MicroMessenger UA + 正则提取 `rich_media_content` 或 `og:description`
> **编写日期：** 2026-06-27

## 方案A：curl直接提取（最快，优先尝试）

```bash
UA="Mozilla/5.0 (Linux; Android 13; SM-S9080) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/112.0.0.0 Mobile Safari/537.36 MicroMessenger/8.0.33"
curl -s -L -A "$UA" "https://mp.weixin.qq.com/s/{ARTICLE_ID}" | \
  python3 -c "
import sys, re, html
data = sys.stdin.read()
# 提取ct变量（编码后的文章内容）
ct_match = re.search(r'var ct\s*=\s*\"([^\"]+)\"', data)
if ct_match:
    content = html.unescape(ct_match.group(1))
    print(content)
"
```

## 方案B：提取rich_media_content div（遇到验证码时备用）

有时`ct`变量只有时间戳(10字符)，真实内容在 `js_content` div中（Vue渲染）。提取HTML中嵌入的元数据：

```bash
UA="Mozilla/5.0 (Linux; Android 13; SM-S9080) AppleWebKit/537.36"
curl -s -L -A "$UA" "https://mp.weixin.qq.com/s/{ARTICLE_ID}" | \
  python3 -c "
import sys, re, html
data = sys.stdin.read()

# 提取OG描述（摘要）
og = re.search(r'<meta[^>]*property=\"og:description\"[^>]*content=\"([^\"]+)\"', data)
if og: print('摘要:', og.group(1)[:200])

# 提取OG标题
title = re.search(r'<meta[^>]*property=\"og:title\"[^>]*content=\"([^\"]+)\"', data)
if title: print('标题:', title.group(1))

# 提取rich_media_content
m = re.search(r'<div[^>]*id=\"js_content\"[^>]*>(.*?)</div>\s*<script', data, re.DOTALL)
if m:
    content = m.group(1)
    clean = re.sub(r'<[^>]+>', '\n', content)
    clean = re.sub(r'\n{3,}', '\n\n', clean).strip()
    print('正文:', clean[:3000])
"
```

## 方案C：查看页面元数据确定文章主题

当正文无法提取时（被CAPTCHA拦截），仍可提取页面meta信息确定文章主题：

```python
import re
# 标题
og_title = re.search(r'<meta[^>]*property="og:title"[^>]*content="([^"]+)"', data)
# 描述 
og_desc = re.search(r'<meta[^>]*property="og:description"[^>]*content="([^"]+)"', data)
# 公众号ID
biz = re.search(r'biz\s*=\s*"([^"]+)"', data)
```

## 坑与注意事项

1. **ct变量可能为时间戳**（10位数字）而非文章内容——此时需要用方案B
2. **rich_media_content需要Vue渲染**——静态HTML提取可能缺失部分段落
3. **文章末尾可能混入广告**（微信号/产品/课程链接）——提取后需人工剔除
4. **微信对IP频率有控制**——同一IP不要连续请求多篇文章，间隔至少3秒
5. **biz参数**=公众号唯一标识，`mid`=文章ID，可用于构造替代链接
6. **页面HTML大小通常2-3MB**——因为包含大量Vue.js框架代码，实际文章内容占很小比例
