# 庚金未月格局精要 —— 逐字精读笔记
来源：命理经典·六月庚金总评
精读日期：20260722

## 一、核心要义

### 格局总纲
庚金生于未月（农历六月），处于三伏天。未月土最旺（辰戌丑未四土中未土最旺）。

**两个关键特征：**
1. 土旺 → 庚金被土埋没，变得坚硬顽固
2. 三伏生寒 → 表面炎热但金气水气开始增强，阴阳转换

### 核心用神：丁火（第一）+ 甲木（第二）

丁火的作用：煅烧庚金 → 使庚金成器。未月庚金如同矿石，需要丁火熔炼才能成器。
甲木的作用：疏松土壤 → 土旺埋金，甲木破土彰显庚金。

### 身强vs身弱的用法差异
- 身强时 → 喜财（甲木）和官（丁火）
- 身弱时 → 喜印（土/金）

## 二、格局层次（逐级递减）

| 配置 | 层次 | 说明 |
|:----|:----|:-----|
| **丁甲双透天干** | 🥇 **清贵显达** | 最高格局，科举成名 |
| 只有丁火无甲木 | 🥈 有才学的秀才 | 有才能但不够顶尖 |
| 只有甲木无丁火 | 🥉 普通人 | 平庸之命 |
| 丁甲皆无 | ❌ 地位低下 | 贫贱 |
| 丁藏支不透 + 不被水伤 | 商贸/衣食足 | 做生意能糊口 |
| 丁藏支 + 地支见水 | 体力劳动 | 苦力之命 |

## 三、特殊格局

### 地支成土局
- 地支组成土局（辰戌丑未数组）→ 先取甲木疏土，后取丁火煅金
- 甲木透干 → 以文章显达，既富且贵

### 四柱多见庚金（比劫多）
- 需要**两个丁火透出**克制庚金 → 他途显达，非正统途径成就
- 只有一个丁火透出 → 文书小吏/舞文弄墨

### 丁火与癸/壬水的禁忌
- 丁火无论是藏是透 → **最忌癸水伤害**
- 未月庚金大忌就是水伤丁火
- 未中本藏丁火，但如果己土当令旺 → 丁火力量泄于土，如同没有

## 四、大暑前后的区别
- **大暑前**：与五月庚金分析方法相同（火更旺）
- **大暑后**：金气水气开始增强，三伏生寒更明显

## 五、关键口诀
> **未月庚金，三伏生寒。**
> **土旺埋金，丁炼甲疏。**
> **丁甲双透，清贵显达。**
> **丁透甲藏，秀才之才。**
> **癸伤丁火，最是要防。**

## 六、对主母成八字的对照判断

八字：丁卯 丁未 庚午 壬午

### 优势
1. **双丁火透天干** ✅ → 丁火为第一用神，双透极佳
   - 原文说庚金多时需要双丁火克制 → 太太八字庚金虽不多，但双丁火煅炼得力
2. **庚金生于未月** ✅ → 未月对应
3. **火旺（双丁+双午）** → 丁火有力煅金

### 不足
1. **无甲木透干** ❌ → 甲木为佐神不足（仅有年支卯中乙木）
   - 原文说「有丁无甲→才学秀才」，层次低于丁甲双透
2. **壬水透时干** ⚠️ → 癸水/壬水伤丁火
   - 原文特别强调「忌癸水伤丁」，壬水也是水同样克火
3. **火过旺** → 庚金被熔而非被炼（火太过则熔金）
   - 火用来煅金正好，火太多则金被熔化

### 综合定位
双丁火透干是太太命中最亮的点——丁火为用神煅庚金。但：
- 无甲木疏土 → 破土之力不足
- 壬水伤丁 → 用神被损伤（虽然壬水也是调候，但原文说水为忌）
- 火炎土燥 → 庚金不是被温和地煅造，而是被烈火包围

定位：中等偏上格局。有丁火煅金之贵，但壬水伤丁、无甲木疏土，贵气不足。
