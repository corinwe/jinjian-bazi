# 大运流年能量分阶段 · 喜忌判定铁律 · 配偶特征断语规则

> 2026-07-07 魏启令报告校准固化

## 一、大运喜忌判断规则（干支分治+能量分阶段）

### 核心原则
大运的喜忌判定，**不能笼统地概括为"纯喜用"或"纯忌神"**——必须分天干、地支判定。

### 公式

```
【前五年（第1~5年）】→ 大运天干70% + 大运地支30%  ← 天干主导
【后五年（第6~10年）】→ 大运天干30% + 大运地支70%  ← 地支主导
```

### 判例

| 大运 | 天干 | 地支 | 前5年体验 | 后5年体验 |
|:----|:----|:----|:---------|:---------|
| 丙戌 | 丙正官(喜用) | 戌正印(忌神) | 丙火70%→官克身·想做事 | 戌土70%→印保守·放不开 |
| 戊子 | 戊正印(忌神) | 子食神(喜用) | 戊土70%→保守压抑 | 子水70%→才华释放 |
| 丁亥 | 丁七杀(喜用) | 亥伤官(喜用) | 丁火70%→被逼成长 | 亥水70%→伤官生财 |

### 🚨 严禁行为
**禁止**：说丙戌大运是"双忌神"（正官是喜用不是忌神）
**✅ 正确**：说丙戌大运"天干喜正官(前5年官制身)+地支忌正印(后5年保守纠结)"

## 二、流年喜忌判断规则（干支分管时间）

### 公式

```
【上半年（1~6月）】→ 流年天干主导 → 看天干五行+十神 + 与原局/大运的互动
【下半年（7~12月）】→ 流年地支主导 → 看地支五行+十神 + 刑冲合害 + 与原局/大运的互动
```

### 实战用法
1. **事件发生在1~6月** → 看流年天干的能量
2. **事件发生在7~12月** → 看流年地支的能量
3. **跨年事件** → 兼顾天干+地支，但以地支产生的刑冲合害为持久影响

### 与大运的能量叠加
> 当前大运所处阶段(前/后5年) × 能量权重 + 流年干支分时(上半年/下半年) = 实际体验

### 判例
| 流年 | 天干 | 地支 | 上半年(天干管) | 下半年(地支管) |
|:----|:----|:----|:--------------|:--------------|
| 2024甲辰 | 甲正财(喜用) | 辰正印(忌神) | 1~6月财运好→收入最高 | 7~12月印保守→花钱多 |
| 2025乙巳 | 乙偏财(喜用) | 巳正官(喜用) | 1~6月偏财机会 | 7~12月官星制身→合同/文书 |
| 2026丙午 | 丙正官(喜用) | 午七杀(喜用) | 1~6月正官立规矩 | 7~12月七杀驱动→拓展 |

## 三、配偶特征断语规则

### 问题溯源
原写法："长相一般（寅申巳亥）" — 列举了四个地支的全组名，但命主日支只有"亥"。

老板指出："哪有寅巳？"

### 修正规则

**断语配偶特征时，只列日支实际所在的那个类，不列全组名称：**

```
✅ 正确做法：
  - 日支子 → "长相好（子为四桃花之一）"
  - 日支午 → "长相好（午为四桃花之一）"  
  - 日支卯 → "长相好（卯为四桃花之一）"
  - 日支酉 → "长相好（酉为四桃花之一）"
  - 日支寅 → "长相一般（寅为四长生之一）"
  - 日支申 → "长相一般（申为四长生之一）"
  - 日支巳 → "长相一般（巳为四长生之一）"
  - 日支亥 → "长相一般（亥为四长生之一）"
  - 日支辰 → "长相敦厚（辰为四墓库之一）"
  - 日支戌 → "长相敦厚（戌为四墓库之一）"
  - 日支丑 → "长相敦厚（丑为四墓库之一）"
  - 日支未 → "长相敦厚（未为四墓库之一）"

❌ 错误做法：
  - "长相一般（寅申巳亥）" ← 列全组名，与日支无关
  - "长相好（子午卯酉）" ← 列全组名
  - "长相敦厚（辰戌丑未）" ← 列全组名
```

**理由**：八字断语要精准对应局中实有地支，列全组名会让外行人困惑（"哪有寅和巳？"）。

## 四、大运年龄显示规则（2026-07-07修正）

### 九龙道长向上取整规则

大运年龄区间显示使用**ceil向上取整**：

```
起运年龄浮点数 → math.ceil() → 整数起始岁数
每步大运 = base + step*10 ~ base + (step+1)*10 - 1  （整10年区间）
```

### 判例
| 起运年龄 | 向上取整 | 第1步 | 第2步 |
|:--------|:--------|:------|:------|
| 0.33岁 | 1岁 | 1~10岁 | 11~20岁 |
| 7.0岁 | 7岁 | 7~16岁 | 17~26岁 |
| 15.56岁 | 16岁 | 16~25岁 | 26~35岁 |
