## 一、身强弱评分体系（九龙道长原始法）

### 位置基础分（满分100分，日干不计）

| 位置 | 基础分 | 原文证据 |
|:-----|:------:|:---------|
| 年干 | 8分 | 素材01行409反推："这里的财有8分" |
| 年支 | 4分 | 素材01行409反推（年干8+月令40+日时各12+年支4=100满分布局） |
| 月干 | 12分 | 素材02行45："天干一透…按这个能量得分12分算" |
| **月令** | **40分** | 素材02行45："**光月令一个都40分了**" ✅ |
| 日支 | 12分 | 素材01行409反推："这里的财有12分" |
| 时干 | 12分 | 同月干/日支逻辑（天干=12分/个） |
| 时支 | 12分 | 同日支逻辑 |
| **合计** | **100分** | |

### 藏干百分比及原文证据

| 藏干层次 | 百分比 | 原文证据 |
|:--------:|:------:|:---------|
| **本气** | **100%** | 月令本气×100%=40分（素材02"光月令一个都40分了"） |
| **中气** | **60%** | 无直接原文，但余气30%反推（中间层=60%） |
| **余气** | **30%** | 素材01行409："**这个是12分的30%，对，30%，大概是有3.6分**" ✅ |

### 🚨 身强弱 = 只算加分，不加不减（2026-06-15老板确认）

```
身强弱 = 八字中所有生助日主的五行（印+比劫）的藏干得分之和
       = 不加不减，不是「生助加分消耗减分」

原文依据（素材文档，非精确引用，概括大意）：
  「只算加分就行了，不用算减分。
  总共100分，你算他加了多少分，不就知道他日元多少分了？」
```

### 三档判定

```
<40分  → 身弱（喜印比帮身，忌财官克泄耗）
40-60分 → 中和（40-49偏弱，50正中和，51-60偏强）
>60分  → 身强（喜食伤财官克泄耗，忌印比帮身）
```

## 二、月令印星/比劫计分规则·原文交叉验证（2026-06-16新增）

### ✅ 月令印星计分 · 确认

**素材01行409（原文直录）**：
> "这个印在月令，一个印都已经**40分**了"

→ **结论：月令印星40分，无争议** ✅

### ✅ 非月令印星同样计分 · 确认

**素材22行81（原文直录）**：
> "只有这一个戌生住他，这样计算的话，他是**12分**"

→ 此戌**不得令**（不是月令），计12分
→ **结论：印星在所有位置都计分，不限月令** ✅

### ✅ 月令比劫同样计分 · 确认（2026-06-16关键修正）

**素材25行93（原文直录）**：
> "比肩和劫财**不入格局**"

⚠️ **此句仅指"格局判定"，不是"身强弱评分"！**

- **格局判定（正八格）**：正官/七杀/正财/偏财/正印/偏印/食神/伤官 — 八种格局，比劫不入格
- **身强弱评分**：完全独立的维度，月令任何对日主生助的五行（包括比劫）都应计分

九龙道长**从未**说过"月令比劫不计分"。他的体系逻辑是：
> 每个位置有基础分 → 看该位置五行对日主是生助还是消耗 → **生助者加该位置分**（不加不减）

→ **结论：月令比劫计分规则正确** ✅

### ✅ 验证案例：亮八字修正前34.4 vs 修正后74.4

```
亮（壬戌 辛亥 癸亥 戊午），癸水日主，亥月帝旺
修正前规则：月令比劫不计分 → 34.4分 → 身弱 ❌（主观不符合，亥月帝旺怎么可能身弱？）
修正后规则：月令比劫计分 → 74.4分 → 身强 ✅（亥月帝旺成立）

老板校正：「月令得令不分印比——印星生身代表得令，比劫同身同样代表得令」
```

## 三、燥土对金的特殊规则（2026-06-15新增）

```
未/戌 = 燥土（含火性）→ 对庚金/辛金日主不计分
  未中己土（印星）→ 燥土中的土性对金不计分 ⚠️
  未中丁火（官杀）→ 本就非加分项 → 不受影响

辰/丑 = 湿土（含水）→ 对庚金/辛金日主正常计分

原文依据（来自课程文档，概括大意）：
  燥土=含有火性的土，火克金，所以燥土中的土性对金不构成生助
  湿土=含水的土，水生金，所以湿土中的土性对金正常生助
```

## 四、从格临界分值

```
得分 > 100分 → 从强反为弱（恒定20分，喜印比）
得分 < 0分  → 从弱反为强（恒定50分，喜财官食）

自坐比劫永不从弱：
  日支为比肩或劫财 → 永远不从弱

从弱50分规则：
  大运流年双消耗 → 保持从弱（50分，喜克泄耗）
  大运流年双生助 → 退出从弱，重新评分
```

## 六、关键术语区分（2026-06-16新增）

### 格局判定 vs 身强弱评分 —— 两码事！

```
格局判定（正八格）：
  月令本气→中气→余气依次看谁透干
  比肩/劫财不入格局（因为同五行）
  目的：看「一个人一生最主要的追求方向」

身强弱评分：
  ✅ 月令本气印 = +40分
  ❌ 月令中/余气印 = 0分
  ❌ 其他位置印（年/日/时/天干）= 0分
  ✅ 月令本气比劫 = +40分
  ✅ 月令中/余气比劫 = +分数
  ✅ 其他位置比劫 = +分数
  ✅ 大运流年印 = +分数（不分位置）
  ✅ 燥土（未/戌）对庚辛金 = 0分（金鉴真人独创补充）
  目的：看「日主是否有足够能量担财官」
```

### 口诀

```
格局定方向，强弱看能量；
比劫不入格，但入强弱账；
月令得令时，印比分都算；
两码事各论，切勿混一谈。
```
