## 一、身强弱评分体系（九龙道长原始法）

### 位置基础分（满分100分，日干不计）

| 位置 | 基础分 | 原文证据 |
|:-----|:------:|:---------|
| 年干 | 8分 | 素材01行409反推："这里的财有8分" |
| 年支 | 4分 | 素材01行409反推（年干8+月令40+日时各12+年支4=100满分布局） |
| 月干 | 12分 | 素材02行45："天干一透…按这个能量得分12分算" |
| **月令** | **40分** | 素材02行45："**光月令一个都40分了**" ✅ |
| 日支 | 12分 | 素材01行409反推："这里的财有12分" |
| 时干 | 12分 | 同月干/日支逻辑（天干=12分/个） |
| 时支 | 12分 | 同日支逻辑 |
| **合计** | **100分** | |

### 藏干百分比及原文证据

| 藏干层次 | 百分比 | 原文证据 |
|:--------:|:------:|:---------|
| **本气** | **100%** | 月令本气×100%=40分（素材02"光月令一个都40分了"） |
| **中气** | **60%** | 无直接原文，但余气30%反推（中间层=60%） |
| **余气** | **30%** | 素材01行409："**这个是12分的30%，对，30%，大概是有3.6分**" ✅ |

### 🚨 月令只看本气评分（2026-07-09老板校准 · 重要！）

```yaml
月令定旺衰，只看本气对日主的生克关系。
中气余气不计入身强弱评分！

✅ 月令本气为印/比劫 → 计40分（生助日主）
❌ 月令中气/余气为印/比劫 → 0分（不计入身强弱评分）
❌ 月令中气/余气为比劫 → 0分（同印星规则，中气余气均不计）

为什么？
  月令的40分代表「月令的能量强度」
  这个强度只体现在「本气」上——本气才是月令的主宰五行
  中气余气只是辅助，不改变月令的根本属性
  所以身强弱评分中月令位置**只问本气对日主生助与否**

对比非月令位置：
  非月令位置（年/日/时支）的藏干按比例计分
  因为非月令位置的「位置分」本身较小，藏干层次体现的是该位置的内部能量分布
```

### 🚨 身强弱 = 只算加分，不加不减（2026-06-15老板确认）

```
身强弱 = 八字中所有生助日主的五行（印+比劫）的藏干得分之和
       = 不加不减，不是「生助加分消耗减分」

原文依据（素材文档，非精确引用，概括大意）：
  「只算加分就行了，不用算减分。
  总共100分，你算他加了多少分，不就知道他日元多少分了？」
```

### 三档判定

```
<40分  → 身弱（喜印比帮身，忌财官克泄耗）
40-60分 → 中和（40-49偏弱，50正中和，51-60偏强）
>60分  → 身强（喜食伤财官克泄耗，忌印比帮身）
```

## 二、月令印星/比劫计分规则·原文交叉验证（2026-06-16新增 · 2026-07-09修正）

### ✅ 月令印星计分 · 确认

**素材01行409（原文直录）**：
> "这个印在月令，一个印都已经**40分**了"

→ **结论：月令印星40分，无争议** ✅

### ✅ 非月令印星同样计分 · 确认

**素材22行81（原文直录）**：
> "只有这一个戌生住他，这样计算的话，他是**12分**"

→ 此戌**不得令**（不是月令），计12分
→ **结论：印星在所有位置都计分，不限月令** ✅

### ⚠️ 月令比劫同样只看本气（2026-07-09修正 · 重要！）

**之前（2026-06-16记录）**：月令中/余气比劫=+分数 ❌
**修正后（2026-07-09老板校准）**：月令中/余气比劫=**0分** ✅

**原则统一**：月令位置的「中气/余气」不论十神属性，**一律不计入身强弱评分**。
**原因**：月令的40分是本气的专属分，中气余气无权使用。
非月令位置（年/日/时支）的藏干才按比例计分。

### ✅ 验证案例

**释道（乙卯 己丑 癸未 戊午）**：
旧判：月令丑中癸水中气计24分 → 身弱24分 ❌
新判：月令只看本气己土（七杀克身）→ 0分 → 全局0分生助 → 从弱格 ✅
老板核：「月令定旺衰，只看本气对日主的生克关系」

## 三、燥土对金的特殊规则（2026-06-15新增）

```
未/戌 = 燥土（含火性）→ 对庚金/辛金日主不计分
  未中己土（印星）→ 燥土中的土性对金不计分 ⚠️
  未中丁火（官杀）→ 本就非加分项 → 不受影响

辰/丑 = 湿土（含水）→ 对庚金/辛金日主正常计分

原文依据（来自课程文档，概括大意）：
  燥土=含有火性的土，火克金，所以燥土中的土性对金不构成生助
  湿土=含水的土，水生金，所以湿土中的土性对金正常生助
```

## 四、从格临界分值

```
得分 > 100分 → 从强反为弱（恒定20分，喜印比）
得分 < 0分  → 从弱反为强（恒定50分，喜财官食）

自坐比劫永不从弱：
  日支为比肩或劫财 → 永远不从弱

从弱50分规则：
  大运流年双消耗 → 保持从弱（50分，喜克泄耗）
  大运流年双生助 → 退出从弱，重新评分
```

## 六、关键术语区分（2026-06-16新增 · 2026-07-09修正）

### 格局判定 vs 身强弱评分 —— 两码事！

```
格局判定（正八格）：
  月令本气→中气→余气依次看谁透干
  比肩/劫财不入格局（因为同五行）
  目的：看「一个人一生最主要的追求方向」

身强弱评分：
  ✅ 月令本气印/比劫 = +40分（只计本气！）
  ❌ 月令中/余气任何十神 = 0分（中气余气不计入身强弱！）
  ✅ 年支/日支/时支藏干 = +分数（按比例）
  ✅ 天干印/比劫 = +分数
  目的：看「日主是否有足够能量担财官」

口诀：月令只看本气缸，中气余气一边晾；
      格局看透干，强弱看本气。
```
