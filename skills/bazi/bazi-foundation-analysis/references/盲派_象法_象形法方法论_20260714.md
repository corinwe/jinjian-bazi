# 盲派·象法（象形法）方法论

> **来源**：泉师兄19篇课程逐字精读
> **本质**：不只看十神技术关系，还要读每个字的**形象/形状/象形意义**
> **位置**：八字分析的最后一步——十神/做功路线分析完后，用象法做具体化判断

## 一、象法的三个层次

| 层次 | 含义 | 例子 |
|:----|:-----|:------|
| **象形** | 天干地支的形象/形状 | 寅木象笔杆子、巳火象炉火 |
| **意象** | 天干地支的意境/气质 | 壬水是大江大河、癸水是涓涓细流 |
| **合象** | 合化后的整体意象 | 卯戌合火象教育、寅丑合象暗箱操作 |

## 三、实战用法

```
分析步骤：
① 先用十神/做功路线分析（技术层面）
② 再用象法做最后的具体化判断（形象层面）
③ 两者交叉验证

案例1：「大家可以从象的角度去读取壬寅年」
壬水是大江大河之水，己土是田园土
→ 田园土怎么约束大江大河？约束不住
→ 所以壬寅年不会被裁，只是有压力

案例2：「癸卯年枭神夺食」
癸水是雨水涓涓细流，己土田园土可以克制
→ 这一年才会真正被裁员
→ 象法解释了同是枭神夺食，不同干支力量不同

案例3：「寅丑合象一种暗箱操作的感觉」
寅=笔杆子=自媒体，丑=暗处
→ 寅丑合=灰色项目/割韭菜
```

## 四、核心口诀

```
十神断完象法收，字形意象看通透
壬水大江己田园，癸水细雨戊高山
寅卯笔杆文化业，巳午炉火餐饮行
申酉机器珠宝响，亥子江河细流长
象法一出定行业，具体判断不迷茫
```
