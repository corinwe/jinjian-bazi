# 月令身强弱只看本气 + 从弱格局优先于正八格

## 触发案例
释道（乙卯 己丑 癸未 戊午）
旧判：七杀格·身弱24分·喜金水
新判：从弱格(从杀格)·50分·喜土火木

## 错误根因
引擎 `shen_qiang_ruo.py` 月令比劫评分对丑中癸水中气计24分（40×0.6）
引擎 `ge_ju.py` 未检测从弱标签，直接返回月令本气定的"七杀格"

## 九龙道长原始规则（2026-07-09老板亲自指出）
**月令身强弱只看本气对日主的生助，中气余气不计。**
- 印在月令本气才计40分，中气/余气印=0分 ✅（代码原已有）
- 比劫在月令同样只看本气，中气/余气比劫=0分 ❌（代码之前未实现，已修复）
- 其他位置的比劫(天干/年支/日支/时支)仍按原规则计分

**格局判定优先级：特殊格局(从弱/从旺) > 月令本气正八格**
- `determine_ge_ju()` 先查 `sqr_label == "从弱"` → 返回"从弱格(从杀/从财/从儿)"
- 从弱子类取克泄耗五行(土/火/木)中能量最强的判定
- 不命中再从弱/从旺→才进入月令本气→透干→正八格逻辑

## 修正文件清单
1. `engine/shen_qiang_ruo.py`
   - 月令比劫评分从"循环所有藏干"改为"只取本气"
   - 燥土规则同步简化
2. `engine/ge_ju.py`
   - v3.0：特殊格局优先判断→月令本气正八格逻辑
3. `skills/bazi/bazi-foundation-analysis/SKILL.md`
   - 核心公式：比劫只在月令本气计分
   - 评分方法：月令中气余气比劫=0分
   - 规则说明：比劫在月令只看本气
   - 假从格规则：补充比劫在月令同样只看本气
4. `HERMES.md` — 新增铁律⑨：有疑问先查原始理论

## 验证方法
```python
# 释道从弱验证
sqr = compute_shen_qiang_ruo(bazi)
print(sqr.label)  # → "从弱"
print(sqr.score)  # → 50.0
print(sqr.details.yue_ling_bi_jie)  # → 0.0

gj = determine_ge_ju(bazi)
print(gj[0])  # → "从弱格"
print(gj[1])  # → "从弱格(从杀格)"
```

## 影响范围
所有月令中气为比劫的八字都会受影响：
- 丑月癸水日主 → 原先多算24分比劫
- 辰月壬水日主 → 辰中癸水余气比劫不计
- 未月己土日主→未中己土余气比劫不计 (不过未中己土只30%->12分)
- 戌月戊土日主→戌中戊土余气比劫不计

## 口诀
月令只看本气算，中气余气皆不沾；
比劫印星同此理，从弱从旺格优先。
