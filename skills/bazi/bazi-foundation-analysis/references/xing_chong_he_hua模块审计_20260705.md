## BUG-1（高）：寅巳申三刑能量系数错误

### 现象
`check_xing()` 对 **寅巳申三刑** 和 **丑未戌三刑** 都使用 `NENG_LIANG["三刑"] = 15.0`：

```python
# xing_chong_he_hua.py L159-173
NENG_LIANG = {
    "三刑": 15.0,  # ← 被两个不同的三刑共享
    "二刑": 8.0,
    ...
}
```

```python
# L187-188: 寅巳申三刑 → 用 NENG_LIANG["三刑"] = 15.0 ❌
# L195-196: 丑未戌三刑 → 用 NENG_LIANG["三刑"] = 15.0 ✅
```

### 正确规则（SKILL.md L2059-2060）
| 类型 | 有引化 | 无引化 |
|:----|:------:|:------:|
| **丑未戌三刑** | **15倍** | 7-8倍 |
| **寅巳申三刑** | **10倍** | 5倍 |

### 修复方案
```python
NENG_LIANG = {
    "丑未戌三刑": 15.0,
    "寅巳申三刑": 10.0,
    "二刑": 8.0,       # 寅巳申/丑未戌的不全形式
    ...
}
```
`check_xing()` 中分别取对应 key。

### 影响范围
- `bazi-fortune-analysis`（命运分析引用能量系数）
- `bazi-engine-workflow`（引擎管线）
- 所有调用 `check_xing()` 的排盘/流年分析

## BUG-2（中）：自刑漏判"辰午酉亥不论组合"

### 现象
`check_xing()` L203-205 仅检查同一字符出现>=2次：

```python
for zhi in ["辰", "午", "酉", "亥"]:
    if zhi_list.count(zhi) >= 2:
        results.append((f"{zhi}{zhi}自刑", NENG_LIANG["自刑"]))
```

只捕获：**辰辰**、**午午**、**酉酉**、**亥亥**

### 正确规则（SKILL.md L1477-1479）
> 辰辰/午午/酉酉/亥亥 = 自刑  
> **辰午酉亥四个字不论组合都按自刑算**

应额外捕获以下6组交叉组合：

| 组合 | 说明 |
|:---|:------|
| 辰+午 | 辰午同现 → 自刑 |
| 辰+酉 | 辰酉同现 → 自刑 |
| 辰+亥 | 辰亥同现 → 自刑 |
| 午+酉 | 午酉同现 → 自刑 |
| 午+亥 | 午亥同现 → 自刑 |
| 酉+亥 | 酉亥同现 → 自刑 |

### 修复方案
```python
chen_wu_you_hai = [z for z in ["辰", "午", "酉", "亥"] if z in zhi_list]
if len(chen_wu_you_hai) >= 2:
    results.append(("辰午酉亥自刑", NENG_LIANG["自刑"]))
# 同时保留原来的同字自刑检查（辰辰/午午/酉酉/亥亥）
```

### 影响范围
- 任何八字中同时包含辰午酉亥中两个不同字的场景
- 大运/流年与原局凑齐交叉组合时的岁运交互
