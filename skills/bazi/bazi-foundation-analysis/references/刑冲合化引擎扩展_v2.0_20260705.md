# 刑冲合化引擎扩展 v2.0（2026-07-05）

## 新增模块

### 1. 天干五合（TIAN_GAN_HE）

```python
TIAN_GAN_HE = {
    ("甲", "己"): "土", ("乙", "庚"): "金",
    ("丙", "辛"): "水", ("丁", "壬"): "木",
    ("戊", "癸"): "火",
}
```

- `check_tian_gan_he(gan1, gan2) -> (bool, wx)`
- `check_all_tian_gan_he(gans) -> list[dict]` — 扫描4天干所有配对
- 能量系数: 10.0

### 2. 三会局（SAN_HUI）

```
寅卯辰→木  巳午未→火  申酉戌→金  亥子丑→水
```

- `check_san_hui(zhis) -> list[dict]`
- 能量系数: **20.0**（最高优先级）

### 3. 拱合（GONG_HE）

12组隔支拱合，每对拱出一个中间地支：

| 地支对 | 拱出 | 合化 |
|--------|------|------|
| 亥+未  | 卯   | 木   |
| 寅+戌  | 午   | 火   |
| 巳+丑  | 酉   | 金   |
| 申+辰  | 子   | 水   |
| 亥+卯  | 未   | 木   |
| 寅+午  | 戌   | 火   |
| 巳+酉  | 丑   | 金   |
| 申+子  | 辰   | 水   |
| 卯+未  | 亥   | 木   |
| 午+戌  | 寅   | 火   |
| 酉+丑  | 巳   | 金   |
| 子+辰  | 申   | 水   |

- `check_gong_he(zhis) -> list[dict]`（含 `missing` 字段标识拱出的地支）
- 能量系数: 10.0

### 4. 暗合（AN_HE）

3组：
```
子巳→水  寅午→火  亥午→木
```

- `check_an_he(zhis) -> list[dict]`
- 能量系数: 0.5（不改变五行能量，仅代表人缘/暗中关系）

## 合化优先级规则

当多种合化共存时，**只保留最高优先级**：

| 优先级 | 类型       | 能量倍数 |
|--------|------------|----------|
| 4 (最高) | 三会局    | 20.0     |
| 3       | 三合局    | 15.0     |
| 2       | 六合/半合/拱合 | 10.0     |
| 1 (最低) | 暗合      | 0.5      |

## 新API: check_all_relations_v2()

```python
def check_all_relations_v2(zhi_list, gan_list=None) -> dict:
```

返回 v1 全部字段 +
- `result["天干五合"]` — list[dict]
- `result["三会"]` — list[dict]
- `result["拱合"]` — list[dict]
- `result["暗合"]` — list[dict]
- `result["highest_priority"]` — 优先级最高的合化关系（含 level/label/items）
- `result["summary"]` — 增强版摘要

调用方式：`check_all_relations_v2(地支列表, 天干列表)` — 天干列表可选。
