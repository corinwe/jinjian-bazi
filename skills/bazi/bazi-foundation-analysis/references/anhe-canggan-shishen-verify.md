# 暗合藏干十神验证铁律

> **编制人**：金鉴真人
> **编制时间**：2026-07-14
> **来源**：家主魏启令盲派报告「伤官合印→伤官合七杀」修正

## 致命错误

魏启令八字（庚申 癸未 辛亥 辛卯）中，日支亥(壬水)+月支未(丁火)→丁壬合暗合，原报告错误标为 **「伤官合印」**。

**根因**：未逐字验证未中藏干的十神关系——未中丁火对辛金是**七杀**（辛被丁克），不是印星（戊己土才是印）。

**后果**：暗象含义从「印制伤官」误判为「七杀伤官互制」，整个暗象解读偏了。

## 强制流程图

```yaml
第1步——列出参与的地支及其藏干
  □ 地支A：名称 + 本气/中气/余气
  □ 地支B：名称 + 本气/中气/余气
  □ 哪些藏干参与了互动？(合/冲/刑/害/穿)

第2步——逐字确认十神关系（最容易被跳过的步骤）
  对每个参与互动的地支藏干，逐字问：
  □ 这个字对日主（X日主）的十神是什么？
  □ 同一五行阴阳不同 → 十神不同
    壬=伤官 vs 癸=食神 / 丁=七杀 vs 丙=正官
  □ **禁止凭五行笼统判断**——必须精确到阴阳属性

  示例验证（辛金日主）：
  ┌──────────┬──────────┬──────────┐
  │ 地支藏干 │ 对辛金是 │ 关系判定  │
  ├──────────┼──────────┼──────────┤
  │ 亥中壬水 │ 伤官     │ 辛生壬=泄  │
  │ 亥中甲木 │ 正财     │ 辛克甲=耗  │
  │ 未中己土 │ 偏印     │ 己生辛=生  │
  │ 未中丁火 │ 七杀     │ 丁克辛=克  │  ← 不是印！
  │ 未中乙木 │ 偏财     │ 辛克乙=耗  │
  └──────────┴──────────┴──────────┘
  
  所以亥中壬(伤官)+未中丁(七杀) → 伤官合七杀 ✅
  不是伤官合印 ❌

第3步——命名总则测试
  □ 名称中的每个十神，是否都经过了第2步验证？
  □ 「伤官合印」→ 伤官(壬)我查了 ✅ / 印(戊己)我查了？ → 未中无戊己 ❌
  □ 「伤官合七杀」→ 伤官(壬)我查了 ✅ / 七杀(丁)我查了 ✅ → 通过
```

## 口诀

```
写暗象先查藏干，逐字推导十神名
丁火七杀非印星，印是戊己查清楚

名字各字都验证，一字不对全盘偏
```
