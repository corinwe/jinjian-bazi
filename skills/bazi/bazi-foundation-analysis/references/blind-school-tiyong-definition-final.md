# 盲派体用定义（最终校准版·2026-07-16）

> **三次修正历史**：
> - 初版（2026-07-14）：体=日主+印+比劫，用=财+官+食伤 ❌
> - 一次修正（2026-07-16）：体=日主+印+比劫+食神，用=财+官+伤官 ❌
> - **最终版（2026-07-16 老板二次校准）**：体=日主+印+比劫+食伤(食神+伤官全算体)，用=财+官(正财偏财正官七杀) ✅

## 正确定义

```
体 = 我自己 + 我所使用的工具
  ├─ 日主（自身 — 「日主就是比肩」— 老板原话）
  ├─ 印星（支持系统/贵人/长辈）
  ├─ 比肩/劫财（帮手/同伴 — 比肩=日主本人，劫财=伙伴）
  ├─ 禄神（身体/根基 — 临官位）
  └─ 食伤（食神+伤官 — 才华/技能/思维/创造力）

用 = 人生追求的目标（外物）
  └─ 财星 + 官杀（正财偏财正官七杀，只有这两个）

做功 = 体（印比食伤）→ 追求 → 用（财官）的方式
  例：食伤生财 = 用才华去赚钱
  例：印化官杀 = 用智慧去得权
  例：比劫扛财 = 用团队去求财
```

## 关键理解

```
为什么食伤在体？
  食神和伤官都是「我的思维/才华/创造力」— 是我的一部分。
  就像木匠手里的刨子（工具）属于木匠本人一样，
  一个人的才华和创造力属于「我」，不属于「我追求的目标」。

为什么用只有财官？
  财是财富（外物），官是地位（外物）— 都是我追求的东西。
  不是「我」本身，而是「我想要的东西」。
```

## 口诀

```

比肩就是我自己，日主也是我自己
禄是身体也是本，食伤才是思维力

用是追求的目标，财官才是真用途
正财偏财和官杀，四个才是真用途

做功是用体去追用，食伤生财印化官
比肩扛财也做功，做功方式看体用
```
