# 三大体系并列架构（2026-07-14 老板校准·泉师兄融合触发）

> 八字命格理论体系下三个独立并列的体系。
> 泉师兄抖音体系 = 盲派 + 象法。与九龙道长体系并列。

## 总览

```
八字命格理论体系
├── 🏆 九龙道长体系（传统+独家倍数法）
│   └─ 基础通用知识库（bazi-foundation-analysis）
│   └─ 子技能：bazi-wealth-analysis / bazi-marriage-analysis / ...
│
├── ⛩️ 盲派体系（做功/体用/归属权/家里家外/寻庚基）
│   └─ 基础规则：bazi-foundation-analysis §3B-§3H
│   └─ 财富规则：bazi-wealth-analysis §19-§29（泉师兄萃取）
│   └─ 婚姻规则：bazi-marriage-analysis §14（鸳鸯门/定份额法）
│
└── 🎨 象法体系（象形读图/取象法）
    └─ 独立技能：bazi-image-method
```

## 使用流程

| 步骤 | 体系 | 做什么 |
|:----|:-----|:-------|
| 1 | 九龙道长 | 定身强弱、用神忌神、主格辅格 |
| 2 | 盲派 | 定做功路线、家里家外、归属权 |
| 3 | 象法 | 把上面全部画成画，看是否和谐 |

## 交叉引用规则

- **盲派→象法**：盲派定「用什么方式得财」，象法定「具体做什么行业/遇到什么事」
- **九龙道长→盲派**：身强弱结果作为盲派体用判断的参考
- **对象法**：先看盲派做功路线，再把路线画成一幅画
- **冲突处理**：三个体系结果不一致时，优先以九龙道长体系为基准，其他两个做补充验证
