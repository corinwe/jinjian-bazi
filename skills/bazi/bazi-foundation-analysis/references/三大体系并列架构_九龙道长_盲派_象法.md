## 三法配合流程

拿到八字后的三法配合路线：

```
第一步（理法）：看做功
  → 废旺衰？不管。直接看谁在动、动什么、动了之后得到什么
  → 定体用、定宾主、定功神废神、定正局反局

第二步（象法）：看画面
  → 把第一步的结果画成一幅画
  → 干支象什么？画面和谐还是冲突？
  → 七维扫描补细节（五行/天干/地支/十神/宫位/神煞/状态）

第三步（技法）：断具体
  → 寻根基定真假
  → 暗象看隐藏信息
  → 三垣看体质/财运/天赋
  → 串宫压运断定流年
```
