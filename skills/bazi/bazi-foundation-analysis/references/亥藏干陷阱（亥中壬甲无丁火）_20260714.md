# 亥藏干陷阱 · 实战致命教训

> **来源**：2026-07-14 家主（庚申 癸未 辛亥 辛卯）实战——我在报告中写了"亥中丁火七杀"，但**亥中根本没有丁火**。
> **亥的正确藏干**：壬（本气100%）· 甲（余气30%）→ **无丁火！**

## 为什么容易搞混

```
❌ 错误记忆："亥中有丁火"（因为亥是木的长生地，和寅申巳合称四生→误以为有火）
✅ 正确：亥为纯水之地，藏壬水（本气）+ 甲木（余气），无任何火元素

易混对比例：
  亥（壬甲）←→ 巳（丙戊庚）—— 只差一位，藏干天差地别
  亥（壬甲）←→ 未（己丁乙）—— 未中有丁火，但亥中无
```

## 十神速查（对辛金日主）

```
亥中壬 → 辛金生壬水（阳水）= 伤官 ✅
亥中甲 → 辛金克甲木（阳木）= 正财 ✅
（亥中无丁 → 无七杀！）
```

## 藏干十神陷阱高频字

以下地支的藏干常被搞混，每个都是实战中踩过的坑：

| 地支 | 正确藏干 | 常被误认为有的字 | 后果 |
|:----|:---------|:----------------|:-----|
| **亥** | 壬·甲 | **丁**（以为有火）| 凭空创造七杀→官杀当财分析全错 |
| **巳** | 丙·戊·庚 | 顺序错（丙庚戊→应为丙戊庚）| 十神名全错 |
| **午** | 丁·己 | **丙**（以为午有丙火）| 比劫错为劫财 |
| **申** | 庚·壬·戊 | 顺序错（庚戊壬→应为庚壬戊）| 伤官错为偏印 |
| **戌** | 戊·辛·丁 | **丙**（以为戌有丙）| 印星错乱 |
