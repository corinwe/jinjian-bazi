# 五行生水链条分析法（2026-07-22）
用于分析八字中"水"的源头和持续力，判断格局层次。

## 核心原理
水不是独立存在的，必须追踪完整的生成链条：
```
土 → 金 → 水
```
缺任何一环，水就无源。

## 分析步骤

### 第1步：定位水的位置
- 壬水/癸水在哪？透干还是藏支？
- 有几个水？单水还是双水？

### 第2步：定位金的位置（生水的源）
- 庚金/辛金在哪？透干还是藏支？
- 藏支的百分比是多少（100%/60%/30%）？
- 注意区分：庚金（阳金·生水之力强）vs 辛金（阴金·生水之力弱）
- 数量可以弥补质量：双辛金 ≈ 单庚金

### 第3步：定位土的位置（生金的源）
- 戊土/己土在哪？
- 戊土（高山·生庚金）vs 己土（田园·生辛金）
- 记住：原文口诀"土生金→金生水"的完整链条

### 第4步：判断链条完整性
```
✅ 完整链条：戊土→庚金→壬水（最高级）
✅ 完整链条：戊土→辛金+庚金→癸水+壬水（中高级）
✅ 完整链条：己土→辛金→癸水（中级）
❌ 缺链条：只有水没有金→水源无根，易涸
❌ 缺链条：只有金没有土→金无源，生水有限
```

## 实战案例：子源（辛卯 癸巳 丙戌 癸巳）

```
日支戌(戊土100%) → 月支巳(庚金30%) + 时支巳(庚金30%) + 年干辛金(透)+日支戌(辛金60%)
                     → 四金生双癸水(月干+时干)
                     → 水源充足，链条完整 ✅
```

## 原文对照
「必须要有庚金，壬水才有源头」
「庚金得禄，能泄去土气而生水」
「亥宫虽然有甲木破土，但甲木力量弱，没办法克制土」

## 结论公式
完整链条 = 土有源 + 金有力 + 水有流
水源力 = 土的力量 × 金的力量 × 水的力量
（每环节弱一档，整体降低半格）
