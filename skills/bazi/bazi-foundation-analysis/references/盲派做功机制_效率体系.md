# 盲派做功机制与效率体系

## 一、做功总论

做功即一个人取得财官名利的手段，代表八字能得到的富贵等级。

- **富贵八字**：有气或势，主位（日柱）与气势一致然后做功
- **普通八字**：未成气成势，或做功效率太低，或功神做的功小
- **差八字**：无功神，不做功的神叫废神；越差废神越多，越好废神越少
- **功神定义**：参与做功的神，消耗能量产生效率
- **废神定义**：消耗能量但不产生效率或效率特别低

### 正向做功 vs 反向做功
- 正向做功：主位的体去追求宾位的用
- 反向做功：主位的用与宾位的体发生作用
- **反向做功效率一般大于正向做功**

### 做功方式总纲
体用、宾主之间进行刑、冲、克、穿、合、墓等都是做功的方式。

## 二、五种做功结构

### ① 制用结构（最关键的做功方式）

#### 基础类型
| 类型 | 核心 | 说明 |
|:----|:-----|:-----|
| 比劫制财 | 比劫旺→制财 | 需看食伤是否参与生扶；我的食伤去生的财才与我有关才能克 |
| 财制印 | 财旺→制印 | 最好制年月的印；制家里的印→身体不好 |
| 印制食伤 | 印旺→制食伤 | 制用结构之一 |
| 食伤制官杀 | 食伤旺→制官杀 | 虚名多；食伤制官杀靠技能权谋 |
| 官杀制比劫 | 官旺→制比劫 | 一定是大人物（管人的），亿万富翁或国家干部 |

#### 双制结构
- 比劫食伤制住财官（制得尽+制住官的原神）→ 格局很大
- 印比制财食伤（制住财的原神）→ 格局很大
- 制住某十神以及它的库 → 大格局
- 双制结构最好是制年月：年支能量最大（国家），月令次之（但制月令易反局→月令是旺点），制日时能量最小

#### 制用能量等级
```yaml
官杀最大 > 印第二 > 比劫第三 > 食伤第四 > 财最小（比劫克财=体力劳动）
```

#### 特殊规则
- 官制比劫、财制印最好制年月的比劫或印；制家里的→身体不好；制日支→不利婚姻；制日时→老苦命
- 家里或体受制→即便大格局有成就，身体也不好，一定有疾病
- 比劫制财：如果没有食伤参与生扶，财与自己无关，克了就是拿了不该拿的钱

### ② 化用结构（杀印相生/官印相生）

| 类型 | 财富级别 | 条件 |
|:----|:--------|:-----|
| 杀印相生 | 亿级别 | 喜官杀在月令 |
| 官印相生 | 千万级别 | 不喜印在月令；印太旺而官杀弱层次就低 |

### ③ 生用结构（食伤生财/食伤泄秀）

- 食伤生财无需看旺衰，只看财是否为我所得、是否通门户
- 通门户不需要身旺
- 忌：年月比劫夺财、印坏食伤、财不在家里、财无法制服
- 食伤生财再生家外的官杀→持续破财（除非官在家里又被我合住）
- 食伤生财又制印→财富级别更高

| 类型 | 方式 |
|:----|:-----|
| 食神生财 | 人脉、资源、艺术 |
| 伤官生财 | 创新、技术、销售 |

### ④ 合用结构

#### 天干合
| 类型 | 方式 |
|:----|:-----|
| 日元合财 | 五阳干合财，需考虑身旺身弱 |
| 日元合官 | 五阴干合官可从弱，官不能弱不能被合化 |
| 劫财合财 | 非劳动收入（股票期货等） |
| 羊刃合杀 | 顽强不屈，步步为营 |
| 伤官合杀 | 权谋 |
| 食神合官 | 口才、人际关系、高情商 |
| 伤官合枭 | 周易、哲学、科研 |
| 正财合枭 | 印合财 |
| 偏财合印 | 印合财 |
| 印合食神 | 控制性 |

#### 合官规则
- 合官要求官要真，官不能弱，也不能被合化掉
- 日主身弱点无所谓，五阴干合官可以从弱
- 合官不喜见财

#### 合财规则
- 合财不喜见官
- 合财的五阳干需考虑身旺身弱
- 财在年月家外、身弱担不住→贫穷
- 财生官来克自己→有灾

#### 印合财规则
- 印合财最忌财制了生身之印
- 月令为印、透月干或时干的是生身之印，不喜财来制
- 日时为家里的印不喜被财制
- 印太多为忌才可用财来制

### ⑤ 墓用结构

| 入库 | 说明 |
|:----|:-----|
| 亥申→丑库 | 亥中壬/申中庚入丑库 |
| 寅巳→未库 | 寅中甲/巳中丙入未库 |
| 丑未戌→辰库 | 丑未戌再入辰库 |

墓用含义：得到、控制、占据、拥有。

## 三、做功效率——合制效率

> 注意：申巳、巳酉等合是干制支，能量最大，算是最高效的制——制住原身本身。

| 等级 | 组合 | 效率 | 说明 |
|:----:|:----|:----:|:-----|
| 1 | 天干五合 | 高 | 前提：合财身要旺，合官官要强 |
| 2 | 申巳合、巳酉合 | 最高 | 一方旺时制住对方的原用神 |
| 3 | 卯戌、亥午、卯申 | 较高 | 卯木制戌土（金存在能量减）；庚克乙；壬克丁 |
| 4 | 子丑、寅丑 | 中等 | 丑是水库制不住子；制丑最好用未戌 |
| 5 | 寅亥、辰酉、午戌 | 低 | 可灭寅中弱丙火/辰中弱乙木/戌中弱辛金；制住一小部分 |
| 6 | 卯未、亥未 | 低 | 卯从未中来；亥未拱卯 |
| 7 | 申子、午未、亥卯、丑酉、寅午 | 不做功 | 可论相生，不做功 |

### 合制能量排序（从高到低）

巳申 > 巳酉 > 子丑 > 卯戌 > 卯申 > 午亥 > 寅亥 > 寅丑 > 午戌

## 四、做功效率——刑冲穿制效率

| 等级 | 组合 | 效率 | 说明 |
|:----:|:----|:----:|:-----|
| 1 | 丑未冲、辰戌冲 | 90% | 墓库能量大，冲库能量高 |
| 2 | 丑戌刑 | 80% | 墓库刑开 |
| 3 | 寅申冲、巳亥冲 | 60% | 效率较高 |
| 4 | 子午冲、卯酉冲 | 50% | 要求一方力量大于另一方，三打一最好 |
| 5 | 丑午穿、子未穿 | 50% | 穿也是重要做功方式 |
| 6 | 酉戌穿、卯辰穿 | 30% | 效率低 |
| 7 | 寅巳、申亥 | 无效 | 生多穿少，不算做功 |

### 冲制财富规则

- 财官临库不冲不发——只有逢刑冲才可能有用
- 丑未冲、辰戌冲：能量高，制得干净，亿以上、省部级
- 未戌刑：制去辛金，几百万级别
- 丑戌刑：戌把丑制干净，几千万、上亿
- 冲制对等（一对一）→ 冲动/对抗，意义不大；成势的冲（二冲一、三冲一）→ 冲伤冲去有意义

### 穿的特殊规则

- 穿是盲派断命最常用的方式，是一种仇恨、难以调节的矛盾
- 穿制不了对方，而是坏；冲合克都可以制，唯独穿制不了
- 穿倒正印当偏印看，穿倒七杀当正官看（反过来看）
- 穿库开不了库，只有冲库、刑库才能开
- 流年穿大运无所谓（大运外来），最怕命局中的被穿（自己的东西）

## 五、多种结构并存

1. 克制、化制（含生制）、合制、墓制四种结构并存时
   - 可串联、不可相悖 → 串联升层次，相悖降层次
2. 复合结构在不同阶段代表换行业/职业，或多种职业同时存在

## 六、正用 vs 反用

| 类型 | 描述 | 含义 |
|:----|:-----|:-----|
| 正用 | 主位的体制宾位的用 | 规矩格局，显示日语规矩正常大众 |
| 反用 | 用占主位、体占宾位，阴阳错位 | 反用：主位的用反制宾位的体；不合常理命局，组合好则有异于常人的才华智慧 |

## 七、正局 vs 反局

| 类型 | 定义 |
|:----|:-----|
| 正局 | 日主（主位）表述的意思与八字意思一致 |
| 反局 | 日主（主位）表述的意思与八字意思相反 |
| 大运反局 | 大运表述的意思和自己的意思反了 → 凶运 |

### 日主表述方式
- 日干合什么 → 表述本意想得到什么；冲则为日主最排斥的东西
- 合为日主需要的，冲为日主不要的
- 日支也为日干的代表
- 气势为日主的表述
- 主位印比最怕被财官破，破必有灾

## 八、制 vs 化 vs 泄 能量层级

```yaml
制格：最高——从能量上干掉对方
  └─ 合制 > 克制 > 冲制
  └─ 制住原神和本神 → 制干净 → 层次最高（巳申、巳酉）
  └─ 只制住本神 → 层次中等（子丑、卯戌）
  └─ 暗合/有通关 → 层次低（寅亥、甲己、午戌）

化格：次高——印化七杀
  └─ 杀印相生 > 官印相生

泄生格：最低——用自己的力量去生
  └─ 食伤生财一般十万百万级别
  └─ 伤官生财靠技能拼搏，层次不如制格
```

### 凶神 vs 吉神
- 凶神（杀伤劫刃枭）为用档次高 → 大人物多为凶神有制化
  - 灵活善变、适应能力强、不择手段达到目的
- 吉神（财官印食）为用成就低 → 太正派受约束
