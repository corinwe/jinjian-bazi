# 象法（象形法）— 盲派核心底层思维

> **来源**：泉师兄19篇文案精读提取（2026-07-14）
> **关联技能**：bazi-wealth-analysis §24.0（完整十干十二支象形表+应用实例）

## 定义

象法即象形法——把天干地支当成有形状有体量的东西，看它们放在一起像一幅什么画。不是技术规则，是**视觉化思维**。

## 核心原则

拿到八字先看它像一幅什么画，各元素之间是和谐共生还是冲突对抗，然后才用技术规则去验证。

## 体量感分级

象法最核心的概念是**体量感**——同样的五行在不同干支上体量完全不同：

| 五行 | 大体量（甲/丙/戊/庚/壬） | 小体量（乙/丁/己/辛/癸） |
|:----|:------------------------|:------------------------|
| 木 | 甲木=参天大树，扎根深 | 乙木=花花草草，根浅柔弱 |
| 火 | 丙火=太阳，强烈普照 | 丁火=烛火，微弱温暖 |
| 土 | 戊土=高山，庞大厚重 | 己土=田园土，松软温和 |
| 金 | 庚金=刀剑大金属，刚硬 | 辛金=珠宝针，精致尖锐 |
| 水 | 壬水=大江大河，奔流 | 癸水=雨露细流，渗透 |

## 场景感

地支提供场景/画布：
- 辰=水库（能养大树也能淹死花草）
- 戌=火库/熔炉
- 寅=山林生机勃勃
- 卯=花草需呵护

## 实战应用

- 辰养甲木=水库养大树→正好（大树扎根深）
- 辰养卯木=水库养花草→淹死（根浅烂根）
- 壬水冲己土=大河冲垮田埂→拦不住
- 壬水遇戊土=高山挡住大河→能约束
- 做功路线定方式，象法定行业（如壬水=互联网/癸水=自媒体）
