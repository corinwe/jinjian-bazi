## 一、核心公式（2026-07-01校准）

```
食伤总分 = 比劫(同我五行)的分数 + 食伤(我生五行)本身的分数
            ↑ 比劫生食伤       ↑ 食伤本身
```

### 食伤五行推导

```
日主金(庚辛) → 食伤=水(壬癸亥子)
日主木(甲乙) → 食伤=火(丙丁巳午)
日主水(壬癸) → 食伤=木(甲乙寅卯)
日主火(丙丁) → 食伤=土(戊己辰戌丑未)
日主土(戊己) → 食伤=金(庚辛申酉)
```

### 比劫五行 = 日主五行（同我者）

```
日主金 → 比劫=金(庚辛申酉) → 金生水 → 生食伤
日主木 → 比劫=木(甲乙寅卯) → 木生火 → 生食伤
日主水 → 比劫=水(壬癸亥子) → 水生木 → 生食伤
日主火 → 比劫=火(丙丁巳午) → 火生土 → 生食伤
日主土 → 比劫=土(戊己辰戌丑未) → 土生金 → 生食伤
```

### 食伤得令 = 月令有比劫（生食伤）或月令有食伤本身

## 三、判定标准

| 分数区间 | 状态 | 含义 |
|:--------:|:----|:-----|
| **≥80分** | **极旺** | 才华横溢，财根极深，赚钱管道非常宽 |
| **55~80分** | **旺** | 才华出众，财根深厚 |
| **30~55分** | **中等** | 有才华有能力，财根中规中矩 |
| **15~30分** | **偏弱** | 财根较浅，需大运补食伤 |
| **<15分** | **极弱** | 财无根，财来财去过手财 |

## 六、关键规律

1. **比劫生食伤是九龙道长能量链条的核心环节**：印→比劫→食伤→财→官
2. **食伤得令**=月令有比劫（生食伤）或月令有食伤本身
3. **食伤=财根**：食伤强则财源稳定，食伤弱则过手财
4. **身强+食伤旺=最佳组合**：泄秀吐秀，才华变财富
5. **身弱+食伤旺=泄身太过**：想法多但执行力差
