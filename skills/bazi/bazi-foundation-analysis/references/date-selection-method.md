# 八字择日方法论 · Python自动化

> **来源**: 2026-07-10 少爷（子源·丙火日主）东北房动工择日实战
> **适用场景**: 装修动工/搬家/开业/签约等需选吉日的场景
> **核心原理**: 日主干支 × 五行喜忌评分 + 建除十二神 + 方位匹配

## 核心算法步骤

### 1. 计算每日干支
```python
from paipan import get_day_gan_zhi
gan, zhi = get_day_gan_zhi(target_date)  # target_date 是 date 对象
```

### 2. 五行喜忌评分（对具体日主）
```
天干得分 + 地支得分 + 藏干加权(×0.3)
喜用→正分 | 忌神→负分
土(食伤) +3 | 金(财) +2 | 水(官) +1 | 木(印) -2 | 火(比劫) -2
```

### 3. 建除十二神检查
月建所在支起"建"，顺数12支定当日建除
- 动工修造最宜：除、定、执、成、开
- 忌：破（诸事不宜）

### 4. 方位匹配
| 方位 | 八卦 | 五行 | 对丙火日主 |
|:-----|:-----|:-----|:----------|
| 东北 | 艮 | 土 | 喜用（食伤泄秀）|
| 西北 | 乾 | 金 | 喜用（财星耗身）|
| 北 | 坎 | 水 | 喜用（官星制衡）|

### 5. 时辰选择
辰时(7-9土) / 巳时(9-11火) / 申时(15-17金) / 酉时(17-19金)

## 口诀
```
择日先排日干支，五行喜忌打分看
建除十二查吉凶，方位匹配再加分
避开冲刑忌神日，最佳时辰定开工
```
