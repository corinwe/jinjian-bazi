# 体用铁律（2026-07-16 老板最终校准）

## 三条铁律（写在§3A.9开篇，子代理不得违反）

### 铁律① — 藏干财官算用，不可排除
盲派定义「用=财+官」不分天干藏干。天干财官、地支藏干财官——只要是财官，全部算用。
禁止任何形式的「天干才算用，藏干不算」排除规则。
常见违规案例：子源报告「藏干财星不算入用」（❌ 已修复）

### 铁律② — 子代理不准发明规则
体用定义已锁定（体=日主+印+比劫+食伤，用=财+官）。
子代理不可创建任何「修正」或「改良」版本。
发现任何与技能库体用定义不一致的规则 → 视为违反铁律。
常见违规案例：立报告「巳中庚不算」（❌ 已修复）

### 铁律③ — 引用口诀前必须校验条件
引用「四柱皆带财」「身弱不担财」等口诀前，先逐条核实本八字是否满足条件。
条件不满足 → 不准引用，须注明差异。
常见违规案例：家主报告「四柱皆带财」（❌ 年柱无财，已修复）

## 大运顺逆规则（易错点）

**年干阴阳+性别**决定顺逆（不是日干）：
- 年干阳+男 → 顺排
- 年干阴+女 → 顺排
- 年干阴+男 → 逆排
- 年干阳+女 → 逆排

常见错误：用日干定顺逆（❌）
