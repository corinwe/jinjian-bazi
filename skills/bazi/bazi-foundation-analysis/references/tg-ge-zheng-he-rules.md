# 天干隔合/争合/遥合规则（2026-07-14新增）

> **来源**：2026-07-14主母成(丁卯丁未庚午壬午)丁壬隔合实战教训
> **规则基础**：bazi-foundation-analysis §3.1 天干五合（第4288行：隔合→合而不化）

## 一、三种合化方式

| 类型 | 距离 | 力量 | 表述 |
|:----|:-----|:----|:-----|
| **邻合** | 紧邻（年+月/月+日/日+时） | 强 | 直接合，可成化 |
| **隔合** | 隔一柱（月+时隔日干/年+日隔月干） | 中弱 | 合而不化，需引动 |
| **遥合** | 隔两柱（年+时隔月干+日干） | 极弱 | 基本不能直接合 |

## 二、争合规则

| 模式 | 条件 | 影响 |
|:----|:-----|:-----|
| **一合多** | 一天干同时合多个天干 | 力量分散，各不成化 |
| **多合一** | 多个天干同时合一天干 | 争合，相互削弱 |
| **妒合** | 两支将合被旁冲 | 猜疑妒忌 |

## 三、报告写法示例

### ❌ 错误写法
```
壬水被丁火合（丁壬合）→ 食神被官合→才华被约束
```
缺了隔合+争合信息，过度肯定合化力量。

### ✅ 正确写法
```
壬水被双丁火隔合（月干丁隔日干庚+年干丁隔两柱→争合+隔合）
→ 合而不化→ 食神与官有合意但力量弱，需大运流年引动
```
