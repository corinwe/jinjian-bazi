## 0. 大运 vs 流年：根本区别

> 核心观点：**大运不会应事，流年才会应事。** 这是流年断事的第一前提。

### 0.1 大运的三大作用

| 序号 | 作用 | 说明 |
|:---:|:---|:---|
| ① | **改变本命局能量** | 身弱走印比大运→变强；身强走财官大运→更平衡 |
| ② | **改变格局** | 月令本气不透→杂气格，大运透出后形成正格（如偏财格满脑子搞钱） |
| ③ | **参与八字与流年的刑冲合化** | 大运+本命局+流年三者互动，大运是中间桥梁 |

### 0.2 大运不会应事

```
大运 = 改变能量的「背景环境」
流年 = 触发事件的「导火索」

比喻：
  大运是土地（肥沃/贫瘠）——决定你能不能长
  流年是天气（风调雨顺/台风冰雹）——决定你这一年收成如何
```

### 0.3 大运的时间分配

```
前5年：天干70% + 地支30%
后5年：地支70% + 天干30%

前半年：天干70% + 地支30%
后半年：地支70% + 天干30%
```

## 0A. 犯太岁系统论

### 0A.1 犯太岁的本质

> 流年天干地支与八字本命局产生**刑冲破害合化**关系 → 叫犯太岁。

其中：
- 冲和害 → **一定是坏事**（冲走喜用更严重）
- 合化 → 看合出的是**喜用还是忌神**（合出喜用=好事，合出忌神=坏事）

**🚨 核心原则：冲不一定坏事，合不一定好事** — 取决于激活的是喜用还是忌神。

### 0A.2 流年太岁是「大哥大」

流年太岁憋了60年才值班一次（60甲子一轮），能量最大，**可以冲破原局合化**：

```
原局有巳酉丑三合金局？流年来了寅→寅巳相害，把三合冲开
原局有亥卯未三合木局？流年来了申→申亥相破/寅申冲，照样破局

🚨 流年＞大运＞原局。流年说了算。
```

### 0A.3 能量伤害量化表

| 互动类型 | 伤害分值 | 能量倍数 | 说明 | 断事建议 |
|:---|:---:|:---:|:---|:---|
| **冲** | -70分 | 辰戌/丑未冲=10倍土 | 直接冲击，最严重 | 🚨 必须断事 |
| 刑（丑未戌） | 能量加强 | 15倍土 | 藏干打架-余气打没-全是土-15倍比劫 | 必须断事 |
| 刑（寅巳申） | 能量加强 | 12倍 | 藏干打架-本气能量翻倍,喜=吉忌=凶 | 必须断事 |
| **害/穿** | -60分 | - | 慢性伤害、暗算、突发之灾 | ⚠️ 必须断事 |
| 刑（子卯） | 能量加强 | 5倍 | 子水生卯木-木能量加强-一般代表吵架 | 可断可不断 |
| **破** | -20~30分 | - | 撕破脸，吵吵架，破小财 | **一般不断事** |

> **实战经验**：破的事我从来不断——顶多吵吵架破点小财，应不出多大事。能量最小时，基本忽略。

### 0A.4 合化能量倍数表

| 组合类型 | 有天干引化 | 无天干引化 | 说明 |
|:---|:---:|:---:|:---|
| **三会**（亥子丑/寅卯辰等） | **20倍** | 10倍 | 能量最大，四季之会 |
| **三合**（申子辰/寅午戌等） | **15倍** | 7倍 | 次之，三合局完整 |
| **半三合/六合** | **10倍** | 5倍 | 合化基础单位 |
| 拱合 | 10倍 | 5倍 | 力量同六合 |

> 例：寅午戌三合火 + 天干丙火引化 = **15倍火能量** → 某命主丙午月进账1.2亿。

### 0A.5 贪合忘冲原则

```
如果流年某字既有「冲」又有「合」的关系 → 优先合化，不冲了。

案例：流年壬水冲日主丙火，
  但丁火与壬水合化（丁壬合木）→ 壬被丁"勾引"走了 → 不冲丙了
  → 这叫"贪合忘冲"
  （好比两只小公狗要打架，来了只小母狗，就不打了）

🚨 优先级(按量化分值)：合化＞冲(-70)≥刑丑未戌(-70)＞害(-60)=刑寅巳申(-60)＞刑子卯(-30)＞破(-20)
```

## 1. 流年断事5步流程

```
第一步：看天干五合
  ├─ 流年天干与命局天干合化 → 看合化出的是喜用还是忌凶
  └─ 合化成功打破原命局组合 → 命主有重大变化

第二步：看地支合化（三会>三合>半三合>六合>拱合）
  ├─ 合出喜用 = 好事，合出忌凶 = 坏事
  └─ 合化可解开刑冲破害

第三步：看刑冲合害
  ├─ 流年冲到某宫位 → 该宫位六亲应事
  ├─ 冲走忌凶 = 好事，冲走喜用 = 灾祸
  └─ 三刑全的人在流年重复见其中一个字 → 多半应灾

第四步：看重大灾祸指标
  ├─ 天克地冲岁运并临（双柱伏吟）→ 不死自己死家人
  ├─ 地支三刑 + 天干七杀或岁运并临 → 大灾
  ├─ 流年七杀过多无制 → 灾祸
  ├─ 血刃/灾煞能量突然增加 → 血光之灾
  └─ 大运流年多个枭神夺食 → 抑郁、自残

第五步：看月份断事
  ├─ 某月能量突然加强 → 极端事情发生
  ├─ 干支相克 → 能量减少，事情不严重
  ├─ 干支相生 → 被生的能量增大
  └─ 干支一气 → 能量全部加强
```

## 2. 重大灾祸查看步骤

```
以坤造庚申、戊寅、丁卯、庚戌为例：

1. 分别查找四柱的天克地冲之大运 + 伏吟之流年
2. 三刑：本命局带申、寅 → 查巳的大运/流年 → 应灾
3. 七杀：查找七杀大运 + 七杀多的流年
4. 查血刃：大运+流年同时为血刃 → 流血事件
5. 查灾煞：大运是否有对应的灾煞流年
6. 查枭印：大运枭神夺本命局食神
7. 查受伤的宫位
```

## 3. 伏吟反吟详解

```
伏吟 = 相同的地支/天干重复出现
反吟 = 天克地冲

伏吟断事原则：
  伏吟的宫位 → 反复、拖延、压抑、旧事重来

  年柱伏吟（年柱在大运/流年重复出现）：
    ├─ 祖业凋零、少年坎坷
    ├─ 父母宫反复有变动
    └─ 远离故乡发展

  月柱伏吟（月柱在大运/流年重复出现）：
    ├─ 事业反复、内心纠结
    ├─ 与父母/兄弟关系反复
    └─ 情绪低潮期

  日柱伏吟（日柱在大运/流年重复出现）：
    ├─ 🚨 最严重的一种伏吟（夫妻宫+日元）
    ├─ 夫妻关系反复、争吵不断
    ├─ 自己情绪反复、身体出问题
    └─ 女命日柱伏吟+七杀流年 → 有离婚/分居风险

  时柱伏吟（时柱在大运/流年重复出现）：
    ├─ 子女事反复、子女离家
    └─ 晚年计划搁置

  四柱全伏吟 → 🚨 大凶之兆（重病/大灾/死亡风险极高）

伏吟应事的程度判定：
  ├─ 本命局两个相同字 = 自伏吟 → 自刑特征
  ├─ 大运/流年与命局某柱相同 → 该年该宫位反复
  ├─ 命局+大运+流年三个相同 → 🚨 重大应灾
  └─ 天干地支同时伏吟 → 力量翻倍

反吟断事原则：
  反吟的天干地支被冲 → 突发性变动、剧烈变化、推翻重来

  ├─ 年柱反吟 → 祖业破败、早年离家
  ├─ 月柱反吟 → 父母离异、工作剧变
  ├─ 日柱反吟 → 🚨 婚灾、本人生命危险
  └─ 时柱反吟 → 子女出意外

伏吟和反吟同时出现 → 🚨 "岁运并临" — 最严重信号之一
```

## 4. 岁运并临深度解析

```
岁运并临 = 大运的干支与流年的干支完全相同

岁运并临规则：
  ├─ 力量判定：伏吟（大运与流年完全一致）× 能量翻倍
  ├─ 🚨 该年发生的事件能量 = 正常流年的3-5倍
  ├─ 吉则大吉（喜用神到位）
  └─ 凶则大凶（忌神到位）

岁运并临的应事类型（金鉴真人实战校准）：

  【岁运并临 + 用神到位】
    断事：该年会有重大突破/转折性事件
    ├─ 事业：升职、创业成功、大项目落地
    ├─ 财富：大财入账、资产翻倍
    ├─ 婚姻：结婚/生子/家庭大喜事
    └─ 健康：无明显问题（用神护身）

  【岁运并临 + 忌神到位】
    断事：该年会有重大灾祸/颠覆性打击
    ├─ 事业：失业、公司倒闭、事业归零
    ├─ 财富：大亏、投资归零、债务危机
    ├─ 婚姻：离婚、丧偶
    └─ 健康：🚨 重病、生命危险

  【岁运并临 + 三刑】
    🚨 最危险组合 — 生命危险信号
    ├─ 本命局带一组三刑 + 岁运并临 → 必应大灾
    ├─ 重伤/重病/官司/牢狱
    └─ 应在三刑最重的宫位

  【岁运并临 + 枭神夺食】
    🚨 次危险组合 — 心理/精神危险信号
    ├─ 严重抑郁、精神疾病
    ├─ 自杀倾向、突发性意外
    └─ 应在枭神食神对应的宫位

  【岁运并临 + 冲夫妻宫】
    ├─ 离婚/分居概率极高
    ├─ 配偶有灾（突发疾病/意外）
    └─ 夫妻宫冲开 → 关系走向终结或重新定义

岁运并临的判断口诀：
  岁运并临，不死自己死他人
  但若用神，吉庆加倍
  看五行，看喜忌，才知生死定夺
```

## 5. 流月断事细化方法

### 5.1 流月断事的基本原则

```
流年能量分12个月，看四季划分和五行旺衰：

春季（寅卯辰月 — 公历2-5月）：
  ├─ 木旺的三个月
  ├─ 寅月（立春后，公历2月4日-3月5日）：木气初升
  ├─ 卯月（惊蛰后，公历3月6日-4月4日）：木气最旺
  └─ 辰月（清明后，公历4月5日-5月5日）：木气渐衰，土开始进气

夏季（巳午未月 — 公历5-8月）：
  ├─ 火旺的三个月
  ├─ 巳月（立夏后，公历5月6日-6月5日）：火气初升
  ├─ 午月（芒种后，公历6月6日-7月6日）：火气最旺
  └─ 未月（小暑后，公历7月7日-8月6日）：火气渐衰，土开始进气

秋季（申酉戌月 — 公历8-11月）：
  ├─ 金旺的三个月
  ├─ 申月（立秋后，公历8月7日-9月7日）：金气初升
  ├─ 酉月（白露后，公历9月8日-10月7日）：金气最旺
  └─ 戌月（寒露后，公历10月8日-11月6日）：金气渐衰，土开始进气

冬季（亥子丑月 — 公历11-2月）：
  ├─ 水旺的三个月
  ├─ 亥月（立冬后，公历11月7日-12月6日）：水气初升
  ├─ 子月（大雪后，公历12月7日-1月5日）：水气最旺
  └─ 丑月（小寒后，公历1月6日-2月3日）：水气渐衰，土开始进气
```

### 5.2 流月断事5步流程

```
第一步：看流年与流月的组合
  流年定一年的基调 → 流月决定什么时候应事

第二步：判断流月的五行是喜用还是忌神
  ├─ 喜用月 → 该月做事顺利
  └─ 忌神月 → 该月有阻碍

第三步：看流月与原命局的互动
  ├─ 流月与原局的字发生刑冲合害 → 在该月应事
  ├─ 流月的能量加强或减弱流年的能量
  └─ 流月与原局形成三合/三会 → 重大事件发生月

第四步：看季节转折点（节气交接前3天后3天）
  ├─ 节气交接前后3天 → 容易发生事件
  ├─ 立春、立夏、立秋、立冬前3天后3天 → 重大转折点
  └─ 某个季节第一个月为进气月 → 事件开始酝酿

第五步：断月份应事类型
  ├─ 春季 → 应事在事业/学业/人际关系（木主）
  ├─ 夏季 → 应事在名声/投资/社交（火主）
  ├─ 秋季 → 应事在财富/合同/兑现（金主）
  └─ 冬季 → 应事在健康/家庭/内省（水主）
```

## 6. 贪合忘冲/忘刑（核心规则）

### 6.1 小公狗打架理论

```
本质：一个天干/地支准备去冲/害/刑另一个字时，
     如果有合化先发生 → 它就"忘记"冲/害/刑了

比喻（经典案例）：
  "两个小公狗或者三只小公狗去打架，
   诶，来一只小母狗，这小公狗就不打架了，
   就跟小母狗去交配去了。"

判定规则：
  ├─ 流年天干要冲八字天干 → 但流年天干先与人合化
  │  → 贪合忘冲，冲不成了
  ├─ 流年地支要刑八字地支 → 但流年地支先与他人合化
  │  → 贪合忘刑，三刑不成立了
  └─ 但如果合化不成立（虚合/遥合）→ 仍以刑冲论

实战案例（本讲稿P5）：
  八字有丙+丁，流年壬寅
  壬去冲丙 → 但有丁壬合 → 贪合忘冲 → 丙没事
  但如果丙有强根 → 就算没有丁壬合 → 冲也没事（大树理论）
```

### 6.2 流年太岁优先级最高

```
流年是60年一轮回的大哥
它可以冲开八字原局的任何合化关系

规则：
  八字原局巳酉丑三合 → 但流年寅来冲申 → 三合被冲开
  原局合化再稳定 → 流年太岁来了也得让路
  流年太岁的能量 = 原局合化能量的3-5倍

验证：寅巳相害可以冲破巳酉丑三合
  → 所以属蛇的人在壬寅年仍然会被"害"到
```

## 7. 分散原则（实战发现）

```
当一个流年字同时冲/害/刑多个八字字时：
  能量被分散 → 每个承受的能量减半 → 应灾轻

公式：
  1个流年字冲1个八字字 → 100%能量 → 应灾重
  1个流年字冲2个八字字 → 各50% → 应灾减轻
  1个流年字冲3个八字字 → 各33% → 应灾更轻
  
案例（本讲稿P22-23）：
  命主有2个巳火 + 天干有丙火
  流年寅去害巳 → 寅害2个巳 → 每个巳只承受50%能量
  再＋天干丙火也分担 → 分散更充分
  → 断：腿不舒服但不严重 → 命主反馈：腿有点不舒服，确实不严重

延伸应用：认干娘解灾法
  某女命八字克母亲 → 风水师建议认多个干娘
  认9个干娘 + 1个亲妈 → 每个只克10%
  → 亲妈应灾极轻 → 被保护起来了
  
  认树当干妈 → 必须找千年古树
  小树能量不够 → 可能把小树克死
```

## 8. 大树理论（应灾轻重判定）

```
判断被冲的字受伤程度的形象化方法

规则：
  有强根的字 = 百年大树
    → 能抵抗8级大风
    → 流年太岁来冲，最多折几根树枝
    → 应灾轻：不舒服、小病，死不了

  无根/微根的字 = 小树/树苗
    → 6级大风就倒了
    → 流年太岁来冲，连根拔起
    → 应灾重：大病、大灾、生命危险

  同类多个 = 树林
    → 林中有多棵大树
    → 一棵受冲，其它分担
    → 整体抗风险能力强

根气的判定标准（用于大树理论）：
  真根（禄位/帝旺）→ 百年大树 → 抗8级风
  中气根 → 30年树 → 抗6级风
  余气根 → 5年小树 → 抗3级风
  无根 → 树苗 → 一场小雨就倒了

案例（本讲稿P23-24）：
  某命主丙火在日元，地支有两个巳火强根
  流年壬冲丙 → 这是百年大树
  → 断：会有心脏血液不舒服，但死不了
  → 如果是孤丙无根 → 要命的大病
```

## 9. 流月应事细化（实战流程）

### 9.1 天干透干法

```
流年断事 → 只能断到"哪一年"
要精确到"哪个月" → 看流月天干

规则：
  流年天干/地支在某个流月"透干"
  → 该天干/地支的能量被引化加强
  → 在这个月应事

操作步骤：
  1. 先确定今年犯太岁类型（冲/害/刑/合）
  2. 找出犯太岁的字（流年壬/寅等）
  3. 看流月中哪一个月的天干是壬/甲/丙等同五行
  4. 该月能量加强 → 应事

案例：
  壬冲丙 → 应事在壬子月（阴历11月）
    因为壬子月天干壬、地支子（壬水强根）
    壬水能量突然爆发 → 冲丙 → 应灾

  寅害巳 → 应事在甲辰月（阴历3月）
    因为甲木透干 → 寅木能量加强
    寅害巳 → 应灾
```

### 9.2 节气交割法

```
节气交割前后3天 → 能量突变，身体不适应
→ 精确到某一天

上半年：阳历每月5号和19号交气（±3天）
下半年：阳历每月6号和21号交气（±3天）

原理：
  人的身体有很强的适应性
  突然换了一个五行能量的环境 → 前3天不适应 → 应事
  过了3天适应了 → 就没事了

案例：
  进入亥月（阴历十月）→ 阳历11月21日±3天
  这时流年壬水在地支有了亥水强根
  → 壬的能量突然爆发 → 去冲丙
  → 应灾就在11月21号前后3天
```
