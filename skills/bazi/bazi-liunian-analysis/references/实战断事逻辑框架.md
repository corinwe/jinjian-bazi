## 一、断事前的必备认知（第0步）

在断任何流年之前，必须已经把八字摸透：

```yaml
必须知道（背出来的）：
  ├─ 日主是什么？五行？阴阳？
  ├─ 身强还是身弱？为什么？
  ├─ 用神是什么？忌神是什么？
  ├─ 原局有什么合化？（亥卯未三合？寅午戌？）
  ├─ 原局有什么三刑？（寅巳申？丑未戌？）
  ├─ 原局有什么暗合？（亥午暗合？子巳暗合？）
  ├─ 每个地支的藏干是什么？
  └─ 每个宫位对应的六亲是什么？

⚠️ 实战派从不拿八字现看，而是在脑子里已经把八字拆解完了
```

## 二、断事七步逻辑（从案例逐字还原）

### 第1步：找「犯太岁」关系

```
流年来了，扫一眼八字，找到流年与八字之间最重要的1-2个关系：

天干方面：
  ┌─ 有没有五合？→ 丙辛、丁壬、戊癸、甲己、乙庚
  └─ 有没有四冲？→ 甲庚、乙辛、丙壬、丁癸

地支方面：
  ┌─ 有没有六冲？→ 子午、丑未、寅申、卯酉、辰戌、巳亥
  ├─ 有没有三刑？→ 寅巳申、丑未戌、子卯
  ├─ 有没有三合？→ 申子辰、寅午戌、亥卯未、巳酉丑
  └─ 有没有六合？→ 子丑、寅亥、卯戌、辰酉、巳申、午未

⚠️ 不是列清单！是扫一眼就知道「今年就是这个关系决定一切」
   实战派脑子里有完整的干支关系网，看到丙马上知道丙辛合丙壬冲
   看到寅马上知道寅申冲寅巳害寅亥破寅巳申三刑寅亥合寅卯辰会寅午戌合
   这些都是条件反射，不用现查表
```

### 第2步：先查「有没有化解」

```
找到核心关系后，第一件事不是分析吉凶，而是检查：
这个关系能被合化/化解吗？

检查顺序：
  ① 被冲的字有没有与流年或大运的另一个字发生合化？
     → 有 → 贪合忘冲 → 这个冲不成立了 → 跳过
     → 案例：壬冲丙，但有丁壬合 → 壬跟丁走了，不冲丙了

  ② 如果没有合化，被冲的字有没有强根？
     → 有真根（禄位/帝旺）= 百年大树 → 冲不动，应灾轻
     → 有中气根 = 30年树 → 会摇晃但不会倒
     → 无根/孤零零 = 小树苗 → 一冲就倒，应灾重
     → 案例：丙有巳火+寅午合火强根 → 壬来冲没事
               丙只有微根 → 壬来冲 → 父亲心脏血液

  ③ 如果原局有同类字分担？
     → 1个流年字冲2个八字字 = 各分担50% → 应灾减半
     → 案例：寅害两个巳 → 每个只承受50% → 腿不舒服但不严重
```

### 第3步：看作用在「什么宫位」

```
这是实战断事最核心的一步！我的报告之前完全忽略了这一步的核心地位。

找到被作用的字 → 它在哪个宫位 → 直接决定断谁的事：

  年干（1-9岁） → 父系家族 + 头顶~鼻梁
  年支（10-18岁） → 母系家族 + 鼻子~锁骨（牙/颈/肩）
  月干（19-27岁） → 父亲/兄姐 + 锁骨~胸
  月令（28-36岁） → 母亲/弟妹 + 胸部~肚脐
  日支（37-45岁） → 配偶/自己 + 小腹/生殖器官
  时干（46-54岁） → 长子/长女 + 大腿
  时支（55岁后） → 次子及后 + 小腿/脚

⚠️ 双重宫位：年干=父亲，月干也有父亲信息
              年支=母亲，月令也有母亲信息

实战案例印证：
  丙在年干 → 父亲宫位 → 断父亲心脏血液 ✅
  申在时柱 → 子女宫位 → 断孩子脚部 ✅
  申在年支 → 母系宫位 → 断母亲不舒服+自己牙/颈 ✅
  巳在日支 → 夫妻宫 → 断夫妻关系出问题 ✅
```

### 第4步：看这个五行对应「什么身体部位」

```
找到被作用的字后 → 这个五行对应什么人体部位？

  火 → 心脏、血液、眼睛、小肠
  金 → 肺、大肠、骨骼、牙齿
  木 → 肝、胆、神经、四肢
  水 → 肾、膀胱、耳朵、血液
  土 → 胃、脾、皮肤、肌肉

结合宫位更要精确：
  丙在年干（头顶~鼻）→ 头部血管、眼睛
  丙在月干（锁骨~胸）→ 心脏、肺部
  丙在日干 → 自己心脏
```

### 第5步：判断三刑/合化的「真实含义」

```
⚠️ 这是我和实战派最大的认知差距：

三刑 ≠ 扣分
三刑 = 地支藏干打架 -> 余气全部打没 -> 只剩本气 -> 本气能量翻倍
例:丑未戌三刑 -> 癸辛丁乙余气全没 -> 全变土 -> 15倍比劫

比如寅巳申三刑：
  → 寅木能量加强
  → 巳火能量加强
  → 申金能量加强
  三个藏干都加强，不是哪个被削弱！

判定标准：
  加强的是喜用五行 → 好事
  加强的是忌神五行 → 坏事

合化 ≠ 一定是好事
  合化只是「加强了合化出来的五行」
  这个五行是喜用 → 好事
  这个五行是忌神 → 坏事

实战案例：
  寅午戌三合火 → 从弱格喜火 → 丙午月进账1.2亿 ✅
  寅亥合木出七杀 → 七杀无制 → 母亲去世 ✅
```

### 第6步：分时段判断「状态是否切换」

```
⚠️ 这是我最致命的漏洞！

同一个八字、同一个流年，上半年和下半年的状态可能完全不一样。

判断方法：
  看大运天干 vs 流年天干 → 决定上半年基调
  看大运地支 vs 流年地支 → 决定下半年基调

从强格的特别处理：
  从强格 = 满盘生助100分，强到极点反变弱 → 恒定20分
  普通流年的克泄耗不会让它破格
  破格条件：必须合化成相反五行（把生助的变成克耗的）

实战案例（假从弱）：
  大运生 + 流年生 → 上半年普通身弱
  → 火为喜用 → 巳火加强 = 好事 🔄
  大运生 + 流年耗 → 下半年从弱
  → 火为忌凶 → 巳火加强 = 坏事 🔄
  同一个巳火，上半年=吉，下半年=凶！
```

### 第7步：给出断语（6要素）

```
实战派的断语永远包含以下6个要素：

1️⃣ 其人 — 谁？（自己/父亲/母亲/配偶/子女）
2️⃣ 其事 — 什么事？（心脏/血液/车祸/破财/升职）
3️⃣ 其时 — 什么时候？（X月/节气/流月透干）
4️⃣ 其位 — 身体哪个部位？
5️⃣ 其度 — 什么程度？（要命/重病/不舒服/小麻烦）
6️⃣ 其因 — 为什么？（五行什么原理）

对比：
  我 ❌："今年注意健康"
  实战 ✅："父亲注意心脏血压，阴历十一月亥子丑会水时应事，程度中等，因为壬水冲丙火，丙在年干=父亲宫位"
```

## 三、完整案例还原（看实战派怎么读命）

### 案例1：壬冲丙 · 不怕冲

```
实战派的脑中思维过程（不是写出来的，是想出来的）：

① 流年壬 → 丙壬冲？ → ✅ 八字有丙
② 有合化解吗？ → 丁壬合 → ✅ 贪合忘冲 → 不冲了
③ 即使没有合化，丙有根吗？→ 丙在地支有巳火+寅午合火=百年大树
④ 冲不动 → 没事

一句话断语：没事。
```

### 案例2：壬冲丙 · 父亲应灾

```
① 流年壬 → 丙壬冲？ → ✅ 八字有丙
② 有合化解吗？ → ❌ 没有
③ 丙有根吗？ → 有微根（巳）但不是强根 = 会被冲坏
④ 丙在哪个宫位？ → 年干 = 父亲宫位 → 父亲应灾
⑤ 丙对应身体什么？ → 火 = 心脏、血液、眼睛
⑥ 什么时候应事？ → 水能量加强的时候：
   - 阴历三月申子辰合水
   - 阴历十一月亥子丑会水

断语：父亲注意心脏血液眼睛问题，阴历三月和十一月应期，程度中等（因为有微根活着不会要命）。
```

### 案例3：寅申冲 · 孩子应灾

```
① 流年寅 → 寅申冲？ → ✅ 八字有申
② 有合化解吗？ → ❌ 没有
③ 申有根吗？ → 孤零零的申，无根无合 = 小树苗
④ 申在哪个宫位？ → 时柱 = 子女宫 → 孩子应灾
⑤ 申对应身体什么？ → 金 = 脚部、骨骼
⑥ 寅申冲的特殊含义 → 驿马冲 = 车祸风险

断语：孩子注意脚部受伤，注意车祸风险，程度严重（申无根）。
```

## 四、我与实战派的根本区别

| 维度 | 我的做法 ❌ | 实战派的做法 ✅ |
|:---|:---|:---|
| **断事起点** | 列清单 → 评分 | **读命** → 先摸透八字本质 |
| **关系筛选** | 列200条关系 | **扫一眼**找1-2个核心 |
| **合化检查** | 列在后面 | **第一件事就查**有没有贪合忘冲 |
| **宫位** | 列着但不用于断语 | **断语的核心来源**—直接决定断谁 |
| **三刑** | 扣-60分 | **能量加强**，喜=吉忌=凶 |
| **合化** | 合化=吉 | **合化=加强某五行**，喜忌决定吉凶 |
| **时段** | 一年一个断语 | **分上/下半年**，状态可能完全相反 |
| **断语** | "注意健康"（泛） | "父亲心脏血压，阴历十一月"（6要素） |
| **思维方式** | 套公式 → 算分 | **条件反射** → 干支关系网刻在脑子里 |
