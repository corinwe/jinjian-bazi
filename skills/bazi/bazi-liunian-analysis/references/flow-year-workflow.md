# 流年断事实战工作流（金鉴真人·实战断事体系·V3）

> **⚠️ 核心原则**：不是「列清单→评分→给等级」的公式游戏。
> 而是「读命→找核心关系→动态分析→给具体断语」的实战过程。
> 断事准不准，取决于你能不能读出命局的本质，而不是你列了多少条关系。

---

## 一、输入准备

```yaml
命主八字: [四柱完整]
命主性别: [男/女]
当前大运: [干支] (第X年，处在第几年)
目标流年: [干支]
目标流月: [干支] (如需要精确到月)
```

---

## 二、断事前必做的「命局本质判定」

### ⚡ 第0步：今年八字处于什么状态？

> 这是最关键的步骤，决定了后续所有的分析逻辑。

```yaml
检查大运+流年对本命局的能量影响：

情形A：大运生助 + 流年生助 → 身强/普通弱
情形B：大运生助 + 流年消耗 → 不变（保持原状态）
情形C：大运消耗 + 流年生助 → 不变
情形D：大运消耗 + 流年消耗 → 从弱/从强（状态切换）

⚠️ 特别：假从弱的八字，情形A→变普通弱，情形D→保持从弱
   同一个五行，在不同状态下喜忌截然不同

判定结论：今年命主处于_____状态 → 喜用五行是_____ → 忌神五行是_____
```

## 三、核心断事流程（三问法）

### ⚡ 第一问：今年最核心的1-2个关系是什么？

```
不是列200条关系，而是找1-2个「决定了今年基调」的核心关系。

怎么找？
  ① 流年天干与八字任何一个天干有「冲」或「合」？ → 这就是核心
  ② 流年地支与八字任何一个地支有「冲」或「三刑」？ → 这就是核心
  ③ 大运与流年有「岁运并临」？ → 这就是核心
  ④ 以上都没有？→ 流年与八字有「合化」？ → 这就是核心
  ⑤ 都没有？→ 今年是平年，按正常轨迹走，没有大事

得到1-2个核心关系后，再查其他次要关系（三会、六合、暗合等）作为辅助。
```

### ⚡ 第二问：这个关系能不能被化解？

```
顺序检查：
  ① 有没有合化可以「贪合忘冲」？
     → 有 → 这个关系不成立了 → 跳过
  ② 如果没有合化，被作用的字有没有根？
     → 有强根（禄位/帝旺）= 百年大树 → 应灾轻
     → 无根/孤零零 = 小树苗 → 应灾重
  ③ 如果有多个同类字分担？
     → 1冲2 = 各50% → 分散减弱
```

### ⚡ 第三问：这个关系作用在谁身上？什么时候应事？

```
宫位断事（这是核心！不是一个附加项）：
  找到被作用的字 → 它在哪个宫位？
  年干 = 父亲/父系家族 + 头顶~鼻
  年支 = 母亲/母系家族 + 鼻~锁骨（牙/颈/肩）
  月干 = 父亲/兄姐 + 锁骨~胸
  月令 = 母亲/弟妹 + 胸~肚脐
  日支 = 配偶 + 小腹/生殖器官
  时干 = 长子/长女 + 大腿
  时支 = 次子及后 + 小腿/脚

应期判断：
  天干透干法 → 流年天干在哪个流月「透干」→ 该月应事
  节气交割法 → 节气前3后3天 → 能量突变
  三合引动法 → 流年某字被三合局加强时 → 该月应事

断语格式：【六亲】+【身体部位】+【应期月份】
  例："父亲注意心脏血液问题，阴历十一月应事"
  例："自己注意胸闷心悸，立秋后三天可能加重"
```

## 四、分时段断事（核心升级）

### ⚡ 每个流年必须分两段分析

```yaml
前半年（立春~立秋）：流年天干主导（70%）
后半年（立秋~立春）：流年地支主导（70%）

而且：命主的能量状态在前半年和后半年可能不同！
  → 大运天干 vs 流年天干 的互动决定前半年基调
  → 大运地支 vs 流年地支 的互动决定后半年基调
  → 如果状态切换，则同一五行在上/下半年的喜忌可能相反

断语必须分时段：
  "上半年：……（具体断语+应期）
   下半年：……（具体断语+应期）"
```

## 五、三刑的正确理解（重要修正）

```
三刑 ≠ 扣分项

三刑的本质：地支藏干「打架」→ 每个藏干的能量都被加强
  → 寅巳申三刑：寅木/巳火/申金的能量都加强
  → 丑未戌三刑：丑土/未土/戌土的能量都加强（15倍）
  → 子卯刑：子水/卯木的能量都加强

判断标准：
  被加强的是喜用五行 → 好事
  被加强的是忌神五行 → 坏事

⚠️ 前提：先检查有没有合化解开三刑（贪合忘冲）
```

## 六、断语标准模板（必须遵守）

```
【年份】丙午

【命局状态】身极弱（假旺真弱），大运戊子后5年子水主导
【状态变化】上半年流年天干丙火主导，下半年流年地支午火主导

【核心关系1】丙辛合水 → 丙合时干辛金
  ├─ 合化了吗？→ 无引化 → 合而不化，以相克论
  ├─ 辛金在时干 → 宫位：子女/大腿
  └─ 断语：注意子女方面的财务支出，大腿部位可能有酸痛

【核心关系2】午未合化火土 → 午合月令未
  ├─ 午未合加强火（忌神）→ 加重身弱
  ├─ 未为月令 → 宫位：母亲/肚脐
  └─ 断语：母亲注意消化系统，立夏后三个月明显

【次要关系】子午冲（大运子冲流年午）
  ├─ 大运子水被冲 → 用神受损
  ├─ 但大运后5年子水仅30%权重 → 影响不大
  └─ 断语：略感精力不足，无大碍

【全年总结】
  上半年（丙主导）：丙辛合水不利，注意子女和财务
  下半年（午主导）：午未合火加重，重点关注母亲健康
  关键应期：乙未月（阴历六月，午未合化最强）
```

## 七、一个准确断语的4个要素

```
✅ 要素1：有其人 — 明确指出是谁（自己/父亲/母亲/配偶/子女）
✅ 要素2：有其事 — 明确指出什么事（心脏/血液/车祸/财运）
✅ 要素3：有其时 — 明确指出何时应事（X月/X节气）
✅ 要素4：有程度 — 明确指出严重程度（要命/不舒服/麻烦/小波折）

错误示范："今年注意健康" ❌
正确示范："父亲注意心脏血压，阴历十一月亥子丑会水时应事"
```
