# 串宫压运 — 流年断事的盲派特法

盲派串宫压运与bazi-liunian-analysis的关系：

## 互补关系

| 维度 | liunian-analysis | 串宫压运 |
|:----|:----------------|:---------|
| 核心方法 | 三力分工（本命局+大运+流年） | 十二神排布+盖头关系 |
| 断事工具 | 十神/刑冲合害/能量倍数 | 太岁起宫/十二神/断诀口诀 |
| 应期定位 | 节气交割/流月/分时段 | 四柱定格（年春夏月夏日秋冬） |
| 核心结论 | 过三关（只说三件事） | 大运十二神为本定一年总概括 |
| 六亲定位 | 宫位对应 | 同宫位系统，另加神之所指 |

## 串宫压运的特色补充

1. **十二神体系** — 以「神」（暗能量/磁场）而非五行十神为断事维度
2. **太岁起宫法** — 以流年地支顺排，一年一换，直观反映当年气场
3. **盖头断事** — 天干之神 vs 地支之神的组合关系决定吉凶程度
4. **断诀口诀** — 12首完整口诀诗，可直接用于断语输出
5. **天地换算** — 天干→地支的映射，扩展了四柱每字的断事范围

## 使用建议

在流年分析时可将串宫压运作为辅助验证系统：
- liunian-analysis 出框架结论（发财/灾祸/婚姻等）
- 串宫压运 出补充细节（贵人/小人/口舌/病符等）
- 两者交叉验证，提高断语的精确度
