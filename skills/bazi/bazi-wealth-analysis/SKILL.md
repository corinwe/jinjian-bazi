### 什么是能量链条？

```
   印（生日主）
    ↓ 生
 比劫（日主本身）→ 生 → 食伤 → 生 → 财
    ↓ 克                     ↓ 克
   财                       官杀
```

**发财的本质逻辑：** 比劫（身）→ 食伤 → 财 这条能量链条必须完整且够力。

| 状态 | 链条状态 | 结果 | 合作策略 |
|:----|:---------|:----|:---------|
| **身强** | 比劫足→能生食伤→链条顺畅 | 能担财 | 以财官食为喜用（消耗多余能量） |
| **身弱** | 比劫弱→生不动食伤→链条断 | 担不住财 | 以印比为喜用（补充能量接上链条） |
| 身强>60分 | 链条过剩反克终点 | 反克财，等级-1 | 比劫年防被劫，不合伙 |
| 食伤旺 | 中间环节强→生财有力 | 财根深厚 | 技能/创意/技术赚钱 |
| 无食伤 | 中间环节断 | 财根不稳 | 靠大运补食伤或财星 |

### 实战运用

```yaml
问：身强+财旺+食伤旺+有禄神 → 什么等级？
答：链条最完整！比劫(强)→食伤(旺)→财(旺)三环节全通+禄神加持
  · 有库+大运配合 = 👑 巨富
  · 有库+大运刑冲 = 💰 大富
  · 无库 = 🥈 中富~大富

问：身强+财弱<40+无库+无刑冲 → 什么等级？
答：起点强但终点弱。中富条件需刑冲(此无)，小富需身弱(此强)
  核定级：中富下段(几千万)，无刑冲=稳，等食伤/财大运可发到大富
```

**口诀：** 财富不看单维度，能量链条看全部；起点终点加中间，三段打通才是富。

### 🚨 5.4 财星五行定向铁律（2026-06-23新增·父亲案例触发）

> **致命错误**：父亲八字（己丑癸酉癸亥戊午）癸水日主，午中己土被误算为偏财7.2分。根因：己土含"财"字且午火中有财，误以为午中所有藏干都是财。
> **正确**：癸水克火为财（丙丁巳午），己土是土，土克水→七杀。财星只有午中丁火12分。

```yaml
【铁律】每次算财星前，先定日主五行，再定财星五行：

日主五行 → 我克者 → 财星五行
金（庚辛）→ 金克木 → 财=木（甲乙寅卯亥卯未）
木（甲乙）→ 木克土 → 财=土（戊己辰戌丑未）
水（壬癸）→ 水克火 → 财=火（丙丁巳午未寅午戌）
火（丙丁）→ 火克金 → 财=金（庚辛申酉巳酉丑）
土（戊己）→ 土克水 → 财=水（壬癸亥子申子辰）

然后只在财星五行中找正财/偏财。

⚠️ 最容易搞混的组合：
  癸水日主 + 午火 → 财=火(丁) ✅，己土是七杀❌
  庚金日主 + 巳火 → 财=木(甲乙寅卯)，巳中丙火是七杀❌不是财
  戊土日主 + 寅木 → 财=水(壬癸亥子)，寅中甲木是七杀❌不是财
```

财星只包含：**正财**和**偏财**。**不包含劫财！**

- 正财 = 我克且阴阳不同（如辛金克甲木）
- 偏财 = 我克且阴阳相同（如辛金克乙木）
- **劫财 = 与我同阴阳的同五行 → ❌ 不是财星！**

Python实现：
```python
caixing = {'正财', '偏财'}  # 仅此两种！劫财（含"财"字但不是财星）不在此列

def is_caixing(rizhu, gan):
    ss = shi_shen[rizhu][gan]
    return ss in caixing  # True=正财/偏财, False=其他（含劫财）
```

## §6 财富等级修正因子（含动态兑现修正）

### 6.1 加分因子
- ✅ 食伤旺（财根稳，等级+1档）
- ✅ 日/时柱有财库（巨富必要条件）
- ✅ 财星得令（≥40分）
- ✅ 从弱格（0分→50分，等级+2档）

### 6.2 减分因子
- ❌ 财星受刑冲迫害（等级-1档）
- ❌ 无食伤（财根不稳）
- ❌ 身过强>60分（自负破财，等级-1档）
- ❌ 四柱无财且身弱（和尚命，贫穷级）

### 🚨 6.3 财星受刑冲迫害（原文依据：素材11行37「财星被合化…财被别人抢走」）

> **原文依据**：素材11行37：「流年的时候，财星受到了冲击，把财给冲走了…财星被合化成其他的能量…也是我们破财的信息」
> 素材11行421：「财星受到刑冲迫害比较多，他就容易破财嘛，挺能赚也挺能花」

**原理：** 财星所在位置被全局围克时，兑现率打折。这是实战校准补充，非原始理论量化分级。

**判定流程：**
```
Step 1 — 找出八字中所有财星所在位置（天干+地支藏干）
Step 2 — 列出每个财星位置被哪些字克制
  - 直克：克该五行的字（如火见水）
  - 晦/耗：同性相害（如火见湿土晦火）
  - 生助对手：生助克星的字（如金生水助克火）
Step 3 — 统计围克程度：
  □ 0-1个克制 → 无围克 → 全额兑现
  □ 2-3个克制 → 围克 → 打折兑现（30-50%）
  □ ≥4个克制 → 重围 → 大运救才兑现
Step 4 — 检查是否有救应（三合救应/天干透印护财/大运补财）
```

**口诀：** 财星围克要看清，一克两克还正常；三克四克围城局，兑现打折要算清；大运救应才发财，平时只是辛苦命。

## §12 🚨 核心判断原则：动态条件体系，非静态等级标签（2026-06-23新增·实战校准）

> **致命误区**：之前将九龙道长的财富体系理解为"五级终身标签"——身强财弱=中富，终身中富。实际上九龙道长的核心公式是「身强财旺=发财」——这是一个**动态条件公式**，条件满足时才发财，不是终身标签。

### 12.1 静态标签 vs 动态条件的根本区别

```
静态标签（❌ 错误用法）：
  身强财弱 → 中富 → "你终身中富，一生总财富几千万"

动态条件（✅ 正确用法）：
  身强财弱 → 底子好但财星弱 → "你遇到财星/食伤大运年份才发中财，
  平时是辛苦钱或小财。因为你财星被围克加上时代限制，
  实际兑现大概在小富上段至中富下段之间"
```

### 12.2 六种状态的正确断语（动态版）

| 原状态 | 动态正确理解 | 错误理解（禁止） |
|:-------|:------------|:----------------|
| **身强财旺** | 天生发财命，不挑大运也能发 | "巨富/大富"（正确，但别忽略大运引化） |
| **身强财弱** | 底子好但财星弱，**等财星/食伤大运才发中财** | "终身中富" ❌ |
| **身弱财旺** | 财在身边但抓不住，**等印比帮身大运才变现** | "终身中富" ❌ |
| **身弱财弱** | 辛苦钱，**遇印比大运才发中财** | "终身小富" ❌ |
| **无财身弱** | 和尚命，对钱看淡 | "贫穷"（这个相对静态） |
| **从弱格** | 特殊格局，**运对方向就爆发** | "亿万富翁"（简单化） |

### 12.3 动态判断五步法

```
Step 1 — 定基础状态
  用§1的六种八字状态表确定基础（身强财旺/身强财弱/身弱财旺等）

Step 2 — 看围克折扣
  用§6.3的财星受围克规则判断兑现率

Step 3 — 看财星刑冲影响（素材11行37）
  用§6.4的时代修正系数判断上限

Step 4 — 看大运窗口
  列出补财/补食伤的大运年份：
  □ 财星大运 → 直接补财，发中财
  □ 食伤大运 → 食伤生财，发中财
  □ 印比大运（身弱者）→ 帮身后担财
  □ 无补财大运 → 不发财，辛苦钱

Step 5 — 综合结论
  "理论等级{五级定位}，但{围克/时代/大运}情况下，
  实际财富在{等级}范围内，集中在{特定大运年份}兑现"
```

## §9 批量多报告财星核查法（0617八人全量审计校准）

> **发现来源**：0617对8份报告（家主/主母/子源/立/晓/印/凤/琼）的全量审计，发现6/8的财星分数有系统性计算错误。
> **核心教训**：单份报告验证不能发现跨报告的系统性误差。需要「基准人法」批量核查。

### 9.1 基准人法（跨报告系统性错误侦测）

```yaml
【原理】当多个报告出现同一种计算错误时（如月令藏干都用了12分而非40分），
  单份报告审计无法发现，但跨报告对比会暴露模式。

Step 1 — 选基准人
  □ 选择一个最熟悉的、八字最清晰的命主作为基准
  □ 基准人必须满足：身强弱/财星/格局已无争议（经实战校准）
  □ 基准人的财星分数逐项手算验证
  示例：主母成（丁卯丁未庚午壬午）→ 财星16分已校准 ✅

Step 2 — 其他人与基准人横向对比
  □ 用相同的计算规则（位置基础分×藏干%）算所有人的财星
  □ 对比报告中的分数 vs 手算分数
  □ 如果多人同时出现同类偏差 → 一定是规则应用错误！

Step 3 — 锁定系统性问题
  □ 所有人都月令藏干写12分不是40分 → 规则记忆有误 ❌
  □ 所有人都把比劫算成财星 → 日主财星五行理解有误 ❌
  □ 部分人有错 → 按个案排查

Step 4 — 修复后全量重算
  □ 修正规则后，重新计算所有人的财星分数
  □ 逐人记录「修正前→修正后→修正原因」
  □ 推送全量更新到知识库

口诀：基准人选一个，手算规则先确认；横向对比八个人，模式偏差一网打
```

### 9.2 日主五行财星定向法（防把比劫当财星）

> **致命教训**：凤（癸水日主）的财星计算中，月支子癸水（比肩）和时支亥壬水（劫财）被当成了财星，实际上：
> - 癸水日主 → 财星是火（丙丁午巳），不是水
> - 水的财 = 火（水克火为财）
> - 子/癸/亥/壬 = 比劫，不是财

```yaml
【强制流程】每算财星前，先确定日主五行的财星是什么：

日主金（庚/辛）→ 财星是木（甲乙寅卯亥卯未）
日主木（甲/乙）→ 财星是土（戊己辰戌丑未）
日主水（壬/癸）→ 财星是火（丙丁巳午未寅午戌）
日主火（丙/丁）→ 财星是金（庚辛申酉巳酉丑）
日主土（戊/己）→ 财星是水（壬癸亥子申子辰）

口诀：日主财星五行定，金木水火土分明；
      金→木 木→土 水→火 火→金 土→水；
      日主定了财星明，比劫财星不会混
```

### 9.3 批量核查自动化清单

```yaml

□ 选基准人 → 手算基础分 → 核对
□ 所有人统一用同一套计算规则
□ 所有人的月令藏干财星=40分×% (不是12分×%)
□ 所有人的年支财星=4分 (不是8分)
□ 全体日主财星五行核对 → 无人把比劫算成财星
```

### 10.0 墓库理论基础——40分法则

```yaml

  辰戌丑未四字，根据全局能量判定是"墓"还是"库":
  ✅ ≥40分 → 库（保险库/仓库）→ 装金银财宝 → 好事
  ❌ <40分 → 墓（坟墓/棺材）→ 装尸体 → 坏事

  - 未 → 看木的能量（亥卯未合木）

口诀：墓库区别四十线，高库低墓两重天；全局能量算清楚，库开好事墓开灾
```

## 🚨 补库 vs 开库 前提条件（2026-07-02 老板校准）

**补库（新建人工库）→ ❌ 不需要财星>40分**
补财库是人工建一个容器（放保险柜、银行开户、摆风水物等），没有前提条件。九龙道长说谁做都可以，不需要任何门槛。

**开库（冲开现有库）→ ✅ 需要财星≥40分**
这是§10.0 40分法则的核心规则：
- 该五行能量≥40分 → 是"库"（保险库/仓库），打开=财库释放=发财 ✅
- 该五行能量<40分 → 是"墓"（坟墓/棺材），打开=灾祸破财 ❌

**口诀：** 补库无前提，谁做都可以；开库看分数，40是分水岭；高分开得财，低分开破财。

### 10.2 墓库开闭机制

```yaml
墓库默认状态：门是关着的（能量不动，无事发生）

开门条件：大运/流年对墓库所在的地支进行刑冲迫害
  - 冲（六冲）：辰戌冲、丑未冲 → 冲开力度最大
  - 刑（三刑）：丑未戌三刑 → 打开但凶险
  - 害（六害）：子未害、丑午害等 → 轻微打开
  - 破（六破）：辰丑破、午破等 → 最轻微

库被打开 = 好事：
  - 财库开 → 发财、进账、投资回报
  - 官杀库开 → 升官、掌权、管理提升
  - 比劫库开 → 合作生财、团队共赢
  - 印库开 → 得合同/证书/权力
  - 食伤库开 → 得福气（获奖/旅游/福利）

墓被打开 = 坏事：
  - 财墓开 → 破财、投资失败
  - 官杀墓开 → 丢官罢职、官司
  - 比劫墓开 → 破财、被人算计
  - 印墓开 → 失去权力、合同被抢
  - 食伤墓开 → 福气被夺

口诀：库开好事墓开灾，冲刑迫害把门开；能量达库定为吉，能量达墓定为灾
```


### 10.5 补财库实操技法（2026-06-19新增·详细版）

> **前提**：必须先诊断财库状态（有无库？库还是墓？在哪个宫位？）

#### 方法一：刑冲生肖催财法（效果最明显）

```yaml
原理：利用十二生肖的刑冲关系，人为冲开财库
      相当于模拟大运/流年的刑冲迫害 + 可控

操作步骤：
  ① 查出日主的财库是什么（查10.1表）
  ② 找到能冲开该财库的生肖物品
     - 辰水库 → 戌狗冲（摆放狗生肖）
     - 戌火库 → 辰龙冲（摆放龙生肖）
     - 丑金库 → 未羊冲（摆放羊生肖）
     - 未木库 → 丑牛冲（摆放牛生肖）
  ③ 将生肖摆件放在财库对应的方位
  ④ 关键节点（谈大生意/签合同前）用该物品去"冲"

风险提示：
  - 刑冲物品可能对家人八字有害
  - 最好放在自己的独立空间（个人办公室/书房）
  - 家里有小孩/老人的需谨慎
  - 用几天后撤走，隔段时间再放 → 能量波动才有效
```

#### 方法二：柜门开关法（辅助方法）

```yaml
原理：人工模拟仓库门的开合
操作：
  ① 在财库方位放一个小柜子或储物盒
  ② 反复进行开门→关门的动作
  ③ 配合意念：开门=财库打开，关上门但不要关死
效果：不如刑冲法明显，但没有副作用
```

#### 方法三：平时补库 + 关键时刻催财

```yaml
日常操作（平时）：
  在财库方位常年放置财库吉祥物
  五行颜色/材质对应该方位五行即可
  目的：储存钱的能力持续加强

关键时刻（谈大生意前/急需钱时）：
  将刑冲生肖物品放在财库方位几天
  目的：主动冲开财库，催财入库
  注意：冲几天就撤走，长期放会失效
  能量波动原理：放几天→适应→没效果→拿走几天→再放→又有效果
```

```yaml
Step 1 — 列出大运地支

Step 3 — 被冲的地支是否是财库？
  □ 是财库 → 冲开财库 = 财富爆发（前提财星≥40分）
  □ 不是财库 → 按常规十神分析
  □ 是财库但财星<40分 → 冲开的是墓不是库 → 反而是破财信号

常见遗漏模式：
  ❌ 己丑运 → 只写「守成期」→ 漏了丑未冲开财库 ✅
```

### 13.5 流年合化/冲库财富触发（深度版）

#### 13.5.1 流年六合出财星五行→直接发财

六合（地支六合）直接引出财星五行，是流年发财最直接的信号之一：

| 财星五行 | 六合组合 | 触发说明 |
|:---------|:---------|:---------|
| **木**（甲乙木日主·财星=土） | 子丑合土、午未合土 | 合出财库五行，直接发财 |
| **火**（壬癸水日主·财星=火） | 寅亥合木、卯戌合火 | 合出财星五行，财运爆发 |
| **土**（甲乙木日主·财星=土） | 辰酉合金、巳申合水 | 合出财星五行 |
| **金**（丙丁火日主·财星=金） | 巳申合水、卯戌合火 | 合出财星五行 |
| **水**（戊己土日主·财星=水） | 午未合土、子丑合土 | 合出财星五行 |

> **核心原理**：流年地支与命局地支六合后，生成的新五行正好是日主的财星五行，相当于天降财运。若合出的财星五行能量强（≥40分），发财幅度大。

#### 13.5.2 流年三合/三会出财星五行→爆发级别

三合局和三会局形成财星五行时，财富爆发力远超六合：

| 财星五行 | 三合局 | 倍数 | 三会局 | 倍数 |
|:---------|:-------|:----:|:-------|:----:|
| **木** | 亥卯未合木 | 15倍 | 寅卯辰会木 | 20倍 |
| **火** | 寅午戌合火 | 15倍 | 巳午未会火 | 20倍 |
| **土** | — | — | 辰戌丑未会土 | 20倍 |
| **金** | 巳酉丑合金 | 15倍 | 申酉戌会金 | 20倍 |
| **水** | 申子辰合水 | 15倍 | 亥子丑会水 | 20倍 |

**判定方法：**
1. 判断流年+原局地支能否凑齐三合/三会局的三个字
2. 检查合成的新五行是否 = 日主的财星五行
3. 若财星得分≥40分 → 爆发级别发财；财星<40分 → 中等级别

> **口诀**：三合爆发一五年，三会更是二倍翻；财星够强才叫发，不够只是小波澜。

#### 13.5.3 流年冲财库→冲开库门=财富释放

流年六冲直接冲开财库，是财富释放的标志性信号：

| 财库 | 冲库流年 | 触发条件 | 财富效果 |
|:----|:---------|:---------|:---------|
| 辰（水库） | 戌年冲辰 | 财星≥40分 | 财富释放，进大财 |
| 戌（火库） | 辰年冲戌 | 财星≥40分 | 财富释放，进大财 |
| 丑（金库） | 未年冲丑 | 财星≥40分 | 财富释放，进大财 |
| 未（木库） | 丑年冲未 | 财星≥40分 | 财富释放，进大财 |

**前置条件（必检）：**
```
□ 财星得分≥40分 → 冲开的是「库」→ 财富释放 ✅
□ 财星得分<40分 → 冲开的是「墓」→ 反而破财 ❌
  详见§10.0 墓库理论基础——40分法则
```

**冲库财富量级判断：**
- 财星≥60分 + 冲库 → 大财~巨富级别
- 财星40~59分 + 冲库 → 中财~大财级别
- 财星<40分 + 冲库 → 破财/投资失败（墓被打开）

#### 13.5.4 流年拱合升级

拱合升级是指原局存在「虚拱」关系，流年补全第三字形成完整三合局：

| 原局拱合 | 拱出的五行 | 流年补全字 | 形成完整三合局 | 效果 |
|:---------|:----------|:-----------|:--------------|:-----|
| 亥未拱木 | 木 | 卯年 | 亥卯未三合木 | 木旺爆发 |
| 寅戌拱火 | 火 | 午年 | 寅午戌三合火 | 火旺爆发 |
| 巳丑拱金 | 金 | 酉年 | 巳酉丑三合金 | 金旺爆发 |
| 申辰拱水 | 水 | 子年 | 申子辰三合水 | 水旺爆发 |

**判定步骤：**
```
Step 1 — 检查原局是否存在拱合关系
  亥+未（拱木）、寅+戌（拱火）、巳+丑（拱金）、申+辰（拱水）
  → 中间缺一字（卯/午/酉/子）

Step 2 — 检查流年是否补全第三字
  若流年是缺的那个字 → 拱合升级为三合局 ✅

Step 3 — 判断新三合局的五行是否为财星
  ✅ 是财星五行 → 财富暴发（15倍能量）
  ❌ 不是财星五行 → 该五行对应的十神领域爆发
```

#### 13.5.5 流年刑开库（丑未戌三刑开库）

丑未戌三刑是特殊的刑开库机制：

| 三刑组合 | 效果 | 吉凶判断 |
|:---------|:----|:---------|
| 丑+未+戌三刑 | 同时冲开丑金库、未木库、戌火库 | 吉：三个库全开→三库能量释放 |
| 丑+未相刑 | 冲开丑金库、未木库 | 吉：两库同开 |
| 丑+戌相刑 | 丑戌相刑破库 | 凶中带吉：刑开但不完整 |

**丑未戌三刑财富判定规则：**
```
Step 1 — 检查原局+流年是否凑齐丑未戌三字
  □ 原局已有丑+未+戌 → 三刑常驻，命局自带动荡
  □ 原局有丑未/丑戌/未戌 + 流年补全剩一字 → 当年三刑触发

Step 2 — 按§10.0 40分法则判断是库还是墓
  □ 该五行≥40分 → 库开 → 三库能量全释放，财运爆发 ✅
  □ 该五行<40分 → 墓开 → 三墓齐开，多领域破财 ❌

Step 3 — 库开的财富效果
  三库全开 = 三倍能量释放（丑金库+未木库+戌火库）
  对财星的影响：若财星五行在其中，爆发程度为普通冲库的3倍
```

#### 13.5.6 能量倍数速查表

| 触发类型 | 能量倍数 | 说明 |
|:--------|:--------:|:-----|
| **三会局** | **20倍** | 最强爆发（如寅卯辰会木） |
| **三合局** | **15倍** | 次强爆发（如亥卯未合木） |
| **六合局** | **10倍** | 中等爆发（如寅亥合木） |
| **六冲库** | **5倍** | 冲库释放（前提财星≥40分） |
| **三刑开库** | **3~5倍** | 三刑开库（最不稳定，需配合全局） |
| **拱合升级** | **15倍** | 拱合→三合升级（等同于三合局） |

> **核心规则**：倍数必须配合财星基础分使用。基础分越高，倍数效果越明显。财星0分者，再高的倍数也无用。

### 13.6 流年十神透藏财富影响

#### 13.6.1 财星透干 vs 藏干

| 状态 | 效果 | 财富表现 |
|:----|:-----|:---------|
| **财星透干** | 财来寻我 | 主动找上门来的钱财机会，轻松获得，别人送钱 |
| **财星藏干** | 求财艰难 | 需要自己主动争取，钱在暗处，需挖掘才能获得 |

**透干分级判断：**
```
□ 天干透正财 → 稳定收入来源显化，正职加薪/固定业务收入增加
□ 天干透偏财 → 意外之财显化，投资/副业/偏门收入机会
□ 天干透财星且得根（地支有本气）→ 财星稳固，到手能存
□ 天干透财星但无根（地支无藏干）→ 虚浮之财，过手财
□ 天干不透财、财全藏地支 → 求财辛苦，需流年引化才能兑现
```

#### 13.6.2 比劫透干→被劫夺

| 状态 | 效果 | 应对策略 |
|:----|:-----|:---------|
| **比肩透干** | 被亲近之人劫财 | 合伙/借款/担保需极其谨慎 |
| **劫财透干** | 被竞争对手劫财 | 市场/职场竞争激烈，利润被分薄 |
| **比劫藏干** | 暗中被劫 | 不易察觉的损失，需警惕细节 |

**比劫透干流年断事法则：**
```
□ 比劫透干+财星在年/月 → 被长辈/上级/合伙人劫财
□ 比劫透干+财星在日/时 → 被配偶/子女/下属劫财
□ 比劫透干+原局有食伤 → 食伤可制比劫，减轻被劫程度
□ 比劫透干+官杀透干 → 官杀制比劫护财，劫财有化解
□ 比劫透干+无食伤无官杀 → 被劫无救，该年不宜投资/合伙
```

#### 13.6.3 正印透干→保守理财

| 状态 | 效果 | 财富行为 |
|:----|:-----|:---------|
| **正印透干** | 稳健保守、不愿冒险 | 储蓄型理财，偏向固定收益 |
| **偏印透干** | 谨慎精明、审查严密 | 投资重分析，但防因多疑错失机会 |

**印星透干流年对财富的影响：**

#### 13.7.1 原局亥未拱木→流年卯=爆发机制

**财富判断：**
```

  ✅ 木是财星 → 直接发财（如庚辛金日主：金克木为财）

  原局已有财星40分 + 流年三合局引化 ×15倍 = 财富爆发级
```

> **口诀**：亥未拱木卯年到，三合木局冲天高；是财是喜不用说，财富爆发在今朝。

#### 13.7.2 原局午未六合→流年影响

```
  - 日主土（戊己）：土为比劫 → 比劫夺财
```

**原局特征：** 八字中地支出现两个巳火（年支+月支，或月支+日支等）。

```
  - 双巳 = 巳火能量≥80分（每个巳40分）
  - 对抗原理：双巳合力 > 单亥冲击力
□ 只有双亥年（流年亥+大运亥 或 大运亥+流年亥）才可能冲开
  - 双亥 = 亥水能量≥80分
  - 双巳 vs 双亥 = 能量对等，冲力平衡但可破局
```

**财富判断：**
```
□ 巳火是日主的什么十神？
  - 癸水日主：巳中丙火=正财、戊土=正官、庚金=正印
    → 双巳抗冲 → 财星在年/月 → 祖上财富守住
    → 双亥年 → 财库被冲开 → 财富释放 ✅

  步骤2 — 双亥冲双巳 → 不论结果是冲开还是冲不动，该年对财富的影响极大
```

## §14 大富格局综合判断体系

### 14.1 大富五要素
真正的大富格局必须从五个维度综合判断，缺一不可：
1. **身强（40-60分）** — 能担财，身过强>60分反克财
2. **财旺（≥40分）** — 有财可得，财透干更好
3. **食神旺（≥40分）** — 有财根，生财有道
4. **禄神** — 十二长生临官位，身强得禄底气足
5. **财库** — 日/时柱有库能蓄财

### 14.2 禄神用法（09号视频原文）

- 日坐禄神：中年不缺钱财

> 八字身弱而没有印星，则禄神作用很大，此时禄神逢冲则忧。

## 15.2 重点规则详解

### 15.2.1 禄神四柱断法（wx_16 核心新增）

禄神在四柱不同位置的完整断语——现有体系仅提及"一禄胜千财"概念，缺此细化：

| 位置 | 名称 | 核心断语 |
|:----|:----|:---------|
| 年支 | — | 喜财官透干，透财则富 |
| 月支 | 建禄 | 性格务实、身体好、适合社会打拼磨练 |
| 日支 | 专禄 | 会享受奢华生活；需羊刃保护；被合=无禄；被冲=克配偶 |
| 时支 | 归禄 | 不喜官星，为后代勤奋；逢合=无禄；逢冲=克子女/寿终 |

**附加规则：**
- 禄冲财 = 忙碌求财
- 禄合财官 = 体力劳动者；制住财官=当官发大财
- 禄空亡 = 财库落空，劳而无功
- 夹禄：寅辰夹卯（乙禄）、卯巳夹辰（无对应天干禄）等顺联夹取

### 15.2.2 盲派非标准当财规则（wx_17 核心新增）

盲派体系下，八字无财或财弱时，多种十神可当财看：

| 当财类型 | 条件 | 细化说明 |
|:---------|:-----|:---------|
| **禄神当财** | 原局占禄 + 无食伤泄 / 八字无财 | 印生禄为条件，以禄取财=辛苦求财 |
| **伤食当财** | 八字无财星但有伤食 | 伤官=谋/经营之财；食神=思想/脑力之财 |
| **官杀当财** | 官杀多财少 / 官杀有制但制服不好 | 官统财或财统官=官杀当财富看；财富级别很高 |
| **比劫为财**（已有，补充深化） | 比劫旺盛且为喜用神 | 合作投资、共同创业、人脉资源变现 |

### 15.2.3 财库位置年限分化（wx_17 新增）

现有体系（§2.3）仅区分"日/时柱=自己的库"和"年/月=祖上的库"。盲派进一步细化：

- **库在年支** → 大库，祖业丰厚，影响一生
- **库在月支** → 中库，青年至中年蓄财
- **库在日支/时支** → 临时之库，近期或个人蓄财

### 15.2.4 特殊格局：外债型 & 空亡型（wx_17 新增）

| 格局 | 特征 | 断语 |
|:----|:-----|:-----|
| **外债型** | 印旺+官旺+日柱太弱+财不强 | 终身带着外债过日子 |
| **财星空亡型** | 财落空亡 | 一生财不强；不见劫财则外债少，一见劫财外债骤增 |

### 15.5.1 萝卜旺财法 详细说明

1. **准备材料**：白萝卜一个、红纸一张（剪1元硬币大小）、红布一块
2. **操作步骤**：
   - 白萝卜切去头部，挖一个洞
   - 红纸上用毛笔写"金银财宝"四字
   - 将红纸塞入萝卜洞中
   - 用红布缠好，复原萝卜
3. **悬挂要求**：
   - 挂在自家门口，持续12天
   - 每次进门时念"招财进宝"（12天内合计念满49遍即可）
   - 找与自己八字三合局的异性朋友，在辰时（7-9点）挂上
4. **收尾**：12天后取下，埋入院中或附近绿化带中

### 15.5.2 日进斗金阵 详细说明

1. **准备材料**：木斗一个、五角铜币100枚、大米、黄豆、小米、黑豆、高粱各适量、朱砂、黄纸
2. **操作步骤**：
   - 木斗中先放入100枚五角铜币
   - 依次铺放大米→黄豆→小米→黑豆→高粱（顺序不可乱）
   - 用朱砂在黄纸上写招财符，埋入米中不露出表面
3. **摆放要求**：
   - 放置在家居/办公室西北角
   - 选择戌日或亥日的戌时（19-21点）或亥时（21-23点）摆放
4. **功效**：进偏财

### 15.5.3 永动机催财 详细说明

1. **布置方法**：先布好财库（按§10财库布法），然后在财库上放置永动机
2. **永动机可选**：摇表器、流水摆件、招财猫、带电池的钟表等能持续运动的物品
3. **原理**：持续的运动产生能量，动生能量催动财库持续入库

## §16 合作伙伴规则（素材11原文逐字）

> **来源：** 素材11-八字财运分析与行业选择讲解（第65~213行原文逐字精读）
> **核心原理：** 身强者和身弱者的合作逻辑完全相反——身强以财官食为喜用（消耗能量），身弱以印比为喜用（补充能量）

### 16.1 核心原则（素材11行65-68）

```
身强之人 → 以财官食为喜用 → 消耗能量
身弱之人 → 以印比为喜用 → 补充能量
```

### 16.2 身强者的三种合作模式

#### 模式一：以财为喜用 → 与有钱的老板合作（素材11行69-105）

**原文：**「你有个好项目，你把它写成一个什么计划书……找这些有钱的人去谈。你不出钱，让他去出钱……叫银股，他出钱，叫银股（银子之意）；你不出钱，你也要占股份，你占的是身股（身体的身）」

**规则：**
```yaml
合作模式：自己出项目/技术/管理，有钱人出资金
股份分配：
  ✅ 出钱一方 = 银股（有查账权、分红权，无管理权）
  ✅ 自己一方 = 身股（负责项目运作，掌握管理权）
  ⚠️ 提前说清楚：出钱≠说了算，出银股者不能干预项目运作

关键要点（素材11行77-77原文）：
  □ 提前说清楚，你出钱，你可以查账
  □ 赚了钱你有分红权
  □ 但是不能干预事情的发展
  □ 整个项目的操作运作由你负责

适用场景：
  □ 有项目但没资金的人（找风投/天使投资）
  □ 有技术但没启动资金的人
  □ 想创业但不想自己投入资金的人
```

#### 模式二：以官杀为喜用 → 与领导/政府合作（素材11行109-145）

**原文：**「以官杀为喜用，代表了上司、领导、官府……你跟领导搞好关系，得到一些领导的资源，利用他的资源，你才好求财」

**规则：**
```yaml
合作模式：与领导/上司/官府搞好关系，利用其手中的权力资源

      ✅ 领导手里有资源、有权力，一句话可以带来很多便利
      ✅ 主动让出一些股份给领导
      ✅ 不卑不亢：越怕领导，领导越欺负你；越不搭理，领导觉得你不跟他一条心
      ✅ 就当他是朋友相处，关系搞好，不是阿谀奉承

      具体方法（素材11行121-131原文）：
      □ 领导手里有权利，稍微给你用点就是资源
      □ 江湖不是打打杀杀，江湖就是人情世故
      □ 有好事想着领导，他有好事也会想着你
```

#### 模式三：以食伤为喜用 → 与下属员工合作 或 学门技术（素材11行157-165）

**原文：**「食伤代表了下属员工还有晚辈……您把自己的生意交给他，让他去打理，股份制嘛」

**规则：**
```yaml
与下属合作模式：
  ✅ 员工兢兢业业、吃苦耐劳、知根知底、人品不错
  ✅ 把自己的生意交给他打理
  ✅ 股份合作：把股份分给下属员工
  ✅ 自己抽出时间开拓新事业

学技术模式：
  ✅ 食伤代表技能、手艺
  ✅ 学会一门技术，万事不求人
  ✅ 靠技术养家：泥瓦匠、木匠、电工、画画、写字、周易等
  ✅ "御自己"——自己管好自己，靠一门手艺走遍天下

适用场景：
  □ 不想跟人打交道的
  □ 不善于驭上、驭中、驭下的
  □ 喜欢独立做事的
```

### 16.3 身弱者的两种合作模式

## §17 什么时候破财（09号视频原文表 + 素材11行33~41确认）

> **来源：** 09号视频「看财运等级与合作伙伴」第0:06:54幻灯片 + 素材11行33~41（破财回顾部分）
> **素材11行33~41原文：** 「流年的时候，财星受到了冲击，把财给冲走了。食伤受冲，食伤是生财的……也是会影响到我们的收入。身强遇到比肩劫财……也是破财的这种信息。财星被合化成其他的能量……也是我们破财的信息……流年遇墓则破财」

### 17.1 破财五种条件（视频原文表）

八字看什么时候破财，共 **5种条件**，满足任一即当年有破财风险：

| # | 条件 | 原文依据 | 应对策略 |
|:-:|:----|:--------|:--------|
| 1 | **流年财星受冲** | 素材11行37「财星受到了冲击，把财给冲走了」 | 不投资、不借出大额资金、不签长期合同 |
| 2 | **流年食伤受冲** | 素材11行37「食伤受冲，食伤是生财的，食伤为财之根……财根给断了」 | 谨慎创业/扩张，食伤=财根，根断则财源枯 |
| 3 | **身强遇比肩劫财** | 素材11行37「身强遇到了比肩劫财，比劫是专门克财星的」 | 不与人合伙、不担保、不借钱出去 |
| 4 | **财星被合化成其他能量** | 素材11行37「财星被合化成其他的能量……财星被合走了，财要被别人抢走」 | 注意合同细节，防范合作方坑骗 |
| 5 | **流年遇墓破财** | 素材11行41「流年遇墓则破财，比劫墓和财财入墓」 | 该年不进大财、不置业、保守理财 |

### 17.2 详细说明

#### 条件①：流年财星受冲

```yaml
判定：天干/地支的财星被流年冲克

具体表现：
  □ 天干财星被流年天干冲 → 例如庚申日主（甲木为偏财），流年庚金冲甲木
  □ 地支财星被流年地支冲 → 例如日支子水为财，流年午火冲子水
  □ 藏干中的财星被流年冲 → 例如时支亥中甲木为财，流年巳冲亥

后果：
  □ 财星在天干被冲 → 收入减少，工资被扣
  □ 财星在地支被冲 → 投资亏损，资产缩水
  □ 财星在年/月柱被冲 → 事业/家庭方面破财
  □ 财星在日/时柱被冲 → 个人/子女方面破财

避坑：财星被冲之年，只进不出，不借钱不担保
```

#### 条件②：流年食伤受冲

```yaml
判定：食神/伤官被流年冲克

原理（素材11行37原文）：
  食伤是生财的，食伤为财之根
  食伤受冲 = 财根被断 = 收入来源切断

后果：
  □ 食神被冲 → 福气财断，靠运气赚钱的渠道关闭
  □ 伤官被冲 → 技能财断，靠专业能力赚钱受阻
  □ 食伤被合化 → 生财管道被改变

避坑：被冲之年不要依靠单一收入来源，准备备用方案
```

#### 条件③：身强遇比肩劫财

```yaml
判定：身强（>60分） + 流年遇比肩或劫财

原理（素材11行37原文）：
  「身强遇到了比肩劫财，多出来的能量无处安放，比劫是专门克财星的」

身强 vs 身弱区别（⚠️ 关键）：
  ✅ 身强遇比劫 → 被劫夺 → 破财
  ❌ 身弱遇比劫 → 帮身 → 反而可合作得财
  （详见§16.3 身强vs身弱铁律）

后果：
  □ 比肩 → 被亲近之人（朋友/亲人/同事）劫财
  □ 劫财 → 被竞争对手劫财
  □ 比劫齐来 → 双重被劫

避坑：身强者比劫年不投资不合伙，身弱者比劫年可合作
```

#### 条件④：财星被合化成为其他能量

```yaml
判定：八字中的财星被天干五合或地支六合/三合/三会合化成别的五行

原理（素材11行37原文）：
  「财星被合化成其他的能量……您身上的财被合走了，那就是您的财要被别人抢走」

天干五合（常见破财组合）：
  □ 甲己合土 → 甲木（对日主可能是财）被合化为土 → 财变忌神
  □ 乙庚合金 → 乙木被合化为金
  □ 丙辛合水 → 丙火被合化为水
  □ 丁壬合木 → 丁火被合化为木
  □ 戊癸合火 → 戊土被合化为火

地支合化（常见破财组合）：
  □ 寅亥合木 → 寅中火（财）被合化为木
  □ 午未合土 → 午中火（财）被合化为土
  □ 辰酉合金 → 辰中水（财）被合化为金

特例：合化出的五行如果是财星 → 反而发财！
  □ 比如原局无财，流年合化后生成了财 → 那一年财运突然提升
  □ 素材11第45行：「如果合化变成财呢？那年会突然财运变好」

避坑：判断合化后的新五行是不是财星 → 是则发财，否则破财
```

#### 条件⑤：流年遇墓破财

```yaml
判定：流年地支遇上辰戌丑未四墓库，且该字对应的五行能量<40分

原理（素材11行41原文）：
  「流年遇墓则破财，什么墓啊？比劫墓和财财入墓」
  「遇到了比劫墓和财财入墓是破财」

与§7.2流年遇库则发的区别（⚠️ 关键！）：
  ✅ 该五行能量≥40分 → 库 → 遇库则发（§7.2）
  ❌ 该五行能量<40分 → 墓 → 遇墓则破财（§17）

五种墓对应的破财（素材11行41原文）：
  □ 比劫墓 → 合作伙伴出问题，被劫财
  □ 财才墓 → 直接破财，投资失败
  □ （官杀墓 → 丢官罢职  ❌ 非破财）
  □ （印枭墓 → 荣誉/证书得不到  ❌ 非破财）
  □ （食伤墓 → 福气得不到  ❌ 非破财）

避坑：墓年要防守，等能量≥40分（库）才出击
```

### 18.1 蓄财双库 = 比劫墓库 + 财才墓库

素材11行365原文：「日柱或者时柱有库是啥意思？就是我们说的有财库或者比劫库」

| **食伤库** | 赚钱动力源：才华/技能/创意变现能力 | 第三层·动力层 |

口诀：蓄财双库比劫先，财才墓库最直接；食伤库是动力源，五大墓库都要看。

### 19.1 当财看的5种字（深化§15.2.2）

泉师兄体系下，代表财富的字按优先级排列：

| 优先级 | 类型 | 条件 | 细化规则 |
|:------|:-----|:-----|:---------|
| ① | **财库** | 辰戌丑未为财库（需查日主五行定库属性） | 有库=有蓄财基础，刑冲开库=爆发 |
| ② | **官杀库** | 官杀多+有库（尤其辰为水库/杀库） | 库中官杀=隐形的权力资源→可转化为财 |

| ④ | **禄** | 临官位（如丙见巳为禄） | 禄=现成之福/稳定收入；无食伤时禄当财看 |

**深化说明**（§15.2.2有基础概念，以下为补充）:

**官杀当财的完整条件**（补充§15.2.2）:
```
条件A: 原局财星弱或不见 + 官杀旺（≥2个）
条件B: 身强能担官杀（身弱则官杀为压力非财）
条件C: 官杀有制（食伤制杀）或官杀有化（印化杀）
  → A+B+C全满足 = 官杀=权=财，财富级别大幅提升
  → A+B满足但C不满足 = 官杀=管理责任，辛苦得财非大富
```

**禄当财的补充条件**:（补充§15.2.2 禄的用法）
```
条件A: 原局无财星或财星极弱
条件B: 有印生禄（印生禄=禄为活钱/稳定收入）
条件C: 无食伤泄禄（食伤泄禄=禄转化为才华/技能而非直接财）
  → 禄被冲 = 破财/调动/奔波求财
  → 禄被合 = 禄被他人分走
```

### 19.2 真假虚实判断（新增·§15未覆盖）

> **核心原则**: 财星必须在合适位置且有根，才为"真财"。虚浮无根之财=假财，看似有实无。

| 类型 | 特征 | 财富意义 |
|:-----|:-----|:---------|
| ✅ **真财（实财）** | 财星当令（月令为财）、有根（地支藏干强）、有食伤生 | 真正能得到的财富，稳定可靠 |
| ⚠️ **假财（虚财）** | 财星虚浮无根、不在月令、远隔日主、被合化 | 看似有机会但实际难到手 |
| ❌ **空亡之财** | 财落空亡 | 财运有就像没有，镜花水月 |

**真假财的实战判断流程**:
```yaml
① 天干财星是否在月令得令？（月令本气是否为财）
  ├─ 是 → 真财，+2级
  └─ 否 → 查地支是否有财星之根
② 地支财星是否透干？
  ├─ 透干 → 真财可用
  └─ 不透 → 虚财，需大运引出
③ 是否有食伤生财？
  ├─ 有 → 财源不断，真财
  └─ 无 → 财为死财，一次性的
④ 财星是否被合化？
  ├─ 被合化 → 财变为其他（被分/被转化）
  └─ 无合化 → 财为自己所有
⑤ 综合判定：
  真财 = ①②③ 满足至少两项
  假财 = ①②③ 满足少于两项
  空亡财 = ④财落空亡 ⇒ 见劫财则财散
```

### 19.3 体用分析（新增·盲派核心框架）

### 19.5 制死/治过概念（2026-07-13新增·泉师兄视频萃取）

> **来源**：泉师兄抖音视频「看八字的思路」
> **核心定义**：五行过旺时，不仅仅是克制另一个五行，而是把它"治死"了——彻底压制到无法发挥作用

#### 19.5.1 制死的判定

```yaml
一般克制 vs 制死：

一般克制（正常五行关系）：
  ├─ 火克金 → 金被限制但还能用
  ├─ 木克土 → 土被限制但还有功能
  └─ 结果是：被克制者功能减弱但仍存在

制死（极端情况）：
  ├─ 火太旺→把唯一的金克死了 → 金完全无法发挥作用
  ├─ 木太旺→把唯一的土克死了 → 土完全无法发挥作用
  └─ 结果是：被克制者功能完全丧失

制死的判定条件：
  ① 克者五行≥3个（太旺）
  ② 被克者只有1个（孤军）
  ③ 被克者无生助（无印/同类帮身）
  ④ 克者与被克者紧邻或同柱
  → 四项全满足 = 制死成立
```

#### 19.5.2 制死的应事

```yaml
制死什么十神，就应什么事：

| 被制死的十神 | 应事方向 | 具体表现 |
|:------------|:---------|:---------|
| **财星被制死** | 财富+婚姻 | 破财/短暂恋爱没有婚姻/经济犯罪 |
| **官星被制死** | 事业+健康 | 丢官/事业无成/重病 |
| **印星被制死** | 学业+后盾 | 学历低/无靠山 |
| **食伤被制死** | 才华+子女 | 无创造力/子女缘薄 |
| **比劫被制死** | 朋友+帮手 | 孤立无援/不善社交 |

口诀：制死什么就应什么，财死无财官死无官
      火多治金财死了，诈骗进去没老婆
      五行极端要看清，制死就是极端事
```

#### 19.5.3 制死与牢狱

```yaml
制死财星 + 阴阳失衡 → 🚨 牢狱命（特别是经济犯罪）

条件：
  ① 财星被制死（火多金死/土多水死等）
  ② 同时有阴阳失衡（群阴独阳/群阳独阴）
  ③ 大运/流年引动比劫

→ 三者全满足 → 因财富问题触犯法律

制死财星的年份分析：
  ├─ 比劫运/流年（身强比劫夺财）→ 容易进去
  ├─ 财星被冲的年份 → 经济和法律双重压力
  └─ 比劫+财星齐被引动的年份 → 最高危

口诀：制死财星加失衡，牢狱之灾躲不掉
      比劫一来抢着花，法律管你叫诈骗
      短暂恋爱没婚姻，财星死了全落空
```

## §20 枭神夺食独立判定（2026-07-13 新增·P0缺口）

> **来源**: 泉师兄视频+九龙道长原始理论交叉验证
> **前提**: 枭神夺食的前提是**八字必须有食神**！无食神则枭神以正印算。

### 20.1 枭神夺食判定流程

```yaml
Step 1 — 检查原局是否有食神
  ├─ 天干食神 → 最危险，枭神在天干可直接夺
  ├─ 地支藏干食神 → 次危险，需大运流年引出
  └─ 无食神 → 枭神按正印论，不构成枭神夺食

Step 2 — 检查枭神位置
  ├─ 枭神与食神紧邻（同柱/相邻） → 🚨 夺食成立
  ├─ 枭神与食神隔位 → ⚠️ 夺食效果减弱
  └─ 枭神远隔 → 不构成夺食

Step 3 — 检查大运流年触发
  ├─ 原局已有枭神夺食 → 大运流年加强枭神 → 应事
  ├─ 原局无但大运流年成 → 该年应事
  └─ 枭神+食神同时出现 → 当年夺食

Step 4 — 应事级别判定
  年轻人（<36岁）：枭神夺食 = 自杀/自残倾向（精神层面）
  老年人（>55岁）：枭神夺食 = 经络淤堵/偏瘫（身体层面）
```

|:--|:-----------|:-----------|
| **比劫自己** | 合作共赢：自己团队赚别人的钱 | 自引外患：自己的比劫护别人的财 |
| **比劫别人** | 虎口夺食：别人的比劫劫你的财（压力） | 外战外行：双方都不占优，财富争夺战 |

### 22.2 完整判定流程

```
Step 1 — 定财星归属
  财星在日/时柱 → 自己的财 ✅
  财星在年/月柱 → 祖上/家族的财（但可利用）
  财星透干且与日主隔位 → 能被看到但不易全得

Step 2 — 定比劫归属
  比自己劫 = 日主+同柱比劫 → 自己阵营
  别人比劫 = 年/月柱之比劫 → 外界竞争/合作方

Step 3 — 落入矩阵单元格，得出合作策略
  □ 财自己+比自己劫 → 合作赚钱，团队共赢
  □ 财别人+比自己劫 → 借祖上资源，但自己人可能内耗
  □ 财自己+别人比劫 → 守财防盗，竞争对手觊觎
  □ 财别人+别人比劫 → 纯竞争格局，不宜在此领域求财

Step 4 — 结合身强弱做最终判断
  身强 → 即使别人比劫多，也能扛住竞争
  身弱 → 别人比劫多时压力大，需印比帮身
```

**口诀**: 财星归属看宫位，日时为已年月祖；比劫归属看阵营，同柱是我年柱他；落入矩阵四象限，合作竞争各不同。

## §23 暗象/暗制理论（新增·盲派核心补充）

> **核心原理**: 五行旺到极致会产生"暗象"——不需要明面上的克制关系，旺极本身就等于克制对宫。

### 23.1 比劫旺到极致 → 暗制财

```
比劫（日主之根）过旺时：
  正常路径：比劫→克→财（明克）
  暗象路径：比劫旺到极致 → 不需要明克 → 暗地压制财星

暗制成立的判定条件：
  ① 比劫≥3个（旺极）
  ② 财星在原局只有1个或没有
  ③ 没有食伤来疏通（食伤可以化比劫生财，给比劫一个出口）
  → 三项全满足 = 暗制财成立
```

### 23.2 暗象的财富表现

| 暗制类型 | 表现形式 | 财富特征 |
|:---------|:---------|:---------|
| **比劫暗制财** | 比劫旺到极致→财不敢出现 | 看似无财，实则暗中有大财；财一露头就被劫，所以藏起来了 |
| **暗财** | 财被比劫压到暗处 | 表面没钱，实际暗地里有钱（隐形富豪） |
| **暗制→明变** | 大运流年引化时 | 若大运/流年帮财（生财/护财），暗财浮出水面→暴发 |

### 23.3 与现有体系的关系

> 此理论补充了§6财星围克和§17破财判断中的极端情况。当比劫太旺时，不是简单的"身强遇比劫→破财"，而是比劫暗制财→财在暗处不出现→"表面无财暗地有财"。

**口诀**: 比劫旺极暗制财，明面无财暗地来；食伤疏通给出口，暗财浮出变明财。

## §24 做功路线优先级（2026-07-14修正·泉师兄原文顺序）

> **核心原理**：八字「做功」是指命局能量实际产生效益的路径。做功路线有严格的优先级顺序。
> **原文依据**：「有禄有库，那我们就要去看禄与库，看看有没有印禄相随...有官则看怎么得到官，没官则看财，实在什么都没有才去看食伤」
> **顺序铁律**：**有好的，肯定不要差的**——禄神是最好最稳的财源，食伤当财是无奈之举

### 24.1 做功路线优先级（从最优到最次）

> **底层逻辑**：**合化 > 生 > 克泄耗**——哪条路最轻松就走哪条
> - 合化（合气生财）→ 最轻松，优先选
> - 生（食伤生财等）→ 中等
> - 克泄耗（食神制杀/比劫克财等）→ 最累，最后选

```
🥇 第一优先级：先看禄（印禄相随）
  → 禄是原局带的天然福气，最轻松
  → 原话：「有禄有库，先看禄与库」

🥈 第二优先级：无禄看库
  → 开库属于合化/冲刑范畴
  → 原话：「有库要看库」

🥉 第三优先级：看官杀
  → 官护财/官合财∈合化范畴
  → 原话：「有官则看怎么得到官」

🏅 第四优先级：看财星（标准财星）

🎯 第五优先级：实在无路才看食伤（最累）
```

### 24.1A 每一条路线内部也有合化>生>克泄耗

| 方式 | 费力 | 例子 |
|:----|:----:|:------|
| **合化** | 🥇最轻松 | 丁壬合(伤官合杀)、乙庚合(体合财) |
| **生** | 🥈中等 | 食伤生财、官生印 |
| **克泄耗** | 🥉最累 | 食神制杀、比劫克财 |

### 24.2 为什么禄神排第一？

禄神（临官/建禄/专禄/归禄）是最优的财源，因为：
- **禄是原局带的**，不是合化出来的（如庚禄在申，原局有申就是有禄）
- 天然自带、无需费力 → **「现成之福」**
- 印禄相随 = **有后盾+有稳定收入**
- 禄当财 = 八字无财星但大富大贵的秘密
- 「即使八字不见财星，但若能见禄神且禄神为用，一样能够荣华富贵」

### 24.3 为什么官杀优先于财星？

官杀 > 财星的逻辑：
```
官护财 = 有地位自然有财富伴随
官带财 = 职位本身带来收入
官当财 = 无财星时官杀可以当财富看
                        → 财富级别很高
```

而财星只是直接的钱——「有好的，肯定不要差的」，官带来的不仅是钱，还有地位和资源。

### 24.4 无官杀时的财富判断

跳过官杀 → 直接看财星：
```
财旺有根 + 身能担 → 直接求财型
财弱 + 食伤旺 → 食伤生财型（间接）
财弱 + 库在日/时 → 蓄财爆发型
```

### 24.5 食伤当财是最后的无奈之举

「食伤当财属于是没有办法的办法，原局但凡有财官，这个命局就是奔着财官使劲的。这也是人之常情——有好的不用差的。」

食伤当财的三大信号：
```
① 原局无财官（无奈之举）
② 食伤在家外（跟我没关系→可以当财看）
③ 食伤旺到极致（太多了，只能当财来治）
```

### 24.6 底层逻辑：合化 > 生 > 克泄耗（按难易程度）

> **来源**：泉师兄原文核心原则 + 老板校准
> **核心**：为什么做功路线按禄→库→官→财→食伤排序？因为底层是按**做事难易程度**排的。

```
🥇 合化（最容易）
  → 合气生财，不用费力
  → 禄神 = 合化出来的福气（现成之福）
  → 官杀合财 / 丁壬合 / 乙庚合 = 用合的方式获取
  → 原话：「合气生财，既然能够合作求财，何必搞得打打杀杀的呢？」

🥈 生（中等）
  → 需要主动去生，但不算太累
  → 食伤生财 = 用才华/技术换钱
  → 印禄相随 = 印生禄

🥉 克泄耗（最累）
  → 需要用力去克制或消耗
  → 食神制杀 = 把七杀制住了才能得财
  → 比劫制财 = 要跟别人抢
  → 原话：「克泄耗是比较累的」

实战口诀：
  合气生财最轻松，生来生去也还行
  克泄耗力最累人，实在没招才用克
```

### 24.7 综合优先级决策树

```
原局有禄？→ 印禄相随 = 最优 ✅
   ↓ 无禄
原局有库？→ 库能被开？→ 开库得财 ✅
   ↓ 无库
原局有官杀？→ 官护财 / 官当财 ✅
   ↓ 无官杀
原局有财星？→ 直接求财 ✅
   ↓ 无财星
食伤当财看（无奈之举）→ 有代价
```

## §25 食伤当财的代价与化解（新增·兼容补充）

> **核心**: 当八字无财或财弱、以食伤当财时（§15.2.2已有基础），食伤当财有代价，需配合化解。

### 25.1 食伤当财的代价

| 代价类型 | 表现 | 严重程度 |
|:---------|:-----|:---------|
| **身体消耗** | 食伤=体力/脑力输出，长期以食伤当财=体力脑力过度消耗 | ⚠️ 中 |
| **官非风险** | 伤官见官=破财官司，食伤当财者容易在法律边缘赚钱 | 🔴 高 |
| **印星破坏** | 食伤旺克官杀、泄日主，极旺时反伤自身根基 | ⚠️ 中 |
| **财不持久** | 食伤当财=当前才华变现，但缺乏财库蓄财，钱来钱去 | ⚠️ 中 |

### 25.2 +和食伤法（化解方法）

```
「+和食伤法」：在食伤当财的基础上，主动加入"和"的元素——
  食伤+财 = 才华变现转化为实际财富（比单纯食伤当财更稳定）
  食伤+官 = 食伤制杀→以才华得权→权再得财
  食伤+印 = 印制的食伤→有底气的才华→踏实赚钱

具体操作：
  ① 食伤当财者 → 主动培养财库思维
     - 开银行账户蓄财
     - 设置固定储蓄比例
     - 将"赚钱→花钱"循环改为"赚钱→储蓄→投资"
  
  ② 食伤+官杀 → 将才华转化为职位/管理权
     - 利用专业能力升职
     - 将个人能力转化为团队生产力
  
  ③ 食伤+印 → 将才华体系化
     - 系统学习、考证
     - 建立知识体系而非靠临时发挥
```

### 25.3 食伤当财的流年把握

```
发财窗口：
  □ 食伤流年 → 才能爆发，赚钱能力强
  □ 财星流年 → 食伤生财→才华变现
  □ 印星流年（有偏财制印）→ 善用贵人资源

风险窗口：
  □ 官杀流年 → 伤官见官，防官司
  □ 比劫流年（身弱） → 被劫财
  □ 印星流年（无偏财制印）→ 枭神夺食，才源被断
```

**口诀**: 食伤当财代价高，身体官非两头烧；+和食伤三法解，财官印合才稳牢。

## §27 财星喜忌前置检查（新增·兼容补充）

> **核心**: 财星是喜神还是忌神，决定了"见财"=发财还是破财。很多人以为财星流年一定发财，这是致命误区。

```
财星喜忌判定：

  ❌ 身弱（<40分）+ 财为忌神 → 财星流年=压力破财（见财不见财）

核心口诀：身强见财=发财，身弱见财=破财；不是所有财年都发财，喜忌决定一切。
```

### 27.2 忌神财流年的具体表现

| 财星状态 | 身弱见财 | 从旺见财 |
|:---------|:---------|:---------|
| **财星透干** | 机会摆在眼前但抓不住，反而成为烦恼 | 诱人的陷阱 |
| **财星藏干** | 暗中破财，不知不觉花钱如流水 | 看似便宜实际亏 |

| **财星生官杀** | 财生官杀克身→工作压力+破财 | 雪上加霜 |

### 27.3 前置检查流程（每次看流年财运前必做）

```
Step 1 — 定喜忌
  □ 身强（40-60分）→ 财为喜用 ✅
  □ 身弱（<40分）→ 财为忌神 ❌
  □ 从弱格 → 财为喜用 ✅
  □ 身过强（>60分）→ 财本身为喜用，但比劫夺财需防

Step 2 — 喜用结论
  ✅ 财为喜用 → 进入正常财星分析流程（§7/§13）
  ❌ 财为忌神 → 跳过「遇财发财」逻辑
    改用「身弱见财→防破财」框架
    重点看：是否有印/比帮身的年份

Step 3 — 组合判断
  □ 财为喜用+财星流年 → 财运上升 ✅
  □ 财为喜用+财星大运 → 十年发财期 ✅
  □ 财为忌神+财星流年 → 见财不见财，防破财 ❌
  □ 财为忌神+财星大运 → 十年压力期，等印比 ❌
```

### 27.4 与现有体系的关系

> 现有§13流年分析表已经列出身弱格局的「财星年→见财不见财」，但缺少**前置喜忌检查**的明确步骤。本§补充为每次流年分析前的强制检查——先定喜忌，再做判断。

**口诀**: 财年未必就发财，身强身弱分喜忌；喜神见财财上门，忌神见财财破门。

## §28 大运流年选择+压制做功路线（新增·盲派做功视角）

> **核心**: 大运流年的选择不仅看十神喜忌，还要看是否"选择"了命局的做功路线，或者是否"压制"了做功路线。

### 28.1 什么是"选择"做功路线

```
大运/流年选择了命局的做功路线 = 该运/年是命局能量产出的管道被激活

判断条件：
  ① 大运地支与原局做功路径的地支有冲/合/刑关系
  ② 大运干支十神与做功路线的十神一致或相生
  ③ 大运引动了做功的关键字（如做功的食伤/财/官）

示例：
  原局做功路线 = 食伤生财
  ├─ 食伤大运/流年 → 选择了做功路线 ✅（才华变现）
  └─ 印星大运/流年 → 压制作功路线 ❌（枭神夺食，断财源）
```

### 28.2 什么是"压制"做功路线

```
大运/流年压制了命局的做功路线 = 该运/年命局的能量管道被堵塞

判断条件：
  ① 大运地支冲/合/刑了做功的关键地支
  ② 大运干支十神克做功路线的十神
  ③ 大运引动了做功路线的忌神

常见压制模式：
  □ 食伤做功 → 被印星大运压制（枭神夺食）
  □ 财星做功 → 被比劫大运压制（比劫夺财）
  □ 官杀做功 → 被食伤大运压制（伤官见官）
  □ 库做功 → 大运不冲不刑（库门不开=白做功）
```

### 28.3 综合判断表

| 原局做功路线 | 选择该路线的大运/流年 | 压制该路线的大运/流年 | 财富效果 |
|:------------|:---------------------|:---------------------|:---------|
| **食伤生财** | 食伤/财星大运 | 印星大运 | 选=发财，压=财源断 |
| **财生官** | 财星/官杀大运 | 比劫/食伤大运 | 选=权财两得，压=破财丢官 |
| **官杀当财** | 官杀/印星大运 | 食伤大运 | 选=权即财，压=丢权 |
| **库开得财** | 冲库/刑库大运 | 无冲无刑大运 | 选=爆发，压=白做功 |
| **禄当财** | 禄神被生被合 | 禄神被冲被刑 | 选=稳定财，压=奔波破财 |

### 28.4 实战流程

```
Step 1 — 确定原局的做功路线（参考§24优先级顺序：禄→库→官→财→食伤）
  □ 有禄先看禄 → 印禄相随
  □ 无禄看库 → 开库得财
  □ 无库看官杀 → 官护财/官当财
  □ 无官看财 → 财星做功
  □ 无财看食伤/库/路 → 食伤/库做功

Step 2 — 检查大运是否选择了该路线
  □ 是 → 该大运是财富爆发期 ✅
  □ 否 → 进入Step 3

Step 3 — 检查大运是否压制了该路线
  □ 是 → 该大运是财富低谷期 ❌
  □ 否 → 该大运对做功中性，主要看流年

Step 4 — 在大运框架内看流年
  □ 大运选路线 + 流年也选 → 双倍爆发 💰💰
  □ 大运选路线 + 流年压制 → 波动但总体向上
  □ 大运压制 + 流年选路线 → 小窗口机会
  □ 大运压制 + 流年也压制 → 防守期
```

**口诀**: 做功路线先定好，大运选它就爆发；大运压它就低谷，流年叠加看强弱。

## §29 非财官库的开库连带效应（新增·盲派库理论补充）

> **核心**: 非财星/官杀库（印库、比劫库、食伤库）被开库时，也会连带影响财富。现有§10.2只涵盖了财库和官杀库的开库效果，本§补充其他库的连带效应。

### 29.1 非财官库的财富连带效应

```
印库（辰/戌/丑/未对应印星）被冲开：
  ├─ 直接影响：证书/合同/权力/文凭
  └─ 财富连带：印生比劫→比劫夺财（身强者）
                印帮身→身强担财（身弱者）
  → 对财富的影响是"间接的"而非直接的

比劫库被冲开：
  ├─ 直接影响：合作/团队/人脉
  └─ 财富连带：比劫库开→合作能量释放
    ├─ 身强：比劫出来夺财→防破财
    └─ 身弱：比劫出来帮身→有机会担财

食伤库被冲开：
  ├─ 直接影响：才华/创意/技能爆发
  └─ 财富连带：食伤生财→才华变现管道全开
  → 食伤库开的财富效果和财库被开差不多，只是"间接生财"而非"直接得财"
```

### 29.2 连带效应判定表

| 被开的库 | 直接效果 | 对财富的间接影响 | 财富方向 |
|:---------|:---------|:----------------|:---------|
| **印库** | 权力/证书/合同 | 印生比劫→间接影响财 | 间接（身弱好/身强差） |
| **比劫库** | 合作/团队/人脉 | 比劫制财/帮身担财 | 间接（身弱好/身强差） |
| **食伤库** | 才华/技能/创意 | 食伤生财→间接暴发 | 间接但接近直接 |
| **官杀库**（已有§10.2） | 权力/地位/管理 | 官护财→间接保财 | 间接（已有覆盖） |
| **财库**（已有§10.2） | 直接得财/进账 | 直接效果 | 直接 |

### 29.3 实战注意事项

```
① 任何库被开都有财富效应
   → 不仅仅是财库！所有五个墓库被冲刑都会影响财富

② 非财官库的财富效应是"连带的"不是"直接的"
   → 印库开 = 权力到手→通过权力得财（官护财）
   → 比劫库开 = 合作达成→通过合作得财
   → 食伤库开 = 才华爆发→变现得财
   
③ 连带效应的财富量级判断
   → 食伤库 ≈ 财库的70-80%（因为间接生财）
   → 比劫库 ≈ 财库的50-60%（看身强身弱）
   → 印库 ≈ 财库的30-40%（最间接）
```

### 29.4 与§10.2和§18的关系

> §10.2已覆盖财库和官杀库的开库效果，§18要求五大墓库全量分析但没展开非财官库的财富连带效应。本§补充其余三种库（印库/比劫库/食伤库）被开后的财富影响方向和量级，供§10.2参考使用。

**口诀**: 五库全开都影响财，不仅是财库一个；印库连权权生财，比劫库合合作财，食伤库开才华财。

### 30.2 核心规则：无禄有印，来禄发财

→ 印为后盾，禄为发力工具
→ 「无禄有印」= 有后盾但自己站不稳
→ 「来禄发财」= 申酉大运/流年到时，印+禄双气齐发

### 30.3 己亥大运的做功路线选择

己土偏印透干 → 己亥自合（印合食伤）+ 身弱见印（帮身）
→ 此大运选择「食伤当财看」路线 = 才华/技术变现
→ **自己给自己打工**（己亥自合=日时=家里做功）
→ 壬寅年：壬水发动合丁火+生寅木 = 打两份工
→ 巳年：巳亥冲驿马+水火相战+伤官见官 = 东奔西跑+口舌是非

### 30.5 核心框架：主要做功 vs 当下做功

> **来源**：泉师兄第19篇案例图2「当下做功与主要做功」
> **核心**：泉师兄的八字分析区分「一生主线」和「当前任务」两个层次

**主要做功（一生的主线）：**
指命主一生中最本质的做功方式，不随大运改变。
- 本案例：**食伤生财 + 伤官合杀**
- 食伤生财 = 靠技术/才华赚钱（本质是技术者）
- 伤官合杀 = 把压力(七杀)通过创意(壬水)转化为机会

**当下做功（当前大运的任务）：**
指当前大运引动的具体做功路线，可能与主要做功不同。
- 己亥大运：**己亥自合（印合食伤）**
- 印合食伤 = 印星介入控制食伤，不让其过度消耗
- 食伤当财看 = 才华变现但控制度，不求大只求稳

**实战应用：**
| 维度 | 主要做功 | 当下做功 |
|:----|:---------|:---------|
| 辛金日主 | 壬水伤官+丁火七杀=伤官合杀 | 己亥自合=印合食伤 |
| 财富方式 | 技术/创意变现（食伤生财） | 控制性变现（食伤当财看） |
| 心理状态 | 有才华有压力（伤官+七杀） | 有后盾但保守（印来控场） |
| 大运影响 | 不随大运变（一生主线） | 随大运变（当前任务） |

**口诀**：主要做功是一生线，食伤生财创业看；当下做功看大运，印合食伤控风险。

此案例中辛金身弱、食伤旺（壬水透干）、食伤生财做功，
完美验证§26「身弱食伤生财底buff」——「心经太弱了，地支的根在丑中离得太远了。」

**补入意义**：OCR文本「丁仁和」「己亥自合合足食伤止谢」→ 有了完整八字+图片标注后才看清五条做功路线同步推理——这是盲派实务分析与学院派逐十神分析最大的区别。泉师兄的案例价值不在八字数据，而在**多路线并行做功的分析方法**。
