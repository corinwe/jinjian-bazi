## §1 看发财的方法总结（四大方法）

### 一、日元的能量和财星的能量

完整六种状态的发财条件：

| 状态 | 发财条件 |
|:----|:---------|
| 身强财旺 | 遇大运流年，见食伤财才则发大财 |
| 身弱财旺 | 遇大运流年，见印比劫则发大财 |
| 身强财弱 | 遇大运流年，见食伤财才则发中财 |
| 身弱财弱 | 遇大运流年，见印比劫则发中财 |
| 四柱无财身又弱 | 不好发财，俗称和尚命 |
| 四柱无财身强 | 遇大运见财则有进账，但财不长久 |

### 二、流年遇库则发

墓库规则：
- 申子辰合水 → 辰为水的墓库
- 寅午戌合火 → 戌为火和土的墓库
- 巳酉丑合金 → 丑为金的墓库
- 亥卯未合木 → 未为木的墓库

⚠️ 戌的五行属性判定（老板2026-07-01校准）：戌本身是土，有丙/丁引化或寅午戌/卯戌合化→当火看（燥土）；无引化合化→当土看

比劫库和财才库跟自己发财有关系（年/月的库是别人的，日/时的库是自己的）

### 三、流年遇五鬼运财则发

五鬼运财 = 月令往后退五位
五鬼神煞：容易得偏财，同时易惹小人、是非、灾害。官府提防，交友小心，否则有刑讼官司、破财牢狱。

### 四、遇流年禄神则发

禄神查法：甲寅乙卯丙巳丁午戊巳己午庚申辛酉壬亥癸子
禄神四柱断语：年柱=祖上父母有钱，月支=青年有钱，日坐=中年不缺钱，时支=晚年富裕

## §3 怎样选择合作伙伴·天乙贵人

口诀：甲戊兼牛羊（丑未） 乙己鼠猴乡（子申） 丙丁猪鸡位（亥酉） 壬癸蛇兔藏（巳卯） 庚辛逢虎马（寅午）

用法：看对方八字任意地支是否有你的天乙贵人；互为贵人=合作融洽；单方是对方贵人=对方不感恩

## §4 身强者赚钱方法

身强→以财官食为喜用
- 财才：与有钱老板合作，自己出项目，对方出资金
- 官杀：与领导搞好关系，让出股份
- 食伤：与下属合作，或学门技术靠手艺养家

---

## §5 身弱者赚钱方法

身弱→以印比为喜用
- 比劫：与朋友/同事合作，沾别人的光得财
- 印枭：与师长/比自己强的人合作，用师者王

铁律：身强遇比劫=被人劫财；身弱遇比劫=可以合作得财
