# 财富文档：补文昌 + 流年分析 模板

> 编制时间：2026-07-01 | 来源：家主/主母/子源三人财富文档实战

## 一、补文昌检查流程

每次写财富判断/补财库文档前，强制检查：

```
Step 1: 查日主文昌
  甲巳乙午丙申丁酉戊申己酉庚亥辛子壬寅癸卯

Step 2: 检查八字有无该地支
  有 → 标注「原局自带文昌✅」
  无 → 需要补文昌 → 加入方案

Step 3: 无文昌者写入方案
  文昌位：文昌地支对应方位
  文昌物：文昌塔 + 四支毛笔
  时机：文昌地支年效果最强
  颜色：文昌地支五行对应色
```

## 二、补文昌在方案中的位置

```
第0步：调身（身强者降到40-60分）
   ↓
第1步：补文昌（文昌位放文昌塔+笔）
   ↓
第2步：补财库（建人工库/保险柜）
   ↓
第3步：开库（财星≥40分后冲库）
```

## 三、流年分析标准结构

### 文档标题
`## 四、大运窗口分析（含流年触发）`

### 3.1 大运表
含10年大运趋势，标注发财窗口（💰）

### 3.2 流年触发规则
按身强弱格局写：

| 流年五行 | 天干/地支 | 对财富的影响 |
|:--------|:----------|:------------|
| **💰财星年** | 日主克者 | 直接发财！ |
| **💰食伤年** | 生财者 | 赚钱管道打开 |
| ✅ 官杀年 | 克日主者 | 管理/自律加强 |
| ⚠️ 印星年 | 生日主者 | 防夺财 |
| ⚠️ 比劫年 | 同日主者 | 守成为上 |

### 3.3 流年+大运组合效应表
- 财星大运+财星流年 → 双倍发财
- 食伤大运+财星流年 → 食伤生财
- 比劫大运+比劫流年 → 守成防夺财

## 四、实战案例

### 家主（辛金身强64分）
- 文昌在子 → 原局亥（非子）→ ⚠️ 需确认是否自带
- 流年：喜火水木，忌土金
- 发财年：2034甲寅/2035乙卯双木财星

### 主母（庚金从弱50分）
- 文昌在亥 → 原局无亥 → ❌ 需补文昌
- 流年：喜火木水（全为喜用），忌土金
- 发财年：2031辛亥/2032壬子食伤年

### 子源（丙火身强55.6分）
- 文昌在申 → 原局无申 → ❌ 需补文昌
- 流年：喜水土金，忌火木
- 发财年：2029己酉/2040庚申财星年
