## 一、墓库基础

### 40分法则
```
能量≥40分 = 库（保险库/仓库） → 装好东西，被打开是好事
能量<40分 = 墓（坟墓/棺材）   → 装糟糕事物，被打开是坏事

判定范围：八字全局（本命局+大运+流年）该五行的总能量
```

### 四墓库对应
| 地支 | 对应三合 | 对应五行 | 对应十神 |
|:----|:---------|:--------|:---------|
| 辰 | 申子辰合水 | 水库 | 官杀库(丙丁火)、食伤库(庚辛金) |
| 戌 | 寅午戌合火 | 火库+土库 | 财库(壬癸水)、印库(戊己土) |
| 丑 | 巳酉丑合金 | 金库 | 官杀库(甲乙木)、比劫库(庚辛金) |
| 未 | 亥卯未合木 | 木库 | 印库(丙丁火)、财库(庚辛金) |

## 二、墓库开闭机制

### 开门条件
大运/流年对墓库地支进行刑冲迫害：
- 冲（力度最大）：辰戌冲、丑未冲
- 刑（有风险）：丑未戌三刑
- 害（轻微）：子未害、丑午害等
- 破（最轻微）：辰丑破等

### 库开=好事的分类
| 库类型 | 打开后的好事 |
|:-------|:------------|
| 比劫库开 | 合作生财、团队共赢 |
| 财库开 | 发财、进账、投资回报 |
| 官杀库开 | 升官、掌权、管理提升 |
| 印库开 | 得合同/证书/权力 |
| 食伤库开 | 得福气（获奖/旅游/福利） |

### 墓开=坏事的分类
| 墓类型 | 打开后的坏事 |
|:-------|:------------|
| 比劫墓开 | 破财、被人算计 |
| 财墓开 | 破财、投资失败 |
| 官杀墓开 | 丢官罢职、官司 |
| 印墓开 | 失去权力、合同被抢 |
| 食伤墓开 | 福气被夺、好事落空 |

## 三、宫位权重

```
日支/时支有库 = 自己的库 ✅ 最有价值
年支/月支有库 = 祖上/亲人的库

自坐财库（日支藏财且≥40分）：
→ 特别能存钱，钱进到口袋出不去
→ 可能赚不多但存得多
```

## 四、入墓诊断

### 财星入墓
八字全局无财，唯一财藏在地支墓中，且该五行能量<40分
→ 男命：与妻子缘分浅，财运不佳
→ 女命：与父亲缘分浅

### 官星入墓
八字全局无官杀，唯一官藏在地支墓中，且该五行能量<40分
→ 女命：婚姻不好，容易离婚
→ 男命：无官运

## 五、补财库实操法（三种）

### 法一：刑冲生肖催财法（效果最明显）
1. 查日主的财库是什么
2. 找能冲开该财库的生肖物品（辰→狗、戌→龙、丑→羊、未→牛）
3. 放在财库方位
4. 关键节点用该物品去"冲"
⚠️ 风险：可能影响家人，放独立空间最好

### 法二：柜门开关法（辅助）
在财库方位放小柜子，反复开合门

### 法三：平时补+关键时刻催
日常放财库吉祥物蓄能
关键节点用刑冲生肖主动冲开
放几天→撤走→再放→保持能量波动

## 六、时间线
- 最快12天见效（月亮周期）
- 常规100天对比提升
- 刑冲催财法：当天几小时可见效
