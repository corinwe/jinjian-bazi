# 官杀当财 · 合用vs制用区分 + 藏干优先级修正

> **来源**：2026-07-14 家主（庚申 癸未 辛亥 辛卯）实战校准——原将丁壬合(伤官合七杀)作为核心做功，但根据藏干同柱优先规则，原局丁壬合不成立。
> **致命教训**：混淆了制用结构(官当财看)和合用结构(官来合身=大格局)，且违反了藏干同柱优先规则。

## 一、制用结构 vs 合用结构——最核心的区分

```
制用结构（官杀当财看的条件）：
  关系：食伤去制官杀 = 对抗关系
  机制：食神制杀 / 伤官去官 / 比劫制官
  条件：官杀被制但不干净 → 官当财看 ✅
  示例：戊土日主 + 甲木七杀 + 丙火食神 → 食神制杀且制不净 → 七杀当财看

合用结构（官合身=大格局，不当财看）：
  关系：官杀合日主或日主合官杀 = 合作关系
  机制：官来合身 / 日主合官
  条件：官主动来找日主 → 大格局信号，不需要"当财看"
  示例：丙火日主 + 双癸水正官 → 官来合身 → 权力主动来找人（非财）

⚠️ 混淆两者导致：
  ❌ 把合用结构当制用结构分析 → 低估格局，误判权力为财
  ✅ 正确：合用结构中官=权力/社会地位，不是直接财富
```

## 二、藏干优先级对官杀当财的影响

```
原局藏干关系判定：

✅ 同柱藏干自动作用（第1优先）：
  - 申中庚金生壬水（比劫生食伤）→ 原局成立
  - 申中壬水泄庚金（伤官泄劫财）→ 原局成立
  - 未中丁火内生己土（七杀生偏印）→ 原局成立

❌ 不同柱藏干需大运引动（第3优先）：
  - 申中壬水 + 未中丁火（丁壬合）→ 原局不自动触发
  - 亥中壬水 + 未中丁火（丁壬合）→ 邻柱月+日，但有争议
  - 大运引动时才激活：如丁亥大运(天干丁+地支亥=同柱丁壬合)

对官杀当财判断的影响：
  ❌ 原局丁壬合→七杀被制→化为权（错误：原局不成立）
  ✅ 丁亥大运(31-40)：大运天干丁火+大运地支亥中壬水=同柱丁壬合
     → 此运官杀当财成立，十年效果期，运过即止
```

## 三、官杀当财完整判定流程

```
Step 1 — 判断是制用还是合用结构
  □ 官杀被食伤制？→ 制用结构 → 进Step 2
  □ 官杀合日主/日主合官杀？→ 合用结构 → STOP，不当财看

Step 2 — 藏干关系确认
  □ 涉及的藏干是同柱还是不同柱？
  □ 同柱 → 原局成立 ✅
  □ 不同柱 → 需要大运/流年引动

Step 3 — 引动判断
  □ 大运/流年有触发字 → 该运/年官杀当财成立
  □ 无触发 → 原局中不成立，不能写

常见错误模式：
  ❌ 直接写"原局丁壬合→伤官合七杀→化为权"
  ✅ 正确："丁亥大运(31-40)大运天干丁+地支亥中壬=同柱引动丁壬合→七杀被制化为权"
```
