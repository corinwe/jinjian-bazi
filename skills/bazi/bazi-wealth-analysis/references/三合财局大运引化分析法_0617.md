# 三合财局/财库大运引化分析法（2026-06-17沉淀）

> 本方法论用于评估八字中财局在大运流年引化下的爆发力，作为财富天花板评估的重要补充维度。

---

## 核心概念

三合/三会财局的财富评估不能只看原局静态分数，必须考虑**大运流年引化后的爆发倍数**。

| 状态 | 倍数 | 效果 |
|:----|:----:|:------|
| 未引化（被动状态） | 7倍 | 财局存在但未激活，财富稳定但不出彩 |
| 被引化（天干透干/大运合化） | **15倍** | 财局激活，财富集中爆发 |
| 整运引化（大运直接引化） | 持续15倍 | 连续10年爆发期，天花板大幅提高 |

## 检查步骤

### Step 1：检查是否有三合/三会财局
```
原局中是否集齐三合/三会财局的三个地支？
例：家主-亥卯未三合木局（财局）
例：无财局则跳过
```

### Step 2：检查引化条件
```
条件A：天干透干引化（甲乙木透干引化木局）
条件B：大运地支合化引化（庚寅运寅亥合木引化木局）
条件C：流年引化（甲寅/乙卯年引化木局）
```

### Step 3：评估爆发量级
```
日常状态：7倍 → 财星分×7 = 等效能量
引化爆发：15倍 → 财星分×15 = 等效能量
整运引化：连续10年15倍 → 集中爆发，可达亿级
```

## 实战案例

### 案例1：家主庚申癸未辛亥辛卯
```
三合局：亥卯未三合木局（日亥+月未+时卯）
身强64分：能担财
引化条件：
  - 己丑运(57-67)：丑未冲，木局能量释放
  - 庚寅运(67-77)：寅亥合木，整运10年15倍引化
财富天花板：从8.5→9.0（几千万~亿级）
类型：爆发型（平时中上，庚寅运集中爆发）
```

### 案例2：立辛卯癸巳甲戌庚午
```
财库：日支戌土偏财库（戊偏财本气）
引化条件：戊子运(44-54)戊土偏财透干引化财库
财富天花板：9.5（几千万~亿级）
类型：高蓄水型（有财库，每年都能存）
```

## 与财富天花板的关系

| 类型 | 特点 | 范例 | 天花板评估 |
|:----|:-----|:-----|:----------|
| 高蓄水型 | 有财库，每年积累 | 立(43.2+戌库) | 稳定9.5 |
| 爆发型 | 三合局+大运引化 | 家主(31.2+三合局) | 爆发9.0 |
| 从弱型 | 从弱格+财星到位 | 主母(16+甲寅运) | 爆发9.0 |
| 微型财库型 | 巳火藏正财+大运引化 | 琼(16.8+巳库) | 稳进6.5 |
