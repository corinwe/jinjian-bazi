# 盲派象法（取象/大象）方法论

> **来源**：泉师兄19篇文案精读
> **编制**：2026-07-14 补入

## 什么是象法？

象法（取象/大象）是盲派的核心方法论之一——不只看十神生克的技术关系，还要读每个字的**形象/意象**，将抽象的天干地支映射到现实生活的具体场景。

## 常见取象表

| 字 | 象 | 应用场景 |
|:--|:---|:---------|
| 壬水 | 大江大河之水 | 力量大但难以约束 |
| 癸水 | 雨水、涓涓细流 | 力量小但可精准克制 |
| 己土 | 田园土 | 温和包容，能约束细流 |
| 戊土 | 高山之土 | 强大有力，能约束大河 |
| 寅木 | 笔杆子/大树 | 文化/写作/办公行业 |
| 卯木 | 花草藤蔓 | 依赖、需要扶持 |
| 巳火 | 炉火/阳光 | 文化/互联网/能源 |
| 午火 | 烈火 | 快速/激烈/爆发 |
| 申金 | 刀剑/传送 | 交通/物流/军警 |
| 酉金 | 珠宝/钟表 | 精密/工艺/金融 |
| 亥水 | 江河湖海 | 流动/贸易/运输 |
| 子水 | 暗流 | 智慧/隐秘/谋略 |

## 十神象意

| 十神 | 象 | 应用 |
|:----|:---|:-----|
| 食神 | 吃喝玩乐、子女、技术 | 福气、才华 |
| 伤官 | 叛逆、口舌、官司 | 争议、创意 |
| 正官 | 领导、规则、约束 | 管理、地位 |
| 七杀 | 压力、挑战、暴力 | 征服、突破 |
| 正印 | 母亲、学历、证书 | 庇护、靠山 |
| 偏印 | 后妈、偏门、玄学 | 冷门技能 |
| 正财 | 工资、妻子、稳定收入 | 脚踏实地 |
| 偏财 | 生意、情人、投资 | 大进大出 |
| 比肩 | 兄弟、朋友、竞争者 | 同行、同伴 |
| 劫财 | 姐妹、合伙人、小人 | 争夺、分担 |

## 使用时机

1. 先用十神生克、做功路线做逻辑分析
2. 再用象法做最后的**具体化判断**（断行业、断事件性质、断应事细象）
3. 原文：「大家可以从象的角度去读取任意年」
