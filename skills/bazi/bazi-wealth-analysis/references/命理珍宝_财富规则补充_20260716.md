# 命理珍宝·财富规则补充（2026-07-16）

## 以禄当财（第4期）

**何时用：** 财星、官杀星、伤食星都无法作用神时，八字又有禄，就可以禄当财，取禄为用。当财、官、食、伤有用时，不可用。

**禄怕什么：**
- 最怕见劫财（分禄之意）
- 怕见财星（口诀：禄神怕财）  
- 最怕复吟（禄神重抑为大凶）

## 伤官去官格（第5期）

以官为忌神，用伤官去官而得官。**不论日主衰旺。**「其实许多八字都不管身弱身强，看身弱身强只是初学者的一个拐杖。」

## 比动去财格（第24期）

以财为忌神，用比动去财也能得财。四种情形：
1. 财多身弱+财为忌+逢比动去财
2. 官杀多身弱+财星虚透反比动克合→主大富
3. 财作忌神被比动制→主富有
4. 比劫为"主"财为"客"→动取他人之财

## 干支虚实（第32期）

有根者实，无根者虚。
- **虚忌神**（天干无根的忌神）→ 吉
- **实用神**（地支有根的用神）→ 吉
- **实忌神**（地支有根的忌神）→ 凶
- **虚用神**（天干无根的用神）→ 凶

## 忌神三制（第38期）+ 贼神（第34期）

**三种制法：**
1. **旺制**（弱忌神→彻底制服，使其为贼神）
2. **平衡制**（略偏旺→相制衡，化敌为友）
3. **弱制**（过强→周旋）

**贼神理论：** 被强力制服的忌神称为贼神。
- **客贼神**：忌神被制（如警察所中的贼）→ 大富贵命
- **主贼神**：喜神被制（自己成了贼）→ 凶恶残疾命

**财富应用：** 客贼神的命行忌神运反而吉——「警察（喜神）等坏蛋（忌神）出现时才抓得住」。这是命理中「冲旺忌神反吉」的深层理论依据。

## 财富分析的世应（第46期）

以日主为世、年干为应——年柱可以代表祖上、外戚、妻（夫）家人或外人。
- 财在年/月（世位）= 祖上/家族的财
- 财在日/时（应位）= 自己的财

## 去印得权（第49期）

印为忌，受财制 = 去印得官 → 财富通过权力而非直接求财获得。
无财星时以本事/能力代财看，权力本身就是财富。
