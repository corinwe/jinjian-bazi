# 双人合参·商业选址与起名工作流

> **来源：** 2026-07-03 老板&老板娘 OPC公司选址实战案例
> **原理：** 综合两人八字喜用神 → 匹配方位五行 → 匹配行业五行 → 起名补运
> **适用场景：** 合伙人/夫妻共同创业时的选址+起名

---

## 核心步骤

### Step 1 — 列出两人八字喜用神

| 人物 | 日主 | 身强弱 | 喜用（排序） | 忌神 |
|:----|:----:|:------:|:-----------|:-----|
| 甲 | X | X | ①>②>③ | X |
| 乙 | Y | Y | ①>②>③ | Y |

**关键：** 找两人的交集喜用。全重合的五行是最强方向。

### Step 2 — 匹配方位五行

| 方位 | 后天卦 | 五行 | 喜用匹配 |
|:----|:-----:|:----:|:--------|
| 东 | 震 | 木 | |
| 东南 | 巽 | **木** | |
| 南 | 离 | 火 | |
| 西南 | 坤 | 土 | |
| 西 | 兑 | 金 | |
| 西北 | 乾 | **金** | |
| 北 | 坎 | 水 | |
| 东北 | 艮 | 土 | |

**规则：** 两人都喜的方位 ✓；一人喜一人忌的方位 ⚠️（需结合谁主导）；两人都忌的方位 ❌

### Step 3 — 匹配行业五行

| 业务板块 | 五行 | 对应十神 |
|:--------|:----|:---------|
| 物流/运输 | 水（流动） | 食伤（智慧/流通） |
| 智能体/AI | 火（电子/科技） | 官杀（系统/管理） |
| 金融 | 金 | 比劫 |
| 教育/文化 | 木 | 印 |
| 地产/建筑 | 土 | 财 |

**规则：** 行业五行最好是两人的共同喜用；如果行业五行是某人的忌神，需要大运流年或名字来补。

### Step 4 — 公司名起名

按优先级选择名字五行组合：

| 优先级 | 名字五行 | 适用场景 |
|:-----:|:---------|:---------|
| 🥇 | **两人共同喜用** | 最理想，全部受益 |
| 🥈 | 共同喜用② + ③ | 次优 |
| 🥉 | 补行业五行 | 行业需求强于个人需求时 |

**字形五行检查：** 每个字的部首/含义五行也需匹配整体方向。

### Step 5 — 综合排序

```
最优选址 = 方位五行 + 行业五行 + 名字五行 = 三人合力
最差选址 = 任何一项落忌神
```

## 实战案例（老板&老板娘·OPC物流智能体公司）

### 八字喜用交集

| 人物 | 喜用 | 共同交集 |
|:----|:-----|:---------|
| 老板（辛金身强64） | 水 > 火 > 木 | ✅ 木 ✅ 火 |
| 老板娘（庚金从弱） | 火 > 木 > 水 | ✅ 火 ✅ 木 |

**共同最喜：🔥火（官杀/管理/科技）> 🟢木（财星/财富）**

### 选址比对

| 位置 | 方位 | 五行 | 老板 | 老板娘 | 结论 |
|:----|:----|:----:|:----:|:------:|:----:|
| 青浦 | 西北 | 金 | ❌忌神 | ❌忌神 | **不推荐** |
| **临港** | **东南** | **木** | ✅**财星** | ✅**财星** | **✅强烈推荐** |

### 行业匹配

| 物流（水） | 智能体（火） | 赚钱（木） |
|:---------:|:-----------:|:---------:|
| ✅ 老板最喜 | ✅ 两人共同喜 | ✅ 两人共同喜 |

### 推荐名字

| 排名 | 名字 | 五行 | 理由 |
|:---:|:----|:----|:------|
| 🥇 | **炜枫智能** | 火+木 | 两人共同最喜用 |
| 🥈 | 煜森物流 | 火+木 | 同上 |
| 🥉 | 澜鼎智能 | 水+火 | 补行业+喜用 |
