# 财星受围克与时代修正 · 父亲/祖母实战校准案例

> 编制时间：2026年06月23日
> 触发场景：老板反馈父亲和祖母的财富断事不准

## 发现的问题

### 问题1：父亲财星计算错误
八字：己丑 癸酉 癸亥 戊午 | 日主癸水
午火藏干：丁(本气)·己(中气)
- **丁火** = 癸水克丁火 = 偏财 ✅ 12分
- **己土** = 己土克癸水 = 七杀 ❌ 之前误算为偏财+7.2分
- **修正**：12分（不是19.2分），财富等级从中富下调为小富上段

### 问题2：静态标签 vs 动态条件
之前用九龙道长五级制作静态终身判断（身强财弱→中富），忽略了：
- 财星被全局围克的兑现率折扣
- 时代背景的天花板限制
- 发财是"大运触发"不是"终身恒定"

## 五层动态判断法（从此案例导出）

### 第1层·原局定调
看身强身弱+财星旺弱→定基础状态（6种）

### 第2层·赚钱辛苦度
看食伤（财根）情况+日主与财星的关系

### 第3层·财星围克
看财星所在位置被多少字克制→定兑现率折扣
- 0-1个克制：全额兑现
- 2-3个克制：围克，打折兑现（30-50%）
- ≥4个克制：重围，大运救才兑现

### 第4层·财库
看日时柱有无辰戌丑未→定蓄财能力
- 有库：能存住钱
- 无库：财来财去，需主动蓄财

### 第5层·大运窗口
看大运是否补财/补食伤→定发财时机
- 遇财星大运：直接补财
- 遇食伤大运：食伤生财
- 无补财大运：辛苦钱维持

### 综合结论 = 5层叠加

## 父亲的具体应用
- 身强66.4 + 财极弱12分 → 底子好但财太少
- 无食伤透干 → 财根不稳，劳碌型
- 午火被亥酉丑三重围克 → 兑现率约30%
- 无财库 → 存不住
- 仅庚午运（27-36）补火 → 唯一发财窗口
- **综合：小富上段（百万级），财富集中在庚午运十年**

## 祖母的具体应用
- 身强73.2 + 财弱24分 → 身强财弱
- 无食伤透干 → 辛苦型
- 巳午被辰申戊四重围克 → 兑现率约30%
- 无财库
- 仅丙午运（1965-1975）补火 → 唯一发财窗口
- **综合：小富上段（百万级），财富集中在丙午运十年**

## 时代天花板修正
九龙道长五级定量（中富=几千万）以当代社会为背景。
对1949-1959年出生者需×0.3-0.5折扣。
