# 财富五层动态判断法（九龙道长原始体系）

> 编制人：金鉴真人
> 编制时间：2026年06月23日
> 来源：九龙道长素材11+17逐字精读，父亲/祖母实战校准触发

## 核心原则

财富判断不是「静态等级标签」（中富/大富），而是「动态条件判断」。

九龙道长核心公式：**发财 = 身强(40~60) + 财旺(≥40)**
两条同时满足=发财；任一条不满足=等大运流年补充。

## 五层法逐层详解

### 第1层：原局定调（六种状态）
看身强身弱 + 财星旺弱 → 定大方向

| 状态 | 财能量 | 正确断语 |
|:----|:-----:|:---------|
| 身强财旺 | ≥40分 | 天生有钱人，不挑大运也能发 |
| 身强财弱 | <40分 | 底子好但财少，等财星/食伤大运才发中财 |
| 身弱财旺 | ≥40分 | 财在眼前抓不住，等印比帮身才变现（最辛苦） |
| 身弱财弱 | <40分 | 辛苦小钱，遇印比大运才发中财 |
| 无财身弱 | 0分 | 和尚命，难发财 |
| 从弱格 | ≥40分 | 特殊格局，运对方向就爆发 |

### 第2层：赚钱辛苦度（食伤定财根）
- 有食神生财 → 福气生财，相对轻松
- 有伤官生财 → 技能生财，劳碌拼搏
- 无食伤 → 财根不稳，靠体力/时间换钱

### 第3层：财星围克（兑现率）
不是看财星分数高低，而是看财星被全局围克的程度：

- 0-1个克制 → 无围克 → 全额兑现
- 2-3个克制 → 围克 → 打折兑现（30-50%）
- ≥4个克制 → 重围 → 大运救才兑现

口诀：财星围克要看清，一克两克还正常；三克四克围城局，兑现打折要算清。

### 第4层：财库（蓄财能力）
- 日/时柱有财库 → 能存钱，财富积累
- 日/时柱无财库 → 财来财去，需主动蓄财
- 年支/月令有库 → 与自身发财关系不大（祖上/家族）

### 第5层：大运窗口（发财时机）
列出每步大运是否补财：
- 财星大运 → 直接补财，发中财
- 食伤大运 → 食伤生财，发中财
- 印比大运（身弱者）→ 帮身后能担财
- 无补财大运 → 不发财，辛苦钱维持

## 综合结论格式

「理论等级{五级定位}，但{围克/时代/大运}情况下，实际财富在{等级}范围内，集中在{特定大运年份}兑现」

## 时代天花板修正（父母辈专用）

- 当代（1980后出生）：×1.0 标准兑现
- 改革开放代（1960-1979）：×0.6-0.8
- 计划经济代（1949-1959）：×0.3-0.5

## 补财库总则

- 无财库者：开户补库（财库方位银行定向储蓄）+ 合作补库（与日时柱带辰戌丑未者合作）
- 有财库被堵者：刑冲生肖冲开（辰→狗、戌→龙、丑→羊、未→牛）
- 身弱财旺者：先补身（印比）再开库，不可身弱未补时强开财库
- 身过强>60分者：先降火气（补水）再谈聚财
- 从弱格：补消耗五行（食伤/财/官杀），忌补印比
