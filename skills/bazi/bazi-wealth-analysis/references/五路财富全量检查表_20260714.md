## ① 禄（稳定收益/工资）

### 检查清单
```
□ 日主禄神查法（甲寅乙卯丙巳丁午戊巳己午庚申辛酉壬亥癸子）
□ 原局是否有禄？（地支找到）  → 有/无
□ 禄在什么位置？（年/月/日/时）→ 位置
□ 禄被冲/合/刑/害？          → 无/有（具体）
□ 大运是否有禄到？           → 有/无（哪个大运）
□ 原局无禄→是否有印生禄？     → 有/无
□ 盲派：八字无财时禄可当财看 → 适用/不适用
```

### 分析要点
```
- 禄=稳定收入、工资、俸禄
- 禄在年：祖上有钱，少年家境好
- 禄在月：青年收入稳
- 禄在日：中年不缺钱（专禄会享受）
- 禄在时：晚年富裕（归禄为后代）
- 禄被冲=工作变动/破财
- 禄被合=禄被分走
- 禄空亡=财库落空，劳而无功
- 盲派：禄当财（原局无财时）
```

## ② 库（财库开闭）

### 检查清单
```
□ 日主财库查表（甲乙-戌/丙丁-丑/戊己-辰/庚辛-未/壬癸-戌）
□ 原局有财库吗？（辰戌丑未）→ 有/无（哪个地支）
□ 财库在什么位置？（年/月/日/时）→ 位置
□ 日/时柱有库=自己的库 ✅
□ 年/月柱有库=祖上的库（财富关联弱）
□ 财库被冲/合/刑？          → 未开/已开（哪个大运/流年开）
□ 财库对应的五行能量≥40分？   → 是=库/否=墓
□ 非财官库的开库连带效应（§29）
```

### 分析要点
```
- 库=保险库（≥40分），墓=坟墓（<40分）
- 库冲开=财富释放，墓冲开=破财
- 冲开方式：六冲>三刑>六害>六破
- 盲派优先级：库做功排在禄之后（第2优先级）
- 非财库开库也有财富连带效应（§29）
```

## ③ 官杀当财（权力变现）

### 检查清单
```
□ 原局有官杀吗？（正官/七杀）→ 有/无（几个）
□ 官杀是制用结构还是合用结构？→ 制用/合用
  制用=食伤去制官杀（对抗关系）→ 制不干净=官当财看
  合用=官杀合日主/日主合官杀（合作关系）→ 官来合身=大格局
□ 身强身弱？                 → 身强/身弱
  身强+官杀有制=官当财看 ✅
  身弱+官杀=压力 ❌ 不是财
□ 官杀无制或制不干净？       → 是=当财看/否=权力
□ 官杀库是否被开？           → 开=官当财/未开
```

### 分析要点
```
- 盲派核心：官杀有制但制服不好=官当财富看，财富级别很高
- 官护财=有地位自然有财富伴随
- 官带财=职位本身带来收入
- 官杀制比劫=官护财
- ⚠️ 制用vs合用必须区分，不能混淆
```

## ④ 财星（直接财富）

### 检查清单
```
□ 财星五行确认（日主克者为财）
□ 原局有正财/偏财？         → 有/无（哪个、位置）
□ 财在年/月/日/时？          → 位置
□ 财有根吗？（地支藏干）      → 有根/虚透
□ 财被比劫争夺？             → 有/无（哪些比劫）
□ 财被合化？                 → 有/无（合化成什么）
□ 财星强弱（≥40分/＜40分）   → 财旺/财弱
□ 财星喜忌（身强=喜/身弱=忌）→ 喜用/忌神
```

### 分析要点
```
- 财星=直接的钱
- 财在日/时=自己能掌控，在年/月=祖上/外部
- 财有根=真财，虚浮无根=假财
- 比劫夺财=财富需与人分享
- 身强财旺=发财格，身弱财旺=富屋贫人
- 三合财局=财富体系化
```

## ⑤ 食伤生财（才华变现）

### 检查清单
```
□ 原局有食神/伤官？         → 有/无（哪个、位置）
□ 食伤生财链条完整？         → 完整/断裂
□ 身强身弱？                 → 
  身强+食伤=生财有源 ✅
  身弱+食伤=泄身过度 ⚠️
□ 枭神夺食风险？             → 有（哪个大运/流年）/无
□ 食伤当财（原局无财时）     → 适用/不适用
```

### 分析要点
```
- 食伤=财根，无食伤=财根不稳
- 食神生财=福气生财（轻松）
- 伤官生财=技能生财（劳碌）
- 身弱食伤生财有底buff（焦虑/内耗）
- 食伤在盲派中排第5优先级（最后）
```
