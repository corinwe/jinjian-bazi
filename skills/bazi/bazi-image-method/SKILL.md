# 象法体系 · 金鉴真人

> **定义**：象法即**象形法/取象法**——把天干地支当成有形状有体量的东西，看它们放在一起像一幅什么画。
> **来源**：泉师兄抖音系列（多篇提及「用象法看」「从象的角度去读取」）
> **与盲派的关系**：并列独立体系，实务中交叉引用。盲派定做功路线，象法定具体行业/事象。

## 第五章：综合案例——象法画图

### 案例① 丁丑 壬寅 辛卯 己亥

画出来：
- 丁火七杀 = 一盏灯
- 壬水伤官 = 一条河
- 辛金日主 = 一颗小珠子
- 己土偏印 = 一块田地
- 寅卯木财 = 花草树木

**整幅画**：一盏灯照着一条河，河边有块田地，田边长着花草，一颗小珠在泥里藏着。
→ 灯照河=丁壬合=伤官合杀，化解压力
→ 河边+花草=壬寅+亥卯=食伤生财，靠水木行业赚钱
→ 珠子藏田=辛金身弱需印帮
→ **结论**：搞文化创意（水木）的自由职业者，压力大但能化解

### 案例② 丙子 丁酉 庚戌 乙酉

画出来：
- 丙火七杀 = 一把大火
- 丁火正官 = 一盏官灯
- 庚金日主 = 一把大刀
- 双酉金 = 两把斧头（阳刃）
- 乙木财 = 一小捆柴
- 子水伤官 = 一盆水
- 戌为火库 = 一个火炉

**整幅画**：两把大斧头旁边放着一把大刀，一捆柴扔在旁边，一把大火烧着，火炉里火焰熊熊，旁边放着一盆水。
→ 大火烧斧头=丙丁官杀制双酉阳刃=阳刃驾杀→管理得权
→ 大刀绑柴=乙庚合=日主合财→稳定收入
→ 斧头蘸水=酉生子=食伤泄秀→有才华
→ **结论**：靠管理/权力吃饭的人（阳刃驾杀），有魄力有执行力，收入稳定

## 第六章：象法进阶（泉师兄原文关键词）

**象法不是单独使用的，而是与盲派做功路线结合使用：**
1. 先用盲派找做功路线（禄→库→官→财→食伤）
2. 再用象法把路线"画出来"——具体做什么行业/遇到什么事
3. 象法验证盲派结论是否合理

**泉师兄原文中的象法应用：**
- 「壬水是大江大河之水，己土是田园土」→ 形象对比决定力量对比
- 「癸水是雨水涓涓细流」→ 与己土可克制
- 「用象法看木在辰为生机孕育之象」→ 五行在特定位置的生长意象
- 「从象的角度去读取」→ 不是算分数，是看画面是否和谐
