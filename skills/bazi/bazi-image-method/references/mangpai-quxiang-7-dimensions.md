# 盲派取象七维度扫描框架 · 外部知识萃取

> **来源：** https://time.actor/zh-CN/guides/mangpai-quxiang-yuanze/ (TIMEACTOR)
> **提取日期：** 2026-07-14
> **与象法体系的关系：** 象法体系（象形读图/画图法）与本框架（结构化七维度逐层扫描）是**互补关系**——象法重在「看画面是否和谐」，盲派取象重在「逐维扫描不遗漏」。实战中可先用七维扫描确定信息点，再画图验证画面是否协调。

## 核心理念

> 「七种取象思路：五行（以水为例——数字1/6、色蓝黑、形圆、味咸）、天干（丙火旺则性急，被壬癸克则牙龈肿痛）、地支（寅为山林、卯为花草）、十神（正财=劳动报酬）、宫位（年祖月父日配偶时子女）、神煞（桃花看祖上/配偶貌美）、生旺死绝状态。」

**取象不是猜——是七个固定维度逐一扫描，有就有没有就没有。** 七个维度分层扫描一个八字，不重不漏。

## 一、五行取象——最基础的扫描层

以水五行为例：
- 数字：一和六
- 颜色：蓝、黑、灰
- 形状：圆
- 味道：咸
- 身体：耳和肾

**实战推断：** 水旺的人可能喜欢蓝黑色系穿搭，口味偏咸，脸型/体型偏圆润，肾脏和听力需关注。

**职业方向：** 水旺→流动性工作（物流交通贸易）或与水相关行业（水利水产）。

### 五行五态体系

| 状态 | 水的表现 | 性格/行为 |
|:----|:---------|:---------|
| **太旺** | 泛滥无制 | 情绪化 |
| **旺** | 力量充足 | 活动力强 |
| **弱** | 力量不足 | 内敛谨慎 |
| **绝** | 几乎消失 | 缺乏该五行特质 |
| **中和** | 流动有序 | 适应力强 |

|:----|:---------|:-----|

| **己土** | 包容滋养 | 田园 |
| **庚金** | 刚健果断 | 刀剑 |

### 天干取象变体规则

天干与日干的关系（十神）产生变体——比如庚金见甲木为偏财，此时的庚金像一把刀在砍树获取木材（资源），性格里会加入「为钱而动」的驱动。

## 三、地支取象——环境与行为模式

| 地支 | 取象 | 延伸 |
|:----|:-----|:-----|
| **寅** | 山林、公园、学校 | 有文化的公共空间 |
| **卯** | 临街花草、蔬果 | 市井生活的美感 |
| **辰** | 水库、龙宫 | 神秘的大机构 |
| **巳** | 驿站、交通网络 | 传播中转 |
| **午** | 光明正大的舞台 | 展示自我的平台 |

### 地支宫位对应

| 位置 | 代表 |
|:----|:-----|

## 四、十神取象——人际关系和资源类型

| 十神 | 基础含义 | 人物象 |
|:----|:---------|:------|
| **正财** | 合法劳动报酬、正式的妻子 | 妻子/财务 |
| **偏财** | 意外之财、父亲、情人 | 商人/投机者 |
| **正官** | 正当权力、上司、丈夫 | 法官/考官 |
| **七杀** | 压力、挑战、小人、武职 | 将军/警察 |
| **正印** | 正统教育、母亲、文化修养 | 老师/贵人 |
| **偏印** | 特殊技能、继母、冷门知识 | 军师/巫师 |
| **食神** | 温和的才华、口福、悠闲 | 厨师/艺人 |
| **伤官** | 犀利的才华、叛逆、不满 | 艺术家/律师 |
| **比肩** | 平等同伴、竞争者 | 兄弟姐妹/同事 |
| **劫财** | 激进的同伴、直接争抢 | 合伙人/竞争者 |

### 十神在不同宫位的含义

例：正官在不同宫位：
- 正官在年 = 祖上有官贵
- 正官在月 = 事业上有领导赏识
- 正官在日 = 配偶是公务员或有权威的人
- 正官在时 = 子女有出息

|:---------|:-----|

| 日支 | 配偶漂亮 |
| 时支 | 晚年有异性缘 |

### 5.3 生旺死绝状态取象

| 状态 | 含义 |
|:----|:-----|

## 多维叠加拆解实战

### 事业与财富
**五行+天干+十神**三层叠加=职业方向：
- 水+壬+正财 = 物流货运或水产贸易的经营者
- 火+丙+食神 = 内容创作或餐饮行业的自由职业者

### 婚恋与人际
**地支+十神+神煞**三层叠加=婚姻细节：
- 日支酉（精致小空间）+七杀（压力挑战）+桃花（吸引力）= 配偶有魅力但关系充满张力

### 性格特质
**天干+五行状态**=性格主导力量：
- 甲木在帝旺状态 = 自信到自大
- 甲木在绝地 = 本来有能力但自我怀疑严重

### 健康表现
**宫位+五行+天干**=身体部位：
- 年柱甲寅（肝胆/头部）被庚申冲 = 头部受伤或肝胆问题有先天倾向
- 月柱丙火（心脏/眼睛）被壬水克 = 心血管或视力需要特别注意

## 实战落地要点

### ① 七个维度逐一扫描，不要跳
拿到八字从五行维度开始，像过安检一样逐层扫：
```
五行 → 天干 → 地支 → 十神 → 宫位 → 神煞 → 状态
```
每层扫完记录信息，最后叠加找交集。**不要只用一个维度下结论。**

### ② 先建基础象库
五行五色五形五味五臓、十天干基础画像、十二地支环境画像、十神人际关系画像——这四张表是取象的字典。

### ③ 用「生旺死绝」做时间轴
同一天干在不同月令状态不同，代表这个人在不同领域的不同「状态」。日干在月令长生=青年发力早成，日干在月令绝=中年后才找到感觉。**这不看旺衰，看的是节奏。**

## 信息矛盾处理

### Q: 七个维度看到的信息矛盾怎么办？
A: 不矛盾——是不同侧面。天干取象说这个人热情（丙火），地支取象说这个人行为保守（坐辰土湿土晦火）——这说的是这个人内心热情但行为上压抑自己。矛盾本身就是信息——表里不一的人。真正的矛盾只有一种：所有维度都找不到共同点的八字——这种八字的人就是没有突出特征。

### Q: 取象有没有优先级？
| 断事类别 | 维度优先级 |
|:---------|:----------|
| 断性格 | 天干>地支>五行 |
| 断行为模式 | 地支>天干>十神 |
| 断职业 | 五行+天干+十神叠加 |
| 断健康 | 宫位+五行+干支生克 |

不同问题用不同的维度组合，没有万能的维度排序。
