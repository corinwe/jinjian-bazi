# 日柱计算·基准日与锚点（已校准·永久固化）

## 🚨 致命错误记录

> **2026-06-06 踩坑**：主母八字日柱全部算错！根因：误认为1900-01-01=甲子日。
> **事实**：1900-01-01 = 甲戌日（gan=0, zhi=10）。
> **后果**：报告重写3版（v5.0废/v5.1废/v5.2才正确），浪费大量时间。
> **铁律**：永远用1900-01-01=甲戌日为基准，每次排盘用代码算日柱，绝不用人脑估算。

## 正确基准

```
1900-01-01 = 甲戌日 (gan=0, zhi=10)
公式：
  gan_idx = delta % 10          # delta = (target_date - date(1900,1,1)).days
  zhi_idx = (10 + delta) % 12
```

## Python验证代码

```python
from datetime import date
tiangan = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸']
dizhi = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥']
base = date(1900, 1, 1)  # 甲戌日

def get_rizhu(d):
    delta = (d - base).days
    return tiangan[delta % 10] + dizhi[(10 + delta) % 12]

# 验证
print(get_rizhu(date(1987,7,20)))   # 庚午
print(get_rizhu(date(2008,8,8)))    # 庚辰
print(get_rizhu(date(1949,10,1)))   # 甲子
```
