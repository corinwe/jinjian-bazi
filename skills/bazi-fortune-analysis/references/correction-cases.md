# Correction Protocol & Common Cases (校正协议与常见案例)

## Golden Rule

User's life facts are GROUND TRUTH. When facts contradict predictions, theory must adjust. Never argue with facts.

## Correction Priority Order

When facts contradict analysis, check in this order:

1. **Body strength (身强身弱)** - Most common error source (~60% of corrections)
2. **Useful god selection (用神)** - Derived from body strength (~20%)
3. **Pattern identification (格局)** - Can change with body strength (~10%)
4. **Ten God interpretation (十神解读)** - Context-dependent (~5%)
5. **Special structures (特殊格局)** - Rare (~5%)

## Common Correction Cases

### Case 1: Body Strength Misjudgment

**Trigger:** User reports opposite experience in predicted good/bad years.

**Example:**
- Predicted: Body strong because born in Earth month (未月)
- User fact: Multiple health crises, wealth loss, career stagnation in "good" years
- Correction: Body is actually weak. Output gods (伤官双透) drain energy completely.
- New analysis: Weak body needs Resource (印) and Sibling (比劫), not Wealth (财)

**Verification method:**
- Resource/Sibling years (印比年) → should be good → verify with user
- Wealth/Authority years (财官年) → should be challenging → verify with user

### Case 2: Hidden Stem Reinterpretation

**Trigger:** User facts don't match month branch surface analysis.

**Example:**
- Month branch 未 appears Earth-supportive
- Hidden stem contains 乙木 (Wood) which attacks 己土 (Day Master)
- Month branch is actually hostile despite surface appearance
- Correction: Body support from month is much weaker than assumed

### Case 3: Double Output Drain

**Trigger:** Chart has 伤官双透 (double output) or strong output with apparent body support.

**Key insight:** Double output can drain body completely even if:
- Born in "strong" season
- Multiple apparent roots
- This is one of the most common misjudgments

**Correction:** Always treat strong output as significant drain factor. Reduce body strength assessment by one level.

### Case 4: Wealth Star Misinterpretation

**Trigger:** User reports wealth patterns different from prediction.

**Example:**
- Wealth star visible and apparently strong
- User reports stable moderate income, not wealth
- Correction: Wealth star is present but DM too weak to hold it
- Wealth becomes burden rather than blessing
- Pattern: "财多身弱" (much wealth, weak body) = wealth cannot be held

### Case 5: Marriage Prediction Error

**Trigger:** User reports harmonious marriage when chart suggests conflict, or vice versa.

**Common resolution:**
- Chart shows conflict pattern BUT both partners have self-regulating mechanisms
- 食神制杀 (Food God controls Killings) = person can self-resolve conflicts
- Hidden harmony patterns not visible in surface analysis
- Spouse selection may have been influenced by chart's filtering mechanism

### Case 6: Luck Period Misalignment

**Trigger:** User experiences don't match predicted luck period quality.

**Common causes:**
- Mixed luck period (first half good, second half bad within same decade)
- Annual stem/branch significantly modifies decade luck
- Hidden interactions between luck and birth chart not initially apparent

### Case 7: Industry/Job Fit Mismatch

**Trigger:** User is successful in industry predicted to be unsuitable.

**Resolution:**
- Industry element may not directly match useful god
- Specific role within industry may match (e.g., Water industry but Fire role)
- User may have compensated through environment or partners
- Check if partner's chart provides missing elements

## Correction Procedure

### Step 1: Acknowledge Discrepancy
Acknowledge the correction explicitly. State what was wrong and what the new understanding is.

### Step 2: Identify Root Cause
Determine which assumption was wrong. Most likely body strength.

### Step 3: Reconstruct Analysis
Rebuild the analysis from the corrected foundation:
- Recalculate body strength
- Reidentify useful god
- Reassess pattern
- Update all downstream analysis

### Step 4: Verify with Additional Facts
Ask the user to verify 2-3 years under the new framework:
- "If body is weak, year X should have been good/bad. Was it?"
- Use this to confirm the correction

### Step 5: Document Correction
Track the correction to maintain consistency:
- Body strength: Weak (corrected from Strong)
- Reason: Double Output drain + hidden hostile stem
- Verified by: User's experience in years X, Y, Z

### Case 8: "假旺真弱" (False Strong, Truly Weak) — Three Stems + Three-Harmony Drain

**Discovery context:** 金鉴真人复核命主①「魏启令」(庚申 癸未 辛亥 辛卯)时发现的身强弱误判典型案例。

**Trigger:** 天干三金透出（庚+辛+辛），表面身强。但地支有亥卯未三合木局暗耗日主。

**Identifying features of 假旺真弱 (False Strong, Truly Weak):**

| Feature | Description | Severity |
|---------|-------------|----------|
| ✅ 天干比劫重重 | 天干有多重比劫/劫财透出 | 表现信号 |
| ✅ 地支无真根 | 日主的禄位（本气根）在地支缺失。注意：劫财的根（如申对辛）≠日主自己的根（酉对辛） | **关键信号** |
| ✅ 地支强力耗泄 | 三合局/六合局形成的强力耗身或泄身力量 | **决定因素** |
| ✅ 月令生力有限 | 燥土（未戌）名义生金实则脆金；或失令 | 加重信号 |

**The Three-Harmony Bureau trap — most dangerous:**

亥卯未三合木局是"假旺真弱"的最高频陷阱。具体分析：
1. **三合局即使不成化**，仍产生70-80%的强力合会效果
2. 合会后的木气持续耗金，日主在地支被持续消耗
3. 天干三金看似威武，但"空中楼阁"——地支才是根基
4. 分析口诀：**"天干看表象，地支看实质；三合局成，天干减半"**

**Critical checklist — Do NOT skip:**

在判定身强时，必须逐条检查以下内容，缺少任何一条都可能导致误判：

- [ ] 日主在地支有**自己的禄位或正根**吗？（如辛金见酉，丙火见巳/午）
- [ ] 劫财根（如庚见申对辛而言）≠ 日主正根
- [ ] 地支是否有三合局/六合局形成的耗身之物？
- [ ] 月支是**真生**日主还是"名义生实则脆"？（特别注意未戌燥土）
- [ ] 天干多重比劫透出是否掩盖了地支真弱？

**Specific traps for each day master:**

| 日主 | 高频误判场景 | 必须检查 |
|:---:|:---|:---|
| 辛金 | 天干三金+亥卯未三合木局 | 全局是否有酉（辛禄）？申是庚根非辛根 |
| 庚金 | 天干双庚+寅午戌三合火局 | 全局是否有申（庚禄）？火局耗身是否被低估 |
| 丙火 | 天干双丙+巳午未三会火局 | 看似极强但是否有双癸/壬水坐强根反制 |
| 甲木 | 天干双甲+亥卯未三合木局 | 全局是否有寅（甲禄）？是否有庚金强根克身 |
| 戊土 | 天干双戊+辰戌丑未四土 | 是否土重埋金？是否有甲乙木疏土 |

**Procedural safeguard:**
1. **天干分析完成** → 得到"表面强弱"的初步判断
2. **强制检查以上 checklist** → 如果"天干比劫重重"且"地支无真根且有三合局" → 标记为**假旺真弱嫌疑**
3. **交叉验证**：查阅用户已发生的3-5个关键年份，看与实际经历是否吻合
4. **标注置信度**：在报告中明确标注"初判待验"，并要求用户提供事件校准

### Case 9: 完整校准循环（V1→V4）—— 魏启令案

**校准事件数：** 19个关键事件，100%吻合

**版本演进：** V1初判→V2修正→V3框架升级→V4事件校准

#### 最关键的四个校准发现

**发现1：假旺真弱的财富密码不是创业，是大平台**
- 三次创业全败（2007兼职、2014小平台、2017初小平台）
- 但在500强大厂年入200-300万
- 结论：大平台=用神土，借力是身弱者的致富之道

**发现2：定性与定量的区别——命理可定性不可定量**
- 定性（用神忌神、格局身强弱）：14事件100%验证 ✅
- 定量（收入数字、财富量级）：理论值50-100万 vs 实际200-300万，差3-5倍
- 结论：命理能告诉你"向什么方向努力"（定性），但需要用户提供真实数据才能精确到数字（定量）

**发现3：真禄年是假旺真弱之人的"命根"**
- 2005乙酉（真禄年）→ offer拿到手软
- 2017丁酉（真禄年）→ 收入暴增至100万+
- 酉金是辛金唯一真根，真禄年效力＞土年数倍
- 结论：对假旺真弱之人，用神有"层次"——真根（禄）＞印（土）

**发现4：大运交接期的力量不可忽视**
- 2000庚辰年高考被干扰：表面看庚辰双用神应吉，但实际被干扰
- 原因：**乙酉大运最后一年（运末交接期），大运乙木忌神余力仍在**
- 结论：大运首尾两年要打折扣，不可单独看流年

**发现5：忌神年≠全凶，用神年≠全吉——命理是三维的**
- 2007丁亥年：双忌神年，创业亏损（忌神应凶✅）
- 但**亥卯未三合木局被引动**→木为妻星→认识了成（未来的配偶）
- 结论：同一个十神在不同维度有相反效应——三合木局耗身致创业失败（凶），但同局引动妻星致姻缘（吉）
- 教训：用神忌神的分析必须**分维度看**——对事业是忌神不等于对婚姻也是忌神

**发现6：三合木局的"蓄能-释放"周期律——想法延迟1-3年实现**
- 魏启令反复验证的规律：心中想法不会立即实现，但1-3年后用神年到就自然发生了
- 机制：亥卯未三合木局=蓄能池，想法被"存"在木局中，等待土金用神年扶身→身得助→从木局取出愿望
- 实例：2000年想上重点大学→2001辛巳年实现(1年)；~2016年想收入提升→2017丁酉年实现(1-2年)
- 三次创业失败的反证：强行立刻做(忌神驱动)→永远不成功
- 对应口诀："刻意追求→求不到；顺其自然→送上门"
- 建议：假旺真弱之人产生想法时记下来等1-3年，用神年到了自然有机会

## Time Verification Protocol

Always verify current year/decade before predictions:
- Confirm exact current year (don't assume)
- Confirm user's current age
- Map correctly to current luck period
- Double-check stem-branch for current year

## Information Quality Assessment

Rate information sources:
- **Exact birth time (精确时辰):** Gold standard, use directly
- **Approximate time (大概时段):** Narrow to 2-3 possible hour pillars, compare with life facts
- **No birth time (无时辰):** Analyze without hour pillar, note limitations
- **Lunar vs Solar date confusion:** Verify by asking about holidays/events around birth date
- **Leap month birth (闰月):** Critical for month pillar, must verify if leap month applies
