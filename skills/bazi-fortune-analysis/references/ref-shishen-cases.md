## 1. 十神组合效应·经典断事（金鉴真人核心）

```
核心原则：两个十神组合产生的效应 ≠ 单个十神的简单相加

【杀印相生】七杀+正印/偏印
  断事：压力转化为动力，贵人化解困境
  ├─ 七杀为忌 → 印星化杀 → 本来要受灾，被贵人化解
  ├─ 女命杀印相生 → 配偶虽强势但体贴
  └─ 身弱杀旺+印为用 → 工作压力大但能出成绩

【食神制杀】食神+七杀
  断事：以智慧制伏困难，以柔克刚
  ├─ 食神得力 → 聪慧过人、善化解危机
  ├─ 食神弱七杀旺 → 制不住，压力反噬
  └─ 男命食神制杀 → 事业有成，子女出息

【伤官配印】伤官+正印/偏印
  断事：才华被规范，狂傲被收服
  ├─ 伤官为忌+印为用 → 本来叛逆惹祸，被印收服成人才
  ├─ 是贵格 — 又聪明又有修养，易出名
  └─ 女命伤官配印 → 有才华但不克夫

【伤官见官】伤官+正官
  断事：才华与制度的冲突
  ├─ 伤官旺+官星弱 → 目中无人、以下犯上
  ├─ 官星旺+伤官弱 → 被压制、抑郁不得志
  └─ 女命伤官见官 → 克夫、婚姻不顺

【枭神夺食】偏印+食神
  断事：好运气被拦截，福气被剥夺
  ├─ 枭神在天干+食神也在天干 → 应灾最大（明着夺）
  ├─ 枭在地支+食在天干 → 暗地里被破坏
  ├─ 枭在月令夺年/月干食神 → 最严重
  └─ 断灾类型：枭金夺木=头手伤，枭木夺土=血光，
      枭土夺水=车祸外伤，枭水夺火=溺水性病，
      枭火夺金=烫伤骨折

【比劫夺财】比劫+财才
  断事：钱被分走、被朋友坑、合伙亏本
  ├─ 男命比劫夺正财 → 克妻、妻子被夺
  ├─ 身强比劫+财弱 → 挣多少花多少
  └─ 合伙生意 → 宜用官杀制比劫可解

【伤官生财】伤官+财才
  断事：靠一技之长赚钱，才华变现
  ├─ 伤官旺+财有根 → 技术/艺术/创意赚钱能力强
  └─ 是发财格局之一

【食神生财】食神+财才
  断事：靠服务/餐饮/福气赚钱
  ├─ 食神旺+财有根 → 轻松赚钱，福气生财
  └─ 比食神生财更温和，适合服务行业
```

## 2. 十神比例·命运质量速断

```
统计八字中十个十神各占多少（精确到天干透出+地支主气）：

吉神（正官/正印/正财/食神）占比：
  ├─ 70%以上 → 命运坦荡，性格平和
  ├─ 40-70% → 普通格局，有吉有凶
  └─ 40%以下 → 人生坎坷多，需大运化解

恶神（七杀/偏印/伤官/劫财）占比：
  ├─ 50%以上 → 大起大落、人生多波澜
  ├─ 30-50% → 有成就但内心焦虑
  └─ 30%以下 → 压力适中，心理相对健康

⚠️ 不可机械套用——需结合身强身弱判断：
  身弱时，吉神挨着挨着也是压力（如身弱官杀混杂）
  身强时，恶神可用（如杀印相生需七杀才能显贵）
```

## 3. 十神格局组合·层次断事

```
相同格局不同组合 → 层次天差地别：

【官格】
  官+印（官印相生）→ 高格局：有名望有权
  官+财（财生官）  → 中格局：有钱有地位
  官+伤（伤官见官）→ 低格局：仕途坎坷
  官+杀（官杀混杂）→ 低格局：心思不定

【杀格】
  杀+印（杀印相生）→ 高格局：大权在握
  杀+食（食神制杀）→ 高格局：英雄豪杰
  杀+劫（杀邀劫）  → 中格局：江湖人物
  杀无制          → 低格局：一生飘零

【财格】
  财+官（财生官）  → 高格局：名利双收
  财+食（食神生财）→ 中格局：稳步致富
  财+伤（伤官生财）→ 中格局：暴发户/偏财
  财+劫（比劫夺财）→ 低格局：财来财去

【印格】
  印+官（官印相生）→ 高格局：文人官贵
  印+杀（杀印相生）→ 中格局：有实权
  印+食（印夺食）  → 低格局：怀才不遇
  印+财（财破印）  → 低格局：学而无用
```

## 4. 实战案例参考

### 4.1 案例1：身强财旺格局（矿老板/首富）

```
乾造：甲辰、戊辰、甲寅、戊辰
决策逻辑：
  1. 排盘后判断身强 → 甲木生于辰月得令，三辰一寅有根
  2. 判断财星 → 戊土偏财三重透干，财旺极
  3. 食伤 → 辰中有水，寅中有火，食伤有根
  4. 结论：身强财旺，大运流年遇食伤财才 → 发大财
  5. 实际：当地首富，开矿+房地产（土相关行业）
```

### 4.2 案例2：身强财旺食伤旺（银行行长+健身房）

```
乾造：丁巳、壬子、壬子、丙午
决策逻辑：
  1. 排盘判断身强
  2. 一心只想着赚钱（财星旺）
  3. 结论：身强+财旺+食伤旺 → 发财
  4. 实际：银行行长 + 连锁健身房合伙人
```

### 4.3 案例3：身弱财旺补比劫

```
坤造：丁巳、戊申、癸卯、丙辰，大运壬子，流年壬寅
决策逻辑：
  1. 判断身强（约90分）→ 从弱
  2. 财星偏弱（约30分）
  3. 食伤强（约75分）
  4. 策略：身强食伤强财弱 → 补财星（火牌60分）
  5. 食伤补木牌15分
```

### 4.4 案例4：三刑+七杀应灾（实战案例）

```
乾造：庚申、己卯、丁酉、壬寅
大运癸未，流年癸巳（33岁）
决策逻辑：
  1. 地支寅巳申三刑 → 巳年触发
  2. 天干两个七杀（癸水）→ 岁运并临
  3. 结论：地支三刑+天干七杀+岁运并临 → 重大灾祸
  4. 实际：因脑梗送医急救
     └─ 可见三刑+七杀+岁运并临的叠加效应极强
```

### 4.5 案例5：伤官配印格（周恩来总理格局）

```
决策逻辑：
  1. 日元身弱 → 伤官为忌凶
  2. 印为喜用 → 以印制伤
  3. 有印克制伤官的狂傲之气 → 又儒雅又有上进心又有爱心
  4. 结论：伤官配印格，容易成名
     └─ 伤官配印是贵格，以印星的克制让伤官的才华得以正向发挥
```

### 4.6 案例6：枭神夺食的应灾判断

```
枭金夺木 → 以金克木的属性判断：
  木在人身类象为肢体端部 → 手脚头部受伤
  木在宫位 → 该宫位六亲应灾
  
应灾大小判断：
  ├─ 多个枭神夺一个食神 → 应灾大
  ├─ 月令枭神夺年干/月干食神 → 应灾大
  ├─ 枭神与食神都在天干透出 → 应灾大
  └─ 隔位力量变小，紧邻力量变大
```

### 4.7 实战断事标准流程（金鉴真人模式）

```
1. 看命主状态（坐姿、语气、衣着）→ 初步判断能量高低
2. 排盘 → 先过三关建立信任
   ├─ 断出生地周围环境（天干地支意象）
   ├─ 断父母兄弟姐妹情况
   └─ 断性格特征
3. 分析日元强弱 → 取喜用忌凶
4. 看格局 → 定性格和一生追求
5. 重点关注：财（何时发财）、婚姻（何时结婚/离婚）、健康（有无大灾）
6. 看大运流年 → 定应期
7. 给出建议（能量调整方案）
8. 结尾用天乙贵人和喜用数字 → 方便收卦金

常见特殊技巧：
  ├─ 八字带墓库辰戌丑未 → 有收藏爱好
  ├─ 子午卯酉方向感强，不容易迷路
  ├─ 八字出现几个同字 → 做连锁扩张生意
  ├─ 八字火旺 → 有烫伤或伤疤
  └─ 八字木超过两个 → 毛发较重
```

## 3. 第三课新增案例

### 案例一：阿廖八字分析（男命现场互动）

**八字信息**：1984年4月14日晚8点（阳历），男命
**排盘结果**：己土日元（土命，得分64分，身强）

**分析链路**：
1. **定身强身弱**：月令得令40分+其他各柱加分→64分，身强
2. **七杀在甲木**：注意肝胆功能
3. **七杀在年干**：头部疾病（偏头痛等）
4. **七杀在日支（夫妻宫）**：配偶霸道厉害，妻子性格强势
5. **偏财在月干**：仗义豪爽，跟朋友抢着请客
6. **正财在月支**：青年时期有稳定工作，中年后开始创业
7. **比肩多（天干得势、地支通根）**：朋友多，朋党成群
8. **八字无劫财**：不会坑朋友，可交
9. **七杀在甲木+甲木在配偶宫**：配偶个子高（甲木为高大树木）
10. **土木交战**：脾胃不好（木克土）
11. **时支有财星**：晚年不缺钱花，子女未来有钱
12. **大运财运**：申子辰三合水局(15倍财星加强)，身强财旺→大运要发财
13. **偏财流年+流月+流日**：选择偏财极旺时刻买彩票可中奖

**断语提炼**：
- "朋友多，肝胆要注意，性格豪爽霸道，跟朋友很正义"
- "老婆厉害，但配偶家族能提供帮助"
- "晚年不缺钱花，孩子未来会有钱"
- "大运要发财，37-47岁走发财大运"

### 案例二：七格格八字分析（女命现场互动）

**八字信息**：1979年7月9日晚11点左右（阳历），女命
**排盘结果**：丁火日元，满盘消耗（0分）→从弱格，物极必反，弱到极点反而身强
**恒定50分，全部为喜用**

**分析链路**：
1. **特殊格局**：从弱格（0分→50分恒定），一生好运
2. **亥亥自刑**：5倍正官，全部为喜用→身边很多男性追求且真心对她好（非渣男体质）
3. **偏财两头挂**：豪爽人，女中豪杰
4. **食神在月令**：一生不愁吃穿，有福气
5. **食神生偏财格**：八字无正财，偏财格
6. **正官在日支**：老公缘分强
7. **食神旺**：投胎投了个游乐场，吃喝玩乐享受生活，性格开朗不会生气
8. **偏财旺**：闲不住，会尝试做生意（野心大）
9. **八字无木**但亥未拱合卯：个子不低（暗藏卯木）
10. **金水两旺出美女**：皮肤白皙，鼻子高挺
11. **丑酉半合（宗教缘分）**：与道教缘分极深
12. **今年印星流年**：开始学习周易

**断语提炼**：
- "全盘喜用，一生好运"
- "身边人对她都好，不是渣男体质"
- "偏财两头挂，定是豪爽人"
- "食神旺，投胎投了个游乐场"
- "几乎完美的八字，一生好运"
- "与宗教缘分很深"

### 案例三：马云八字十神分析

**背景**：2019年辞去阿里巴巴董事会主席
**分析**：当时大运是食神、流年是食神、流月是食神→三个食神同时克到官星→不想操心、不想当官→辞去职务
**断语提炼**："食神一到，官运要丢"

### 案例四：助理培养失败案例（道长自述）

**背景**：道长培养助理6年想让他做管理，始终培养不起来
**八字分析**：助理八字食神旺、没有官星→满脑子想玩，不爱当官
**结论**：应该培养他做销售、接待（食神旺的人能说会道），而不是管理岗位
**教训**：每个人的人生追求不同——食神旺要玩，官星旺要当官，财旺要搞钱，印旺要传播思想
