### 1.1 男命婚姻铁律

**核心规则：正财为妻，无正财则偏财为妻**
- ✅ 有正财 → 正财=妻，偏财=情人/偏房
- ✅ 无正财有偏财 → **则偏财直接为妻**（偏财取代正财成为妻星）
- ✅ 正偏财皆无 → 看夫妻宫（日支）与日主的关系断配偶

- 正财被合 = 妻子可能有外遇
- 对妻子越好财运越旺

**判断口诀**：有正看正，无正看偏，两样皆无看日支

### ⚡ [黄金规则] 婚姻/子女/事件断事的终极逻辑——一切回到能量

> **老板点拨**：不用搞复杂。所有判断的底层只有两样东西——①能量的倍数（强弱）②能量的喜忌（方向）。刑冲合害只是改变能量的工具。一以贯之，万事通用。

**断任何事的终极公式**：
```
结果 = 能量倍数 × 喜忌方向

能量倍数 = 合/会/冲/刑/害给出的倍数（5倍~20倍）
喜忌方向 = 喜用（+正向）/ 忌凶（-负向）
```

**配偶星判断示例**：
| 流年场景 | 能量倍数 | 喜忌 | 结论 |
|:---------|:--------:|:----|:-----|
| 逢正官+三合（有引化） | 15倍🔥 | 喜用 | 结婚概率极高 |
| 逢正官+六冲 | 冲10倍但冲走了 | — | 等于没来，不结 |
| 逢正官+三刑（喜用） | 15倍🔥 | 喜用 | 刑得厉害但反而为吉 |
| 逢正官+三刑（忌凶） | 15倍🔥 | 忌凶 | 结婚了也不幸福 |
| 逢正官+空亡 | 减半 | — | 虚的，不靠谱 |

**所有事件通用该公式**：
- 发财 = 财星能量×喜忌 + 帮身能量×喜忌
- 生子 = 子女星能量×喜忌 + 时柱加强
- 灾祸 = 七杀能量×倍数 + 有无化解
- 升职 = 官杀能量×喜忌 + 印星配合

**口诀**：
```
万事不离能量论，倍数喜忌定乾坤
合冲刑害都只是，能量加减的工具
莫记散乱小规则，回到根本断得准
```
