# Calibration Insights — 定性 vs 定量 + 核心发现

> **来源**: 金鉴真人对魏启令案(庚申 癸未 辛亥 辛卯)的四版迭代校准(V1→V4)
> **核心结论**: 14个真实事件100%验证用神土金判断，但收入数字差3-5倍

## 一、定性 vs 定量 — 最重要的方法论发现

**命理学的准确定位：**

| 能力 | 可靠性 | 证据 |
|:---|:---:|:---|
| **定性**（用神忌神、格局身强弱） | **极高** 14/14事件100%验证 | 魏启令案 |
| **定量**（收入数字、财富量级） | **低** 理论值50-100万 vs 实际200-300万 | 魏启令案收入差3-5倍 |
| **方向指引**（适不适合创业、何时发力） | **高** 三次创业失败+大平台成功验证 | 魏启令创业史 |

```
结论：命理能告诉你"向什么方向努力"（定性），
      但需要用户提供真实数据才能精确到数字（定量）。
      所以每一份报告必须标注：⚠️ "初判待验，需事件校准"
```

### 实战协议

1. 初判报告只给出**方向性判断**（用神、格局、宜忌）
2. 财富量级标注"此为初判，需用户提供实际收入校准"
3. 收集到真实数据后，再做定量精修
4. 定量校准后，用神判断不变（定性稳定），但数字修正（定量可变）

## 二、用神层级 — 并非所有用神力量相等

从魏启启令案中发现的用神力量层次（按效力排序）：

| 层级 | 类型 | 效果 | 案例 |
|:---:|:---|:---:|:---|
| 🥇 | **真根（禄位）** | 数倍于其他用神 | 酉金年（2005乙酉/2017丁酉）= offer到手软/收入暴增 |
| 🥈 | **劫财根（帮身）** | 强于理论预期 | 申金年（2016丙申）= 扛大项目成功 ✅ |
| 🥉 | **真生（湿土印星）** | 有效但有限 | 辰丑土年 = 稳固提升（1997丑土中考第一） |
| ⚠️ | **名义生（燥土脆金）** | 远低于理论预期 | 未戌土年 = 效果极有限，切忌当完整用神 |

**实战口诀：**
- 对辛金日主：酉（真禄）＞ 申（劫财根）＞ 辰丑（湿土）＞ 未戌（燥土）
- 对丙火日主：午（阳刃）＞ 巳（禄位）＞ 寅（长生）
- 对甲木日主：寅（禄位）＞ 卯（劫财根）＞ 亥（长生）

**关键：如果真禄年都效果不佳，那一定是用神判断反了。**

**新增校准（2026-05-30）——大运能量打折：**
- 大运首年（刚交运）：力量打7折
- 大运末年（即将换运）：力量打6折 ← **最易忽视的陷阱**
- 中间六年（稳定期）：100%全力发挥
- ⚠️ 2009己丑年（双土用神+丙戌大运末）效果被错误夸大——实际只是因为大运末力量打折，只是小调整而非大跳槽

**新增校准（2026-05-30）——事件推导三步骤：**
推导已发生事件时，必须遵守：
1. 先用已知事件反推命主的时间线和社会身份
2. 按常识建立正常轨迹基线
3. 只有八字有强烈特殊信号，才偏离基线描述特殊事件

## 三、平台 > 创业 — 假旺真弱的核心财富密码

| 经济行为 | 魏启令的实际结果 | 命理解读 |
|:---|:---:|:---|
| 三次独立创业 | ❌ 全部失败 | 身弱不担财，木火忌神年行动必败 |
| 大平台技术高管 | ✅ 年入200-300万 | 大平台=土用神，借力致富 |
| 未来己丑大运 | ⏳ 一生最佳窗口 | 湿土生金，真正的财富跃升期 |

**实战判断：**
- 如果用户创业多次失败但在大平台成功 → **假旺真弱嫌疑极高**
- 对假旺真弱之人：大平台 = 用神土，借力是致富之道
- 不宜建议"创业突破"——应建议"在大平台内深耕+股权变现"
- 年收入达到200万+时，达亿路径是"工资+股权+投资"而非"纯创业"

## 四、真禄年 vs 忌神年的极端对比

假旺真弱之人最显著的标志：

```
真禄年 → 人生大转折（起飞）
忌神年 → 灾难性事件（跌落）
二者之间的落差极大，远超过真强/真弱之人的正常波动
```

### 魏启令案的极端对比（最具说服力的验证）

| 对比组 | 年份 | 结果 | 落差值 |
|:---:|:---:|:---|:---:|
| 双忌神年 | 2007丁亥 | 创业连亏 | 年收入≈0 |
| 真禄年 | 2005乙酉 | 毕业offer拿到手软 | 起点即高点 |
| 双忌神年 | 2014甲午 | 车祸+官司+创业失败 | 一生最凶 |
| 真禄年 | 2017丁酉 | 收入暴增至100万+ | 年收入飙升 |

这种"凶年大凶、吉年大吉"的极端对比 = **假旺真弱的DNA级特征**。  
如果用户反馈凶年只是"平平"而非"灾难"，吉年只是"稍好"而非"飙升" → 假旺真弱判断可能有误。

大运首尾两年（交接期）的力量需要打折扣：

**案例**：2000庚辰年，表面看庚辰双用神应吉，但高考被干扰。原因：**乙酉大运最后一年，乙木忌神余力仍在。**

## 六、校准流程（推荐标准）

```
【初判】基于命盘做定性分析 → 标记"初判待验"
   ↓
【事件收集】向命主收集5-10个关键人生事件（含年份）
   ↓
【逐条验证】将事件映射到对应流年/大运
   ↓
【吻合度统计】≥80%吻合 → 用神确认；<80% → 重新审视
   ↓
【数字校准】根据用户实际收入修正财富量级
   ↓
【层级优先】真禄年 > 真生年 > 名义生年
   ↓
【报告重构】基于校准后判断重写报告
   ↓
【交付定稿】标注"校准后最终版"
```
