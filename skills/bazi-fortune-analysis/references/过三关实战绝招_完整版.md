> **验证派原则：** 各派知识实战验证后准的保留，不准的抛弃。

## 一、王牌绝招：土→身上必有痣

八字中土对应哪个宫位，该宫位对应身体部位必有痣：
- 年柱带土 → 后脑勺/头顶偏后有痣
- 月令带土 → 胸口/肋骨有痣
- 日支带土 → 小腹/后背有痣
- 时柱带土 → 腿上/脚上有痣
- 土越多→痣越多/越明显
- 准确率：100%（九龙道长自述从未失手）
- 原理：土主肌肉、皮肤、痣、斑

## 二、十大高频绝招（实战排序）

### ① 偏印=家中带羽毛的飞物
- 偏印在哪个宫位 → 家中该方位有羽毛/鸡毛掸子/鸟类装饰品
- 宫位对应方位：年干=北/年支=东北…按宫位方位找

### ② 卯=电脑/显示器/屏幕
- 八字有卯 → 家中该宫位有显示器/电脑/电视/计算器/电子屏幕

### ③ 酉=音响/喇叭/酒瓶/钟表
- 八字有酉 → 家中该宫位有音响/喇叭/酒瓶/大钟表

### ④ 庚金=暴力倾向/霸气
- 庚金在哪个宫位 → 该六亲有暴力倾向（不一定打人但性格刚硬）

### ⑤ 丁火=熬夜/晚睡
- 八字有丁火（特别是时柱/日支丁火）→ 必熬夜
- 身强丁火 → 熬夜不累；身弱丁火 → 熬夜伤身但戒不掉

### ⑥ 身强破财 / 身弱不担财
- 身强(>60分)+比劫旺 → 破财存不住钱
- 身弱(<40分)+财星旺 → 财来财去担不住

### ⑦ 甲+金=头部受伤
- 甲方（甲木/卯木）遇庚金/辛金冲克 → 头部受过伤

### ⑧ 名字带十字交叉=对应部位疼痛
- 名字中有「十」「乂」「×」等交叉结构 → 该字对应身体部位长期酸痛
- 例：「田」=腹部脾胃，「文」=头痛偏头痛

### ⑨ 名字带「丽」=婚姻不好
- 丽字=双鹿皮=美丽外表下藏着不为人知的伤痛
- 女性名字带「丽」→ 婚姻多半不好

### ⑩ 十二大败日出生=花钱如流水
- 口诀：正巳二寅三猴乡，四亥五申六午详，七卯八辰九戌位，十丑子十一未十二酉
- 正月巳日、二月寅日、三月申日、四月亥日、五月申日、六月午日、七月卯日、八月辰日、九月戌日、十月丑日、十一月子日、十二月酉日

## 三、辅助知识点

### ① 三鬼一害（三癸一害）— 火灾必应
- 口诀：三癸一害全，烈火焚房屋
- 条件：天干三个癸 + 地支一个亥
- 后果：必遇火灾（烧房子/烧山/楼下着火等）
- 案例：某命主19岁三癸一害，烧山400多亩

### ② 手机图标补五行法
- 把五行补能量图标设成手机桌面 → 24小时补能量
- 每天看手机就在补 → 效果比佩戴还好

### ③ 住满三年才准
- 一个人住进某个房子必须满三年，房子磁场才会与人的八字磁场完全同步
- 不满三年 → 风水断事可能不准

### ④ 藏干同字=夫妻同单位/行业
- 八字两个以上地支藏干相同 → 夫妻可能在同一个单位/行业

### ⑤ 五行补能量最优50分（不是60分）
- 九龙道长纠正：每个能量补到50分最好
- 40-60分是及格范围，50分是最优状态
- 超过60分会招灾
