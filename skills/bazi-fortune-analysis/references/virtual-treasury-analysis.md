# 虚库分析框架 — 库的非标准形态

> **来源**：子源（辛卯 癸巳 丙戌 癸巳）戌库分析 — 少爷盘 · 2026-06-07
> **核心概念**：库不一定都是"实库"——当库被合局熔化、或所藏元素被严重克泄时，即为"虚库"（看似有库，实则为空）

---

## 一、什么是虚库

### 标准库 vs 虚库

| 类型 | 定义 | 示例 |
|:----|:------|:-----|
| **实库** | 辰戌丑未正常发挥存储功能，库中元素可被调用 | 丑为金库（真财库），辰为水库 |
| **虚库** | 库被合局改变性质、库中元素被压制无法释放 | 戌被卯戌合火"熔化"，辛金困在熔炉中 |

### 虚库的三种成因

```
成因①：合局熔化（最常见）
  库被三合/六合/三会改变五行性质
  → 例：卯戌合火 → 戌从火库变为纯火 → 失去库的存储功能
  → 例：寅午戌三合火 → 戌变纯火 → 戌中辛金被熔化

成因②：月令压制
  月令五行与库的存储功能相克
  → 例：未月木休+亥卯未木局 → 未既是木局成员又压制木局

成因③：藏干被克
  库中藏干（财富/官星等）被天干或其它地支克制
  → 例：戌中辛金被全局旺火克制 → 虽有辛金实则无用
```

---

## 二、虚库的判断流程

### 三步判断法

```
Step 1：识别库的性质
  辰/戌/丑/未分别是什么库？
  对日主而言是财库/官库/比劫库/印库/食伤库？
  
Step 2：检查合局状态
  该库是否参与了合局？（三合/六合/三会）
  合局是否改变了库的五行性质？
  合局有天干引化吗？（有引化则能量更强）

Step 3：检查藏干状态
  库中的财富/官星/印星藏干是否被克制？
  被克的来源是什么？（天干/月令/其它地支）
  藏干是否能被释放？（需要什么条件）
```

### 虚库判定矩阵

| 检查项 | 正常（实库） | 异常（虚库） |
|:-------|:-----------|:-----------|
| 库之本气 | 未被合化改变 | 被合局改为其它五行 |
| 库中藏干（财/官等） | 不受严重克泄 | 被全局旺势克制 |
| 开库条件 | 简单（冲即可用） | 复杂（需冲+降温+引化） |

## 三、虚库的"激活"条件

要让虚库发挥实库作用，需要同时满足以下条件。条件越多、配合越精准，激活越彻底。

### 条件A：开库（打开库门）

| 方式 | 适用 | 倍数 | 说明 |
|:----|:----|:----:|:------|
| **冲**（辰戌冲/丑未冲） | 单库激活 | 10倍 | 最直接的开库方式 |
| **三刑**（丑未戌三刑） | 三库联动 | 15倍 | 能量最大，但伴随动荡 |
| **刑**（丑戌刑/未戌刑） | 半开库 | 5-7倍 | 温和但不够彻底 |

### 条件B：保护藏干（保护其中的财富/官星）

虚库之所以"虚"，是因为藏干被压制。要激活，**必须先解除压制**：

```
例：戌为火库 + 卯戌合火（熔炉状态）
→ 戌中辛金（正财）被火熔化
→ 需要"水"来降温（水克火，保护辛金）
→ 降温后才可能释放辛金

降温的优先级：
① 子水（正印）> 亥水（偏印）> 壬水 > 癸水
② 辰为湿土也能降温（蓄水制火）
③ 丑为湿土也能助金（生金）
```

### 条件C：引化（天干透出）

库中藏干如果能在天干透出，即为有引化，更容易激活：

```
有引化：藏干在天干显露 → 激活更容易
无引化：藏干只在地支 → 激活更难

例：子源（丙戌日）
  戌中辛金 → 年干辛金透出 → 有引化 ✅
  但辛金被火克 → 仍需条件B（降温）才能生效
```

## 四、虚库 vs 真财库的区别（实战关键）

这是最容易被混淆的概念。以丙火日主为例：

| 类型 | 地支 | 对丙火含义 | 何时发挥财库作用 |
|:----|:----|:----------|:----------------|
| **真财库** | **丑**（金库） | 丙火的真正财库 | **任何时候冲/刑都能开库** |
| **虚财库** | **戌**（火库+辛金藏） | 比劫库/火库，但有辛金残留 | **需同时满足条件A+B+C** |

**核心口诀：**
```
真财库，冲就开，金银珠宝往外抬
虚财库，冲不开，水火到位才出来
库若不辨真与假，冲了也是白冲他
```

## 五、实战案例：子源（辛卯 癸巳 丙戌 癸巳）

### 戌库诊断

| 检查项 | 结果 |
|:-------|:------|
| 戌是什么库？ | 火库/比劫库（对丙火日主） |
| 戌中有什么？ | 戊(本气) · 辛(中气·正财) · 丁(余气) |
| 合局状态？ | **卯戌合火**，有丙火引化 → 10倍 → 戌变纯火 |
| 戌的库功能？ | **被熔化** — 库门锁死，辛金困在熔炉中 |
| 判定 | **虚库** — 有库但空 |

### 激活条件

| 条件 | 需要 | 何时到位 |
|:-----|:------|:---------|
| A. 开库 | 辰戌冲（辰来） | 壬辰大运(8-17)已过，下次2036丙辰(26岁) |
| B. 保护辛金 | 水来降温 | 壬辰大运有壬水，戊子大运有子水 |
| C. 引化 | 天干辛金已透 | 年干辛金自透 ✅ |

**最佳激活窗口：**
- 第一次：壬辰大运(8-17岁) — 辰戌冲+壬水，但年幼
- 第二次：2036丙辰(26岁) — 辰戌冲+辛卯大运辛金双透
- **第三次：2051辛未(41岁)** — 丑未戌三刑(15倍)+辛金透+己丑大运真财库到位 ⭐

## 六、虚库的通用分析模板

```yaml
【虚库分析四步法】

第1步：定位
  库位：__（年/月/日/时支）
  库型：__（辰/戌/丑/未）
  对日主：__库（财/官/比劫/印/食伤）
  藏干：__（本气）+ __（中气）+ __（余气）

第2步：评估库状态
  □ 有合局改变性质？__（三合/六合/三会/无）
  □ 有引化？__（天干透出？倍数？）
  □ 藏干被克？__（被什么克制？强度？）
  → 判定：__（实库/虚库/半虚库）

第3步：激活条件分析
  □ 开库条件：__（辰戌冲/丑未冲/丑未戌三刑）
  □ 保护条件：__（什么五行能解压）
  □ 引化条件：__（天干需不需要透出）
  → 最早激活年份：__

第4步：与真财库对比
  真财库是什么：__（对日主的真财库地支）
  真财库何时到：__（大运/流年）
  虚库 vs 真财库的配合：__（同步还是先后）
```
