## 🔴 铁律：断事前必须先做的三件事（杜绝硬凑）

**这是本技能的第一原则，也是金鉴真人在魏启令案中付出惨痛代价后才学会的法则。**

> 🚨 **老板原话**："我觉得你现在就是硬凑——根据我提供的一些事实去硬凑断事。不行的。你必须先根据理论实打实地研究清楚，根据大运流年理论知识架构整体框架来断，而不是我输入一些东西你来硬凑事实。"

**分析八字不是「用户给事实→我去解释」的对答案游戏，而是「我先搭理论框架→用户用事实来验证」的科学方法。**

### 第一步：已知事件 → 反推时间线（全生命周期覆盖）

> ⚠️ **覆盖范围要求**：流年断事必须覆盖从出生到80岁的**全生命周期**，而非仅截取最近10-20年。只做部分年份会丢失关键的大运转换点和重要犯太岁年份。

```
拿到八字 + 命主已知信息（出生年/职业/重大事件）
    ↓
反推命主完整的时间线（每一年在社会中的身份角色）：
    ↓
覆盖全生命周期：出生 → 童年(1-20) → 青年(21-40) → 中年(41-60) → 老年(61-80)
    ↓
按大运分段：每10年一段，先写大运基调(大运干支与本命局的互动判断能量状态)
    ↓
逐年分析：重点年份按五步法详细展开，普通年份1-2句带过(按正常轨迹基线)

约龄参考：
  6-7岁 → 小学        15-18岁 → 高中
  12-15岁 → 初中       18-22岁 → 大学
  22-23岁 → 大学毕业   25岁+ → 职场人

⚠️ 特殊学制（复读/硕博等）通过已知事件校准

确定每一年社会身份：
  IF 在学 → 以学生角色分析
  IF 在职场 → 以职场人角色分析
  IF 在婚育期 → 以家庭角色分析
```

#### 评分方法速查（只算加分不加不减）

> 🚨 **2026-06-15修正**：原文明确「只算加分就行了，不用算减分」。之前写成「生助加分，消耗减分」是错的。
> 正确做法：只算印+比劫的得分，食伤/财/官杀不扣分。详见 bazi-foundation-analysis §评分方法

```yaml
【位置基础分】（满分100分，日干不计）
  年干=8分  年支=4分
  月干=12分 月令=40分
  日支=12分 时干=12分 时支=12分

【藏干百分比】
  本气=100%×位置分 | 中气=60%×位置分 | 余气=30%×位置分
  天干不分藏干，直接按位置分

【计分逻辑（只算加分！）】
  只看生助日主的五行（印+比劫）：
  ✅ 比劫在所有位置都算（天干+地支藏干本/中/余气均计）

  ❌ 食伤/财/官杀 → 不扣分（不加不减）

【判定标准】
  >60分 → 身强
  40-60分 → 中和
  <40分 → 身弱
  得分为0分（全无印比）→ 从弱反为强，恒定50分
  得分>100分（满盘印比）→ 从强反为弱，恒定20分
```


### 6.3 格局深化·官杀担力精确规律（家主子源实战校准·2026-06-16）

> **⚠️核心认知**：正官和七杀是两回事——正官是温和的规则压力，七杀是凶暴的生存压力。
> 能担两个正官的人，不一定能担一个七杀。
>
> **实战案例**：
> - **子源**（丙火55分+双癸水正官）→ 舒适区内的承受，表现为**高标准自律**
> - **家主**（辛金64分+丁火七杀+亥水伤官）→ 超出舒适区的承受，表现为**极苦历练**
>
> **边界条件**：杀印相生/食神制杀等特殊格局可豁免此规则（§7.10特殊豁免规则）

#### 核心公式

```
官杀担力 = 身强分数 ÷ 40分（基准中和线）

正官担力阈值：1.0倍 = 单正官刚好扛住（正常压力）
             1.3倍 = 双正官刚好扛住（自律，不出问题）
             
七杀担力阈值：1.2倍 = 单七杀刚好扛住（有压力但可控）
             1.5倍 = 双七杀勉强扛住（会非常痛苦）

  ❗ 正官和七杀不能简单相加——一个七杀 ≈ 两个正官的扛力需求
```

#### 关键发现

```yaml
【七杀 vs 正官的核心差异】
  正官是规则压力 → 有标准、有方向、知道该怎么做
  七杀是生存压力 → 被威胁、被挑战、不知能否扛住

  同一分数下，扛七杀远比扛正官困难：
  55分扛双正官 → 只是自律，并不痛苦
  55分扛单七杀 → 已经喘不过气了
  64分扛双正官 → 轻松驾驭
  64分扛单七杀 → 有压力但可控
  64分扛七杀+伤官 → 极苦（家主的丁亥运）

【为什么七杀这么难扛？】
  七杀的本质是「克身者，阴阳相同」— 同性相斥所以更猛烈
  正官的本质是「克身者，阴阳不同」— 异性相吸所以更温和

  → 同样的身强分数，扛七杀需要多付出30%~50%的能量
```

#### 通用断语规则

```yaml

  □ 只有七杀 → 走「生存压力」路线
  □ 官杀混杂 → 取压力最大的那个为主

  □ 正官扛力 = 身强分数 ÷ (40 × 正官个数) ≥ 1.0 → 能担
  □ 七杀扛力 = 身强分数 ÷ (40 × 七杀个数 × 1.4) ≥ 1.0 → 能担
  （七杀乘以1.4系数，因为七杀难度约是正官的1.4倍）

  例：身强55分+双正官 → 55分足以覆盖双正官的需求
  → 因为「双正官」需要的分数约在50-55分即可

  → 单七杀需要的分数约在55-60分，64分能扛但苦

简化口诀：
  50分=单正官底线，55分=双正官或单七杀底线
  60分=双七杀/七杀+其他用神底线
```

### 7.2 财星评分量化公式（统一规则·2026-06-16修正）

> 🚨 **之前写的是旧版规则（"财星得令+40分""地支有相同五行+4分"等），与引擎不一致，导致分数变来变去。**
> **修正后的规则与身强弱评分使用同一套位置基础分体系，确保评分一致性。**

#### 评分规则（与身强弱同体系）

```yaml
【位置基础分】（与身强弱评分完全一致）
  年干=8分  年支=4分
  月干=12分 月令=40分
  日支=12分 时干=12分 时支=12分

【藏干百分比】（与身强弱评分完全一致）
  本气=100%×位置分 | 中气=60%×位置分 | 余气=30%×位置分
  天干不分藏干，直接按位置分

【计分逻辑】
  天干财星：天干是正财/偏财 → 直接加该位置的基础分
            例：年干辛金正财 → +8分
            例：时干癸水偏财 → +12分

  藏干财星：地支藏干是正财/偏财 → 位置基础分 × 藏干百分比
            例：月令巳中庚金偏财(余气30%) → 40×30%=12分
            例：日支亥中甲木正财(中气60%) → 12×60%=7.2分
            例：年支卯中乙木正财(本气100%) → 4×100%=4分

【判定标准】
  原局财星总分仅反映命局中财星的客观存在量
  不等于「能赚多少钱」——还要看身强弱（能否担财）+大运配合
```

#### 财星分数与身强弱的关系

```yaml
身强(>40分) + 财星高分 → 能担财，财富等级高
身弱(<40分) + 财星高分 → 财旺身弱，财来财去（需印比大运帮身）
身强(>40分) + 财星低分 → 有能力但财运有限（需大运补财）
身弱(<40分) + 财星低分 → 财弱身弱，普通水平

⚠️ 格外注意：
  财星分数 ≠ 财富等级的直接值
  财富等级 = 财星分数 × 身强系数 × 食伤生财路径 × 大运杠杆
```

#### 财星评分示例

```yaml
家主（庚申 癸未 辛亥 辛卯，辛金身强64分）：
  月支未藏乙木偏财30%=40×30%=12分
  日支亥藏甲木正财60%=12×60%=7.2分
  时支卯藏乙木偏财100%=12×100%=12分
  → 财星总分 = 31.2分

子源（辛卯 癸巳 丙戌 癸巳，丙火中和55分）：
  年干辛金正财=8分
  月支巳藏庚金偏财30%=40×30%=12分
  日支戌藏辛金正财60%=12×60%=7.2分
  时支巳藏庚金偏财30%=12×30%=3.6分
  → 财星总分 = 30.8分
```

#### 常见错误（以前犯过的）

```yaml
❌ 把月令藏干按12分基础算（实际应为40分）
   例：月令巳中庚金偏财30% → 不是12×30%=3.6分，是40×30%=12分

❌ 漏算年支的财星（年支基础分4分，往往被忽略）
   例：年支卯中乙木正财100% → 4×100%=4分

❌ 漏算天干透出的财星（直接加位置分）
   例：年干辛金正财 → +8分

✅ 统一规则：与身强弱评分完全一致的位置基础分+藏干百分比
```

**口诀：**
```
财星评分不靠猜，位置基础定下来
年干8分年支4，月干12月令40
日支12时干时支各12，藏干百分百算清楚
本气全分中六成，余气三成不能忘
天干直接加位置，别把月令当日支
```

**财的真假（实战关键认知）：**
```

判断：透干之财需在地支有根才为真；无根则虚
```

```
  → 戌为比肩库 → 财富等级比一般人高

身弱→财来财去（身弱不担财），喜见印枭比劫

各类组合的财运逻辑：
```

### 7.4 发财方法详解

#### 方法A：身强财旺法（基础法）

操作逻辑：
1. 看日元能量是否在40-60分之间（或通过大运流年调到这个区间）
2. 看财星能量是否足够（得令加分、周围有助）
3. 看大运流年中是否有财星/食伤星来助
4. 三条件同时满足 → 发财时机到

大运流年量化加分规则：
- 大运一运管10年，前5年天干主导（+10分），后5年地支主导（+10分）
- 流年再加分（+5~10分）
- 超过60分 → 多出来的比劫去克财 → 破财

#### 方法B：五鬼运财法（偏财神煞）

**推算方法**：月令向后推五位（掐左手手指）

| 层级 | 持续时间 | 效果 |
|:---|:---:|:---|
| 流月五鬼运财 | 1个月 | 小笔偏财，不大 |
| 流年五鬼运财 | 1年 | 有可观偏财，种子生根开花结果 |
| 大运五鬼运财 | 后5年 | 持续5年偏财不断，可能创造巨额财富 |

**五鬼运财吉凶判断**：

| 五鬼运财遇到 | 性质 | 结论 |
|:---|:---:|:---|
| 食神/正印/正官/财星 | ✅ 吉 | 得财有福，没事 |
| 七杀（五鬼杀） | ⚠️ 凶 | 得财同时犯小人/得病/官非 |
| 偏印/枭神（五鬼枭） | ⚠️ 凶 | 得财但不开心，被人陷害 |
| 伤官（五鬼伤） | ⚠️ 凶 | 得财伴随口舌官司纷争 |
| 劫财（五鬼劫） | ⚠️ 凶 | 得了财又被劫走，等于没得 |

**本质**：意外之财，不是辛苦努力赚来的。代表找到赚钱"窍门"，人一开窍钱就好赚。

#### 方法C：禄神发财法

- 日元在地支的比肩为禄（甲的禄在寅，乙的禄在卯，丙戊在巳，丁己在午，庚在申，辛在酉，壬在亥，癸在子）
- 遇禄年/禄运 → 财运好
- 临官又称禄神，是人生最好的状态之一

#### 方法D：墓库发财法

##### 财库量级体系（年=亿/月=千万/日=百万/时=十万）

```yaml
财库所在宫位决定财富量级：
  年柱财库 → 亿级（祖上积累与自己关系不大）
  月柱财库 → 千万级（父母/家族背景）
  日支财库 → 百万级（自己打拼，最实际的财库）
  时柱财库 → 十万级（子女/晚年）

⚠️ 这是基准量级，还要结合格局、大运杠杆、时代红利调整
  
实战口诀：
  财库在年看祖业，财库在月看父母
  财库在日靠自己，财库在时看子女
  日时财库最得力，年月的财库听听而已
```

##### 开库条件（辰戌丑未冲才叫开库）

```yaml
开库 = 辰戌冲 或 丑未冲 → 库门打开 → 财富释放
锁库 = 没有冲 → 库门关闭 → 财富存着但用不了

开库规则：
  ✅ 辰戌冲 → 水库火库同时开，能量最大（10倍土）
  ✅ 丑未冲 → 金库木库同时开
  ❌ 辰与辰/戌与戌 → 同字不冲，不开库
  ❌ 有合化优先 → 合化后不开库（贪合忘冲）

从弱八字特别看：
  从弱反为强 → 喜克泄耗 → 开库=忌神能量释放
  → 开库年份反而不好，需要具体分析
```

- **库**的定义：某五行能量（本命局+大运+流年）≥40分 → 入库（吉）
- **墓**的定义：某五行能量<40分 → 入墓（凶）
- **与发财相关的库**：仅财库和比劫库
- **与发财无关的库**：官杀库（升官）、印枭库（文凭证书）、食伤库（福气）
- **库的宫位规则**：
  - 财库在年支/月令 → 祖上/家族有钱，与自己关系不大
  - 财库在日支/时支 ✅ → 自己有钱，财富等级高一档
- 墓库的五行对应：丑=金库、辰=水库、未=木库、戌=火库（也兼土库）

### 7.5 赚钱合作模式（身强 vs 身弱天壤之别）

#### 身强之人的三种赚钱模式（以财/官/食为喜用）

| 模式 | 核心 | 合作对象 | 操作方式 |
|:---|:---|:---|:---|
| 财为用 | 出项目 | 有钱老板 | 自己出项目/技术（身股），对方出资金（银股），对方有查账/分红权但无管理权 |
| 官为用 | 驭上 | 领导/政府 | 与领导搞好关系，利用领导手中的资源求财，适当让出股份 |
| 食伤为用 | 驭下 | 下属/员工 | 分股份给员工，让他们帮你管理；或学门技术靠手艺吃饭 |

**身强绝对禁忌**：❌ **不可与朋友/平辈合作** → 合作会被抢夺财产（比劫为忌）

#### 身弱之人的两种赚钱模式（以印/比为喜用）

| 模式 | 核心 | 合作对象 | 操作方式 |
|:---|:---|:---|:---|
| 比为用 | 合伙 | 朋友/同辈 | 与人合作共谋发展，依靠别人的能力和实力沾光 |
| 印为用 | 用师 | 贵人/师长 | 尊奉贤能之人为老师，借老师智慧成就大功 |

**身弱遇比劫 = 内劫（劫别人的财）** — 如打麻将赢三家钱
**身强遇比劫 = 外劫（被别人劫财）** — 如合伙被朋友坑

> 经典名句（曾子）：**用师者王，用友者霸，用徒者亡**

## 8. 断行业逻辑

### 8.1 核心原则

- **格局定方向**：正官格→管理/公职，偏财格→生意，食神格→餐饮/文艺，印格→教育/服务
- **五行定行业**：喜用五行对应的行业最容易成功

## 11. 断疾病决策树

### 11.2 疾病诊断流程（实战完整版）

#### 核心原理：五行能量失衡

```yaml
某个五行能量超过60分 = 太强
  → 多出来的能量无处安放 → 去克其他五行 → 导致被克的五行出问题
  → 例子：土68分→克水→肾/妇科问题；土本身代表脾胃→脾胃不好

某个五行能量低于20分 = 太弱
  → 该五行对应的器官功能弱 → 容易出问题

最佳状态：40-50分（平衡状态，不易得病）

五行超过3个以上 → 该五行能量过强 → 容易导致对应器官疾病
  → 金太多(3+个)→肺/鼻子/咽炎/咳嗽/哮喘
  → 土太多→脾胃/消化
  → 水太多→肾/膀胱
  → 火太多→心脏/血液/眼睛
  → 木太多→肝胆/神经

⚠️ 「超过3个的五行」包含日元在内（疾病诊断时

  → 例：金5个→太强→金主肺→肺与鼻子相表里→鼻炎/哮喘/咽炎/气虚
  → 例：金多火灭→反克（金太强可以把火灭了）
     火本来克金，但金太多→火克不动→反被金克→火受伤
  → 例：强金宜泄不宜克
     用火去克强金→克不动，反被伤
     用水去泄强金→金生水，把能量卸掉→水为调和
     强金能量太大，补火不如补水

燥湿平衡（疾病诊断的关键视角）：
  燥湿不平衡 → 身上容易有炎症
  炎症 → 血液流速变慢 → 血液出问题 → 高血压/糖尿病/痛风
  火太旺 → 燥热体质 → 阴虚火旺 → 血热 → 血液疾病
  水太旺 → 寒性体质

⚠️ 地支藏干的水≠五行流通：
  八字有藏干之水只能计算能量（例如24分），但无法流通
  无法流通=水起不到金生水·水生木的作用
  所以八字不缺水 ≠ 五行流通好
```

```yaml

  七杀为实疾之病痛 → 具体、实在的疾病
  七杀在胆→胆囊息肉/胆囊炎
  七杀在肝→肝病
  七杀在乙木→胆
  七杀在甲木→肝
  七杀在地支某宫位→该宫位对应的器官有病

  超过60分就出问题
```

```yaml
  位置：胸部（锁骨~胸之间）戴青金石或黑曜石
```

## 12. 流年断事方法

### 🔴 实战断事铁律（V3方法论 · 2026-06-01核心升级）

> 🚨 **方法论的根本修正**：之前把流年断事当成了「列清单→评分→公式算等级」的公式游戏，这是错的。
> 实战断事是**读命→找核心→给具体断语**的过程。不是你的公式有多全，而是你能不能读出这个八字的本质。

#### 7个要点（必须同时满足）

```yaml
要点① 分时段断事（上半年/下半年分开）— 同五行在不同时段喜忌可能完全相反
要点② 先判断今年命局的真实状态 — 大运+流年可能改变身强弱，不能沿用静态结论
要点③ 三刑=藏干打架=能量加强，不是扣分 — 被加强的字喜则吉、忌则凶
要点④ 断语必须包含六亲 + 身体部位 + 应期月份 + 严重程度
   错误："注意健康" ❌   正确："父亲注意心脏血压，阴历十一月应事" ✅
要点⑤ 宫位是断语的起点 — 找到被作用的字 → 它在哪个宫位 → 直接出断语
要点⑥ 大运+流年组合不同 → 年内可能发生状态切换，断语必须对应调整
要点⑦ 先找1-2个核心关系 — 不是列200条，核心关系决定80%的断语
```

#### 断事前必答自查清单

```yaml
□ 这个八字今年处于什么状态？（大运+流年作用下的真实状态）
□ 今年的核心关系是什么？（冲/合/三刑中的哪一个？）
□ 这个关系能被化解吗？（贪合忘冲？大树理论？）
□ 被作用的字在哪个宫位？（六亲是谁？身体哪个部位？）
□ 今年需要分时段吗？（上半年/下半年状态是否不同？）
□ 断语有4要素吗？（六亲+身体+应期+程度）
□ 我是在「读命」还是在「套公式」？
```

#### 常见错误自查

```yaml
❌ 列200条关系 → 只需1-2个核心
❌ 评分定等级 → 先定状态再看能量
❌ 笼统断语   → 必须有4要素
❌ 三刑当扣分 → 三刑是能量加强
❌ 一年一个断 → 分上半年/下半年
❌ 静态用神   → 命局状态年内可切换
```

### 12.1 核心框架：三力分工模型 + 九龙道长过三关法

> ⚠️ **重要更正**：九龙道长原始素材中不存在「断事七步法」或「断事八步法」。之前的版本是我自创的，不准确。九龙道长的实战方法叫「过三关」（只说三件事）。详见 bazi-liunian-analysis §12.1.0。

### 12.1.1 三力各司其职

```
本命局 = 出厂设置 — 先天能量烙印（八字8个字）
  ↓ 不动
大运 = 修改本命局能量 + 修改本命局格局 + 参与刑冲合害
  ↓ 改变状态但不硬事
流年太岁 = 60年轮一次，肆意发挥权力
  ↓ ✅ 唯此应事——决定好事坏事何时发生

记忆口诀：
  本命局是出厂设置，大运是系统升级
  流年是新上任的总统，一换人就换政策
  大运改能量改格局，流年才真正搞事情
```

#### 12.1.2 流年太岁本质

```
流年太岁 = 天上值班的星星，每个流年地支就是当年的"总统"
60甲子一轮回，好不容易轮到它值班
→ 它要肆意发挥手里的权力
→ 它制定这一年的"政策"
→ 它与你的八字产生互动 → 你就"犯太岁"
```

### 12.2 犯太岁两大类别

#### 第一类：刑冲破害 → 多为凶

| 类型 | 能量扣分 | 比喻 | 典型事件 |
|:---|:---:|:---|:---|
| **六冲** | -70分（最重） | 直接猛烈冲突 | 激烈矛盾、突发灾祸、车祸 |
| **六害（六穿）** | -60分 | 利刃刺穿身体 | 突发疾病、小人陷害 |
| **六破** | -20分（几乎不断） | 撕破脸 | 吵架、破小财 — 一般不作为断事依据 |

**冲的核心规则**：
```
流年冲八字 → 只有八字受伤，流年（天上的星星）不可能受伤
冲走忌凶（不好的能量）→ 好事
冲走喜用（需要的能量）→ 坏事
有合化则贪合忘冲 ↔ 小公狗打架，来了小母狗就不打了
```

**害的核心规则**：
```
害 = 穿 = 拿利刃刺穿身体
能量仅次于冲，代表突发疾病或突发灾祸
```

#### 第二类：合化 → 吉凶看喜忌（合出什么五行）

| 合化类型 | 无天干引化 | 有天干引化 |
|:---|:---:|:---:|
| 三会（寅卯辰等） | 10倍 | 20倍 |
| 三合（寅午戌等） | 7倍 | 15倍 |
| 半三合/六合 | 5倍 | 10倍 |

```
合化本质：加强合化出来那个五行的能量

合出喜用 → 大好事（比如合出财+身强=发财1.2亿案例）
合出忌凶 → 大坏事（比如合出七杀且制不住=母亲去世案例）

关键判断：合出来的五行，是命主喜用还是忌凶？
```

### 12.3 宫位·身体·六亲对应体系

```
年干（1-9岁）→ 头顶~鼻子       → 父系家族
年支（10-18岁）→ 鼻子~锁骨（牙/颈/肩） → 母系家族
月干（19-27岁）→ 锁骨~胸部     → 父亲/兄姐
月令（28-36岁）→ 胸部~肚脐     → 母亲/弟妹
日支（37-45岁）→ 小腹/生殖器官  → 配偶
时干（46-54岁）→ 大腿          → 长子/长女
时支（55岁+）  → 小腿/脚       → 次子及后

身体对应记忆法：一扎一个宫位，从头顶往下
  年干→鼻梁    年支→锁骨
  月干→胸      月令→肚脐
  日支→小腹/胯  时干→大腿
  时支→小腿/脚

六亲双重宫位：
  年干=父亲，月干也有父亲信息
  年支=母亲，月令也有母亲信息
```

### 12.4 应灾轻重判定

```
要素一：被冲/害的字有没有强根？
  ├─ 有强根 = 百年大树 → 抵抗8级风 → 应灾轻，死不了
  ├─ 无根/孤零零 = 小树 → 6级风就倒 → 应灾严重
  └─ 特别：天干强根比地支根弱

要素二：有没有同类分担？
  ├─ 1个流年字冲1个八字字 → 100%能量 → 应灾重
  ├─ 1个流年字冲2个八字字 → 各50% → 应灾轻
  └─ 案例：寅害两个巳→每个分50%→腿不舒服但不严重

要素三：原局是否有合化解开？
  ├─ 有合化则贪合忘冲/忘刑 → 冲不成了
  ├─ 有多个合化，流年是大哥 → 可能冲开原局合化
  └─ 流年最大，流年说了算
```

### 12.4.1 贪合忘冲 vs 流年冲破原局 — 边界规则

> ⚠️ 这两个规则看似矛盾，实际上各有适用范围，核心区别在于**谁是合化的发起方**。

#### 适用场景判定

```
场景A：流年某字同时有「冲」和「合」两种选择
  → 优先合化，不去冲了 = 贪合忘冲
  → 例：流年壬冲丙，但有丁壬合 → 壬跟丁走了，不冲丙了
  → 本质：流年自己的选择

场景B：原局有稳定合化，流年来冲/害破坏它
  → 流年憋了60年，它可以冲开原局合化 = 流年是大哥
  → 例：原局巳酉丑三合，流年寅来害巳 → 冲开三合
  → 本质：流年主动攻击原局
```

#### 一句话区分

```
贪合忘冲 = 流年自身的选择（内部优先级）
流年破局 = 流年对外界的干预（外部破坏力）
两回事，不矛盾。
```

### 12.5 流月应事方法（何时发生）

```
天干应事法：
  某个天干在流月"透干"（能量加强）→ 那个月应事
  例：壬冲丙 → 亥月（亥子丑三会+壬水强根）→ 阴历11月应事
  例：寅害巳 → 甲辰月（甲木透干加强寅）→ 阴历3月应事

节气交割法：
  节气交割前后3天 → 能量突变，身体不适应 → 精确到日
  上半年：阳历每月5号和19号交气（前后3天）
  下半年：阳历每月6号和21号交气（前后3天）
  例：阴历十月中（阳历11月21日±3天）→ 进入亥月→能量爆发

全年看：流月引化流年天干/地支 → 该月才真正应事
```

### 12.6 大运流年时间权重

> ⚠️ **这是流年断事的核心基础规则之一。不懂这个，断事必然偏差。**
> 大运的10年不是平均分配的——天干地支各管一段，各有70%的话语权。

#### 12.6.1 大运能量配比

```yaml
大运10年生命周期：
  ┌─── 前5年 ───┬─── 后5年 ───┐
  天干70% + 地支30% │ 地支70% + 天干30%
  └───────────────┴───────────────┘

实战含义：
  前5年 → 大运天干是主角，看天干五行与八字互动来决定前5年基调
  后5年 → 大运地支是主角，看地支五行与八字互动来决定后5年基调

举例：
  大运戊子 → 前5年戊土主导（重印/官），后5年子水主导（重劫财/食伤）
  大运己丑 → 前5年己土主导，后5年丑土主导（同样是土但燥湿不同）
```

#### 12.6.2 流年能量配比

```yaml
流年12个月生命周期：
  ┌─── 前半年（立春~立秋）─┬─── 后半年（立秋~立春）─┐
  天干70% + 地支30%       │ 地支70% + 天干30%
  └────────────────────────┴────────────────────────┘

实战含义：
  前半年 → 流年天干的影响力更大，先看天干互动
  后半年 → 流年地支的影响力更大，再看地支互动
  但全年来看，天干地支都会发挥作用，只是权重不同

节气切换点：
  立春（2月4日±1天）→ 新一年开始
  立秋（8月7日±1天）→ 前半年→后半年切换
  节气交割前3天后3天 → 能量突变期，最容易应事
```

#### 12.6.3 综合运用法则（金鉴实战）

```yaml
断某一年的关键步骤：
  ① 先定这一年在大运中的位置
     → 前5年？还是后5年？
     → 前5年：优先看大运天干与流年的互动
     → 后5年：优先看大运地支与流年的互动
  ② 再定这一年在流年中的位置
     → 前半年：优先看流年天干与大运/八字的互动
     → 后半年：优先看流年地支与大运/八字的互动
  ③ 大运的*第一年*（刚换运）和*最后一年*（准备换运）
     → 第一年：新大运的能量刚到，旧大运还有惯性 → 7折
     → 最后一年：准备交接给下个大运 → 6折

简化口诀：
  大运十年分两半，前看天干后看支
  流年一年分两半，前看天干后看支
  换运首年打七折，末年打六折不要忘
```
