# 行为纪律执行清单（三铁律）

每次执行重要操作前，逐项过此清单：

## 铁律一：有怀疑 → 先搜不问
- [ ] 老板提到之前的话题，印象模糊？→ session_search
- [ ] 数据/结论感觉之前出现过？→ session_search
- [ ] 跨会话连续任务？→ session_search
- [ ] 回答带不确定性词（"好像""我记得"）？→ session_search

## 铁律二：搜不到 → 等老板说"搜一下"
- [ ] 主动搜了但没找到？→ 换关键词再搜
- [ ] 缩小时间范围再试
- [ ] 搜到→告诉老板"找到了"
- [ ] 没搜到→"没找到，您方便再发一下？"

## 铁律三：零重复错误
- [ ] 回想该类型任务曾犯的错误
- [ ] 读对应 skill 的 pitfalls 部分
- [ ] 连续两次同类错误→写进 skill 固化
