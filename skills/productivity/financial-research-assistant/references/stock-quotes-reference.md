## OCR提取屏幕持仓数据的已验证流程

当老板发送持仓截图时：

1. 安装 tesseract：`apt-get install tesseract-ocr tesseract-ocr-chi-sim -y && pip install pytesseract`
2. 图片缩小到宽度600再OCR（原图通常1260+宽，太大字体识别模糊）
3. lang参数用 `chi_sim+eng`（中英文混合）
4. 多张截图表示多账户，需要分别提取后合并

## 注意事项

- 以上股价数据为特定采集日参考，每日晨报需重新获取最新数据
- 买入价为老板口头告知的参考成本，非精确交易记录
- 港股市场显示的成本价为2020年12月11日后买入交易计算，之前持仓按当日收市价计算，可能与实际成本有出入
