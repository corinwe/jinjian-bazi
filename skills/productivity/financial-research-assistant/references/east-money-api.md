# 东方财富API — A股实时数据获取参考

## Base URL
```
https://push2delay.eastmoney.com/api/qt/stock/get
```

## 常用参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `secid` | 市场代码.股票代码 | `0.002594`（深市）, `1.000001`（沪市） |
| `fields` | 需要返回的字段 | 常用：`f43,f44,f45,f46,f47,f48,f50,f51,f52,f57,f58,f60,f169,f170` |

## 字段含义

| 字段 | 含义 | 说明 |
|:----:|:----:|:----:|
| f43 | 当前价 | 整数，需÷100；指数直接为点数×100？ |
| f44 | 当日最高 | 同上 |
| f45 | 当日最低 | 同上 |
| f46 | 开盘价 | 同上 |
| f47 | 成交量 | 手（股票）/ 手（指数） |
| f48 | 成交额 | 元 |
| f50 | 量比 | 百分比数字 |
| f57 | 代码 | 字符串 |
| f58 | 名称 | 字符串 |
| f60 | 昨收 | 同上÷100规则 |
| f169 | 涨跌额 | 整数，实际值÷100 |
| f170 | 涨跌额(备用) | 同上 |

## A股重要股票secid映射

| 股票 | secid |
|:----|:-----:|
| 比亚迪 002594 | `0.002594` |
| 美的集团 000333 | `0.000333` |
| 上证指数 000001 | `1.000001` |
| 深证成指 399001 | `0.399001` |
| 沪深300 000300 | `1.000300` |

## 实际使用示例

```bash
# 获取比亚迪实时数据
curl -sL "https://push2delay.eastmoney.com/api/qt/stock/get?secid=0.002594&fields=f43,f44,f45,f46,f47,f48,f50,f51,f52,f57,f58,f60,f169,f170" \
  -H "User-Agent: Mozilla/5.0"

# 返回示例（简化）：
# {"data":{"f43":9860,"f44":9925,"f45":9781,"f60":9923,"f169":-63,"f57":"002594","f58":"比亚迪"}}
# -> 当前价 ¥98.60, 昨收 ¥99.23, 涨跌 -¥0.63 (-0.63%)
```

## 注意事项
- push2.eastmoney.com 会重定向到 push2delay.eastmoney.com，需用 `-sL`
- 免费接口，可能有延迟（约15分钟延迟？盘中验证约1-3分钟延迟）
- 不可用于港股（HKEX数据不同源）
- 200+ 请求/分钟可能被限流，建议间隔1秒以上
