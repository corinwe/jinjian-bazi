## 每次财富判断的8步流程

```
链条判定 → 五级定级 → 发财五法 → 破财检查
       → 合作伙伴 → 五层动态 → 补文昌 → Checker验证
```

## 发财五法（老板校准为5法，非4法）

| # | 方法 | 说明 |
|:-:|:----|:-----|
| ① | 身强财旺 | 原局已满足，直接发 |
| ② | 遇库则发 | 库被引化 |
| ③ | 五鬼运财 | 月令后退五位 |
| ④ | 遇禄神则发 | 流年地支=日主禄神 |
| ⑤ | 大运流年补足 | 原局不满足→等大运流年补→分6种状态 |

## 第⑤法六种状态（最常用）

身强财旺→遇食伤财才发大财 | 身弱财旺→遇印比劫发大财
身强财弱→遇食伤财才发中财 | 身弱财弱→遇印比劫发中财
无财身弱→和尚命 | 无财身强→短暂小财

## 大运开库检查（常见遗漏）

每次分析大运对财富的影响，必须检查：
**大运地支冲/合/刑/害月支** → 若月支是财库则冲开爆发
（己丑运丑未冲案例教训）

## 合作伙伴

身强→以财官食为喜用（财=有钱人/官=领导/食=下属技术）
身弱→以印比为喜用（比=朋友合伙/印=贵人师长）
天乙贵人合作永远是对的
