# 2026-07-07 大运喜忌判断 + 配偶特征 错误修正记录

## 错误一：丙戌大运 喜忌误判

### 错误描述
家主v28.0报告将丙戌大运(20~30岁)标记为"双忌神(正官+正印)"。

### 根因
误将丙火(正官)归为忌神，但引擎显示喜用神为['木','水','火']，丙火=火=喜用神。

**正确判定：**
```
丙 = 正官 = 火 = 喜用神 ✅
戌 = 正印 = 土 = 忌神 ✅
→ 一喜一忌(喜正官+忌正印)，非双忌神
→ 体验：官制身(想做事)+印保守(放不开)=纠结运
```

### 修复
- 报告：`丙戌(20~30岁)` 从"双忌神"修正为"一喜一忌(喜正官+忌正印)"
- 模板：bazi-report-template §17 新增铁律「每步大运干支分别判定喜忌，不可笼统说'纯忌神'或'双忌神'」
- 关键数据速查表：丙戌从"纯忌神运"修正为"一喜一忌（非纯忌）"

### 教训
**每步大运的天干和地支必须分别判定喜忌**，不能笼统说"纯忌神运"或"双忌神运"。身强辛金丙火(正官)是第三喜用，戌土(正印)才是忌神。

## 错误二：配偶特征"寅申巳亥"

### 错误描述
报告写"长相一般（寅申巳亥）"，但命主八字地支为申/未/亥/卯，寅和巳不在局中。

### 根因
把四长生(寅申巳亥)全类列出的写法——虽然日支亥确实属于寅申巳亥类，但罗列全类会让读者困惑"哪里来的寅和巳？"

### 修复
修正为"长相一般（日支亥为四长生之一）"，只写实际存在的那个地支。

### 教训
配偶特征描述只列存在于命局中的具体地支，不列全类。写"酉"不写"子午卯酉"，写"亥不写寅申巳亥"。

## 错误三：大运年龄浮点显示

### 错误描述
引擎输出和报告显示"0.3~10.3岁"等浮点年龄区间。

### 根因
引擎用 `round(qi_yun_age + step * 10, 1)` 保留浮点小数。

### 修复
改为 `math.ceil(qi_yun_age)` + 整数区间：
- 0.33岁→1岁起运
- 第1步: 1~10岁, 第2步: 11~20岁
- 引擎修复: da_yun.py lines 178-182
- 模板: bazi-report-template 新增「起运年龄显示」铁律

### 教训
起运年龄向上取整是常识（九龙道长规则），每个命理师都知道。写报告时不应用浮点小数显示大运年龄区间。
