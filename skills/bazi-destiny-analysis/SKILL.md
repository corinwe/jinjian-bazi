# 八字命理深度分析 Skill

## 核心能力

1. **八字排盘与格局判定**：严格按月令透干规则判定正八格/杂气格/特殊格局
2. **日主强弱分析**：金鉴真人原始评分规则（月令印40分/天干+地支印比计分）
3. **五行十神分析**：藏干十神逐字推导，确认正偏阴阳无误
4. **大运流年推演**：推算大运并标注关键流年
5. **事象深度分析**：财富/事业/婚姻/子女/健康/灾祸逐项基于引擎数据展开

## 🚨 格局判定规则（每次写§2前必须过此规则）

### 判定流程

```
第一步：看月令藏干
  月令地支 → 本气藏干 → 中气藏干 → 余气藏干

第二步：检查透干
  本气是否在天干中出现？（年/月/日/时四柱）
  ├─ 是 → 以本气十神定格局 ✅
  └─ 否 → 中气或余气是否透干？
      ├─ 是 → 以透出者定格局 ✅
      └─ 否 → 杂气格 ⚠️

第三步：命名
  - 正八格：正官格/七杀格/正印格/偏印格/正财格/偏财格/食神格/伤官格
  - 月令不透：杂气格（月令藏干全不透天干）
  - 特殊格局：从格/化气格/伤官伤尽格等
```

### 实例对照

| 八字 | 月令 | 藏干 | 透干情况 | 正确定格 |
|:----|:----|:------|:---------|:---------|
| 庚申 癸未 辛亥 辛卯 | 未 | 己(偏印)/丁(七杀)/乙(偏财) | 己丁乙全不透 | **杂气格** |
| 丁卯 丁未 庚午 壬午 | 未 | 己(正印)/丁(正官)/乙(正财) | 丁火透月干 | **正官格** |

### ⚠️ 常见错误（本会话致命教训）

```
❌ 错误1：把月干的十神当格局名
   例：月干癸水(食神) → 叫"食神配印格"
   ✅ 正确：必须看月令藏干是否透出，月干≠月令透出

❌ 错误2：月令不透时硬安一个正八格名
   例：未月己不透 → 叫"偏印格"
   ✅ 正确：全部不透就是杂气格

❌ 错误3：混用食神/伤官
   例：癸水对辛金是食神（阴生阴），不是伤官
   ✅ 正确：逐字推导十神，不凭记忆
```

## 📋 标准分析流程（配合bazi-paipan-sop执行）

### Phase 0：数据准备
1. 加载技能：skill_view('bazi-engine-workflow')（引擎规则）
2. 加载技能：skill_view('bazi-report-template')（模板规则）
3. 加载技能：skill_view('bazi-calibration')（校准规则）
4. 确认引擎数据就绪（/tmp/{姓名}_engine.json）

### Phase 1：格局判定（严格按照上方规则）
1. 查月令地支 → 确认藏干
2. 查四柱天干 → 确认哪些藏干透出
3. 透干 → 正八格名；不透 → 杂气格
4. 引擎ge_ju字段仅作参考，以手动验证为准

### Phase 2：身强弱+喜用神
1. 从引擎result提取sec_3_shen_qiang_ruo
2. 确认喜用/忌神与身强弱逻辑一致
3. 身强喜克泄耗（财官食），身弱喜生扶（印比）

### Phase 3：大运分析（8步至约80岁）
1. 从引擎提取sec_17_da_yun_detail.list
2. 只取前8步（至约80岁），去掉81岁后的
3. 标注最佳/最差/现行大运

### Phase 4：事象分析（按模板§§展开）
1. 财富：财星评分+补财库方案（充库/蓄库/开库）
2. 婚姻：夫妻宫+配偶星+特征+桃花窗口
3. 子女：子女星+子女宫+出息程度+性别倾向
4. 性格：五重人格+十神底色
5. 文昌：仅≤25岁未成年人输出改进方案

### Phase 5：报告生成
1. 按bazi-report-template精简版（800~1000行）
2. 禁止废话，每句话有引擎数据支撑
3. 大运只到80岁
