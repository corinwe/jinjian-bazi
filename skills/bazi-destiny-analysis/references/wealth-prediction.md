## 财富预测核心逻辑

### 命理财富要素
财富预测基于以下命理要素综合分析：

| 要素 | 影响 | 权重 |
|-----|------|------|
| 财星状态 | 正财/偏财透干、藏支、有无根气 | 30% |
| 日主强弱 | 身强能担财，身弱财来财去 | 20% |
| 大运配合 | 是否走财运、印运护财 | 25% |
| 格局层次 | 格局越高，财富上限越高 | 15% |
| 神煞助力 | 禄神、天乙贵人、将星 | 10% |

### 财星与财富关系

**正财**：稳定收入、工资、固定资产
- 正财透干+身强：年收入可达数十万到数百万
- 正财藏支：稳定但数额一般
- 正财被劫：财来财去，存不住钱

**偏财**：投资收入、意外之财、商业利润
- 偏财透干+身强：投资眼光好，可能暴富
- 偏财藏支：偶有偏财，不稳定
- 偏财被劫：投资失利，不宜投机

**财多身弱**：
- 表现：八字中财星多但日主弱
- 结果：看得到钱但赚不到/存不下，为财所累
- 建议：用印比生身，增强自身能力后再求财

**身强财弱**：
- 表现：日主强但财星少
- 结果：能力强但赚钱机会少，或赚钱辛苦
- 建议：用食伤生财，发挥才华创造财富

**身财两停**：
- 表现：日主与财星力量相当
- 结果：能赚能存，财富稳定增长
- 这是最佳状态

## 财富量级评估模型

### 评估公式
财富潜力 = 命格基础财富 × 大运系数 × 时代系数 × 个人努力系数

### 命格基础财富（根据命格等级）

| 命格等级 | 基础财富区间（人民币） | 财富特征 |
|---------|-------------------|---------|
| 10级（极品） | 10亿以上 | 大富大贵，财富自由 |
| 9级（上等偏上） | 1-10亿 | 富豪级别，产业领袖 |
| 8级（上等） | 5000万-1亿 | 富裕阶层，财务自由 |
| 7级（中等偏上） | 1000-5000万 | 中产偏上，生活优渥 |
| 6级（中等） | 300-1000万 | 中产阶层，小康生活 |
| 5级（中等偏下） | 100-300万 | 普通中产，略有积蓄 |
| 4级（下等偏上） | 30-100万 | 温饱有余，积蓄不多 |
| 3级（下等） | 10-30万 | 勉强温饱，为钱奔波 |
| 2级（下等偏下） | 1-10万 | 贫困线附近 |
| 1级（极品凶） | <1万 | 极度贫困 |

### 大运系数
- 连续走用神运（尤其是财运）：系数1.5-2.0
- 用神运与忌神运交替：系数1.0-1.3
- 连续走忌神运：系数0.5-0.8

### 时代系数
- 改革开放初期（1980-2000）：系数1.2-1.5（机遇多）
- 互联网时代（2000-2020）：系数1.0-1.3
- 当前时代（2020-）：系数1.0-1.2
- 特殊机遇（如房地产黄金期、股市牛市）：额外+0.3-0.5

### 个人努力系数
- 顺势而为+努力：系数1.2-1.5
- 逆势而为+努力：系数0.8-1.0
- 随波逐流：系数1.0

### 校准调整
用户提供实际收入数据后，用实际值修正模型参数：
- 若实际收入 > 预测值：可能命局中有未发现的财富线索（如藏干财星、暗财）
- 若实际收入 < 预测值：可能大运配合不佳，或个人努力不足
- 根据偏差调整后续预测

## 身家过亿路径规划

### 适用条件
命格等级 ≥7级，且财星有力，才有可能身家过亿。

### 四大杠杆模型
身家过亿通常需要至少2-3个财富杠杆协同作用：

| 杠杆 | 说明 | 贡献度 |
|-----|------|--------|
| **工资积累** | 高薪职位的长期积累 | 20-30% |
| **股权变现** | 公司上市/并购的股权变现 | 30-50% |
| **投资收益** | 复利投资的长期增长 | 20-30% |
| **房产增值** | 核心城市房产的长期增值 | 10-20% |
| **配偶创业** | 配偶事业成功的家庭财富增量 | 5-15% |

### 路径规划步骤
1. **评估当前净资产**：确定起点
2. **确定目标年限**：通常需要20-30年
3. **选择主攻杠杆**：根据命格选择最适合的1-2个核心杠杆
4. **制定分阶段计划**：每5年一个里程碑
5. **标注关键时间节点**：大运交接年、用神到位年
6. **风险预案**：健康风险、行业风险、政策风险

### 命理禁忌（身家过亿的阻碍）
- **比劫旺者创业**：必破财，已验证案例需特别警惕
- **财多身弱投资**：越投资越亏，应先强身（提升能力）
- **七杀无制冒险**：高风险投资必败
- **伤官见官年变动**：此年不宜重大投资决策

## 社会名望层级评定

### 名望层级体系

| 层级 | 名称 | 特征 | 命理对应 |
|-----|------|------|---------|
| 1 | 无名 | 社会普通大众，无特殊影响力 | 命格低，无特殊格局 |
| 2 | 圈内知名 | 所在行业/领域内有一定知名度 | 有吉神（天乙、文昌） |
| 3 | 行业精英 | 行业内公认专家，有话语权 | 正官/七杀格成格 |
| 4 | 公众人物 | 跨行业知名度，媒体曝光 | 将星+官印相生 |
| 5 | 社会名流 | 广泛社会影响力，公众人物 | 从格/化气格+时代机遇 |

### 名望命理要素

**官星**：社会地位、职位、名誉
- 正官透干：体制内名望，按部就班晋升
- 七杀透干：体制外名望，靠能力打拼

**印星**：学问、文凭、贵人提携
- 正印旺：学术名望，靠知识立身
- 偏印旺：技术名望，靠专业能力

**将星**：领导才能、权威
- 将星到位：有领导力，能成团队核心

**文昌**：文才、考试、学术
- 文昌到位：学术/文化领域名望

**禄神**：福禄、衣食无忧
- 禄神坐命：稳定的社会地位

### 名望提升路径
根据命格特点选择名望提升路径：

| 命格特点 | 名望路径 | 达成方式 |
|---------|---------|---------|
| 官印相生 | 体制内晋升 | 考公/考编，按部就班升职 |
| 食伤生财 | 技术创业 | 技术创业，靠产品赢得名声 |
| 七杀有制 | 行业专家 | 在专业领域深耕，成为权威 |
| 伤官配印 | 学术文化 | 著书立说，靠作品立身 |
| 比劫成群 | 团队领袖 | 带团队，靠领导力成名 |

### 事业方向判定流程
1. 确定用神五行 → 对应行业方向

4. 结合命局禁忌 → 排除不适合方向

## 创业适宜性评估

### 创业适宜度评分

| 评估维度 | 适合创业 | 不适合创业 |
|---------|---------|-----------|
| 日主强弱 | 身强（能担财担官） | 身弱（经不起折腾） |
| 财星状态 | 偏财旺（敢冒险） | 正财弱（求稳） |
| 官杀状态 | 七杀有制（有魄力） | 正官旺（喜稳定） |
| 食伤状态 | 食伤旺（有创意） | 印旺（保守） |
| 比劫状态 | 比劫适中（有团队） | 比劫旺（分财） |
| 大运配合 | 食伤/财运（利创业） | 印/官杀运（利稳定） |

### 关键判定规则

**绝对不适合创业的信号**：
1. 比劫旺+比劫运：创业利润必被分走
2. 财多身弱：看得到机会但没能力抓住
3. 七杀无制：冒险必败
4. 印星太旺：保守被动，不敢冒险
5. 已验证两次以上创业失败：命局明确指向稳定工作

**适合创业的信号**：
1. 身强+偏财旺：能担财，敢冒险
2. 食伤生财：有创意，能变现
3. 七杀有制：有魄力，能控制风险
4. 大运走食伤/财运：时机配合

### 创业时机选择
即使命格适合创业，也要选对时机：
- 最佳时机：食伤运、偏财运、七杀有制运
- 次佳时机：正财运（稳健创业）
- 最差时机：比劫运（必破财）、印运（保守）、官杀混杂运
