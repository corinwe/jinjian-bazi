## 大运排法

### 起运岁数计算
1. 确定出生日期与最近节气的天数差
2. 阳年男命/阴年女命：顺排，从出生日算到下一个节气
3. 阴年男命/阳年女命：逆排，从出生日算到上一个节气
4. 3天=1岁，1天=4个月

### 阳年/阴年判定
- 阳年：甲、丙、戊、庚、壬年
- 阴年：乙、丁、己、辛、癸年

### 大运排列方向
| 性别 | 阳年 | 阴年 |
|-----|------|------|
| 男 | 顺排 | 逆排 |
| 女 | 逆排 | 顺排 |

### 顺排/逆排
- **顺排**：从月柱开始，按地支顺序排列（子→丑→寅→卯→辰→巳→午→未→申→酉→戌→亥）
- **逆排**：从月柱开始，按地支逆序排列（子→亥→戌→酉→申→未→午→巳→辰→卯→寅→丑）

### 大运数量
一般排8个大运，覆盖约80年人生。

### 大运示例
假设月柱为丙寅，男命阳年顺排：
- 大运1：丁卯（1-10岁）
- 大运2：戊辰（11-20岁）
- 大运3：己巳（21-30岁）
- 大运4：庚午（31-40岁）
- 以此类推...

## 大运吉凶判定

### 用神运 vs 忌神运
- **用神运**：大运天干/地支为命局用神，此十年整体吉利
- **忌神运**：大运天干/地支为命局忌神，此十年整体不利
- **闲神运**：大运五行对命局影响不大，此十年平平

### 大运与命局的作用关系
1. **大运天干**：主外在事件、社会关系、事业变动
2. **大运地支**：主内在状态、身体健康、家庭变化
3. **大运与命局天干**：合、冲、生、克，产生具体事件
4. **大运与命局地支**：合、冲、刑、害，影响根基和内在

### 大运吉凶判定表

| 大运十神 | 身强喜 | 身弱喜 | 一般影响 |
|---------|--------|--------|---------|
| 正官 | 吉（事业稳定） | 凶（压力太大） | 事业、名誉 |
| 七杀 | 吉凶参半（有魄力但风险） | 凶（压力过大） | 挑战、变动 |
| 正印 | 凶（过于保守） | 吉（贵人相助） | 学问、贵人 |
| 偏印 | 凶（孤僻多疑） | 吉（机智应变） | 技术、偏门 |
| 比肩 | 凶（竞争多） | 吉（朋友助） | 合作、竞争 |
| 劫财 | 凶（破财） | 吉（得助） | 财务、人际 |
| 食神 | 吉（才华发挥） | 吉凶参半 | 才华、福气 |
| 伤官 | 吉凶参半（创新但叛逆） | 凶（耗身） | 创造、变动 |
| 正财 | 吉（财运好） | 凶（身弱不胜财） | 正财、妻子 |
| 偏财 | 吉（偏财佳） | 凶（身弱不胜财） | 偏财、机遇 |

### 大运转折点
每个大运的前两年和后两年是过渡期，运势波动较大，中间六年相对稳定。

## 流年分析

### 流年定义
流年即每年的天干地支，如2025年为乙巳年。

### 流年作用关系
1. 流年天干作用于大运天干和命局天干
2. 流年地支作用于大运地支和命局地支
3. 流年是应期，触发大运和命局的潜在信息

### 流年吉凶判定
- 流年为用神：此年吉利
- 流年为忌神：此年不利
- 流年冲命局：变动之年（冲年柱主长辈，冲月柱主事业，冲日柱主婚姻/自身，冲时柱主子女）
- 流年合命局：合作/感情之年

### 重要流年类型

**天克地冲年**：流年与命局某柱天干相克、地支相冲，主重大变动、灾病、破财
**岁运并临**：流年与大运相同，力量加倍，吉凶放大
**用神到位年**：流年为命局第一用神，主大吉、好事临门
**忌神到位年**：流年为命局第一忌神，主大凶、灾祸临门
**三合局年**：流年与命局两柱合成三合局，主大事发生
**桃花年**：流年逢子午卯酉，主感情变动

## 冲合刑害详解

### 天干四冲
甲庚冲、乙辛冲、丙壬冲、丁癸冲、戊己不冲

### 地支六冲
| 六冲 | 影响 |
|-----|------|
| 子午冲 | 心肾不交，动荡变动 |
| 丑未冲 | 脾胃不和，房产变动 |
| 寅申冲 | 筋骨损伤，事业变动 |
| 卯酉冲 | 肝胆问题，感情变动 |
| 辰戌冲 | 脾胃问题，祖业变动 |
| 巳亥冲 | 心脑血管，思想变动 |

### 地支六合
合主牵绊、合作、感情。合化成功则力量转化。

### 地支三合
合局力量强大，主重大事件。如申子辰合水局，主智慧、流动、变化。

### 地支三刑
刑主刑伤、官非、疾病。寅巳申三刑最凶，丑戌未次之，子卯刑较轻。

### 地支六害
子未害、丑午害、寅巳害、卯辰害、申亥害、酉戌害。害主暗害、小人、慢性病。

## 吉凶时间节点标注

### 人生大事预测方法
1. 找出命局中的关键十神（婚姻看官星/财星，事业看官杀，财富看财星）
2. 查看该十神在大运中何时出现
3. 查看流年中何时触发（冲、合、生、克）
4. 综合判定具体年份

### 婚姻应期
- 男命：正财/偏财星到位的年份，或日支被合/冲的年份
- 女命：正官/七杀星到位的年份，或日支被合/冲的年份
- 桃花年（子午卯酉）也是婚姻高发期

### 事业应期
- 官星/印星到位的年份，主升职、稳定
- 七杀/伤官到位的年份，主变动、跳槽、创业
- 食神/正财到位的年份，主平稳发展

### 财富应期
- 财星到位的年份，主财运佳
- 比劫旺的年份，主破财、投资失利
- 印星旺的年份，主置业、学习投资

### 健康应期
- 冲克日主的年份，主身体欠安
- 忌神到位的年份，主对应脏腑疾病
- 刑害年份，主慢性病、暗疾
