## 五行生克

### 相生链
木生火 → 火生土 → 土生金 → 金生水 → 水生木

### 相克链
木克土 → 土克水 → 水克火 → 火克金 → 金克木

### 五行力量计算
将四柱天干+地支五行+藏干五行全部统计：
1. 天干五行：每个天干计1份
2. 地支五行：每个地支计1.5份（地支力量大于天干）
3. 藏干五行：本气计1份，中气计0.5份，余气计0.3份
4. 月令加倍：月柱五行力量乘以2（月令为提纲，力量最大）

### 五行平衡标准
- 理想状态：五行力量均匀分布，各约20%
- 实际要求：无一行超过40%，无一行低于5%
- 过旺：某行>45%，需要克泄耗
- 过弱：某行<5%，需要生扶

## 十神格局判定

### 格局定义
格局是八字中十神组合的特定模式，决定命主的人生大方向。

### 正格（普通格局）

**正官格**：月令本气为日主正官，或月干透出正官
- 特点：正直守法，适合体制内、管理层
- 成格条件：官星透干，不被伤官克制
- 破格：伤官见官、官杀混杂

**七杀格**：月令本气为日主七杀，或月干透出七杀
- 特点：果断有魄力，适合军警、创业、高风险行业
- 成格条件：有印星化杀或食神制杀
- 破格：杀无制、杀重身轻

**正印格**：月令本气为日主正印，或月干透出正印
- 特点：学问深厚，适合教育、研究、文化
- 成格条件：印星透干，不被财星破印
- 破格：财破印、印重身轻

**偏印格**：月令本气为日主偏印，或月干透出偏印
- 特点：机敏独创，适合技术、艺术、玄学
- 成格条件：偏印有制（财星制偏印）
- 破格：偏印无制、枭神夺食

**食神格**：月令本气为日主食神，或月干透出食神
- 特点：才华横溢，适合艺术、餐饮、娱乐
- 成格条件：食神透干，不被偏印克制
- 破格：枭神夺食、食神生财太过

**伤官格**：月令本气为日主伤官，或月干透出伤官
- 特点：聪明叛逆，适合创新、技术、自由职业
- 成格条件：伤官配印或伤官生财
- 破格：伤官见官、伤官无制

**正财格**：月令本气为日主正财，或月干透出正财
- 特点：务实节俭，适合金融、会计、实业
- 成格条件：财星透干，身强能担财
- 破格：身弱财多、比劫夺财

**偏财格**：月令本气为日主偏财，或月干透出偏财
- 特点：慷慨有眼光，适合投资、销售、经商
- 成格条件：偏财透干，身强能担财
- 破格：身弱偏财多、比劫夺财

### 比肩/劫财格的特点
- **特点**：自我独立，适合个体经营、技术工作
- **成格条件**：身强有官杀制或食伤泄
- **破格**：比劫重重无制

### 实用格局→职业速查表

以下表格将特定双子星组合（两个十神共同作用）映射到具体的行业和职业方向。比单一格局分析更精准。

| 格局组合 | 核心逻辑 | 适合行业/方向 | 不适合 |
|:--------|:--------|:-------------|:------|
| **食神 + 七杀** | 食神制杀 → 以精妙的"食神"技术和才华，去对抗"七杀"的高风险 | 军警、外科医生、风险投资、特种工程、竞赛选手 | 安稳的体制内工作（埋没才华） |
| **七杀 + 偏财** | 七杀的魄力 + 偏财的冒险精神 → 商业开拓 | 自主创业、做生意、创业者、投资人 | 受雇于人的普通岗位 |
| **伤官 + 偏财** | 伤官的创新 + 偏财的变现能力 → 用创意赚钱 | 设计师、技术创业、自媒体、自由职业、专利发明 | 按部就班的大公司螺丝钉 |
| **正印 + 正官** | 印的学识 + 官的位阶 → 体制内稳步上升 | 公务员、国企管理、学术机构、教育管理 | 创业、高风险行业 |
| **伤官 + 正印** | 伤的才华 + 印的约束 → 被社会接受的创新 | 学术研究、发明家、作家、导演、学者 | 纯技术执行岗位 |
| **食神 + 正财** | 食的才华 + 财的稳定 → 靠手艺稳定赚钱 | 厨师、手工艺人、专业顾问、技术工匠 | 投机生意 |
| **七杀 + 正印** | 杀的压力 + 印的化解 → 化压力为权力 | 高级管理、CEO、掌控型职位 | 被人管的基层岗位 |
| **比肩 + 七杀** | 比的独立 + 杀的进攻 → 孤胆英雄 | 自由战士、单干创业者、个体精英 | 团队协作型工作 |
| **劫财 + 偏财** | 劫的行动力 + 偏的灵活 → 机会型生意 | 销售、中间商、经纪人、贸易 | 固定工资的稳定岗位 |

### 格局组合分析的三步法

```
第一步：找出格局中**最重要的两个十神**（通常是月令定格 + 透干最强者）
    ↓
第二步：查上表，找到对应的"格局组合 → 职业方向"
    ↓
第三步：结合日主强弱校准
    ├── 身强 → 可以大胆走对应路径
    └── 身弱 → 需要先增强自身（用印比），再考虑该路径
```

### 示例：一个"七杀 + 偏财"格局的判断

```
格局：月令七杀 + 偏财透干
速查：七杀+偏财 = 创业/做生意
    ↓
身强校准：如果身强能担杀担财 → 适合创业、经商、做老板
          如果身弱（杀重身轻）→ 必须先增强自身（印星生身），
          否则创业=自取灭亡
```

### 特殊格局

**从格**：日主极弱，四柱中克泄耗日主的五行极旺，日主无法自立，只能顺从旺势
- 从杀格：七杀极旺，日主顺从杀星
- 从财格：财星极旺，日主顺从财星
- 从儿格：食伤极旺，日主顺从食伤
- 从强格：比劫印星极旺，日主顺从强旺

**化气格**：天干五合化气成功，如甲己合化土、乙庚合化金
- 成格条件：合神紧邻、月令支持化神、无破局字

**专旺格**：四柱中某一行极旺（>70%），日主顺从该行
- 曲直格（木专旺）、炎上格（火专旺）、稼穑格（土专旺）、从革格（金专旺）、润下格（水专旺）

## 用神取用法则

### 用神定义
用神是八字中对日主最有益的五行，能补偏救弊、平衡命局。

### 扶抑用神
- 日主强：用克泄耗（官杀、食伤、财星）
- 日主弱：用生扶（印星、比劫）

### 调候用神
- 夏生（巳午未月）：八字偏热，用水调候
- 冬生（亥子丑月）：八字偏寒，用火调候
- 春秋月生：一般不需特别调候

### 通关用神
当八字中两五行相战（如金木相克），用中间五行（水）通关化解。

### 用神层次
| 层次 | 定义 | 影响 |
|-----|------|------|
| 第一用神 | 最需要的五行，命局中恰好有且有力 | 大富大贵 |
| 第二用神 | 次需要的五行，命局中有但力量一般 | 中等富贵 |
| 第三用神 | 辅助用神，命局中弱或藏而不透 | 普通层次 |
| 无用神 | 命局中找不到用神 | 命途坎坷 |

### 忌神定义
忌神是对日主最不利的五行，加重命局失衡。
- 日主强：印星、比劫为忌
- 日主弱：官杀、食伤、财星为忌

## 命格等级评定

### 评定标准（10级制）

| 等级 | 名称 | 格局特征 | 一生成就 |
|-----|------|---------|---------|
| 10级 | 极品 | 从格/化气格成格，用神到位 | 大富大贵，名垂青史 |
| 9级 | 上等偏上 | 特殊格局，用神有力 | 大富大贵，行业领袖 |
| 8级 | 上等 | 正格成格，用神有力，调候得宜 | 富裕，社会名流 |
| 7级 | 中等偏上 | 正格成格，用神稍弱 | 中产偏上，行业精英 |
| 6级 | 中等 | 正格有成有破，用神一般 | 中产，稳定生活 |
| 5级 | 中等偏下 | 正格破格，忌神干扰 | 普通，起伏较大 |
| 4级 | 下等偏上 | 格局混乱，用神无力 | 贫寒，劳碌奔波 |
| 3级 | 下等 | 从格破格，化气不成 | 贫苦，多灾多难 |
| 2级 | 下等偏下 | 五行严重失衡，无药可救 | 极贫，短寿 |
| 1级 | 极品凶 | 天克地冲，五行灭绝 | 夭亡或极凶 |

### 评定要素
1. **格局成败**：格局是否成格，有无破格（权重30%）
2. **用神力量**：用神是否透干、得地、得势（权重25%）
3. **五行平衡**：五行分布是否相对均衡（权重15%）
4. **神煞吉凶**：吉神多还是凶煞多（权重10%）
5. **大运配合**：大运是否走用神运（权重20%）
