## 主要警示项详解

### 🔴 严重缺失：白话🗣️格式缺失（3人）
**症状**：全文无`🗣️`标记，琼/凤最严重（全文无🗣️段落），英有"白话解读"文字但未用🗣️格式
**修复**：在§1 25字段表后插入 `> **🗣️ 白话：** {不少于4句话的通俗解读}`

### 🟡 主要警告：§6五重人格格式错误（11人）
**常见错误模式**：
- 写成了十神分布表（子源/华/栾/英）
- 写成了身材外貌分析（印）
- 人格数量不足5个（宽/祖母）
- 无编号/无具体人格特质（家主/晓/立）

**修复**：§6应改为5个以上明确编号的人格特质（每个≥50字），使用【核心特质名】格式：
```markdown
### 6.2 五重人格特质
**特质一：【XXX】**（{十神依据}）
{≥50字描述}
```

### 🟡 §9置业分析不足300字（3人）
**常见错误**：家主(99字)/立(101字)/晓(98字) — 仅版本说明提及"§9置业分析"，无实际内容
**修复**：补充置业最佳年份+方位建议+房产类型建议+风水搭配+风险提示

### 🟡 §16事件总表<70行（5人）
**常见错误**：家主/晓(0行·parser无法识别格式)、华(~50行)、栾(~29行)、英(~17行)
**修复**：用标准Markdown表格格式，按大运分段，覆盖9类事件

### 🟡 §17大运精析不足10步（4人）
**常见错误**：家主/晓(格式无法识别)、栾/英(仅有部分年份分析)
**修复**：从起运年龄开始，连续10步至100岁，每步有起止年份+干支+运象分析+关键年份

## 实战经验：审计效率技巧

### 并行审计法
```
对14份报告，使用 execute_code 统一做grep扫描：
for report in 报告列表; do
    # 同时扫描多个checkpoint
    grep -c "🗣️" "$report"  # 白话缺失
    grep -c "特质[一二三四五]" "$report"  # 人格特质
    grep -c "三决断" "$report"  # 决断完整性
done
```
耗时：手动逐份读 ≈ 3-5分钟/份 × 14份 = 42-70分钟
耗时：grep自动扫描 ≈ 0.5-1分钟/14份

### 模范报告对标法
找到全量通过的报告（本轮=静v2.0 100%），以其结构和格式为标杆逐项对比其他报告。

### 分层修复策略
```
P0（阻塞级·必须立即修复）：
  白话🗣️缺失 → §17 §1后无法对接非专业读者
  §6人格缺失 → 性格分析维度质量不足

P1（标准级·建议修复）：
  §9置业不足 → 缺少独立板块
  §16事件表少行 → 未覆盖全生命周期

P2（优化级·可后续修）：
  §17步数不足 → 大运精析深度不够
```

```python
def audit_report(path, name):
    import subprocess, re
    checks = {
        "§1_25字段": f"grep -c '25字段' {path}",
        "🗣️白话": f"grep -c '🗣️' {path}",
        "§5神煞": f"grep -c '元辰\|灾煞\|天罗' {path}",
        "§6人格特质": f"grep -c '特质[一二三四五]' {path}",
        "§8财富五级": f"grep -c '五级\|身强财旺' {path}",
        "§10恶神制化": f"grep -c '恶神制化' {path}",
        "§11三档法": f"grep -c '第0层\|三档\|六步' {path}",
        "§12结婚窗口": f"grep -c '结婚窗口\|四大信号' {path}",
        "§15六亲": f"grep -c '年柱.*祖上\|六亲' {path}",
        "§18三决断": f"grep -c '决断.*一二三\|三决断' {path}",
        "§19运程曲线": f"grep -c 'ASCII\|运程曲线' {path}",
        "§21人生建议": f"grep -c '21\.\|人生建议\|补财库' {path}",
    }
    results = {}
    for k, cmd in checks.items():
        r = subprocess.run(cmd, shell=True, capture_output=True, text=True)
        results[k] = int(r.stdout.strip()) > 0
    passed = sum(1 for v in results.values() if v)
    return passed, len(checks), results
```
