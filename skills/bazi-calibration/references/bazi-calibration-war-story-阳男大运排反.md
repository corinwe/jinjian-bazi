# 实战踩坑记录：阳男大运顺逆排反（2026-06-04）

## 错误经过

**场景**：为家主（庚申年·阳男）排大运。

**我的错误操作**：
1. 在记忆里写了「家主1980庚申阳男逆排」
2. 排出的错误大运：壬午→辛巳→庚辰→己卯→戊寅→丁丑→丙子→乙亥
3. 将当前大运标为「丁丑运（2021-2030）」
4. 生成了完整报告 v3.1 并推知识库

**用户指出**：
- "下一个大运是己丑，不是丁丑"
- "当前大运是戊子，不是丁丑"
- "起运年1岁，大运完全是错的"

## 根因分析

### 原因①：规则记忆错误
我的记忆里写的是「阳男逆排」，这本身就是错的。

**正确规则**：
```
阳男 → 顺排
阴男 → 逆排  
阳女 → 逆排
阴女 → 顺排

口诀：阳男阴女顺，阴男阳女逆
```

庚申年，庚为阳，男性 → 阳男 → 顺排 ✅

### 原因②：没有做交叉验证
如果我在排完大运后问自己一句：
> 「这个序列从月柱癸未出发，逆排是壬午辛巳庚辰... 顺排是甲申乙酉丙戌... 我确定我用了正确的方向吗？」

我就会发现问题。

### 原因③：没有用已知事件反推
用户说「起运1岁，1981年」，如果我用常识检验：
- 41-50岁 = 2021-2030年
- 如果是逆排，41-50是戊寅（2021-2030）
- 如果是顺排，41-50是戊子（2021-2030）
- 用户确认当前大运=戊子 → 所以是顺排

## 事后补救措施

1. 永久记忆修正：「阳男顺排，阴男逆排」
2. 大运排好后，先用「月柱→第一步大运」的干支关系验证方向
3. 排完后用 zydx.top 在线排盘验证
4. 将大运序列写入报告前，问自己一遍：这个序列对得上起运年龄和当前年份吗？
