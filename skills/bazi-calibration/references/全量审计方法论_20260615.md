## 一、触发条件

任何时候满足以下任一条件，应触发全量审计：

1. **老板明确要求**「全面审计」「全部盘点」「找出瞎编的内容」
2. **老板纠正了你**某条规则不对
3. **新知识吸收前**，该知识与身强弱/评分体系相关
4. **定期维护**（建议每月至少1次，或每次吸收新课程后）

### Step 3 — 逐层对比（关键步骤）

对每个技能文件的每个规则，问三个问题：

**问题①：这条规则在原始课程原文中能找到吗？**
- ✅ 能 → 记录原文位置（文档编号+行号）
- ❌ 不能 → 标记为「⚠️ 待确认」

**问题②：如果是自己推理出来的，有交叉验证过吗？**
- ✅ 验证过 → 记录验证依据
- ❌ 没验证 → 标记为「⚠️ 待验证」

**问题③：这条规则和已有体系有矛盾吗？**
- ✅ 一致 → 保留
- ❌ 矛盾 → 标记冲突，以原始理论为准

### Step 4 — 冲突排查清单

| 检查项 | 原始理论 | 常见自创错误 |
|:-------|:---------|:-------------|
| 身强弱评分法 | 7字位置分(8/4/12/40/12/12/12)，看五行生克加减分 | 得令+20/透干+10/合局+15等额外加分项 |
| 合局计算 | 5倍=50分/10倍=100分，独立于基础分 | 合局+15分这种小分 |
| 三档判定 | <40弱/40-60中/>60强 | 7种状态细分 |
| 月令计算 | 看月令地支的五行对日主生克 | 只看本气印星/翻藏干 |
| 印星计分 | 其他宫位的五行对日主一样算生克 | 印不在月令不计分（传统派规则混入） |
| 财星评分 | 没有独立评分体系，用身强弱+大运匹配 | 自创七步评分法 |
| 固定分 | 每个位置基础分不同（4/40/12） | 统一用12分 |

### Step 5 — 补入遗漏

对确认缺失、但有原始理论依据的知识点，补入对应子skill。

格式：
```

【已修正】#X条
【待确认（需询问老板）】#Y条
【结论】
```

## 三、铁律：严禁自由发挥

```
任何写入技能文件的新规则，必须满足以下至少一项：
① 出自九龙道长原始课程原文（截图/录音/文字稿）
② 经过老板书面确认同意
③ 经过交叉验证与已有体系不矛盾，且标注为「实战校准」

满足以上三项中任意一项后，还必须：
④ 在写入前问一句：这是根据原始理论推导出来的，还是我编的？
⑤ 如果是编辑/发挥 → ❌ 不准写，先问老板
```
