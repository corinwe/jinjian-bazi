# 校准案例：主母八字版本混乱（2026-06-07）

## 问题描述

主母八字曾在多个版本中混淆，最终正确版本为：
**丁卯 丁未 庚申 壬午（庚金日主，1987-07-10）**

历史上曾出现的错误版本：丁卯 丁未 辛巳 癸巳（辛金日主）
根因：生辰信息在知识库和记忆中存在旧版未清理。

## 致命教训

家庭成员八字一旦校准，必须同步：
1. 更新 memory 中对应条目（含"校准"标记）
2. 更新所有引用该八字的 skill 中的示例/参考
3. 通知所有依赖该八字的下游流程

## 规避方案

建立"一人一八字"原则：
- 每个家庭成员只有一个最新版本八字
- 旧版本必须显式标记为"作废"
- 每次引用前检查是否有校准标记
