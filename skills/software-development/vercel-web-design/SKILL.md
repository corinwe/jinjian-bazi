# Web Design Guidelines (Vercel官方网页设计指南)

## 概述

Vercel工程团队认证的网页设计最佳实践。打造专业、一致、可用的界面。

## 布局

### 使用CSS Grid与Flexbox
- Grid用于二维布局（页面、卡片网格）
- Flexbox用于一维布局（导航、工具栏、按钮组）
- 避免使用浮动布局

### 间距和留白
- 使用一致的间距刻度
- 内容周围使用足够的留白（但不是太多）
- 相关元素彼此靠近，不相关的元素分开
- 使用gap而非margin处理flex/grid项之间的间距

## 颜色

### 调色板原则
- 使用语义色彩系统：primary/success/warning/error
- 中性色用于背景、文本、边框
- 保持调色板简洁——颜色表达越多，传达的越少
- 确保足够的对比度

### 深色模式
- 设计系统应原生支持深色模式
- 使用CSS变量切换主题
- 测试所有组件在两种模式下的表现

## 排版

### 层次
- h1: 页面标题（每页一个）
- h2: 章节标题
- h3: 子章节标题
- body: 默认文本
- small: 次要文本

### 排版原则
- 限制一行长度：45-75字符（包括空格）
- 使用项目的字体系统
- 保持行高舒适（1.5-1.6 body文本）
- 不要使用过多不同的字体（最多2-3种）

## 响应式设计

### 断点
- 移动端: 320px-767px
- 平板: 768px-1023px
- 桌面: 1024px-1439px
- 大屏: 1440px+

### 移动优先
- 先设计最小的屏幕
- 逐步添加更大的断点
- 使用min-width媒体查询

## 可用性
- 触摸目标 ≥ 44x44px
- 表单标签在输入上方而不是左侧
- 清晰的错误和成功状态
- 加载状态（骨架屏）
- 适当的过渡动画（微妙的、有目的的）
