# 投资炒股与科技分析大模型推荐指南（摘要）

来源：老板提供的文档（2026年5月12日）

## 国内券商AI原生应用

| 产品 | 定位 | 适用场景 |
|:----|:----|:---------|
| **AI涨乐**（华泰证券） | 人格化智能交易伙伴 | 选股-盯盘-交易全链路，散户首选 |
| **同花顺i问财** | 数据覆盖度极高的证券数据商AI | 条件筛选查询（如"近5年ROE>20%的白酒股"） |
| **东方财富AI** | 盘面工具中的AI助手 | 情绪感知和快速信息获取，盯盘时使用 |

## 专业金融数据终端

| 产品 | 核心优势 |
|:----|:---------|
| **朝阳永续AI小二** | 盈利预测数据库为核心，基本面研究深度强 |
| **Wind金融AI助手** | 宏观数据核对、财务指标快速提取 |
| **慧博智能投研** | 研报内容自动生成图表 |

## 通用大模型

| 模型 | 特点 |
|:----|:-----|
| **DeepSeek**（九九在用） | 成本极低（约为GPT-4的1/100），128K上下文，开源 |
| **通义千问 Qwen-Plus** | 带搜索能力，中文金融友好 |
| **GPT-5 / GPT-5 Thinking** | 金融基准测试最佳（准确率88.23%） |
| **Gemini 2.5 Pro** | 股票交易预测第一梯队 |

## 多智能体/量化系统

| 系统 | 特点 |
|:----|:-----|
| **TradingAgents** | 7个LLM Agent协作（基本面/技术/新闻/情绪/多空/交易/风控），支持DeepSeek V4、Qwen-Plus切换 |
| **Stocked AI** | 标普500每月荐2股，年化约35.06% |
| **Hikyuu / Abu** | 开源量化框架，百万级K线秒级回测，覆盖A股/港股/美股 |

## 选型建议

| 投资者类型 | 首选工具 | 搭配 |
|:---------|:--------|:----|
| 价值投资者 | AI涨乐 + 朝阳永续AI小二 | 雪球 |
| 技术/量化派 | 同花顺i问财 + Hikyuu/Abu | 大智慧 |
| 美股/全球配置 | Stocked AI | TradingAgents |

## 风险提示
- 所有大模型存在"幻觉"风险，包括金融概念误解和虚构信息
- 2026年学术测试表明在严格人类监督下大模型有跑赢大盘的潜力
- 绝不能完全替代独立判断
