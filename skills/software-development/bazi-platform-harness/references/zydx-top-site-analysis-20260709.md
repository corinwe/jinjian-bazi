# ZYDX.top (周易大学堂) 排盘网站分析

> **分析日期**：2026-07-09
> **用途**：参考竞品输出结构，评估自建同类网站可行性

## 网站基本信息

- **URL**：https://zydx.top/
- **排盘页面**：https://zydx.top/paipan.php（iframe内嵌）
- **表单提交**：POST到 `paipan.php`，参数含 `act=ok` + 出生数据
- **品牌**：「周易大学堂四柱八字排盘」+「九龙先生」

## 可以自建且比ZYDX更好的理由

1. **引擎数据比ZYDX多**：我们有21§完整分析、从弱/从旺判定、流年评分
2. **ZYDX缺少的**：身强弱评分、格局认定、喜用神、8维度评分
3. **前端差距可补齐**：只需FastAPI暴露API + HTML/CSS/JS前端
4. **需补充的模块**：称骨算命（标准算法，易加）

## 关键技术参考

- 五行颜色映射：金=#FD7C04, 木=green, 水=blue, 火=red, 土=#995001
- 十神简写：才(偏财)/官(正官)/杀(七杀)/印(正印)/枭(偏印)/劫(劫财)/比(比肩)/食(食神)/伤(伤官)
- 页面布局：表格+div双结构，深色分隔行(#999/#CCC)
- 交互：JS动态填充流年/流月/流日（Swiper轮播）
- 动态数据：`dayunliunianData` 对象存储每步大运中10年的流年数据
