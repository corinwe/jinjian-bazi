## 3. 字间互动规则库

### 3.1 天干五合速查

```
甲己合化土 | 乙庚合化金 | 丙辛合化水 | 丁壬合化木 | 戊癸合化火

隔合/遥合 → 不算合化，以相克论
合而不化 → 紧邻但无根/月令不生，以相克论
合而成化 → 紧邻+地支有根/月令生
争合 → 一天干与多天干合，多角关系
妒合 → 两天干将合被旁冲，猜疑妒忌
```



### 3.2 地支合化速查

#### 合化优先级规则（实战核心）

```
当多个合化同时出现时，只计算优先级最高的那个：

优先级：三会(20/10倍) > 三合(15/7倍) > 半三合(10/5倍) > 六合(10/5倍) > 拱合(10/5倍)

实战案例：
  某年同时有子辰半三合 + 辰酉六合
  → 只计算子辰半三合（优先级更高），不计算辰酉六合
  → 加强了水，不是金

  某年同时有午未六合 + 寅午戌三合
  → 只计算寅午戌三合（优先级更高）
  → 加强了火，不是火+土的混合
```

```yaml
力量层级：三会(20/10倍) > 三合(15/7倍) > 半三合(10/5倍) > 六合(10/5倍) > 拱合(10/5倍)

三会：亥子丑→水 | 寅卯辰→木 | 巳午未→火 | 申酉戌→金
三合：申子辰→水 | 寅午戌→火 | 亥卯未→木 | 巳酉丑→金
半三合：申子/子辰/寅午/午戌/亥卯/卯未/巳酉/酉丑
六合：巳申合水 | 辰酉合金 | 卯戌合火 | 寅亥合木 | 子丑合土 | 午未合火/土
拱合：申辰/寅戌/亥未/巳丑

合化条件：两支紧邻+天干引化（或大运流年补引化）
合化解刑冲破害，合出喜用=吉，合出忌凶=凶
```

### ⚠️ 2026-06-16新增：原局拱合局的长期效应（实战校准）

> **实战来源**：家主八字庚申 癸未 辛亥 戊子，原局亥未拱木（日支亥+月令未紧邻）
> 这个拱木局是原局自带的，一生都在起作用——财星能量长期存在但不爆发，
> 直到卯年补全亥卯未三合才猛烈释放。
> **⚠️注意**：拱合的"爆发"方向取决于大运——若大运方向是忌神，拱合升级也可能以压力形式出现

```yaml
【拱合的本质】拱合 = 三合局缺一字（两支之间有间隙）
  如亥未拱木 = 亥+未中间缺卯 → 木局「虚设」但没有补全
  三合是「完全体」，拱合是「未完成体」

【拱合 vs 三合的力量差异】
  三合（完全体）：15倍（有引化）/ 7倍（无引化）
  拱合（未完成体）：10倍（有引化）/ 5倍（无引化）
  → 拱合的力量约是三合的2/3

【拱合需紧邻】——重要！
  拱合的两个地支必须紧邻（中间不能有第三个地支阻隔）
  ✅ 家主亥（日支）+未（月令）= 紧邻 → 拱木成立
  ❌ 年支未+时支亥 = 隔位（月日柱在中间）→ 不成立
  ⚠️ 隔支不拱合——这是致命的易错点

【拱合局的四个组合】
  申辰拱水、寅戌拱火、亥未拱木、巳丑拱金

【拱合局的长期影响——最关键的知识点】
  原局有拱合 = 该五行长期处于「半激活」状态
  → 原局自带的拱合，一生都在起作用
  → 但不是猛烈爆发，而是「持续存在但不明显」
  → 直到大运/流年补全第三字时，才从「拱合」升级为「三合」

【拱合→三合的升级条件】
  大运或流年出现三合局缺失的第三个字时：
  □ 拱合升级为三合 → 力量翻倍（从5倍到7倍/从10倍到15倍）
  □ 该五行的能量从「持续但有节制」变为「猛烈爆发」

【拱合升级的实际体验——取决于大运方向】
  拱合升级释放的能量本身是中性的，吉凶取决于：
  
  ✅ 大运喜用该五行 → 拱合升级 = 该领域的好事爆发
     例：家主亥未拱木(财喜) + 2023癸卯(补全三合木)
     → 财能量爆发
  
  ⚠️ 大运忌神该五行 → 拱合升级 = 该领域的压力集中释放
     例：家主2011辛卯(补全三合木) + 丁亥大运(七杀为忌)
     → 木能量虽然爆发，但被七杀大运转化为压力

【断事前必须检查的拱合局】

Step 1 — 找原局拱合（必须紧邻）
  拿到八字后，先查是否有以下组合：
  □ 申+辰（紧邻）= 拱水
  □ 寅+戌（紧邻）= 拱火  
  □ 亥+未（紧邻）= 拱木（家主案例）
  □ 巳+丑（紧邻）= 拱金

Step 2 — 定拱合五行的喜忌
  拱合出来的五行是喜用还是忌凶？
  ✅ 喜用 → 长期有该五行的支持力量
  ❌ 忌凶 → 长期被该五行的负面能量影响

Step 3 — 找补全年份 + 结合大运方向
  三合局缺的第三个字是什么？
  大运/流年遇到这个字时 → 拱合→三合升级
  → 该年份该五行的能量从5倍→15倍（或10倍→15倍）
  → ⚠️ 必须结合大运方向判断最终体验：
     大运喜用→好事爆发
     大运忌神→压力释放

口诀：
  原局拱合半激活，紧邻才算隔位废
  申辰拱水寅戌火，亥未拱木巳丑金
  大运流年补全字，拱合升级变三合
  吉凶喜忌看大运，方向定了才能断
```

### ⚠️ 2026-07-02新增：三合局完整度检查——三字齐全但不成立的诊断（实战校准：老板亥卯未案例）

> **实战来源**：老板（魏启令）八字庚申 癸未 辛亥 辛卯，原局亥卯未三字齐全但仅虚邀，原因=申亥害+卯空亡。
> 这是一个「看似完整实则降级」的典型案例，极易被忽略——看到三字齐了就直接给15倍，却忘了检查破局因素。

```yaml
【核心原则】
三合局「三字齐全」≠「完整三合局」
必须逐项检查三合局每个参与字的状态，任何一字被破坏都导致整体降级：

三合局完整度等级（从高到低）：
  完整三合 = 三字齐全+无破局因素 → 15倍（有引化）/ 7倍（无引化）
  虚邀三合 = 三字齐全+有破局因素 → 约5~7倍（能量不到完整三合一倍）
  半三合 = 两字齐全 → 10倍（有引化）/ 5倍（无引化）
  拱合 = 两字隔位/缺中神 → 10倍（有引化）/ 5倍（无引化），需紧邻

【破局因素检查流程——三合局降级诊断】

拿到一个原局三合局（三字皆在）后，按以下顺序检查：

Step 1 — 检查中神是否空亡（最致命）
  中神（子午卯酉）落空亡 → 三合局无核心 → 直接降为虚邀三合
  例：老板亥卯未，中神卯在时支，辛亥日→空亡寅卯→卯空亡→虚邀三合
  ⚠️ 空亡对三合局的破坏力因位置而异：
    中神空亡 → 🚨 最致命（合局无核心，降级最大）
    生方空亡（寅申巳亥）→ ⚠️ 中等（三合力量受损但尚存）
    墓库空亡（辰戌丑未）→ ⚠️ 轻微（仍可成局但能量减半）

Step 2 — 检查是否有六害破坏参与字（次致命）
  六害穿伤三合局的某个参与字 → 该字「受伤」→ 三合局降级
  六害直接关系（速查）：
    申亥害 → 穿伤亥（影响亥+卯+未中的亥）
    寅巳害 → 穿伤巳（影响巳+酉+丑中的巳）
    卯辰害 → 穿伤卯（影响亥+卯+未中的卯）
    酉戌害 → 穿伤酉（影响巳+酉+丑中的酉）
    丑午害 → 穿伤午（影响寅+午+戌中的午）
    子未害 → 穿伤未（影响亥+卯+未中的未）
```

### 12.0 实战断事铁律（V3方法论 · 2026-06-01核心升级）

> 🚨 **方法论的根本修正**：之前把流年断事当成了「列清单→评分→公式算等级」的公式游戏，这是错的。
> 实战断事是**读命→找核心→给具体断语**的过程。不是你的公式有多全，而是你能不能读出这个八字的本质。

#### 7个要点（必须同时满足）

```yaml
要点① 分时段断事（上半年/下半年分开）— 同五行在不同时段喜忌可能完全相反
要点② 先判断今年命局的真实状态 — 大运+流年可能改变身强弱，不能沿用静态结论
要点③ 三刑=藏干打架=能量加强，不是扣分 — 被加强的字喜则吉、忌则凶
要点④ 断语必须具体到六亲+身体+应期+程度 — 六亲 + 身体部位 + 应期月份 + 严重程度

要点⑤ 宫位是断语的起点 — 找到被作用的字 → 它在哪个宫位 → 直接出断语
要点⑥ 大运+流年组合不同 → 年内可能发生状态切换，断语必须对应调整
要点⑦ 先找1-2个核心关系 — 不是列200条，核心关系决定80%的断语
```

### 12.1.0 实战断事后方法论：过三关（九龙道长原始法）

> ⚠️ **重要更正**：九龙道长原始素材中不存在「断事八步法」或「实战断事七步法」。我之前的「八步法」是自创的，不准确。九龙道长的实战方法是叫「过三关」——只说三件事，每件事精确到年份/事件/宫位，不说第四件事。以下为九龙道长原始实战方法论：

#### 过三关核心原则

```yaml
过三关 = 只给命主说三件事，把这三件事断准
  第一件事：最核心/最重大的（发财/升官/灾祸等）
  第二件事：次核心的（健康/家庭/搬迁等）
  第三件事：再次的（小事但能验证的）
  
说三件的理由：
  ① 说太多会冲淡核心断语的精度
  ② 命主只记得住三件事
  ③ 三件事全准 = 建立信任，后续深度咨询
```

#### 过三关的断事步骤（替代旧版七步法/八步法）

```yaml
Step 1 — 排盘看这一年的核心关系
  看流年天干 → 五合/四冲
  看流年地支 → 三合/三会/六合/刑/冲/害/破
  
Step 2 — 找出1-2个最强的互动
  按能量倍数排序，取最大的：
  三会20倍 > 三合15倍 > 半三合10倍 > 六合10倍 > 冲/害/刑5-10倍
  
Step 3 — 定喜忌→定吉凶
  冲走忌凶=好事，冲走喜用=坏事
  合出喜用=好事，合出忌凶=坏事
  
Step 4 — 看宫位→定对象/部位
  被冲击的宫位 = 应事对象
  年柱=父母/头颈 月柱=兄弟姐妹/胸腹
  日支=配偶/小腹 时柱=子女/腿脚
  
Step 5 — 下断语（只说三句）
  断语需精准：哪件事+什么感受/结果+什么时间+严重程度
```

#### 与旧方法的区别

| 旧做法（自创·已废弃） | 新做法（九龙道长原始法）|
|:---------------------|:----------------------|
| 列出所有刑冲合害（8项以上）| 只找1-2个最强的 |
| 评分定等级（六维法）| 只看喜忌+能量倍数 |
| 对每个恶神做排查 | 只断核心关系 |
| 说一大堆综合分析 | **过三关**——只说三件事 |

#### 口诀

```yaml

选最核心的关系，定喜忌和倍率数
看宫位出断语，四要素不能少
一句准了全部准，三句过了就是高人
```

### 12.2 犯太岁两大类别

#### 第一类：刑冲破害 → 多为凶

|| 类型 | 比喻 | 典型事件 |
|:---|:---|:---|
|| **六冲** | 直接猛烈冲突 | 激烈矛盾、突发灾祸、车祸 |
|| **六害（六穿）** | 利刃刺穿身体 | 突发疾病、小人陷害 |
|| **六破** | 撕破脸 | 吵架、破小财 — 一般不作为断事依据 |

**冲的核心规则**：
```
流年冲八字 → 只有八字受伤，流年（天上的星星）不可能受伤
冲走忌凶（不好的能量）→ 好事
冲走喜用（需要的能量）→ 坏事
有合化则贪合忘冲 ↔ 小公狗打架，来了小母狗就不打了
```

**害的核心规则**：
```
害 = 穿 = 拿利刃刺穿身体
能量仅次于冲，代表突发疾病或突发灾祸
```

#### 第二类：合化 → 吉凶看喜忌（合出什么五行）

| 合化类型 | 无天干引化 | 有天干引化 |
|:---|:---:|:---:|
| 三会（寅卯辰等） | 10倍 | 20倍 |
| 三合（寅午戌等） | 7倍 | 15倍 |
| 半三合/六合 | 5倍 | 10倍 |

```
合化本质：加强合化出来那个五行的能量

合出喜用 → 大好事（比如合出财+身强=发财1.2亿案例）
合出忌凶 → 大坏事（比如合出七杀且制不住=母亲去世案例）

关键判断：合出来的五行，是命主喜用还是忌凶？
```
