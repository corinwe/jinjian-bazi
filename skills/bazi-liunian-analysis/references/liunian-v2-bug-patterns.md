## Bug模式①：十神喜忌硬编码

```python
# ❌ 错误做法（旧代码）
xi_yong_ss = ["正印", "偏印", "比肩", "劫财", "食神"]  # 硬编码"喜用十神"
ji_shen_ss = ["七杀", "正官", "伤官", "正财", "偏财"]  # 硬编码"忌神十神"

# 然后用流年十神去匹配这两个列表来判断吉凶
if liu_nian_shi_shen in xi_yong_ss:
    score += 1.5
```

**错误原因**：十神本身不是喜用/忌神的判定标准。喜忌取决于身强弱——同一个十神在不同命局中可能是喜用也可能是忌神。skill基本原则：用传入的 `xi_yong`/`ji_shen`（五行列表）判断流年天干的五行是否喜用。

```python
# ✅ 正确做法（修复后）
ln_wx_gan = TIAN_GAN_WU_XING[liu_nian_gan]
is_xi = ln_wx_gan in xi_yong   # 五行是否在喜用列表中
is_ji = ln_wx_gan in ji_shen   # 五行是否在忌神列表中
if is_xi:
    score += 1.5
elif is_ji:
    score -= 1.5
```

## Bug模式②：印星加分硬编码

```python
# ❌ 错误做法
if liu_nian_shi_shen in ["正印", "偏印"]:
    if "水" in xi_yong or "金" in xi_yong:  # 硬编码水/金
        score += 2.0
```

**错误原因**：印星加分取决于印星所在五行是否在喜用中，而不是检查 xi_yong 列表里有水或金。

```python
# ✅ 正确做法
if liu_nian_shi_shen in ["正印", "偏印"]:
    yin_wx = TIAN_GAN_WU_XING[liu_nian_gan]  # 印星的五行
    if yin_wx in xi_yong:
        score += 1.5  # 印星为喜用
    elif yin_wx in ji_shen:
        score += 0.5  # 印星为忌神（印星本身有护身作用，小幅加分）
```

## Bug模式③：能量倍数用小数而非整数

```python
# ❌ 错误做法（旧xing_chong_he_hua.py）
NENG_LIANG = {
    "三合": 1.5,   # 应该是15倍！
    "六合": 0.8,   # 应该是10倍！
    "六冲": 1.0,   # 应该是10倍！
    "三刑": 1.2,   # 应该是15倍！
}
```

**错误原因**：skill §3.9明确指定能量倍数是10倍/15倍/5倍等整数级。小数0-2范围的倍数会导致后续任何乘算都失去意义。

```python
# ✅ 正确做法（修复后）
NENG_LIANG = {
    "六冲": 10.0,   # 辰戌/丑未冲=10倍(有引化)/5倍(无引化)
    "三刑": 15.0,   # 丑未戌三刑=15倍(有引化)/10倍(无引化)
    "二刑": 8.0,    # 两刑=7~8倍
    "自刑": 10.0,   # 辰午酉亥自刑=10倍(有引化)/5倍(无引化)
    "六害": 5.0,    # 六害=5倍
    "六破": 5.0,    # 六破=5倍
    "六合": 10.0,   # 六合=10倍(有引化)/5倍(无引化)
    "三合": 15.0,   # 三合=15倍(有引化)/7倍(无引化)
    "半合": 10.0,   # 半合=10倍(有引化)/5倍(无引化)
}
```

## Bug模式④：三刑当扣分

```python
# ❌ 错误做法
if "三刑" in all_rels["summary"]:
    score -= 1.0  # 三刑不是扣分！
```

**错误原因**：skill §3.3明确——三刑的本质是地支藏干"打架"→余气打没→本气能量翻倍。三刑=能量加强，不是伤害。被加强的字喜则吉、忌则凶。

```python
# ✅ 正确做法
if all_rels.get("刑"):
    score += 0.5  # 三刑=能量加强，不扣分
```
