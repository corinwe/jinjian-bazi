## 🔴 严重问题（BUG级）

### BUG-1：冲突标签引用不存在的十神标签名（3处无效冲突检测）

**位置：** character.py L175-180

```python
if "规则型" in ss_labels and "叛逆型" in ss_labels:  # ❌ "叛逆型"不存在
    conflicts.append("内心有规则与叛逆的冲突")
if "权威型" in ss_labels and "享受型" in ss_labels:
    conflicts.append("强势与安逸的矛盾")
if "开拓型" in ss_labels and "保守型" in ss_labels:  # ❌ "开拓型"不存在
    conflicts.append("进取与守成的拉锯")
```

**问题：**
- `"叛逆型"` — 查遍 SHI_SHEN_CHARACTER 字典，没有任何十神的 label 是"叛逆型"。伤官的 label 是"才华型"。
  → **此冲突条件永远无法触发 ❌**
- `"开拓型"` — 没有任何十神的 label 是"开拓型"。比肩的 label 是"独立型"。
  → **此冲突条件永远无法触发 ❌**
- `"保守型"` — 同样不存在于任何十神 label 中。
  → **此冲突条件永远无法触发 ❌**

**影响：** `analyze_character()` 的三条性格冲突检测中，两条（规则vs叛逆、开拓vs保守）永远不触发，性格维度分析严重失真。只有"权威型vs享受型"这一条实际生效。

### BUG-2：日主被重复计入十神性格特质

**位置：** character.py L149-162

**问题：** `bazi_gans` 包含日主本身（来自 `all_gans = [年干, 月干, 日干, 时干]`）。日主通过 `get_shi_shen_for_gan(ri_zhu, ri_zhu)` 计算得到 比肩/劫财，使日主自身的性格被重复计入 `shi_shen_traits`。

**例：** 甲日主，天干[庚,辛,甲,丙]
- 日主底色 = RI_ZHU_CHARACTER["甲"] ✅（正确）
- 十神遍历时，甲日主 → get_shi_shen_for_gan("甲","甲") → 比肩 → "独立型" 被加入 shi_shen_traits
- 同时 庚/辛/丙 正确得到各自的十神
- 结果：**日主的比肩特质被重复计入（一次在底色，一次在十神）**

**影响：** shi_shen_traits 中多了一条无意义的 日主→比肩 条目，personality_type 中多了一个"独立型"标签。

**建议修复：** 在遍历时跳过日主：`for g in bazi_gans: if g == ri_zhu: continue`

## 🟡 重要缺失

### MISS-1：身强弱性格修正过于简化

**当前实现：** (character.py L165-170)
```python
身强 → ["自信外放", "能量充沛"]
身弱 → ["内敛敏感", "心思细腻"]
```

**基础规则应有（bazi-foundation-analysis §29 身强弱实战）：**

| 身强性格 | 身弱性格 |
|---------|---------|
| 主观性强，独立有主见 | 自我意志相对弱，随和 |
| 偏于自我，不易服从 | 不太坚持己见，善于配合 |
| 强势（女命尤甚） | 喜欢热闹扎堆群居 |
| 喜欢独立发展、偏好陌生环境 | 不情愿离开熟悉环境 |
| 天生自我感觉良好 | 害怕人生地不熟 |
| 可承担命中财富和压力 | 缺少温情关爱，内心无安全感 |
| 男身旺稍好（干事业） | 女身弱稍好（持家温和） |

**性别差异化缺失：** foundation 中有「男性身旺稍好（干事业）；女性身弱稍好（持家个性温和）」及「阳干偏强点好，阴干偏弱点好」，character.py 完全没有处理性别和阴阳体性。

### MISS-2：地支午火脾气急躁规则缺失

**基础规则（bazi-foundation-analysis 地支表注释）：**
> ⚠️ 午火：被加强机会最多的地支——寅午合·戌午合·午未合·午午自刑4种加强方式(12年占4年)。**地支午火者脾气必然急躁，午火大运也会让脾气变爆**。

**当前实现：** 完全没有检查日支或其他地支是否为午火/巳火，缺少火旺急躁的性格信号。character.py 的 ri_zhu_base 只有日主五行的通用描述，没有考虑地支藏干/地支本身对性格的细化影响（如午火急躁）。

### MISS-3：庚金巳申合水性格细化缺失

**基础规则（bazi-foundation-analysis 天干意象表）：**
> 庚：刚强、果断、锐利。⚠️ 若有巳申合水→金水相生→**刚中带柔、精致细腻**

**当前实现：** RI_ZHU_CHARACTER["庚"] 只有固定描述"果断、刚毅、有魄力、锐利"，没有根据命局组合做动态调整。

### MISS-4：酉金多维性格规则缺失

**基础规则（bazi-foundation-analysis 地支注释）：**
> 酉金多维：酉+伤官→口舌官司；酉+偏印→抒情忧郁音乐；酉+财星→靠声音/口才赚钱；酉+七杀→对声音敏感苛刻；酉=酒瓶→能喝酒；酉=鸡→喜欢唱歌

**当前实现：** 完全未考虑。

### MISS-5：羊刃/孤辰寡宿等神煞影响性格

**当前实现：** (character.py L187-198) 只处理了文昌/桃花/驿马/华盖/天乙 5 种神煞的天赋，没有处理其他神煞对性格的负面影响。

**基础规则应有：**
- 羊刃 → 刚烈、好斗、容易受伤
- 孤辰/寡宿 → 孤独、不合群
- 元辰 → 内心矛盾
- 劫煞 → 好冲动
- 灾煞 → 容易惹事
- 阴阳差错 → 性格古怪、婚姻不顺
- 金舆 → 优雅有礼

### MISS-6：六合/三合/害/破/伏吟对性格的影响缺失

**当前实现：** (character.py L182-186) 仅检查了"冲"和"三刑"两个关键词。

**基础规则应有（bazi-foundation-analysis §27）：**
- 六合 → 性格圆融，善于合作
- 三合 → 贵人相助，人缘好
- 六害（穿）→ 内心矛盾多，表里不一
- 六破 → 性格分裂
- 伏吟 → 固执、重复模式
- 自刑 → 自我纠结、内耗

## 🟠 一般问题

### ISSUE-1：十神性格描述过于通用

**当前实现：** SHI_SHEN_CHARACTER 中每个十神只有一句正面+一句负面，缺乏：
- 十神在不同柱位的性格差异（年柱伤官 vs 时柱伤官性格不同）
- 十神旺弱对性格的影响
- 十神喜忌对性格的影响

**基础规则有（bazi-foundation-analysis §33 真伤官柱位解析）：**
- 月干伤官 → 聪慧但清高傲慢；兄弟缘薄；对父母不孝
- 日支伤官 → 面貌易有疤，性格高傲；男晚育，女克夫

**而且 §29.9 比肩性格有细化：**
- 比肩→自尊心强，不容许被轻视
- 比肩→独立性，相信自己的能力
- 比肩→争强好胜，过度则人际关系紧张

### ISSUE-2：di_zhi_summary 字符串匹配脆弱

**位置：** character.py L182-185
```python
if "冲" in di_zhi_summary:
    conflicts.append("地支多冲→性格波动大，内心矛盾多")
if "三刑" in di_zhi_summary:
    conflicts.append("地支三刑→内在压力大，容易自我消耗")
```

**问题：** 纯文本子串匹配。如果有其他包含"冲"字的摘要内容（如"冲破"、"冲动"等无关词语）也会误触发。同样，"三刑"可能因分隔符不同而漏配。

建议改为解析结构化数据而非摘要字符串。

### ISSUE-3：八种主要冲突仅检测三类，覆盖面不足

**当前实现：** 仅检测：
1. 规则型 vs 叛逆型（永不触发）
2. 权威型 vs 享受型
3. 开拓型 vs 保守型（永不触发）

**应补充的常见冲突：**
- 七杀 vs 正印（冲动vs保守）
- 伤官 vs 正官（叛逆vs规则）
- 食神 vs 正印（享受生活vs学习压力）
- 比肩 vs 劫财（合作vs竞争）
- 正财 vs 偏财（保守vs冒险）

### ISSUE-4：personality_type 信息丢失

**位置：** character.py L201
```python
personality_type = ", ".join(list(set([s["label"] for s in shi_shen_traits] + [ri_char.get("base", "")]))[:4])
```

**问题：**
1. 使用 `set()` 丢失顺序（虽然问题不大，但会导致每次输出顺序不一致）
2. 截断到前4个，可能丢失重要标签
3. 如果 shi_shen_traits 有日主自带的比肩，会多出一个"独立型"
4. 如果有重复十神（如两个天干都是正财），label 会去重，但可能丢失重要的性格维度信息
