#### ❌ 实现错误示例

**问题：五行过三判定逻辑完全错误**

理论原文（金鉴八字疾病分析体系）：
```
一个为真，两个为假，三个就为病，四五六都不好。
含义：某个五行在八字中（原局）出现3个及以上（含藏干），即为「过多」。
藏干：藏干中的五行计入总数
```

报告生成器实现（report_generator_simple.py line 4622）：
```python
over = pct > 30  # 使用能量百分比
```

**后果：** 一个命局可能有3个火（天干1个+地支藏干2个）但能量占比不到30%，按照理论应判为"火多"，但报告判断为"不过旺"——**漏判**。
